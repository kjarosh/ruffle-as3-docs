类 x


mx.data 包中包含用于使用 Adobe Digital Enterprise Platform Data Services for Java EE 的类。


接口
 接口说明
 IAccessPrivileges 那些必须限制访问的对象必须实现 IAccessPrivileges 接口。
 IChangeObject 提供在检测本地冲突时所需的方法和属性。
 IItemReference IItemReference 接口为 ItemReference 提供合同。
 IManaged IManaged 接口为托管对象提供契约。
 IPagedList IPagedList 是列表实现（它们实现了列表自己的分页功能）的一个标记接口。
 ITokenResponder 包含附加的结果标记 AsyncToken 字段的 mx.rpc.IResponder 的扩展名。
类
 类说明
 AccessPrivileges 封装允许的权限，并提供对允许哪些操作的简单描述的访问。
 AsyncTokenResponder 显示结果标记的 AsyncResponder 的子类。
 CacheDataDescriptor CacheDataDescriptor 类提供有关缓存数据特性的信息。
 ChangedItems 描述自指定时间后发生更改的数据。
 Conflict 表示由更新操作引发的错误。
 ConflictDetector DataService 实例使用 ConflictDetector 实例确定从远程目标推送的操作何时发生冲突。
 Conflicts Conflicts 类表示一个冲突列表。
 DataManager DataManager 类为特定数据类型的所有实例提供与服务器上的对象保持同步的功能。
 DataService DataService 类为 Data Management Service 提供顶级功能。
 DataServiceAdapter DataServiceAdapter 类是一个抽象基类，用于实现一种新的 DataService 实现。
 DataServiceOfflineAdapter 提供持久保存 DataManager 的脱机缓存项。
 DataStore DataStore 成批处理一个或多个 DataService 实例的事务。
 DataStoreEventDispatcher 调度与数据存储相关的事件。
 DynamicManagedItem 指定为动态类型的托管项应扩展其类，以便在更改动态属性时执行事件。
 DynamicProperty 描述动态属性的名称和类型。
 ItemReference ItemReference 实例从 DataService.getItem() 和 DataService.createItem() 方法返回，以提供一种管理对远程对象的引用的方式。
 ManagedAssociation 此类用于将关联表示为数据管理元数据的一部分。
 ManagedEntity 此类用于将受管理实体表示为 Remoting 数据管理元数据的一部分。
 ManagedObjectProxy 使用 ManagedObjectProxy 类可以跟踪对 DataService 管理的匿名项所做的更改。
 ManagedOperation ManagedOperation 类用于表示使用客户端数据管理的 RPC 服务的托管服务操作。
 ManagedQuery 使用此类的实例可定义一个由 Data Management Service 管理的查询。
 ManagedRemoteService ManagedRemoteService 类可以通过 LCDS 的数据管理功能为您提供对远程应用程序服务器上的 Remoting 类的访问。
 ManagedRemoteServiceOperation 专门由 RemoteObjects 使用的 Operation。
 ManualSyncConfiguration 如果需要对客户端如何与其它客户端共享数据的控制比由 autoSyncEnabled 功能提供的控制更严格，则使用此类。
 MessageBatch 表示一组更改：创建、更新、删除和更新集合消息。
 MessageCacheItem 表示 MessageBatch 的项属性的一个元素。
 MessagingDataServiceAdapter LiveCycle Data Services 后端的 DataServiceAdapter 实现。
 PageInformation PageInformation 类提供有关以下方面的详细数据：已加载集合的哪些页、集合中包含多少页以及当前页的大小。
 PropertySpecifier 此类指定对于特定的 findItem 或填充调用，应包含哪些属性。
 RPCDataManager RPCDataManager 是一个组件，可用于管理使用某个服务（如 RemoteObject、WebService 和 HTTPMultiService）检索得到的对象。
 RPCDataServiceAdapter 这是一个抽象基类，用于实现一种新的 DataService 实现。
 SQLiteOfflineAdapter 为 DataManager 的脱机缓存项提供基于 SQLite 的持久格式。
 UpdateCollectionRange UpdateCollectionRange 类提供有关对集合执行的一系列添加和删除操作的特定信息。

《ActionScript 3.0 参考》中的内容以英语显示
