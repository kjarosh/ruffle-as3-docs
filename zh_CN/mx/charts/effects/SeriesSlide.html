类public class SeriesSlide
继承SeriesSlide SeriesEffect TweenEffect Effect EventDispatcher Object

语言版本: ActionScript 3.0
产品版本: Flex 3
运行时版本: Flash Player 9, AIR 1.1

SeriesSlide 效果将数据系列滑向或滑离图表边界。您可以使用 direction 属性指定系列滑动的起始位置。

如果您将 SeriesSlide 与 hideDataEffect 效果触发器一同使用，系列将沿指示的方向从屏幕上的当前位置滑动到屏幕外的某个位置。如果您将 SeriesSlide 与 showDataEffect 一同使用，系列将沿指示方向从屏幕外的位置滑动到屏幕上的某个位置。

隐藏 MXML 语法

The <mx:SeriesSlide> tag inherits all the properties of its parent classes, and adds the following properties:

  <mx:SeriesSlide
    Properties
    direction="left|right|up|down"
  />

direction

定义系列开始滑动的位置。有效值为 "left"、"right"、"up" 和 "down"。默认值为 "left"。

initInstance

()方法
override protected function initInstance(inst:IEffectInstance):void

语言版本: ActionScript 3.0
产品版本: Flex 3
运行时版本: Flash Player 9, AIR 1.1

将效果的属性复制到效果实例。

Flex 将从 Effect.createInstance() 方法调用此方法；您不必亲自调用它。

创建自定义效果时覆盖此方法，将属性从 Effect 类复制到效果实例类。进行覆盖时，请调用 super.initInstance()。

参数

inst:IEffectInstance — 要初始化的效果实例。
