以下是 FLVPlaybackCaptioning 组件支持的 Timed Text XML 标签的精确子集。

tt 标签 -- 顶级标签

忽略 tt 标签的所有属性。

允许使用 metadata 标签，但在文档的所有级别上忽略该标签。

允许使用 set 标签，但在文档的所有级别上忽略该标签。

支持以下时间值：

  • 完整时钟格式

    （如 00:03:00.1）

  • 部分时钟格式

    （如 03:00.1）

  • 偏移时间

    （如 100.1s 或 20h）不带单位的偏移时间（如 10）假定以秒为单位。

    • 不支持任何含有帧或计时单元的时间格式（如：00:03:00:05、00:03:00:05.1、30f、30t）。

    • begin、dur 和 end 仅允许在 p 标签中使用，因此所有给定的时间都与媒体相关。

忽略 xml:lang 属性。

忽略 xml:space 属性，FLVPlaybackCaptioning 组件假定 xml:space=default。

head 标签可选，可使用一个或不使用。

  • styling 标签需要 id 属性；支持其中有一个或多个 style 标签。

    • 支持的属性：

      • tts:backgroundColor

        整个文本区域具有相同的背景颜色。可以在任何支持的标签级别上设置它，但会影响当前可见的字幕文本中的所有段落、范围等。

        支持所有合法的颜色表示形式，但除非 alpha 组件设置为 0，否则忽略该组件。如果 alpha 为 0，将不显示背景颜色。值 transparent 被视为 alpha 为 0。

      • tts:color

        支持所有合法的颜色表示形式，但忽略 alpha 组件。值 transparent 被视为黑色。

      • tts:fontFamily

      • tts:fontSize

        仅支持第一个（垂直）值（如果存在两种 fontSize 属性）。单位是可选的（假定为像素）。支持绝对值（如 12）和相对值（如 +2）。不支持百分比值。

      • tts:fontWeight

      • tts:textAlign

      • tts:wrapOption

        wrapOption 属性应用于整个文本区域（例如当前可见的所有段落、范围等）。

    • 不支持的属性：

      • tts:direction

      • tts:display

      • tts:displayAlign

      • tts:dynamicFlow

      • tts:opacity

      • tts:origin

      • tts:overflow

      • tts:padding

      • tts:showBackground

      • tts:textOutline

      • tts:unicodeBidi

      • tts:visibility

      • tts:writingMode

      • tts:zIndex

忽略 layout 标签，包括其中含有的任何 region 标签。

body 标签是必需的；并且只允许一个 body 标签。

  • body 标签支持属性样式和任何样式属性。

  • 忽略 div 标签中的 begin、dur、end、id、region、timeContainer 及其他任何属性。

  • 支持零个或多个 p 标签

    • 支持 p 标签中的 begin、dur、end、style 及任何样式属性。

    • 忽略 p 标签中的 id、region、timeContainer 及其他任何属性。

    • 属性 begin 是必需的。如果省略 begin 属性，将出现无提示故障的情况。

    • 如果没有 dur 或 end 属性，则字幕将被下一个字幕取代。

    • 如果同时有 dur 和 end 属性，则使用 end 属性。

    • p 标签支持 span 标签、br 标签和文本的混合（根据 W3C 规范）。

    • 允许使用 metadata 和 set 标签，但将它们忽略。

    • span 标签

      -不支持嵌套的 span。

      -支持 span 标签中的属性样式和任何样式属性。

      -忽略 span 标签中的 begin、dur、end、id、region、timeContainer 及其他任何属性。

    • br 标签。忽略 br 标签的所有属性和内容。

FLVPlaybackCaptioning 组件分析 Timed Text XML 文件时，将开头字幕计时信息变成 ActionScript 提示点，添加到 FLVPlayback 实例的列表中。提示点采用以前缀“fl.video.caption2.0”开头的命名约定。除非要将提示点用于添加字幕，否则不要创建任何具有这些前缀名称的提示点。

[ X ]为什么显示为英语？
《ActionScript 3.0 参考》中的内容以英语显示

《ActionScript 3.0 参考》中的部分内容未翻译成所有语言。当某个语言元素未翻译时，将显示为英语。例如，ga.controls.HelpBox 类未翻译成任何语言。因此在简体中文版的参考中，ga.controls.HelpBox 类显示为英语。
