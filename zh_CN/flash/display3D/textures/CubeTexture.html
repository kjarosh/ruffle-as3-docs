包flash.display3D.textures
类public final class CubeTexture
继承CubeTexture TextureBase EventDispatcher Object

CubeTexture 类表示上载到渲染上下文的立方体纹理。

定义一个立方图纹理，以便在渲染期间使用。立方体贴图可用于多种渲染技术，例如环境图、skyboxes 和 skylight 光照。

不能直接创建 CubeTexture 对象，而应使用 Context3D createCubeTexture()。

uploadCompressedTextureFromByteArray

()方法
public function uploadCompressedTextureFromByteArray(data:ByteArray, byteArrayOffset:uint, async:Boolean = false):void

语言版本: ActionScript 3.0
运行时版本: Flash Player 11, AIR 3

从字节数组上载 Adobe 纹理格式 (ATF) 的立方体纹理。

该字节数组必须包含纹理的所有表面和 mipmap。

参数

data:ByteArray — 包含 mipmaps 的压缩立方体纹理的字节数组。ByteArray 对象必须使用 little endian 格式。
 
byteArrayOffset:uint — 开始读取纹理数据的可选偏移。
 
async:Boolean (default = false) — 如果为 true，此函数将立即返回。在成功完成上传前，尝试使用纹理的任何绘图方法都将失败。成功上传后，此 Texture 对象将调度 Event.TEXTURE_READY。默认值：false。


引发
TypeError — Null 指针错误：当 data 为 null 时。
 
ArgumentError — 纹理解码失败：当无法从 data 中的压缩数据的格式派生此对象的压缩格式时。
 
ArgumentError — 纹理需要为正方形：当解压缩的纹理的宽度和高度不相等时。
 
ArgumentError — 纹理大小不匹配：如果解压缩的纹理的宽度和高度与纹理的边长不相等时。
 
ArgumentError — Miplevel 过大：如果解压缩的纹理的 mip 级别大于纹理边长所隐含的级别。
 
ArgumentError — 纹理格式不匹配：如果解码的 ATF 字节不包含与此纹理格式兼容的纹理或不是立方体纹理。
 
Error — 3768：后台执行期间不可使用 Stage3D API。
 
RangeError — 输入大小错误：如果 byteArrayOffset 存在整数溢出或 byteArrayOffset +6 大于 data 的长度，或者从 byteArrayOffset 到 data 字节数组末尾的可用字节数少于 ATF 纹理所需的数据量。

相关 API 元素

flash.events.Event.TEXTURE_READY

uploadFromBitmapData

()方法 
public function uploadFromBitmapData(source:BitmapData, side:uint, miplevel:uint = 0):void

语言版本: ActionScript 3.0
运行时版本: Flash Player 11, AIR 3

从 BitmapData 对象上载立方图纹理的组件。

此功能上载立方图一侧的一个 mip 级别。根据需要调用 uploadFromBitmapData() 以上载立方图的每个 mip 级别和每一面。

参数

source:BitmapData — 一个位图。
 
side:uint — 一个代码，表示要上载的立方体的一侧：

    正 X 轴：0
    负 X 轴：1
    正 Y 轴：2
    负 Y 轴：3
    正 Z 轴：4
    负 Z 轴：5

 
miplevel:uint (default = 0) — 要加载的 mip 级别，级别零是高级全分辨率图像。默认值为零。


引发
TypeError — Null 指针错误：如果 source 为 null。
 
ArgumentError — Miplevel 过大：如果指定的 mip 级别大于纹理尺寸所隐含的级别。
 
ArgumentError — 无效的立方体面：如果 side 大于 5。
 
ArgumentError — 无效的 BitmapData 错误：如果 source BitmapData 对象不包含有效的立方体纹理表面。图像必须为正方形，面数必须是 2 的幂数，并具有 miplevel 所指定的正确大小。
 
ArgumentError — 纹理格式不匹配：如果纹理格式是Context3DTextureFormat.COMPRESSED 或 Context3DTextureFormat.COMPRESSED_ALPHA，且代码是在不支持运行时纹理压缩的移动平台上执行。
 
Error — 3768：后台执行期间不可使用 Stage3D API。
