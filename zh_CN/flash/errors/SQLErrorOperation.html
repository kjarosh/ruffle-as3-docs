包flash.errors
类public class SQLErrorOperation
继承SQLErrorOperation Object

语言版本: ActionScript 3.0
运行时版本: AIR 1.0

该类包含一些常量，它们表示 SQLError.operation 属性的可能值。这些值表示导致发生错误的尝试性操作。

每个值表示 SQLConnection 类或 SQLStatement 类的一项操作。

相关 API 元素

flash.errors.SQLError.operation
