Classepublic class SQLIndexSchema
EreditarietàSQLIndexSchema SQLSchema Object

Versione linguaggio: ActionScript 3.0
Versioni runtime: AIR 1.0

Un'istanza SQLIndexSchema fornisce informazioni che descrivono un indice specifico in un database. Le informazioni disponibili includono il nome della tabella associata (la proprietà table), l'istruzione SQL utilizzata per creare l'indice (la proprietà sql) e il nome dell'indice (la proprietà name).

Per ottenere le informazioni sullo schema dell'indice per un database, utilizzate il metodo SQLConnection.loadSchema() per caricare le informazioni sullo schema, assicurandovi di utilizzare null o SQLIndexSchema come valore dell'argomento type. Nell'istanza SQLSchemaResult risultante, la proprietà indices contiene un array di istanze SQLIndexSchema che rappresentano gli indici nel database.

Di solito, il codice dello sviluppatore non costruisce le istanze SQLIndexSchema direttamente.

SQLIndexSchema

()Funzione di costruzione
public function SQLIndexSchema(database:String, name:String, sql:String, table:String)

Versione linguaggio: ActionScript 3.0
Versioni runtime: AIR 1.0

Crea un'istanza SQLIndexSchema. Di solito, il codice dello sviluppatore non chiama direttamente la funzione di costruzione SQLIndexSchema. Per ottenere le informazioni sullo schema per un database, chiamate il metodo SQLConnection.loadSchema().

Parametri
database:String — Il nome del database associato.
 
name:String — Il nome dell'indice.
 
sql:String — L'istruzione SQL utilizzata per creare l'indice.
 
table:String — Il nome della tabella a cui questo indice è associato.

Il contenuto della Guida di riferimento di ActionScript 3.0 appare in inglese

Non tutte le parti della Guida di riferimento di ActionScript 3.0 sono tradotte in tutte le lingue. Quando un elemento del linguaggio non è tradotto, viene riportato in inglese. Ad esempio, la classe ga.controls.HelpBox non è tradotta in nessuna lingua. Pertanto, nella versione italiana della guida di riferimento, la descrizione della classe ga.controls.HelpBox è riportata in inglese.
