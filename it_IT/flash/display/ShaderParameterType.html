Classepublic final class ShaderParameterType
EreditarietàShaderParameterType Object

Versione linguaggio: ActionScript 3.0
Versioni runtime: Flash Player 10, AIR 1.5

Questa classe definisce le costanti che rappresentano i possibili valori per la proprietà type della classe ShaderParameter. Ogni costante rappresenta uno dei tipi di dati disponibili in Flash Player per i parametri nel linguaggio per shader Pixel Bender.

Descrizione delle costanti

BOOL

Costante
public static const BOOL:String = "bool"

Versione linguaggio: ActionScript 3.0
Versioni runtime: Flash Player 10, AIR 1.5

Indica che il parametro shader è definito come valore bool, che equivale a un'unica istanza Boolean in ActionScript.

Sebbene il parametro preveda solo un valore singolo, la proprietà ShaderParameter.value è un array, quindi il valore singolo deve essere il solo elemento di un array assegnato alla proprietà value simile al seguente:

         // assumes the shader has a parameter named "param"
         // whose data type is bool
         myShader.data.param.value = [true];
