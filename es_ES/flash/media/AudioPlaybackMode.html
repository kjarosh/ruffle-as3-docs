Clasepublic final class AudioPlaybackMode
HerenciaAudioPlaybackMode Object

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 3.0

La clase AudioPlaybackMode define constantes para la propiedad audioPlaybackMode de la clase SoundMixer.

Problema conocido: no se reproduce audio en modo de reproducción multimedia para StageVideo, cuando el conmutador Silenciar está activado.

Cada una de estas constantes representa un conjunto de comportamientos para audio móvil adaptado a un uso particular.

AMBIENT

Constante
public static const AMBIENT:String = "ambient"

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 3.4

Un modo para la reproducción de sonido ambiental. Utilice este modo para reproducir sonidos que están asociados con el objetivo real de la aplicación y no son necesarios para su uso correcto. Cuando se usa este modo, el audio se enruta a través el altavoz. Si hay auriculares conectados, anulan el altavoz. El sistema operativo utiliza valores predeterminados para sonido ambiente cuando decida el comportamiento y la prioridad para el audio.

Este ajuste no tiene ningún efecto especial en el escritorio o en el televisor.

En iOS, el uso de este modo hace que el audio se silencie al bloquear la pantalla (o al enviar la aplicación a segundo plano por cualquier otro medio) y al activar el interruptor físico de silencio mientras que el micrófono no está en uso. Si el micrófono está en uso, el audio no se ve afectado por estas acciones.

Nota: tenga en cuenta que el audio no se silencia al pulsar el interruptor físico de silencio si hay conectado un auricular.

    

MEDIA

Constante 
public static const MEDIA:String = "media"

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 3.0

Un modo para reproducir sonidos multimedia. Cuando se usa este modo con en dispositivos móviles, el sonido se pasa a través el altavoz. Si hay auriculares conectados, anulan el altavoz. El sistema operativo utiliza valores predeterminados para audio multimedia cuando decide el comportamiento y la prioridad para el audio.

Este ajuste no tiene ningún efecto especial en el escritorio o en el televisor.

Este es el valor predeterminado.

    

VOICE

Constante 
public static const VOICE:String = "voice"

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 3.0

Un modo para reproducir audio de voz. Cuando se utiliza este modo, el audio pasa a través del auricular del teléfono de forma predeterminada. Si hay auriculares conectados, anulan el auricular. Si la propiedad SoundMixer.useSpeakerphoneForVoice se establece, el sonido llega al altavoz, anulando los auriculares. El sistema operativo utiliza valores predeterminados para audio de voz cuando decide el comportamiento y la prioridad para el audio.

Intente reducir el uso de AudioPlaybackMode.VOICE e intente cambiar a AudioPlaybackMode.MEDIA tan pronto como sea posible tras la finalización de la llamada de voz. Esto permitirá que las aplicaciones puedan reproducir contenido en modo multimedia.

Este ajuste no tiene ningún efecto especial en el escritorio o en el televisor.
