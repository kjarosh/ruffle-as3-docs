Clasepublic class HTMLWindowCreateOptions
HerenciaHTMLWindowCreateOptions Object

Versiones de motor de ejecución: AIR 1.0

Esta clase las opciones que pueden especificarse cuando el código JavaScript ejecutado en el objeto HTMLLoader intenta crear una nueva ventana HTML llamando al método window.open().

Esta clase define las propiedades y métodos correspondientes a las opciones del parámetro features transferido al método window.open() en JavaScript.

Por ejemplo, el código JavaScript de un documento HTML (en un objeto HTMLLoader) puede incluir la siguiente llamada a window.open(), donde el parámetro features (el tercer parámetro) muestra varias opciones:

window.open("http://www.adobe.com", "AdobeWindow", "scrollbars=1,menubar=1,location=0,status=0")

Utilice la clase HTMLWindowCreateOptions para anular el método createWindow() de una subclase de la clase HTMLHost. El objeto HTMLLoader transfiere un objeto HTMLWindowCreateOptions como parámetro windowCreateOptions del método createWindow() del objeto HTMLHost.

Información sobre propiedades
    

fullscreen

propiedad
public var fullscreen:Boolean = false

Versiones de motor de ejecución: AIR 1.0

Especifica si la ventana puede cambiar a pantalla completa. Esta propiedad se establece en true si la cadena features de la llamada JavaScript al método window.open() incluye "fullscreen", "fullscreen=1" o "fullscreen=y".

    

height

propiedad 
public var height:Number = NaN

Versiones de motor de ejecución: AIR 1.0

Especifica la altura inicial deseada de la nueva ventana. Se establece en el valor height de la cadena features de la llamada JavaScript al método window.open(). Si el valor es NaN (valor predeterminado si no se especifica ningún valor de height en la cadena features), se utilizará la altura predeterminada de la ventana.
