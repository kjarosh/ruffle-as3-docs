Clasepublic class MatrixTransformer
HerenciaMatrixTransformer Object

Versión del lenguaje: ActionScript 3.0
Versión de producto: Flash CS3
Versiones de motor de ejecución: Flash Player 9.0.28.0, AIR 1.0

La clase MatrixTransformer contiene métodos para la modificación de propiedades individuales de una matriz de transformación: la escala horizontal y vertical, el sesgado horizontal y vertical, y la rotación. Esta clase también tiene métodos para la rotación en torno a un punto de transformación determinado, en lugar del típico punto (0, 0).

public static function matchInternalPointWithExternal(m:Matrix, internalPoint:Point, externalPoint:Point):void

Versión del lenguaje: ActionScript 3.0
Versión de producto: Flash CS3
Versiones de motor de ejecución: Flash Player 9.0.28.0, AIR 1.0

Mueve una matriz según sea necesario para alinear un punto interno con un punto externo. Esto se puede utilizar para hacer coincidir un punto de un clip de película transformado con otro punto del clip de película principal.

Parámetros

m:Matrix — Instancia de Matrix.
 
internalPoint:Point — Instancia de Point que define una ubicación dentro del espacio de transformación de la matriz.
 
externalPoint:Point — Instancia de Point que define una ubicación de referencia fuera del espacio de transformación de la matriz.

setRotation

()método 
public static function setRotation(m:Matrix, rotation:Number):void

Versión del lenguaje: ActionScript 3.0
Versión de producto: Flash CS3
Versiones de motor de ejecución: Flash Player 9.0.28.0, AIR 1.0

Cambia el ángulo de rotación en una matriz. Si el sesgado horizontal y el vertical no son iguales, el sesgado vertical se establece con el valor de rotación y el sesgado horizontal se aumenta con la diferencia entre el valor de rotación antiguo y el nuevo. Esto coincide con el comportamiento de rotación de Flash Player.

Parámetros

m:Matrix — Instancia de Matrix.
 
rotation:Number — Ángulo de rotación, expresado en grados.
