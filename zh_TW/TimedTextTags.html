Timed Text 標籤

類別 x

下列為 FLVPlaybackCaptioning 組件支援的 Timed Text XML 標籤確切子集。

tt 標籤 - 最上層標籤

tt 標籤的所有特質都會遭到忽略。

允許使用 metadata 標籤，但是會在文件的任何層級遭到忽略。

允許使用 set 標籤，但是會在文件的任何層級遭到忽略。

支援下列時間值：

  • 完整時鐘格式

    (例如，00:03:00.1)

  • 部分時鐘格式

    (例如， 03:00.1)

  • 偏移時間

    (例如，100.1s 或 20h)不含單位的偏移時間 (例如，10) 會假設以秒為單位。

    • 不支援含有影格或刻度的任何時間格式 (例如： 00:03:00:05、00:03:00:05.1、30f、30t)。

    • begin、dur 和 end 都只允許出現在 p 標籤中，所以全部指定的時間都是相對於該媒體而言。

xml:lang 特質會遭到忽略。

xml:space 特質會遭到忽略，FLVPlaybackCaptioning 組件將假設 xml:space= default。

head 標籤是選擇性的，只能使用一個或是不使用。

  • styling tag 需要有 id 特質；支援在內部使用一或多個 style 標籤。

    • 支援的特質：

      • tts:backgroundColor

        整個文字區域都具有相同的背景顏色。您可以在任何支援的標籤層級設定這個屬性，但是設定它將會影響目前顯示之註解文字的每個段落、範圍等等。

        支援所有有效的顏色表示，但是除非設定為 0，否則將會忽略 Alpha 組件。如果 Alpha 為 0，則不會顯示背景顏色。 transparent 的值會視為 Alpha 0。

      • tts:color

        支援所有有效的顏色表示，但會忽略 Alpha 組件。 transparent 的值會視為黑色。

      • tts:fontFamily

      • tts:fontSize

        只支援第一個 (垂直) 值 (如果有兩個 fontSize 特質)。單位是選擇性的 (假設以像素為單位)。支援絕對 (例如，12) 和相對 (例如，+2) 值，但不支援百分比值。

      • tts:fontWeight

      • tts:textAlign

      • tts:wrapOption

        wrapOption 特質適用於整個文字區域 (例如，目前顯示的每個段落、範圍等等)。

    • 不支援的特質：

      • tts:direction

      • tts:display

      • tts:displayAlign

      • tts:dynamicFlow

      • tts:opacity

      • tts:origin

      • tts:overflow

      • tts:padding

      • tts:showBackground

      • tts:textOutline

      • tts:unicodeBidi

      • tts:visibility

      • tts:writingMode

      • tts:zIndex

layout 標籤會遭到忽略，包括其中包含的任何 region 標籤。

需要 body 標籤；只允許一個 body 標籤。

  • body 標籤支援 style 特質和任何樣式特質。

  • begin、dur、end、id、region、timeContainer 及任何其它特質都會在 div 標籤中遭到忽略。

  • 支援零個或多個 p 標籤

    • p 標籤支援 begin、dur、end、style 特質和任何樣式特質。

    • id、region、timeContainer 及任何其它特質都會在 p 標籤中遭到忽略。

    • begin 特質是必要的。如果省略 begin 特質，將產生靜音失敗。

    • 如果沒有 dur 或 end 特質，下一個註解就會關閉此註解。

    • 如果同時遇到 dur 和 end 特質，就會使用 end 特質。

    • p 標籤支援 span 標籤、br 標籤和文字混用 (依 W3C 規格)。

    • 允許使用 metadata 和 set 標籤，不過會遭到忽略。

    • span 標籤

      - 不支援巢狀範圍。

      - span 標籤支援 style 特質和任何樣式特質。

      - 在 span 標籤中會忽略 begin、dur、end、id、region、timeContainer 及任何其他特質。

    • br 標籤。br 標籤的所有特質和內容都會遭到忽略。

當 FLVPlaybackCaptioning 組件剖析 Timed Text XML 檔時，會將 begin 的註解時間資訊轉換成可加入至 FLVPlayback 實體清單的 ActionScript 提示點。提示點會使用以前置詞 "fl.video.caption2.0" 為開頭的命名慣例。除了用於註解之外，請不要建立含有這類前置詞名稱的提示點。
