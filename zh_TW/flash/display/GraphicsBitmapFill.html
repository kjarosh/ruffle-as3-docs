類別public final class GraphicsBitmapFill
繼承GraphicsBitmapFill Object
實作 IGraphicsFill, IGraphicsData

語言版本: ActionScript 3.0
執行階段版本: Flash Player 10, AIR 1.5

定義點陣圖填色。點陣圖可以透過平滑化、重複或拼貼處理來填滿區域，或是使用變形矩陣來操作。

與 Graphics.drawGraphicsData() 方法搭配使用 GraphicsBitmapFill 物件。繪製 GraphicsBitmapFill 物件相當於呼叫 Graphics.beginBitmapFill() 方法。

相關 API 元素

matrix

語言版本: ActionScript 3.0
執行階段版本: Flash Player 10, AIR 1.5

這是一個矩陣物件 (屬於 flash.geom.Matrix 類別)，此矩陣物件會定義點陣圖的變形。例如，下列矩陣會將點陣圖旋轉 45 度 (pi/4 弧度)： 

     matrix = new flash.geom.Matrix(); 
     matrix.rotate(Math.PI / 4);

repeat

屬性 
public var repeat:Boolean

語言版本: ActionScript 3.0
執行階段版本: Flash Player 10, AIR 1.5

指定是否要以並排方式重複顯示點陣圖影像。

若為 true，將以並排模式重複顯示點陣圖影像。 若為 false，則不會重複顯示點陣圖影像，而且會使用點陣圖邊緣的最外部像素來填滿任何超出此點陣圖邊界的填色區域。

smooth

屬性 
public var smooth:Boolean

語言版本: ActionScript 3.0
執行階段版本: Flash Player 10, AIR 1.5

指定是否要將平滑化演算法套用至點陣圖影像。

若為 false，則使用最近鄰點法呈現放大後的點陣圖影像，影像會呈現像素化。 若為 true，則使用雙線性插補法呈現放大後的點陣圖影像。 使用最近鄰點法顯示影像通常較為快速。

GraphicsBitmapFill

()建構函式
public function GraphicsBitmapFill(bitmapData:BitmapData = null, matrix:Matrix = null, repeat:Boolean = true, smooth:Boolean = false)

語言版本: ActionScript 3.0
執行階段版本: Flash Player 10, AIR 1.5

建立新的 GraphicsBitmapFill 物件。

參數
bitmapData:BitmapData (default = null) — 包含要顯示的位元之透明或不透明點陣圖影像。
 
matrix:Matrix (default = null) — 這是一個矩陣物件 (屬於 flash.geom.Matrix 類別)，可以用來定義點陣圖的變形。
 
repeat:Boolean (default = true) — 若為 true，將以並排模式重複顯示點陣圖影像。 如果是 false，則不會重複顯示點陣圖影像，而且會使用點陣圖的邊緣來填滿任何超出此點陣圖的填色區域。
 
smooth:Boolean (default = false) — 若為 false，則使用最近鄰點法顯示放大後的點陣圖影像，影像會呈現像素化。如果是 true，則使用雙線性插補法呈現放大後的點陣圖影像。使用最近鄰點法顯示影像通常較為快速。
