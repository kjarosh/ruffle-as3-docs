類別 x


flash.system 套件包含可用來存取系統層級功能的類別，例如安全性、記憶體回收等等。


函數
 函數說明
 fscommand 允許 SWF 檔與 Flash Player 或用來裝載 Flash Player 的程式 (如網頁瀏覽器) 進行通訊。
類別
 類別說明
 ApplicationDomain ApplicationDomain 類別是分離之類別定義群組的容器。
 Capabilities Capabilities 類別能提供屬性，這些屬性會描述裝載應用程式的系統和執行階段。
 ImageDecodingPolicy     ImageDecodingPolicy 類別會針對 LoaderContext 類別中的 imageDecodingPolicy 提供值。
 IME IME 類別可以讓您在用戶端電腦上執行的 Flash 執行階段應用程式內，直接操作作業系統的輸入法編輯器 (IME)。
 IMEConversionMode 此類別含有與 IME.conversionMode 屬性搭配使用的常數。
 JPEGLoaderContext JPEGLoaderContext 類別包含一個屬性，可在載入 JPEG 影像時啟用消除馬賽克濾鏡。
 LoaderContext LoaderContext 類別會藉由使用 Loader 類別，提供載入 SWF 檔和其他媒體的選項。
 MessageChannel MessageChannel 類別提供一個機制，可供 Worker 彼此通訊。
 MessageChannelState 這個類別會針對 MessageChannel 類別的 state 屬性定義代表其可能值的常數。
 Security Security 類別能讓您指定不同網域中的內容相互通訊的方式。
 SecurityDomain SecurityDomain 類別代表目前的安全性「安全執行程序」，又稱為安全性網域。
 SecurityPanel SecurityPanel 類別會提供用來指定您要顯示之「安全性設定」面板的值。
 System System 類別包含與本機設定及作業相關的屬性。
 SystemUpdater SystemUpdater 類別可讓您更新 Flash Player 的模組 (例如 Adobe Access 的 DRM 模組)，就像 Flash Player 一樣。
 SystemUpdaterType SystemUpdaterType 類別提供常數給系統更新。
 TouchscreenType TouchscreenType 類別是列舉類別，可以提供不同觸控螢幕類型的值。
 Worker Worker 物件代表 Worker，亦即 Flash 執行階段的虛擬實體。
 WorkerDomain WorkerDomain 類別提供建立並存取 Worker 物件的方式。
 WorkerState 此類別會定義代表 Worker 類別 state 屬性之可能值的常數。
