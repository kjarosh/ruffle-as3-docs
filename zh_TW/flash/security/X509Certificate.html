套件flash.security
類別public class X509Certificate
繼承X509Certificate Object

語言版本: ActionScript 3.0
執行階段版本: Flash Player 11, AIR 3

X509Certificate 類別代表 X.509 憑證。這個類別會定義 RFC2459 中指定的 X.509 屬性。在您成功呼叫 SecureSocket.connect() 之後，伺服器的憑證資料會以 X509Certificate 實體的形式儲存在 SecureSocket.serverCertificate 屬性中。

在建立安全通訊端連線之後，這個類別可用來檢查伺服器憑證。這個類別中的屬性提供 X.509 憑證中最常用特質的存取權。如果您必須存取伺服器憑證的其他部分 (例如擴充功能)，完整的憑證可以在 encoded 屬性中取得。儲存在 encoded 屬性中的憑證經過 DER 編碼。

語言版本: ActionScript 3.0

version

屬性 
version:uint  [唯讀]

語言版本: ActionScript 3.0
執行階段版本: Flash Player 11, AIR 3

提供憑證格式的版本號碼。這個屬性指出憑證是否具有擴充功能、唯一識別名稱或者只有基本欄位。

  • version = 2：表示 X.509 版本 3 - 擴充功能存在
  • version = 1：表示 X.509 版本 2 - 擴充功能不存在，但有唯一識別名稱。
  • version = null：表示 X.509 版本 1 - 只有基本憑證欄位
