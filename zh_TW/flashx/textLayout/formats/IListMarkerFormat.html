content

屬性 
content:*  [唯讀]

控制標記的內容。

此字串的有效值為：

  • none - 無標記。
  • counter(ordered) - 顯示標記。
  • counter(ordered,ListStyleType) - 顯示標記，但將 listStyleType 變更為指定的值。
  • counters(ordered) - 從最上層的父輩 ListElement 開始，在每個計數器中建立值，為順序計數器的字串指定了以每一字尾分隔的 listStyleType。這適用於外框編號，例如 I.1. 和 I.2. 等。
  • counters(ordered,"<string>") - 類似上一個值，只是每個順序計數器的字尾會取代為 <string>。
  • counters(ordered,"<string>",ListStyleType) - 類似上一個值，只是每個計數器的 listStyleType 會取代為指定的值。

如果未定義，則此屬性的預設值為 "counter(ordered)"。

counterIncrement

屬性 
counterIncrement:*  [唯讀]

控制遞增計數器的值。

此字串的有效值為：

  • none - 無遞增。
  • ordered - 以 1 遞增計數器。
  • ordered integer - 以 integer 遞增計數器。

如果未定義，則此屬性的預設值為 "ordered 1"。

注意：counterIncrement 屬性會在 counterReset 屬性之前套用。



實作
    public function get counterIncrement():*

counterReset

屬性 
counterReset:*  [唯讀]

控制重設計數器的值。

此屬性的有效值為：

  • none - 無重設。
  • ordered - 將計數器重設為零。
  • ordered integer - 將計數器重設為 integer。

如果為 undefined，則此屬性的預設值為「none」。

注意：counterReset 屬性會在 counterIncrement 屬性之前套用。

suffix:*  [唯讀]

控制 ListItemElement 中所產生文字的字尾應用。
