flashx.textLayout.elements

O pacote flashx.textLayout.elements contém as classes centrais usadas para representar o conteúdo de texto no Text Layout Framework.


Interfaces
 InterfaceDescrição
 IConfiguration Interface exclusiva para leitura de um objeto de configuração.
 IFormatResolver Interface com um conversor de formato.
Classes
 ClasseDescrição
 BreakElement A classe BreakElement define uma quebra de linha, que permite a criação de uma quebra de linha no texto sem criar um novo parágrafo.
 Configuration A classe Configuration é um ponto principal de integração entre o Text Layout Framework e um aplicativo.
 ContainerFormattedElement ContainerFormattedElement é a classe de raiz de todos os elementos de bloco em nível de contêiner, como objetos DivElement e TextFlow.
 DivElement A classe DivElement define um elemento para agrupamento de parágrafos (objetos ParagraphElement).
 FlowElement O texto em um fluxo é armazenado na forma de árvore, cujos elementos representam as divisões lógicas dentro do texto.
 FlowGroupElement A classe FlowGroupElement é a classe básica dos objetos FlowElement que podem ter uma matriz de descendentes (filhos).
 FlowLeafElement Classe básica de FlowElements exibida no nível inferior da hierarquia do fluxo.
 GlobalSettings A configuração que se aplica a todos os objetos TextFlow.
 InlineGraphicElement A classe InlineGraphicElement gerencia objetos gráficos exibidos linearmente no texto.
 InlineGraphicElementStatus A classe InlineGraphicElementStatus define um conjunto de constantes para verificar o valor de InLineGraphicElement.status.
 LinkElement A classe LinkElement define um link para um URI (Identificador Universal de Recursos), executado quando o usuário clica nele.
 LinkState A classe LinkState define um conjunto de constantes para a propriedade linkState da classe LinkElement.
 ListElement A classe List é usada para agrupar itens em uma lista numerada ou não numerada.
 ListItemElement ListItemElement é um item em uma lista.
 OverflowPolicy A classe OverflowPolicy define um conjunto de constantes para a propriedade overflowPolicy da classe IConfiguration.
 ParagraphElement A classe ParagraphElement representa um parágrafo na hierarquia de fluxo de texto.
 ParagraphFormattedElement A classe ParagraphFormattedElement é uma classe básica abstrata das classes FlowElement que têm propriedades de parágrafo.
 SpanElement A classe SpanElement representa uma sequência de texto com um único conjunto de atributos de formatação aplicado.
 SpecialCharacterElement A classe SpecialCharacterElement é uma classe básica abstrata de elementos que representam caracteres especiais.
 SubParagraphGroupElement O SubParagraphGroupElement é um elemento que se agrupa de FlowLeafElements e outras classes que estendem SubParagraphGroupElementBase.
 SubParagraphGroupElementBase A classe SubParagraphGroupElementBase agrupa os FlowLeafElements.
 TabElement A classe TabElement representa um <tab/> no fluxo de texto.
 TCYElement A classe TCYElement (Tatechuuyoko - ta-tae-chu-yo-ko) é uma subclasse de SubParagraphGroupElementBase que faz com que o texto seja desenhado na horizontal em uma linha vertical.
 TextFlow A classe TextFlow é responsável por gerenciar todo o conteúdo do texto de uma matéria.
 TextRange Uma classe do tipo ‘somente leitura’ que descreve um intervalo de texto contíguo.

[ X ]Por que inglês?
O conteúdo na Referência do ActionScript 3.0 aparece em inglês

Nem todas as partes da Referência do ActionScript 3.0 estão traduzidas em todos os idiomas. Quando um elemento de um idioma não está traduzido, ele é exibido em inglês. Por exemplo, a classe ga.controls.HelpBox não está traduzida em nenhum idioma. Por isso, na versão da referência em português do Brasil, a classe ga.controls.HelpBox é exibida em inglês.
