Pacotefl.events
Classepublic class RSLEvent
HerançaRSLEvent Event Object

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS5
Versões de runtime: Flash Player 10.1, AIR 2

A classe RSLEvent define eventos despachados pelo RSLPreloader. Estes eventos indicam o progresso do download de RSLs, incluindo informações sobre o número de arquivos e o número de bytes carregados. Quando uma repetição de pré-carregamento de evento personalizada é usada e o código RSLPreloader é gerado automaticamente pela autoria do Flash, estes eventos são despachados pela linha de tempo principal. Neste caso, use addEventListener sem precisar fazer referência ao RSLPreloader.

RSLEvent

()Construtor
public function RSLEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, rslsLoaded:int = 0, rslsFailed:int = 0, rslsTotal:int = 0, bytesLoaded:int = 0, bytesTotal:int = 0)

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS5
Versões de runtime: Flash Player 10.1, AIR 2

Cria um objeto Event com informações específicas pertinentes a eventos RSL.

Parâmetros
type:String — O tipo de evento. Os valores possíveis são: RSLEvent. RSL_LOAD_COMPLETE e RSLEvent. RSL_PROGRESS.
 
bubbles:Boolean (default = false) — Determina se o objeto Event participa do estágio de bubbling do fluxo de eventos.
 
cancelable:Boolean (default = false) — Determina se o objeto Event pode ser cancelado.
 
rslsLoaded:int (default = 0) — O número de arquivos cujo download foi realizado com sucesso.
 
rslsFailed:int (default = 0) — O número de arquivos cujo download falhou.
 
rslsTotal:int (default = 0) — O número total de arquivos que foram carregados com sucesso ou não.
 
bytesLoaded:int (default = 0) — O número total de bytes baixados em arquivos que carregaram com sucesso.
 
bytesTotal:int (default = 0) — O número total de bytes baixados em arquivos que foram carregados com sucesso ou não.
Detalhes da constante

RSL_LOAD_COMPLETE

Constante
public static const RSL_LOAD_COMPLETE:String = "rslLoadComplete"

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS5
Versões de runtime: Flash Player 10.1, AIR 2

Evento despachado pelo RSLPreloader quando todos os RSLs foram carregados. As propriedades bytesLoaded e bytesTotal as são sempre 0 para este tipo de evento. Ao pré-carregar com uma repetição de evento personalizada, o conteúdo de usuário DEVE escutar este evento para determinar quando sair da repetição de animação de carregamento.

Esse evento tem as seguintes propriedades:

PropriedadeValor
bubblestrue
cancelablefalse; não há comportamento padrão a ser cancelado.
currentTargetO objeto que está processando ativamente o objeto Event com um ouvinte de evento.
targetA ocorrência DisplayObject que está sendo adicionada à lista de exibição. target não é sempre o objeto na lista de exibição que registrou o ouvinte de evento. Use a propriedade currentTarget para acessar o objeto na lista de exibição que está processando o evento no momento.
rslsLoadedO número de arquivos cujo download foi realizado com sucesso.
rslsFailedo número de arquivos cujo download falhou.
rslsTotalo número total de arquivos que foram carregados com sucesso ou não.
bytesLoadedO número total de bytes baixados em arquivos que carregaram com sucesso.
bytesTotalo número total de bytes baixados em arquivos que foram carregados com sucesso ou não.
         import fl.events.RSLEvent;
         myPreloader.addEventListener(RSLEvent.RSL_LOAD_COMPLETE, loadComplete);
         function loadComplete(e:RSLEvent) {
         gotoAndPlay('startOfContent');
         }

         import fl.events.RSLEvent;
         myPreloader.addEventListener(RSLEvent.RSL_PROGRESS, progressHandler);
         function progressHandler(e:RSLEvent) {
         trace('rslsLoaded = ' + e.rslsLoaded);
         trace('rslsFailed = ' + e.rslsFailed);
         trace('rslsTotal = ' + e.rslsTotal);
         trace('bytesLoaded = ' + e.bytesLoaded);
         trace('bytesTotal = ' + e.bytesTotal);
         }
