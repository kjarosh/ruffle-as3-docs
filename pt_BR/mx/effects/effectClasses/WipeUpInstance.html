Classepublic class WipeUpInstance

Versão da linguagem: ActionScript 3.0
Versão de produto: Flex 3
Versões de runtime: Flash Player 9, AIR 1.1

The WipeUpInstance class implements the instance class for the WipeUp effect. Flex creates an instance of this class when it plays a WipeUp effect; you do not create one yourself.
