Classepublic class HTMLWindowCreateOptions
HerançaHTMLWindowCreateOptions Object

Versões de runtime: AIR 1.0

Esta classe define as opções que podem ser especificadas quando o JavaScript em execução em um objeto HTMLLoader tenta criar uma nova janela HTML chamando o método window.open().

Esta classe define as propriedades e os métodos que correspondem às opções no parâmetro features transmitidas para o método window.open() no JavaScript.

Por exemplo, o JavaScript em um documento HTML (em um objeto HTMLLoader) pode incluir a seguinte chamada para window.open(), em que o parâmetro features (o terceiro parâmetro) lista várias opções:

window.open("http://www.adobe.com", "AdobeWindow", "scrollbars=1,menubar=1,location=0,status=0")

Use a classe HTMLWindowCreateOptions na substituição do método createWindow() de uma subclasse da classe HTMLHost. O objeto HTMLLoader transmite um objeto HTMLWindowCreateOptions como parâmetro windowCreateOptions do método createWindow() do objeto HTMLHost.

Detalhes da propriedade
    

fullscreen

propriedade
public var fullscreen:Boolean = false

Versões de runtime: AIR 1.0

Especifica se a janela deve ser tela cheia. Essa propriedade é definida como true se a string features da chamada JavaScript para o método window.open() incluir "fullscreen", "fullscreen=1" ou "fullscreen=y".

    

height

propriedade 
public var height:Number = NaN

Versões de runtime: AIR 1.0

Especifica a altura inicial desejada da nova janela. Isso é definido para o valor height na string features da chamada JavaScript para o método window.open(). Se o valor for NaN, o padrão quando não há especificação do valor height na string features, então será usada uma altura de janela padrão.
