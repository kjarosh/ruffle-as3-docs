Classepublic class SoftKeyboardEvent
HerançaSoftKeyboardEvent Event Object

Versão da linguagem: ActionScript 3.0
Versões de runtime: Flash Player 10.2, AIR 2.6

Um objeto SoftKeyboardEvent é despachado quando um teclado acionado por software é ativado ou desativado em um dispositivo ou sistema operacional. Um objeto SoftKeyboardEvent é despachado por um TextField ou InteractiveObject que tem a propriedade needsSoftKeyboarddefinida como true.

Detalhes do construtor

SoftKeyboardEvent

()Construtor
public function SoftKeyboardEvent(type:String, bubbles:Boolean, cancelable:Boolean, relatedObjectVal:InteractiveObject, triggerTypeVal:String)

Versão da linguagem: ActionScript 3.0
Versões de runtime: Flash Player 10.2, AIR 2.6

Cria um objeto de evento que contém informações sobre eventos de desativação e ativação de teclado virtual. Objetos Event são transmitidos como parâmetros para ouvintes de eventos.

Parâmetros
type:String — O tipo do evento como um valor constante (tal como SOFT_KEYBOARD_ACTIVATE). Ouvintes de eventos podem acessar essas informações por meio da propriedade type herdada.
 
bubbles:Boolean — Determina se o objeto Evento participa da fase bubbling do fluxo de eventos. Ouvintes de eventos podem acessar essas informações por meio da propriedade bubbles herdada.
 
cancelable:Boolean — Determina se o objeto Event pode ser cancelado. Ouvintes de eventos podem acessar essas informações por meio da propriedade cancelable herdada.
 
relatedObjectVal:InteractiveObject — Uma referência a um objeto de lista de exibição relacionado ao evento.
 
triggerTypeVal:String — Indica se o evento keyboard foi disparado pelo aplicativo ou pelo usuário.

Detalhes da constante

SOFT_KEYBOARD_ACTIVATE

Constante
public static const SOFT_KEYBOARD_ACTIVATE:String = "softKeyboardActivate"

Versão da linguagem: ActionScript 3.0
Versões de runtime: Flash Player 10.2, AIR 2.6

A constante SoftKeyboardEvent.SOFT_KEYBOARD_ACTIVATE define o valor do objeto SoftKeyboardEvent da propriedade type quando um teclado virtual é exibido. Para este evento, cancelável = false.

Esse evento tem as seguintes propriedades:

PropriedadeValor
tipoSOFT_KEYBOARD_ACTIVATE
bubblestrue
cancelablefalse; não há comportamento padrão a ser cancelado.
relatedObjectValUma referência a um objeto de lista de exibição relacionado ao evento.
triggerTypeValIndica se o evento keyboard foi disparado pelo aplicativo ou pelo usuário.
currentTargetO objeto que está processando ativamente o objeto Event com um ouvinte de evento.

SOFT_KEYBOARD_ACTIVATING

Constante 
public static const SOFT_KEYBOARD_ACTIVATING:String = "softKeyboardActivating"

Versão da linguagem: ActionScript 3.0
Versões de runtime: Flash Player 10.2, AIR 2.6

A constante SoftKeyboardEvent.SOFT_KEYBOARD_ACTIVATING define o valor objeto SoftKeyboardEvent da propriedade type imediatamente antes de um teclado virtual ser exibido. Para este evento, cancelável = true.

Esse evento tem as seguintes propriedades:

PropriedadeValor
tipoSOFT_KEYBOARD_ACTIVATING
bubblestrue
cancelableverdadeiro a não ser que o evento seja despachado por um objeto StageText. Para objetos StageText, o evento não pode ser cancelado.
relatedObjectValUma referência a um objeto de lista de exibição relacionado ao evento.
triggerTypeValIndica se o evento keyboard foi disparado pelo aplicativo ou pelo usuário.
currentTargetO objeto que está processando ativamente o objeto Event com um ouvinte de evento.
