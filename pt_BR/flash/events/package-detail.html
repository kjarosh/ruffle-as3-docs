O pacote flash.events suporta o novo modelo de evento DOM e inclui a classe básica EventDispatcher.


Interfaces
 InterfaceDescrição
 IEventDispatcher A interface IEventDispatcher define métodos para adicionar ou remover ouvintes de eventos, verifica se tipos de ouvinte de evento específicos estão registrados, além de enviar eventos.

É a responsabilidade do aplicativo lidar explicitamente com os eventos de erro. Estes eventos incluem caixas onde os usuários digitam credenciais válidas, mas o voucher que protege o conteúdo criptografado restringirá o acesso ao conteúdo.
 DRMDeviceGroupEvent Emitido por DRMManager quando uma chamada relacionada ao grupo de dispositivos é concluída com sucesso.

 DRMLicenseRequestEvent O DRMManager envia um objeto DRMLicenseRequestEvent antes de cada chamada para os métodos loadVoucher() ou loadPreviewVoucher() do objeto DRMManager ter sucesso.
 DRMMetadataEvent AVSegmentedSource despacha DRMMetadataEvent quando carrega e analisa arquivos.

 ErrorEvent Um objeto envia um objeto ErrorEvent quando um erro causa falha em uma operação assíncrona.

 FocusEvent O objeto envia um objeto FocusEvent quando o usuário altera o foco de um objeto na lista de exibição para outro.

 HTMLUncaughtScriptExceptionEvent     Um objeto HTMLLoader envia um objeto HTMLUncaughtScriptExceptionEvent sempre que uma exceção do JavaScript é descartada e não é tratada com uma instrução catch.
 HTTPStatusEvent O aplicativo envia objetos HTTPStatusEvent quando uma solicitação de rede retorna um código de status HTTP.
