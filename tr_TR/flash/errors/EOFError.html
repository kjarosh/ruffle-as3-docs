public function EOFError(message:String = "")

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: AIR 1.0, Flash Player 9

Yeni EOFError nesnesi oluşturur.

Parametreler
message:String (default = "") — Error nesnesi ile ilişkili bir dize.
Örnekler Bu örnek nasıl kullanılır?
EOFErrorExample.as

Aşağıdaki örnek, varolan verilerin sonunun ötesini okumaya çalışmanız durumunda oluşan hatayı göstermek amacıyla EOFErrorExample sınıfını kullanır. Bu işlem, aşağıdaki adımlar uygulanarak gerçekleştirilir:
  1. Yapıcı, bir ByteArray nesnesi byteArr oluşturur ve writeBoolean() kullanarak bayt akışına false olan bir Boolean değeri yazar.
  2. byteArr konumu, 0 olarak sıfırlanır. (veri akışının başlangıcı).
  3. readBoolean() kullanılarak tek bir bayt veri akışından kaldırılır. Veri akışı şu anda hiçbir veri içermiyor.
  4. EOFError nesnelerini yakalayacak şekilde ayarlanan hata işleme kod parçası içinde readBoolean() ikinci kez çağrılır, EOFError yakalanarak trace() ifadesine iletilir; böylece, EOFError nesneleriyle ilişkili hata mesajı dışa verilir. 
package {
    import flash.display.Sprite;
    import flash.errors.EOFError;
    import flash.utils.ByteArray;

    public class EOFErrorExample extends Sprite {        
        public function EOFErrorExample() {
            var byteArr:ByteArray = new ByteArray();

            byteArr.writeBoolean(false);
            trace(byteArr.length);        // 1
            
            byteArr.position = 0;
            try {
                trace(byteArr.readBoolean());    // false
            }
            catch(e:EOFError) {
                trace(e);
            }
            try {
                trace(byteArr.readBoolean());
            } 
            catch(e:EOFError) {
                trace(e);        // EOFError: Error #2030: End of file was encountered.
            }
        }
    }
}
