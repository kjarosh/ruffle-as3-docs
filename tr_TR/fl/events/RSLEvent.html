Paketfl.events
Sınıfpublic class RSLEvent
Miras AlmaRSLEvent Event Object

Dil Sürümü: ActionScript 3.0
Ürün Sürümü: Flash CS5
Çalışma Zamanı Sürümleri: Flash Player 10.1, AIR 2

RSLEvent sınıfı, RSLPreloader tarafından gönderilen olayları tanımlar. Bu olaylar, yüklenen dosya sayısı ve bayt sayısı ile birlikte, RSL'lerin indirilmesinin devam ettiğini belirtir. Döngüyü önyükleyen bir özel olay kullanıldığında ve RSLPreloader kodu Flash geliştirme tarafından otomatik olarak oluşturulduğunda, bu olaylar ana zaman çizelgesi tarafından gönderilir. Bu durumda, addEventListener öğesini kullanın (RSLPreloader öğesine başvurmanız gerekmez).

public function RSLEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, rslsLoaded:int = 0, rslsFailed:int = 0, rslsTotal:int = 0, bytesLoaded:int = 0, bytesTotal:int = 0)

Dil Sürümü: ActionScript 3.0

RSL olaylarıyla ilgili belirli bilgileri içeren bir Olay nesnesi oluşturur.

Parametreler
type:String — Olayın türü. Olası değerler: RSLEvent.RSL_LOAD_COMPLETE ve RSLEvent.RSL_PROGRESS olur.
 
bubbles:Boolean (default = false) — Event nesnesinin olay akışının köpürme aşamasında yer alıp almayacağını belirler.
 
cancelable:Boolean (default = false) — Event nesnesinin iptal edilip edilemeyeceğini belirler.
 
rslsLoaded:int (default = 0) — Başarıyla indirilen dosyaların sayısı.
 
rslsFailed:int (default = 0) — İndirilemeyen dosyaların sayısı.
 
rslsTotal:int (default = 0) — Başarılı veya başarısız olarak indirilen dosyaların toplam sayısı.
 
bytesLoaded:int (default = 0) — Başarıyla indirilen dosyalardaki indirilen toplam bayt sayısı.
 
bytesTotal:int (default = 0) — Başarılı veya başarısız olarak indirilen dosyalardaki indirilen toplam bayt sayısı.

RSL_LOAD_COMPLETE

Sabit
public static const RSL_LOAD_COMPLETE:String = "rslLoadComplete"

Dil Sürümü: ActionScript 3.0
Ürün Sürümü: Flash CS5
Çalışma Zamanı Sürümleri: Flash Player 10.1, AIR 2

Tüm RSL'lerin indirilmesi tamamlandığında, RSLPreloader tarafından gönderilen olay. bytesLoaded ve bytesTotal özellikleri bu olay türü için her zaman 0'dır. Bir özel olay döngüsüyle önyükleme yapılırken, kullanıcı içeriği yüklenen animasyon döngüsünden ne zaman çıkılacağını belirlemek için MUTLAKA bu olayı dinlemelidir.

Bu olay şu özelliklere sahiptir:

ÖzellikDeğer
bubblestrue
cancelablefalse; iptal edilecek varsayılan davranış yoktur.
currentTargetBu Event nesnesini olay dinleyicisiyle etkin olarak işleyen nesne.
targetGörüntüleme listesine eklenen DisplayObject örneği. target her zaman olay dinleyicisini kaydeden görüntüleme listesindeki nesne değildir. Geçerli olarak olayı işleyen görüntüleme listesindeki nesneye erişmek için currentTarget özelliğini kullanın.

rslsTotalbaşarılı veya başarısız olarak indirilen dosyaların toplam sayısı.
bytesLoadedBaşarıyla indirilen dosyalardaki indirilen toplam bayt sayısı.
bytesTotalbaşarılı veya başarısız olarak indirilen dosyalardaki indirilen toplam bayt sayısı.
         import fl.events.RSLEvent;
         myPreloader.addEventListener(RSLEvent.RSL_LOAD_COMPLETE, loadComplete);
         function loadComplete(e:RSLEvent) {
         gotoAndPlay('startOfContent');
         }

         trace('rslsFailed = ' + e.rslsFailed);
         trace('rslsTotal = ' + e.rslsTotal);
         trace('bytesLoaded = ' + e.bytesLoaded);
         trace('bytesTotal = ' + e.bytesTotal);
         }
