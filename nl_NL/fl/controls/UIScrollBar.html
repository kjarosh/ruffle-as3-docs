Klassepublic class UIScrollBar
OverervingUIScrollBar ScrollBar UIComponent Sprite DisplayObjectContainer InteractiveObject DisplayObject EventDispatcher Object

Taalversie: ActionScript 3.0
Productversie: Flash CS3
Runtimeversies: Flash Player 9.0.28.0, AIR 1.0

De klasse UIScrollBar omvat alle functionaliteit van de schuifbalk maar voegt een methode scrollTarget() toe, zodat u deze aan een TextField- of TLFTextField-instantie kunt koppelen.

Opmerking: wanneer u ActionScript gebruikt om eigenschappen van de TextField- of TLFTextField-instantie bij te werken die van invloed zijn op de tekstlay-out, moet u de methode update() aanroepen voor de instantie van de component UIScrollBar om de schuifeigenschappen te vernieuwen. Voorbeelden van eigenschappen voor tekstlay-out die bij de TextField- of TLFTextField-instantie behoren, zijn width, height en wordWrap.

Hiermee wordt een waarde opgehaald of ingesteld die aangeeft of de schuifbalk horizontaal of verticaal schuift. Geldige waarden zijn ScrollBarDirection.HORIZONTAL en ScrollBarDirection.VERTICAL.

De standaardwaarde is ScrollBarDirection.VERTICAL.

Verwante API-elementen

ScrollBarDirection

Voorbeeld  ( Hoe dit voorbeeld te gebruiken )
In het volgende voorbeeld wordt tekst dynamisch in een tekstveld geladen en worden een horizontale en verticale schuifbalk toegevoegd: 

import fl.controls.UIScrollBar;
import fl.controls.ScrollBarDirection;

var url:String = "http://www.helpexamples.com/flash/text/lorem.txt";

var uLdr:URLLoader = new URLLoader(new URLRequest(url));
uLdr.addEventListener(Event.COMPLETE, completeHandler);

var tf:TextField = new TextField();
tf.width = 320;
tf.height = 140;
tf.x = 10;
tf.y = 10;
tf.border = true;
tf.multiline = true;
tf.wordWrap = false;
addChild(tf)

var hScrollBar:UIScrollBar = new UIScrollBar();
hScrollBar.direction = ScrollBarDirection.HORIZONTAL;
hScrollBar.move(tf.x, tf.y + tf.height);
hScrollBar.width = tf.width;
hScrollBar.scrollTarget = tf;
addChild(hScrollBar);

var vScrollBar:UIScrollBar = new UIScrollBar();
vScrollBar.direction = ScrollBarDirection.VERTICAL;
vScrollBar.move(tf.x + tf.width, tf.y);
vScrollBar.height = tf.height;
vScrollBar.scrollTarget = tf;
addChild(vScrollBar);

function completeHandler(event:Event):void {
    tf.text = event.target.data;
    hScrollBar.update();
    vScrollBar.update();
}
In het volgende voorbeeld wordt een instantie van de component UIScrollBar gemaakt om te schuiven in een dynamisch tekstveld: 

import fl.controls.ScrollBarDirection;
import fl.controls.UIScrollBar;

var myTextField:TextField = new TextField();
myTextField.width = 160;
myTextField.height = 120;
myTextField.border = true;
myTextField.x = 10;
myTextField.y = 10;
addChild(myTextField);

var hScrollBar:UIScrollBar = new UIScrollBar();
hScrollBar.direction = ScrollBarDirection.HORIZONTAL;
hScrollBar.scrollTarget = myTextField;
hScrollBar.width = myTextField.width;
hScrollBar.move(myTextField.x, myTextField.y + myTextField.height);
addChild(hScrollBar);

var myURLLoader:URLLoader = new URLLoader();
myURLLoader.addEventListener(Event.COMPLETE, completeHandler);
myURLLoader.load(new URLRequest("http://www.helpexamples.com/flash/lorem.txt"));

function completeHandler(event:Event):void {
    var txt:String = URLLoader(event.currentTarget).data as String;
    myTextField.text = txt;
    vScrollBar.update();
}

setScrollProperties

()methode 
override public function setScrollProperties(pageSize:Number, minScrollPosition:Number, maxScrollPosition:Number, pageScrollSize:Number = 0):void

Taalversie: ActionScript 3.0
Productversie: Flash CS3
Runtimeversies: Flash Player 9.0.28.0, AIR 1.0

Hiermee wordt het bereik en de kadergrootte van de component ScrollBar ingesteld. De component ScrollBar werkt de toestand van de pijlknoppen en de grootte van het schuifblokje dienovereenkomstig bij. Alle schuifeigenschappen zijn relatief ten opzichte van de schaal van de minScrollPosition en de maxScrollPosition. Elk getal tussen de maximum- en minimumwaarden stelt één schuifpositie voor.

Parameters

pageSize:Number — Grootte van één pagina. Hiermee wordt de grootte van het blokje bepaald en de stap waarmee de schuifbalk verplaatst wanneer op de pijlen wordt geklikt.
 
minScrollPosition:Number — Laagste waarde van het schuifbereik.
 
maxScrollPosition:Number — Hoogste waarde van het schuifbereik.
 
pageScrollSize:Number (default = 0) — Grootte van verplaatsing wanneer in het vak wordt geklikt, in pixels.

Runtimeversies: Flash Player 9.0.28.0, AIR 1.0

Hiermee wordt de schuifbalk gedwongen om de schuifeigenschappen onmiddellijk bij te werken. Dit is noodzakelijk nadat tekst in het opgegeven tekstveld scrollTarget is toegevoegd met ActionScript en de schuifbalk moet worden vernieuwd.

Verwante API-elementen

scrollTarget

Voorbeeld  ( Hoe dit voorbeeld te gebruiken )

In het volgende voorbeeld wordt een instantie van de component UIScrollBar gemaakt om te schuiven in een dynamisch tekstveld: 

import fl.controls.UIScrollBar;

var myTextField:TextField = new TextField();
myTextField.wordWrap = true;
myTextField.width = 160;
myTextField.height = 120;
myTextField.border = true;
myTextField.x = 10;
myTextField.y = 10;
addChild(myTextField);

var vScrollBar:UIScrollBar = new UIScrollBar();
vScrollBar.scrollTarget = myTextField;
vScrollBar.height = myTextField.height;
vScrollBar.move(myTextField.x + myTextField.width, myTextField.y);
addChild(vScrollBar);

var myURLLoader:URLLoader = new URLLoader();
myURLLoader.addEventListener(Event.COMPLETE, completeHandler);
myURLLoader.load(new URLRequest("http://www.helpexamples.com/flash/lorem.txt"));

function completeHandler(event:Event):void {
    var txt:String = URLLoader(event.currentTarget).data as String;
    myTextField.text = txt;
    vScrollBar.update();
}
Voorbeelden Hoe dit voorbeeld te gebruiken
UIScrollBarExample.as

In het volgende voorbeeld wordt getoond hoe u een UIScrollBar-instantie aan een TextField toevoegt.

U kunt als volgt het voorbeeld uitvoeren:

  1. Voeg de component UIScrollBar toe aan de bibliotheek.
  2. Sla deze code op als UIScrollBarExample.as, in dezelfde map als het FLA-bestand.
  3. Stel de klasse Document in het FLA-bestand in op UIScrollBarExample.
package 
{
    import fl.controls.UIScrollBar;
    import flash.display.Sprite;
    import flash.text.TextField;
    
    public class UIScrollBarExample extends Sprite
    {
        private var tf:TextField;
        private var sb:UIScrollBar = new UIScrollBar();
        private var sampleText:String = "Lorem ipsum dolor sit amet, consectetuer adipiscing elit. Phasellus vestibulum. Praesent in velit. Fusce scelerisque. Nullam fringilla. Aenean ut lacus at mi rutrum elementum. Suspendisse sit amet ligula et ipsum rhoncus sagittis. Aenean quis lacus. Nullam quis lorem. Integer pellentesque dictum lorem. In hendrerit lectus non orci. Sed vestibulum. Maecenas ornare, libero eu laoreet tristique, felis odio tincidunt nibh, et faucibus nibh tellus at metus. In hac habitasse platea dictumst. Nullam aliquam, dolor ut commodo dictum, ante justo dictum nisi, id suscipit elit sapien nec pede."
        
        public function UIScrollBarExample() {
            createTextField();
            assignScrollBar();
        }
        private function createTextField():void {
            tf = new TextField();
            tf.width = 200;
            tf.height = 100;
            tf.wordWrap = true;
            tf.multiline = true;
            tf.border = true;
            tf.text = sampleText;
            addChild(tf);            
        }
        private function assignScrollBar():void {
            sb = new UIScrollBar();
            sb.move(tf.x + tf.width, tf.y);
            sb.setSize(sb.width, tf.height);
            sb.scrollTarget = tf;
            addChild(sb);            
        }
    }
}
