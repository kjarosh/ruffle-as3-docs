Wersja produktu: Adobe Digital Enterprise Platform Data Services for Java EE 3
Wersje środowiska wykonawczego: Flash Player 9, AIR 1.1

Adds units of time to a date. Units are:

  • yyyy: Year
  • q: Quarter
  • m: Month
  • y: Day of year
  • d: Day
  • w: Weekday
  • ww: Week
  • h: Hour
  • n: Minute
  • s: Second
  • l: Millisecond

Parametry

part:String — Type of units to add
 
units:int — Number of units
 
date:Date — Date

Zwraca
Date — New date; null if date is null
