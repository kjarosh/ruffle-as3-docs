accelerationX

właściwość
accelerationX:Number

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 2, Flash Player 10.1, Flash Lite 4

Przyspieszenie wzdłuż osi x, jako wielokrotność G. (1 G wynosi około 9,8 m/s2) Oś x przebiega z lewej na prawą stronę urządzenia, jeśli jest ono ustawione pionowo. Przyspieszenie jest dodatnie, jeśli urządzenie jest przesuwane w prawą stronę.

Konstruktor Szczegół

AccelerometerEvent

()Konstruktor
public function AccelerometerEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, timestamp:Number = 0, accelerationX:Number = 0, accelerationY:Number = 0, accelerationZ:Number = 0)

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 2, Flash Player 10.1, Flash Lite 4

Tworzy obiekt AccelerometerEvent, który zawiera informacje o przyspieszeniu wzdłuż trzech osi wymiaru. Obiekty Event są przekazywane jako parametry do detektorów zdarzeń.

Parametry
type:String — Typ zdarzenia. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości type. Istnieje tylko jeden typ zdarzenia aktualizacji: AccelerometerEvent.UPDATE.
 
bubbles:Boolean (default = false) — Określa, czy obiekt Event jest w fazie propagacji. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości bubbles.
 
cancelable:Boolean (default = false) — Określa, czy obiekt Event może zostać anulowany. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości cancelable.
 
timestamp:Number (default = 0) — Znacznik czasowy aktualizacji danych przyspieszeniomierza.
 
accelerationX:Number (default = 0) — Wartość przyspieszenia w g (9,8 m/s/s) wzdłuż osi x.
 
accelerationY:Number (default = 0) — Wartość przyspieszenia w g (9,8 m/s/s) wzdłuż osi y.
 
accelerationZ:Number (default = 0) — Wartość przyspieszenia w g (9,8 m/s/s) wzdłuż osi z.
