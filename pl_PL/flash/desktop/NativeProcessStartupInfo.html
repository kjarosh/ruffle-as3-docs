Klasapublic class NativeProcessStartupInfo
DziedziczenieNativeProcessStartupInfo Object

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 2

Ta klasa udostępnia podstawowe informacje używane do uruchamiania procesów w systemie operacyjnym hosta. Obiekty tej klasy są konstruowane i przekazywane do metody start() obiektu NativeProcess.

Dostęp do procesów rodzimych jest możliwy tylko z aplikacji AIR zainstalowanych za pomocą rodzimych programów instalacyjnych (aplikacji o rozszerzonym profilu aplikacji lokalnej).

Powiązane elementy interfejsu API

Argumenty wiersza poleceń, które zostaną przekazane do procesu podczas uruchamiania.

Każdy ciąg znaków w obiekcie Vector o nazwie arguments zostanie przekazany jako osobny argument do programu wykonywalnego, niezależnie od tego, jakie zawiera znaki. Innymi słowy, elementy wektora odpowiadają jeden-do-jednego argumentom; nie są w żaden sposób interpretowane. Środowisko AIR automatycznie poprzedza symbolem zmiany znaczenia wszelkie znaki, które tego wymagają.

executable

właściwość 
executable:File

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 2

Obiekt File odwołujący się do pliku wykonywalnego w systemie operacyjnym hosta. Powinna to być pełna ścieżka do pliku wykonywalnego, obejmująca ewentualne wymagane rozszerzenie.

Uwaga: Aby w systemie Mac OS można było uruchomić plik wykonywalny w pakunku aplikacji, ścieżka obiektu File musi zawierać pełną ścieżkę tego pliku (w pakunku), a nie tylko ścieżkę pliku APP.
