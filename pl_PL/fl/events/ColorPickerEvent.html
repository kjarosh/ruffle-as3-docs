Klasapublic class ColorPickerEvent
DziedziczenieColorPickerEvent Event Object

Wersja języka: ActionScript 3.0
Wersja produktu: Flash CS3
Wersje środowiska wykonawczego: Flash Player 9.0.28.0, AIR 1.0

Klasa ColorPickerEvent definiuje zdarzenia skojarzone ze składnikiem ColorPicker. Są to następujące zdarzenia:
  • ColorPickerEvent.CHANGE: wywoływane, gdy użytkownik kliknie inny kolor w składniku ColorPicker.
  • ColorPickerEvent.ENTER: wywoływane, gdy użytkownik naciśnie klawisz Enter po wprowadzeniu wartości w polu składnika ColorPicker.
  • ColorPickerEvent.ITEM_ROLL_OUT: wywoływane, gdy wskaźnik urządzenia znajdzie się poza próbką koloru w składniku ColorPicker.
  • ColorPickerEvent.ITEM_ROLL_OVER: wywoływane, gdy wskaźnik myszy porusza się nad próbką koloru w składniku ColorPicker.

Przykład  ( Sposób korzystania z tego przykładu )

Poniższy przykład ilustruje wyświetlanie wartości szesnastkowej odpowiadającej wybranemu kolorowi po jego zmianie. 

import fl.controls.ColorPicker;
import fl.events.ColorPickerEvent;

var myColorPicker:ColorPicker = new ColorPicker();
myColorPicker.addEventListener(ColorPickerEvent.CHANGE, changeHandler);
myColorPicker.move(10, 10);
addChild(myColorPicker);

function changeHandler(event:ColorPickerEvent):void {
    trace("color changed:", event.color, "(#" + event.target.hexValue + ")");
}
