Klasapublic class SliderEvent
DziedziczenieSliderEvent Event Object

Wersja języka: ActionScript 3.0
Wersja produktu: Flash CS3
Wersje środowiska wykonawczego: Flash Player 9.0.28.0, AIR 1.0

Klasa SliderEvent definiuje zdarzenia skojarzone ze składnikiem Slider. Są to następujące zdarzenia:
  • SliderEvent.CHANGE: wywoływane po zmianie pozycji suwaka.
  • SliderEvent.THUMB_DRAG: wywoływane, gdy użytkownik przeciąga uchwyt suwaka.
  • SliderEvent.THUMB_PRESS: wywoływane, gdy użytkownik naciska uchwyt suwaka.
  • SliderEvent.THUMB_RELEASE: wywoływane, gdy użytkownik zwalnia przycisk suwaka.

public function SliderEvent(type:String, value:Number, clickTarget:String, triggerEvent:String, keyCode:int = 0)

Wersja języka: ActionScript 3.0
Wersja produktu: Flash CS3
Wersje środowiska wykonawczego: Flash Player 9.0.28.0, AIR 1.0

Tworzy nowy obiekt SliderEvent z określonymi parametrami.

Parametry
type:String — Typ zdarzenia; ta wartość identyfikuje operację, która wywołała zdarzenie.
 
value:Number — Nowa wartość suwaka.
 
clickTarget:String — Określa, czy naciśnięto uchwyt lub ścieżkę suwaka. Wartość SliderEventClickTarget.THUMB oznacza, że naciśnięto uchwyt suwaka; wartość SliderEventClickTarget.TRACK oznacza, że naciśnięto ścieżkę suwaka.
 
triggerEvent:String — Obiekt String określający źródło interakcji. Wartość InteractionInputType.MOUSE oznacza, że źródłem interakcji była mysz; wartość InteractionInputType.KEYBOARD oznacza, że źródłem interakcji była klawiatura.
 
keyCode:int (default = 0) — Jeśli zdarzenie zostało wywołane naciśnięciem klawisza, ta wartość jest kodem klawisza identyfikującym ten klawisz.
