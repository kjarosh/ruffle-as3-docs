Klasapublic class CaptionChangeEvent
DziedziczenieCaptionChangeEvent Event Object

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 1.0, Flash Player 9.0.28.0

Zdarzenie CaptionChangeEvent jest wywoływane w dowolnym momencie po dodaniu lub usunięciu podpisu w docelowym polu tekstowym podpisu.

To zdarzenie jest również wywoływane w przypadku spełnienia następujących warunków:

  • właściwość captionTargetName nie została ustawiona
  • właściwość captionTarget nie została ustawiona
  • instancja FLVPlaybackCaptioning utworzyła obiekt TextField automatycznie dla umieszczenia podpisu.

CaptionChangeEvent

()Konstruktor
public function CaptionChangeEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, added:Boolean = true, captionCuePointObject:Object = null)

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 1.0, Flash Player 9.0.28.0

Tworzy obiekt Event, który zawiera informacje o zdarzeniach captionChange. Obiekty Event są przekazywane jako parametry do detektorów zdarzeń.

Parametry
type:String — Typ zdarzenia. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości type. Istnieje tylko jeden typ zdarzenia captionChange: CaptionChangeEvent.CAPTION_CHANGE.
 
bubbles:Boolean (default = false) — Określa, czy obiekt Event uczestniczy w etapie propagacji przepływu zdarzeń. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości bubbles.
 
cancelable:Boolean (default = false) — Określa, czy obiekt Event może zostać anulowany. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości cancelable.
 
added:Boolean (default = true) — Wartość typu Boolean, która wskazuje, czy podpis został dodany lub usunięty z obszaru wyświetlanego.
 
captionCuePointObject:Object (default = null) — Obiekt punktu wskazującego dla tego podpisu.
