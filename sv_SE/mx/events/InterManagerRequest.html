InterManagerRequest

()Konstruktor
public function InterManagerRequest(type:String, bubbles:Boolean = false, cancelable:Boolean = false, name:String = null, value:Object = null)

Språkversion: ActionScript 3.0
Produktversion: Flex 3
Körningsmiljöversioner: Flash Player 9, AIR 1.1

Constructor. Does not return anything, but the value property can be modified to represent a return value of a method.

Parametrar
type:String — The event type; indicates the action that caused the event.
 
bubbles:Boolean (default = false) — Specifies whether the event can bubble up the display list hierarchy.
 
cancelable:Boolean (default = false) — Specifies whether the behavior associated with the event can be prevented.
 
name:String (default = null) — Name of a property or method or name of a manager to instantiate.
 
value:Object (default = null) — Value of a property, or an array of parameters for a method (if not null).
