HTMLWindowCreateOptions  - AS3

Paketflash.html
Klasspublic class HTMLWindowCreateOptions
ArvHTMLWindowCreateOptions Object

Körningsmiljöversioner: AIR 1.0

Denna klass definierar de alternativ som kan anges när JavaScript som körs i ett HTMLLoader-objekt försöker skapa ett nytt HTML-fönster genom att anropa metoden window.open().

Den här klassen definierar de egenskaper och metoder som stämmer överens med de alternativ som kan anges i parametern features som skickas till metoden window.open() i JavaScript-kod.

Exempelvis kan JavaScript i ett HTML-dokument (i ett HTMLLoader-objekt) innehålla följande anrop till window.open(), där parametern features (den tredje parametern) innehåller ett antal olika alternativ:

window.open("http://www.adobe.com", "AdobeWindow", "scrollbars=1,menubar=1,location=0,status=0")

Du använder klassen HTMLWindowCreateOptions för att åsidosätta metoden createWindow() för en underklass till klassen HTMLHost. HTMLLoader-objektet skickar ett HTMLWindowCreateOptions-objekt som parametern windowCreateOptions för metoden createWindow() för HTMLHost-objektet.

Egenskapsdetaljer
    

fullscreen

egenskap
public var fullscreen:Boolean = false

Körningsmiljöversioner: AIR 1.0

Anger om fönstrets storlek ska vara helskärm. Denna egenskap anges till true om strängen features för JavaScript-anropet till metoden window.open() innehåller "fullscreen", "fullscreen=1" eller "fullscreen=y".

    

height

egenskap 
public var height:Number = NaN

Körningsmiljöversioner: AIR 1.0

Anger den önskade ursprungliga höjden på det nya fönstret. Anges till värdet för height i strängen features för JavaScript-anropet till metoden window.open(). Om värdet är NaN, vilket är standard när inget värde för height har angetts i strängen features, används en standardhöjd på fönstret.
