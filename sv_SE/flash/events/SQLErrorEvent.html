Klasspublic class SQLErrorEvent
ArvSQLErrorEvent ErrorEvent TextEvent Event Object

Språkversion: ActionScript 3.0
Körningsmiljöversioner: AIR 1.0

En SQLErrorEvent-instans skickas av en SQLConnection-instans eller en SQLStatement-instans om ett fel inträffar medan en databasåtgärd utförs i asynkront körningsläge. SQLErrorEvent-instansen som har passerat som ett händelseobjekt till avlyssningar ger åtkomst till information om felets orsak och den åtgärd som försökte utföras.

Felets specifika detaljer hittas på SQLError-objektet i SQLErrorEvent-instansens error-egenskap.

public function SQLErrorEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, error:SQLError = null)

Språkversion: ActionScript 3.0
Körningsmiljöversioner: AIR 1.0

Skapar ett SQLErrorEvent-objekt som fungerar som ett argument till händelseavlyssnare.

Parametrar
type:String — Händelsens typ finns i type-egenskapen. SQLErrorEvent definierar en händelsetyp, error-händelsen representeras av SQLErrorEvent.ERROR-konstanten.
 
bubbles:Boolean (default = false) — Bestämmer om Event-objektet deltar i bubblingssteget av händelseflödet. Standardvärdet är false.
 
cancelable:Boolean (default = false) — Avgör om Event-objektet kan avbrytas. Standardvärdet är false.
 
error:SQLError (default = null) — SQLError-objektet innehåller detaljer om felet.

Konstantdetaljer
    

ERROR

Konstant
public static const ERROR:String = "error"

Språkversion: ActionScript 3.0
Körningsmiljöversioner: AIR 1.0

Konstanten SQLErrorEvent.ERROR definierar värdet på type-egenskap i en felhändelse som skickas när ett anrop till en SQLConnection-metod eller SQLStatement-instans som slutar i ett fel. Error-händelsen har följande egenskaper:

EgenskapVärde
bubblesfalse
cancelablefalse; det finns inget standardbeteende att avbryta.
errorEtt SQLError-objekt innehåller information om felet som inträffade och åtgärden som har orsakat felet.
currentTargetDet objekt som aktivt behandlar event-objektet med en händelseavlyssnare.
targetSQLConnection eller SQLStatement-objektet rapporterar felet.
