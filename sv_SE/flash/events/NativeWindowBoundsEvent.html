Klasspublic class NativeWindowBoundsEvent
ArvNativeWindowBoundsEvent Event Object
Underklasser FlexNativeWindowBoundsEvent

Språkversion: ActionScript 3.0
Körningsmiljöversioner: AIR 1.0

Ett NativeWindow-objekt skickar ett NativeWindowBoundsEvent-objekt när fönstrets storlek eller placering ändras.

Det finns fyra typer av händelser:

  • NativeWindowBoundsEvent.MOVING
  • NativeWindowBoundsEvent.MOVE
  • NativeWindowBoundsEvent.RESIZING
  • NativeWindowBoundsEvent.RESIZE

Egenskapsdetaljer
    

afterBounds

egenskap
afterBounds:Rectangle  [skrivskyddad]

Språkversion: ActionScript 3.0
Körningsmiljöversioner: AIR 1.0

Fönstrets gränser efter ändringen.

Om händelsen är moving eller resizing har gränserna inte ändrats ännu; afterBounds visar de nya gränserna om händelsen inte avbryts. Om händelsen är move eller resize visar afterBounds de nya gränserna.

Språkversion: ActionScript 3.0

Fönstrets gränser före ändringen.

Om händelsen är moving eller resizing, har gränserna inte ändras ännu; beforeBounds visar de aktuella gränserna. Om händelsen är move eller resize visar beforeBounds de aktuella gränserna.

Skapar ett nytt Event-objekt med specifik information som är relevant för fönstrets gränshändelser. Event-objekt skickas som parametrar till händelseavlyssnare.

Parametrar
type:String — Händelsens typ. Möjliga värden är:
  • NativeWindowBoundsEvent.MOVING
  • NativeWindowBoundsEvent.MOVE
  • NativeWindowBoundsEvent.RESIZING
  • NativeWindowBoundsEvent.RESIZE
 
bubbles:Boolean (default = false) — Anger om Event-objektet deltar i händelseflödets bubblingssteg.
 
cancelable:Boolean (default = false) — Anger om Event-objektet kan avbrytas.
 
beforeBounds:Rectangle (default = null) — Anger gränserna före den senaste ändringen eller väntande ändringen.
 
afterBounds:Rectangle (default = null) — Anger gränserna efter den senaste ändringen eller väntande ändringen.
