Klasspublic final class LastOperationStatus
ArvLastOperationStatus Object

Språkversion: ActionScript 3.0
Körningsmiljöversioner: Flash Player 10.1, AIR 2

Klassen LastOperationStatus räknar upp konstanta värden, som representerar statusen för den senaste globaliseringstjänståtgärden. De här värdena kan hämtas med den skrivskyddade egenskapen lastOperationStatus, som är tillgänglig i de flesta globaliseringsklasserna.

ILLEGAL_ARGUMENT_ERROR

Konstant 
public static const ILLEGAL_ARGUMENT_ERROR:String = "illegalArgumentError"

Språkversion: ActionScript 3.0
Körningsmiljöversioner: Flash Player 10.1, AIR 2

Anger att ett argument som skickades till en metod var ogiltigt.

Följande kod visar till exempel att en felstatus för ogiltigt argument anges när egenskapen CurrencyFormatter.grouping får det ogiltiga värdet "3;". 

        var cf:CurrencyFormatter = new CurrencyFormatter("en-US");
        cf.groupingPattern = "3;";
        trace(cf.lastOperationStatus); // "illegalArgumentError"

PARSE_ERROR

Konstant 
public static const PARSE_ERROR:String = "parseError"

Språkversion: ActionScript 3.0
Körningsmiljöversioner: Flash Player 10.1, AIR 2

Anger att tolkningen av ett tal misslyckades. Det här statusvärdet kan returneras av tolkningsmetoder för formateringsklasser, som exempelvisCurrencyFormatter.parse() och NumberFormatter.parseNumber(). Om till exempel värdet "12abc34" skickas som parameter till metoden CurrencyFormatter.parse() blir returvärdet "NaN" och värdet lastOperationStatus anges till LastOperationStatus.PARSE_ERROR.

PATTERN_SYNTAX_ERROR

Konstant 
public static const PATTERN_SYNTAX_ERROR:String = "patternSyntaxError"

Språkversion: ActionScript 3.0
Körningsmiljöversioner: Flash Player 10.1, AIR 2

Anger att mönstret för att formatera ett tal, ett datum eller en tid är ogiltigt. Detta statusvärde anges när användarens operativsystem inte har stöd för det givna mönstret.

Följande kod visar till exempel värdet på egenskapen lastOperationStatus efter att ett ogiltigt "xx"-mönster har använts för datumformatering:

        var df:DateTimeFormatter = new DateTimeFormatter("en-US");
        df.setDateTimePattern("xx");
        trace(df.lastOperationStatus); // "patternSyntaxError"
