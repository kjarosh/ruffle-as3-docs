Paketet flash.display3D innehåller de huvudklasser som Flash Player använder för att tillhandahålla högpresterande återgivningskontext.


Klasser
 KlassBeskrivning
 Context3D Klassen Context3D ger ett sammanhang för återgivningen av geometriskt definierad grafik.
 Context3DBlendFactor Definierar värdena som ska användas för att ange blandningsfaktorer för källan och målet.
 Context3DBufferUsage Definierar värdena som används för att ange typ av buffertanvändning.
 Context3DClearMask Definierar det värden som ska användas för att ange rensningsmaskar för Context3D.
 Context3DCompareMode Definierar värdet som ska användas för att ange 3D-buffertjämförelser i metoderna setDepthTest() och setStencilAction() i en Context3D-instans.
 Context3DFillMode     Definierar värden som ska användas för att ange fyllningsläget för 3D-återgivning i metoderna setFillMode() i en Context3D-instans.
 Context3DMipFilter Definierar värdena som ska användas för läget för provmipmap-filter
 Context3DProfile Definierar de värden som ska användas för att ange profilen för Context3D.
 Context3DProgramType Definierar värdena som ska användas för att ange om ett skuggningsprogram är ett fragment- eller ett vertex-program.
 Context3DRenderMode Definierar de värden som ska användas för att ange återgivningsläget för Context3D.
 Context3DStencilAction Definierar värden som ska användas för att ange stencilåtgärder.
 Context3DTextureFilter Definierar värdena som ska användas för samplerfilterläget.
 Context3DTextureFormat Definierar värden som ska användas för att ange ett texturformat.
 Context3DTriangleFace Konstanter för att ange orienteringen för en triangel relativt till visningspunkten.
 Context3DVertexBufferFormat Definierar värden som används för att ange vertex-buffertar.
 Context3DWrapMode Definierar värdena som ska användas för provbrytningsläget
 IndexBuffer3D IndexBuffer3D används för att representera listor med vertex-index bestående av grafiska element kvarhållna i det grafiska undersystemet.
 Program3D Klassen Program3D representerar två återgivningsprogram (även kallade "skuggningsprogram") överförda till återgivningssammanhanget.
 VertexBuffer3D Klassen VertexBuffer3D representerar en uppsättning vertex-data skickade till ett återgivningssammanhang.

[ X ]Varför på engelska?
Innehåll i Referenshandbok för ActionScript 3.0 i Adobe Flash-plattformen är på engelska

Det är inte alla delar av Referenshandbok för ActionScript 3.0 i Adobe Flash-plattformen som översätts till alla språk. Om ett språkelement inte översätts visas det på engelska. Klassen ga.controls.HelpBox översätts till exempel inte till något språk. I den svenska versionen av referenshandboken visas därför klassen ga.controls.HelpBox på engelska.
