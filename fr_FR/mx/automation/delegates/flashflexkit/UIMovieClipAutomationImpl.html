addLayoutCompleteSynchronization

()méthode
protected function addLayoutCompleteSynchronization():void

Version du langage: ActionScript 3.0
Version du produit: Flex 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Configure une synchronisation d’automatisation avec l’événement UPDATE_COMPLETE du gestionnaire de mises en forme. Lorsque certaines actions sont réexécutées, l’automatisation doit attendre pour pouvoir réexécuter l’événement suivant. Cette attente est nécessaire pour que la structure puisse terminer les actions requises par le composant. Normalement, un événement update_complete du gestionnaire de mises en forme signale la fin de toutes les mises à jour. Cette méthode ajoute une synchronisation signalée comme terminée lorsque l’événement update_complete est reçu.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
