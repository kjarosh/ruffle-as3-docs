Classepublic class MediaFactoryItem
HéritageMediaFactoryItem Object

Version du langage: ActionScript 3.0
Version du produit: OSMF 1.0
Versions du moteur d’exécution: Flash Player 10, AIR 1.5

MediaFactoryItem est l’encapsulation de toutes les informations requises pour créer de façon dynamique et initialiser un MediaElement à partir d’un MediaFactory.

Les objets MediaFactoryItem sont exposés par des modules externes (dans la classe PluginInfo) et utilisés par la structure pour créer les MediaElements spécifiés par le module externe.

public function MediaFactoryItem(id:String, canHandleResourceFunction:Function, mediaElementCreationFunction:Function, type:String = null)

Version du langage: ActionScript 3.0
Version du produit: OSMF 1.0
Versions du moteur d’exécution: Flash Player 10, AIR 1.5

Constructeur.

Paramètres
id:String — Un identifiant qui représente ce MediaFactoryItem. Les identifiants doivent contenir le nom du fabricant du module externe et le nom de l’élément qu’il génère. La convention consiste à utiliser le plan de l’espace de noms du package. Deux exemples : com.example.MyAdPlugin com.example.MyAnalyticsPlugin Remarque : org.osmf doit être évité car le MediaFactory donne la priorité aux modules externes non osmf.
 
canHandleResourceFunction:Function — Fonction qui est utilisée pour déterminer si ce MediaFactoryItem est capable de gérer une ressource. La fonction doit prendre un seul paramètre de type MediaResourceBase et renvoyer une valeur booléenne.
 
mediaElementCreationFunction:Function — Fonction qui crée une nouvelle instance du MediaElement désiré. La fonction ne doit accepter aucun paramètre et renvoyer un MediaElement.
 
type:String (default = null) — Le type de ce MediaFactoryItem. Si null, la valeur par défaut est MediaFactoryItemType.STANDARD.

Valeur émise
ArgumentError — Si un argument (excepté type) est null.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
