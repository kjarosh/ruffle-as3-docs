Packageflash.system
Classepublic final class IMEConversionMode
HéritageIMEConversionMode Object

Version du langage: ActionScript 3.0
Versions du moteur d’exécution: AIR 1.0, Flash Player 9

Cette classe contient des constantes à utiliser avec la propriété IME.conversionMode. La définition de conversionMode sur ALPHANUMERIC_FULL ou JAPANESE_KATAKANA_FULL force le lecteur à utiliser une police complète, tandis que ALPHANUMERIC_HALF ou JAPANESE_KATAKANA_HALF utilise une police de demi-largeur.

public static const UNKNOWN:String = "UNKNOWN"

Version du langage: ActionScript 3.0
Versions du moteur d’exécution: AIR 1.0, Flash Player 9

La chaîne "UNKNOWN" qui peut être renvoyée par un appel à la propriété IME.conversionMode. Cette valeur ne peut pas être définie et n’est renvoyée que si le lecteur n’est pas en mesure d’identifier l’IME actif. Utilisez la syntaxe IMEConversionMode.UNKNOWN.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
