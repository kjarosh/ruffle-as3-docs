content:Object

Version du langage: ActionScript 3.0
Version du produit: Flex 4.5
Versions du moteur d’exécution: Flash Player 10, AIR 1.5

Référence au contenu. Il peut s’agir (entre autres facteurs), d’une instance de LoaderInfo, BitmapData, ou tout autre contenu générique. Lorsque l’événement complete est lancé et/ou que complete() renvoie true, le contenu est considéré comme valide.

ContentRequest

()Constructeur
public function ContentRequest(contentLoader:IContentLoader, content:*, shared:Boolean = false, complete:Boolean = false)

Version du langage: ActionScript 3.0
Version du produit: Flex 4.5
Versions du moteur d’exécution: Flash Player 10, AIR 1.5

Constructeur.

Paramètres
contentLoader:IContentLoader — Objet IContentLoader.
 
content:* — Référence au contenu.
 
shared:Boolean (default = false) — La valeur true indique que cette demande est actuellement partagée par d’autres demandes précédentes
 
complete:Boolean (default = false) — La valeur true indique que quelqu’un a appelé le chargement sur une mémoire cache et que la mémoire cache a immédiatement renvoyé un résultat de chargement complet

[ X ]Pourquoi existe-t-il du contenu en anglais ?
Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
