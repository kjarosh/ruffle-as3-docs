Clasepublic class StageVideoAvailabilityEvent
HerenciaStageVideoAvailabilityEvent Event Object

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: Flash Player 10.2, AIR 3

Este evento se activa cuando el estado de la propiedad Stage.stageVideos cambia. Esta propiedad puede cambiar cuando el usuario amplía a modo de pantalla completa desde un modo wmode que no admita StageVideo (por ejemplo, wmode=normal, wmode=opaque o wmode=transparent). Ampliar a pantalla completa puede hacer que el vector Stage.stageVideos se llene. A la inversa, salir del modo de pantalla completa puede provocar que el vector Stage.stageVideos se vacíe.

NOTA: esta notificación sólo se produce cuando el estado de la propiedad Stage.stageVideos cambia. Como resultado, el comportamiento puede variar según la plataforma y el navegador. En Windows, por ejemplo, el evento stageVideoAvailability no se distribuye cuando se pasa al modo de pantalla completa si wmode se ajusta en direct. En otras plataformas, sin embargo, el mismo comportamiento hace que Flash Player reasigne recursos. En estos casos, la propiedad Stage.stageVideos cambia de estado y se activa el evento. Se pueden detectar cambios en el modo de pantalla completa detectando el evento flash.events.FullScreenEvent. Este evento lo distribuye el objeto Stage.
