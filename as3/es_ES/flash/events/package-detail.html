El paquete flash.events admite el nuevo modelo de evento DOM e incluye la clase base EventDispatcher.


Interfaces
 InterfazDescripción
 IEventDispatcher La interfaz IEventDispatcher define métodos para añadir o quitar detectores de eventos, comprueba si hay registrados tipos específicos de detectores y distribuye eventos.
