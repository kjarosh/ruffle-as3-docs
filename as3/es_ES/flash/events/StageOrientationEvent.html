Clasepublic class StageOrientationEvent
HerenciaStageOrientationEvent Event Object

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 2

Un objeto Stage distribuye un objeto StageOrientationEvent cuando cambia la orientación del escenario. Esto puede suceder cuando el dispositivo se gira, cuando el usuario abre un teclado deslizante o cuando se llama al método setAspectRatio() del escenario.

Existen dos tipos de evento StageOrientationEvent: orientationChanging (StageOrientationEvent.ORIENTATION_CHANGING) se distribuye antes de la pantalla cambie a una nueva orientación. Llamar al método preventDefault() del objeto de evento distribuido para orientationChanging impide que el escenario cambie de orientación. orientationChange (StageOrientationEvent.ORIENTATION_CHANGE:) se distribuye una vez que la pantalla cambia a una nueva orientación.

Nota: si la propiedad autoOrients es false, la orientación del escenario no cambia cuando se gira un dispositivo. Por consiguiente, StageOrientationEvents sólo se distribuye cuando gira el dispositivo si autoOrients es true.

StageOrientationEvent

()Información sobre
public function StageOrientationEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, beforeOrientation:String = null, afterOrientation:String = null)

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 2

Crea un objeto de evento StageOrientationEvent con información específica relacionada con los eventos de orientación del escenario. Los objetos de evento se transfieren como parámetros a los detectores de eventos. Generalmente no se crea este evento utilizando la función constructora. En su lugar, se añade un detector de eventos en el objeto Stage para detectar estos eventos a medida que se producen.

Parámetros
type:String — El tipo del evento: StageOrientationEvent.ORIENTATION_CHANGE
 
bubbles:Boolean (default = false) — Indica si el objeto de evento participa en la fase de propagación del flujo del evento.
 
cancelable:Boolean (default = false) — Indica si se puede cancelar el objeto Event.
 
beforeOrientation:String (default = null) — Indica la orientación antes del cambio.
 
afterOrientation:String (default = null) — Indica la orientación tras el cambio.
