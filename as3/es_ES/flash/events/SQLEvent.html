SQLEvent

()Información sobre
public function SQLEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false)

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 1.0

Crea un objeto SQLEvent para transmitirlo como un parámetro a los detectores de eventos.

Parámetros
type:String — Tipo de evento. Se puede acceder desde la propiedad type.
 
bubbles:Boolean (default = false) — Determina si el objeto de evento participa en la fase de propagación del flujo del evento. El valor predeterminado es false.
 
cancelable:Boolean (default = false) — Determina si se puede cancelar el objeto Event. El valor predeterminado es false.

ANALYZE

Constante
public static const ANALYZE:String = "analyze"

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 1.0

La constante SQLEvent.ANALYZE define el valor de la propiedad type de un objeto de evento analyze. Este tipo de evento se distribuye cuando finaliza correctamente una llamada al método SQLConnection.analyze(). El evento analyze tiene las siguientes propiedades:

PropiedadValor
bubblesfalse
cancelablefalse; no hay ningún comportamiento predeterminado que cancelar.
currentTargetObjeto que procesa de forma activa el objeto de evento con un detector de eventos.
targetObjeto SQLConnection que ha llevado a cabo la operación.
