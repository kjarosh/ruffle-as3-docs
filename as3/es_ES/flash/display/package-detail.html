El paquete flash.display contiene la clases principales que Flash Player utiliza para crear presentaciones visuales.


Interfaces
 InterfazDescripción
 IBitmapDrawable La interfaz IBitmapDrawable se implementa mediante objetos que pueden pasarse como parámetro source del método draw() de la clase BitmapData.
 IGraphicsData Esta interfaz se utiliza para definir objetos que pueden usarse como parámetros en los métodos flash.display.Graphics, incluidos rellenos, trazos y trazados.
 IGraphicsFill Esta interfaz se utiliza para definir objetos que se pueden utilizar como parámetros de relleno en los métodos flash.display.Graphics y en las clases de dibujo.
 IGraphicsPath Esta interfaz se utiliza para definir objetos que se pueden utilizar como parámetros de relleno en los métodos flash.display.Graphics y en las clases de dibujo.
 IGraphicsStroke Esta interfaz se utiliza para definir objetos que se pueden utilizar como parámetros de trazo en los métodos flash.display.Graphics y en las clases de dibujo.

 AVLoader 
 AVM1Movie AVM1Movie es una clase simple que representa clips de película AVM1, que utilizan ActionScript 1.0 o 2.0.

 BitmapDataChannel La clase BitmapDataChannel es una enumeración de valores constantes que indican el canal que debe utilizarse: rojo, azul, verde o transparencia alfa.
 BitmapEncodingColorSpace La clase BitmapEncodingColorSpace define las constantes que especifican cómo se muestrean los canales de color por el método flash.display.BitmapData.encode() al especificar el compresor como flash.display.JPEGXREncoderOptions.
 BlendMode Clase que proporciona valores constantes para efectos visuales de modos de mezcla.
 CapsStyle La clase CapsStyle es una enumeración de valores constantes que especifican el estilo de los extremos que debe utilizarse al dibujar líneas.
 ColorCorrection La clase ColorCorrection proporciona valores para la propiedad flash.display.Stage.colorCorrection.
 ColorCorrectionSupport La clase ColorCorrectionSupport proporciona valores para la propiedad flash.display.Stage.colorCorrectionSupport.
 DisplayObject La clase DisplayObject es la clase base de todos los objetos que pueden situarse en la lista de visualización.
 DisplayObjectContainer La clase DisplayObjectContainer es la clase base de todos los objetos que pueden actuar como contenedores de objetos de visualización en la lista de visualización.
 FocusDirection     La clase FocusDirection enumera valores que se utilizan en el parámetro direction del método assignFocus() de un objeto Stage y en la propiedad direction de un objeto FocusEvent.
 FrameLabel El objeto FrameLabel contiene propiedades que especifican un número de fotograma y el correspondiente nombre de etiqueta.
 GradientType La clase GradientType proporciona valores para el parámetro type en los métodos beginGradientFill() y lineGradientStyle() de la clase flash.display.Graphics.
 Graphics La clase Graphics contiene un conjunto de métodos que puede utilizar para crear una figura vectorial.

 GraphicsEndFill Indica el final de un relleno gráfico.

 GraphicsPathCommand Define los valores que se utilizan para especificar comandos de dibujo de trazados.
 GraphicsPathWinding La clase GraphicsPathWinding proporciona valores para la propiedad flash.display.GraphicsPath.winding y el método flash.display.Graphics.drawPath() para determinar la dirección de dibujo de un trazado.
 GraphicsShaderFill Define un relleno sombreado.
 GraphicsSolidFill Define un relleno sólido.
 GraphicsStroke Define un estilo de línea o un trazo.
 GraphicsTrianglePath Define un conjunto ordenado de triángulos que se pueden representar con coordenadas de relleno (u,v) o con un relleno normal.

 InterpolationMethod La clase InterpolationMethod proporciona valores para el parámetro interpolationMethod en los métodos Graphics.beginGradientFill() y Graphics.lineGradientStyle().
 JointStyle La clase JointStyle es una enumeración de valores constantes que especifican el estilo de juntas que debe utilizarse al dibujar líneas.

 LineScaleMode La clase LineScaleMode proporciona valores para el parámetro scaleMode en el método Graphics.lineStyle().
 Loader La clase Loader se utiliza para cargar archivos SWF o archivos de imagen (JPG, PNG o GIF).
 LoaderInfo La clase LoaderInfo proporciona información sobre un archivo SWF cargado o un archivo de imagen cargado (JPEG, GIF o PNG).
 MorphShape La clase MorphShape representa objetos MorphShape en la lista de visualización.
 MovieClip La clase MovieClip hereda de las siguientes clases: Sprite, DisplayObjectContainer, InteractiveObject, DisplayObject y EventDispatcher.

 NativeWindowRenderMode     La clase NativeWindowRenderMode define constantes para la propiedad renderMode del objeto NativeWindowInitOptions utilizado para crear una ventana nativa.
 NativeWindowResize     La clase NativeWindowResize define constantes para los valores posibles del parámetro edgeOrCorner del método NativeWindow startResize().
 NativeWindowSystemChrome     La clase NativeWindowSystemChrome define constantes para la propiedad systemChrome del objeto NativeWindowInitOptions utilizado para crear una ventana nativa.
 NativeWindowType     La clase NativeWindowType define constantes para la propiedad type del objeto NativeWindowInitOptions utilizado para crear una ventana nativa.
 PixelSnapping La clase PixelSnapping es una enumeración de valores constantes para establecer las opciones de ajuste en píxeles a través de la propiedad pixelSnapping de un objeto Bitmap.

 Scene La clase Scene incluye propiedades para identificar el nombre, las etiquetas y el número de fotogramas de una escena.
 Screen     La clase Screen proporciona información sobre las pantallas de visualización disponibles en la aplicación.
 ScreenMode     El objeto del ScreenMode proporciona la información sobre la anchura, la altura y la frecuencia de actualización de una pantalla.
 Shader Una instancia de Shader representa un kernel de sombreado de Pixel Bender en ActionScript.
