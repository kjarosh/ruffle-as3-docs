Información sobre constantes

EVEN_ODD

Constante
public static const EVEN_ODD:String = "evenOdd"

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: Flash Player 10, AIR 1.5

Establece el tipo de número de vueltas par/impar. El tipo de número de vueltas par/impar es la regla utilizada por todas las API de dibujo originales y es el tipo predeterminado del método flash.display.Graphics.drawPath(). Todos los trazados superpuestos alternan trazados abiertos y cerrados. Si dos cuadrados dibujados con el mismo relleno intersecan, el área de intersección no se rellena. Las áreas adyacentes no son las mismas (ni rellenas ni sin rellenar).

NON_ZERO

Constante 
public static const NON_ZERO:String = "nonZero"

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: Flash Player 10, AIR 1.5

Establece el tipo de número de vueltas distinto de cero. El tipo de número de vueltas distinto de cero determina que, cuando intersecan trazados con dirección de trazado opuesta, el área de intersección no se rellena (como ocurre con el tipo de número de vueltas par/impar). En el caso de trazados con la misma dirección de trazado, el área de intersección sí se rellena.
