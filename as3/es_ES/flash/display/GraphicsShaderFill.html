Información sobre propiedades

matrix

propiedad
public var matrix:Matrix

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: Flash Player 10, AIR 1.5

Objeto matrix (de la clase flash.geom.Matrix) que puede utilizar para definir transformaciones en el sombreado. Por ejemplo, es posible utilizar la siguiente matriz para girar un sombreado 45 grados (pi/4 radianes): 

     matrix = new flash.geom.Matrix(); 
     matrix.rotate(Math.PI / 4);

Las coordenadas recibidas en el sombreado se basan en la matriz especificada para el parámetro matrix. Para una matriz (null) predeterminada, las coordenadas del sombreado son coordenadas de píxeles locales que se pueden utilizar para crear una entrada de muestra.

Elementos de API relacionados

flash.geom.Matrix

shader

propiedad 
public var shader:Shader

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: Flash Player 10, AIR 1.5

El sombreado que se utiliza en el relleno. Esta instancia de Shader no se requiere para especificar una entrada de imagen. No obstante, si se especifica una entrada de imagen en el sombreado, la entrada debe proporcionarse manualmente mediante la configuración de la propiedad input de la propiedad ShaderInput correspondiente de la propiedad Shader.data.

Cuando se transfiere una instancia de Shader como un argumento, el sombreado se copia internamente y la operación de relleno de dibujo utiliza la copia interna, no una referencia al objeto shader original. Todos los cambios realizados en el objeto shader, como la modificación de los valores de los parámetros, la entrada o el código de bytes, no se aplican al sombreado copiado que se utiliza en el relleno.
