Clasepublic final class IMEConversionMode
HerenciaIMEConversionMode Object

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 1.0, Flash Player 9

Esta clase contiene constantes que se utilizan con la propiedad IME.conversionMode. Si define conversionMode como ALPHANUMERIC_FULL o JAPANESE_KATAKANA_FULL, obligará al reproductor a utilizar una fuente de anchura completa, mientras que al usar ALPHANUMERIC_HALF o JAPANESE_KATAKANA_HALF se utiliza una fuente de media anchura.
