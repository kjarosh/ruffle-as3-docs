El paquete flash.system contiene clases para el acceso a funcionalidad a nivel del sistema, como, por ejemplo, seguridad, eliminación de datos innecesarios, etc.


Funciones
 FunciónDescripción
 fscommand Permite que el archivo SWF se comunique con Flash Player o con el programa que lo aloja, por ejemplo, un navegador web.
Clases
 ClaseDescripción
 ApplicationDomain La clase ApplicationDomain es un contenedor para grupos discretos de definiciones de clases.
 Capabilities La clase Capabilities ofrece propiedades que describen el sistema y el motor de ejecución que alojan la aplicación.
 ImageDecodingPolicy     La clase ImageDecodingPolicy proporciona valores para imageDecodingPolicy en la clase LoaderContext.
 IME La clase IME permite manipular directamente el editor de método de entrada (IME) del sistema operativo en la aplicación del motor de ejecución de Flash que se ejecuta en un equipo cliente.
 IMEConversionMode Esta clase contiene constantes que se utilizan con la propiedad IME.conversionMode.
 JPEGLoaderContext La clase JPEGLoaderContext incluye una propiedad para habilitar un filtro de desbloqueo cuando se carga una imagen JPEG.
 LoaderContext La clase LoaderContext ofrece opciones para cargar archivos SWF y otros medios con la clase Loader.
 MessageChannel La clase MessageChannel proporciona un mecanismo para que un programa de trabajo pueda comunicarse con otro.
 MessageChannelState Esta clase define las constantes que representan los posible valores de la propiedad state de la clase MessageChannel.
 Security La clase Security permite especificar cómo puede comunicarse entre sí contenido de diferentes dominios.
 SecurityDomain La clase SecurityDomain representa el "entorno limitado" de seguridad actual, también conocido como dominio de seguridad.
 SecurityPanel La clase SecurityPanel ofrece valores para especificar qué panel Parámetros de seguridad desea mostrar.
 System La clase System contiene propiedades relacionadas con la configuración local y las operaciones.
 SystemUpdater La clase SystemUpdater permite actualizar módulos del Flash Player como, por ejemplo, el módulo DRM para Adobe Access, así como el propio Flash Player.
 SystemUpdaterType La clase SystemUpdaterType proporciona constantes para una actualización del sistema.
 TouchscreenType La clase TouchscreenType es una clase de enumeración que proporciona valores para los distintos tipos de pantalla táctil.
 Worker Un objeto Worker representa un programa de trabajo, que es una instancia virtual del motor de ejecución de Flash.
 WorkerDomain La clase WorkerDomain proporciona un modo de crear objetos Worker y acceder a ellos.
 WorkerState Esta clase define las constantes que representan los posible valores de la propiedad state de la clase Worker.

Cierto contenido de la Referencia de ActionScript 3.0 se muestra en inglés

No todo el contenido de la Referencia de ActionScript 3.0 se traduce a todos los idiomas. Si un elemento del lenguaje no se traduce, aparecerá en inglés. Por ejemplo, la clase ga.controls.HelpBox no está traducida en ningún idioma. Por lo tanto, en la versión en español de la referencia, la clase ga.controls.HelpBox aparecerá en inglés.
