Clasepublic class X500DistinguishedName
HerenciaX500DistinguishedName Object

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: Flash Player 11, AIR 3

La clase X500DistinguishedName define propiedades de Distinguished Name (DN) que se utilizan en la clase X509Certificate. El protocolo Distinguished Name se especifica en RFC1779.

Esta clase resulta útil para cualquier código que necesite examinar el emisor y el asunto de un certificado de servidor tras haberse establecido una conexión de socket segura. Los datos de Subject e Issuer DN están accesibles en las propiedades subject e issuer de la clase X509Certificate. Estas propiedades se llenan con objetos X500DistinguishedName después de que SecureSocket.connect() establezca una conexión con el servidor.

Esta clase almacena atributos DN como propiedades de cadena. Se puede utilizar el método toString() para obtener todas las propiedades DN individuales en una cadena. Las propiedades con valor no DN están configuradas con el valor null.

Nota: las propiedades de X500DistinguishedName almacenan sólo la primera aparición de cada atributo DN, aunque el protocolo DN permite varios atributos del mismo tipo.
