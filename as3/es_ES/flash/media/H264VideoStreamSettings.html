HerenciaH264VideoStreamSettings VideoStreamSettings Object

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: Flash Player 11, AIR 3

La clase H264VideoStreamSettings permite especificar ajustes de compresión de vídeo para cada NetStream. Las propiedades serán validadas una vez que el objeto Camera se conecte a NetStream y haya comenzado la compresión.

setProfileLevel

()método
public function setProfileLevel(profile:String, level:String):void

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: Flash Player 11, AIR 3

Establece el perfil y el nivel de codificación de vídeo. Los valores posibles del perfil son H264Profile.BASELINE y H264Profile.MAIN. El valor predeterminado es H264Profile.BASELINE. Otros valores se omiten y el resultado es un error. Consulte la clase H264Profile para obtener más información. Los niveles admitidos son 1, 1b, 1.1, 1.2, 1.3, 2, 2.1, 2.2, 3, 3.1, 3.2, 4, 4.1, 4.2, 5 y 5.1. El nivel puede incrementarse si es necesario por la resolución y la velocidad de fotogramas. Consulte la clase H264Level para obtener más información.

Parámetros

profile:String — El perfil de codificador solicitado como una cadena. Consulte la clase H264Profile para conocer los posibles valores.
 
level:String — El nivel de codificador solicitado como una cadena. Consulte la clase H264Level clase para conocer los posibles valores.


Emite
Error — Si se especifica un perfil o nivel no válido.
