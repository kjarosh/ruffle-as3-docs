Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 3.4

La clase RemoteNotifier permite suscribirse a notificaciones push y anular la suscripción.

Esta clase incluye métodos para consultar qué elementos RemoteNotificationStyles se admiten en la plataforma de destino. Distribuye eventos RemoteNotificationEvent y StatusEvent.

    subscribe

()método
public function subscribe(options:RemoteNotifierSubscribeOptions = null):void

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 3.4

Suscribe/registra la aplicación para recibir notificaciones remotas de APN.

  • RemoteNotificationEvent (TOKEN) se activa una vez que el símbolo está disponible. Este símbolo se utiliza en los proveedores de notificaciones remotas para enviar notificaciones remotas a esta aplicación en el dispositivo mediante APN.
  • RemoteNotificationEvent (NOTIFICATION) se activa cuando el SO de la plataforma comunica la notificación a la aplicación.
  • StatusEvent (STATUS) se activa cuando se produce un error durante la suscripción a notificaciones remotas.

Parámetros

options:RemoteNotifierSubscribeOptions (default = null) — Opciones necesarias para el proceso de suscripción en determinadas plataformas.

notification

 Evento
Tipo de objeto de evento: flash.events.RemoteNotificationEvent
propiedad RemoteNotificationEvent.type = flash.events.RemoteNotificationEvent

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 3.4

Se distribuye como respuesta a una nueva notificación recibida por la aplicación.

En iOS, este evento se activa en las siguientes dos situaciones:

  • La aplicación estaba en primer plano cuando se recibió la nueva notificación.
  • A través de una ventana emergente de notificación, el usuario toca el botón de acción de la alerta o toca (hace clic) en el icono de la aplicación. En este caso, no está disponible ninguna información sobre la carga de notificación.
