Add

()método
public function Add(inMatrix:DynamicMatrix):Boolean

Versión del lenguaje: ActionScript 3.0
Versión de producto: Flash CS3
Versiones de motor de ejecución: Flash Player 9, AIR 1.0

Suma la matriz actual a una matriz especificada. La matriz actual se convierte en el resultado de la suma (dicho de otro modo, el método DynamicMatrix.Add() no crea una nueva matriz para el resultado).

Parámetros

inMatrix:DynamicMatrix — La matriz que se añade a la matriz actual.

Valor devuelto
Boolean — Un valor booleano que indica si la suma se ha realizado correctamente (true) o no (false). Si las dimensiones de las matrices no coinciden, DynamicMatrix.Add() devuelve false.

Multiply

()método 
public function Multiply(inMatrix:DynamicMatrix, order:int):Boolean

Versión del lenguaje: ActionScript 3.0
Versión de producto: Flash CS3
Versiones de motor de ejecución: Flash Player 9, AIR 1.0

Multiplica la matriz actual por una matriz especificada que, o bien incorpora como prefijo o como sufijo. Utilice el segundo parámetros del método DynamicMatrix.Multiply() para incorporar la matriz especificada como prefijo o como sufijo.

Parámetros

inMatrix:DynamicMatrix — La matriz que se añade a la matriz actual.
 
order:int (default = NaN) — Especifica si se incorpora como prefijo o como sufijo la matriz del parámetro inMatrix: MATRIX_ORDER_APPEND o MATRIX_ORDER_PREPEND.

Valor devuelto
Boolean — Un valor booleano que indica si la multiplicación se ha realizado correctamente (true) o no (false). El valor es false si la matriz actual o la matriz especificada (el parámetro inMatrix) es null, o si se debe anexar y la anchura actual de la matriz no coincide con la altura de la matriz suministrada; o si se debe colocar delante y la altura actual de la matriz no coincide con la anchura de la matriz suministrada.
