El paquete fl.controls contiene clases de componentes de nivel superior, como List, Button y ProgressBar. Este paquete también contiene clases abstractas como SelectableList, que amplían todos los componentes basados en listas. En este paquete también puede encontrar clases que definen constantes que los componentes utilizan o clases que admiten componentes individuales.


Clases
 ClaseDescripción
 BaseButton BaseButton es la clase base de todos los componentes button y define las propiedades y métodos que son comunes a todos los componentes.
 Button El componente Button representa un botón rectangular de uso frecuente.
 ButtonLabelPlacement La clase ButtonLabelPlacement define las constantes de los valores de la propiedad labelPlacement de un componente Button, CheckBox o RadioButton.
 CheckBox El componente CheckBox muestra un pequeño cuadro que puede incluir una marca de verificación.
 ColorPicker El componente ColorPicker muestra una lista de una o varias muestras en las que el usuario puede seleccionar un color.
 ComboBox El componente ComboBox contiene una lista desplegable en la que el usuario puede seleccionar un único valor.
 DataGrid La clase DataGrid es un componente basado en listas que proporciona una cuadrícula de filas y columnas.
 Label Un componente Label muestra una o varias líneas de texto sin formato o con formato HTML a las que se puede dar formato para alineación y tamaño.
 LabelButton La clase LabelButton es una clase abstracta que amplía la clase BaseButton al añadirle una etiqueta, un icono y la funcionalidad de activación.
 List El componente List muestra información basada en listas y resulta idóneo para la visualización de conjuntos de información.
 NumericStepper El componente NumericStepper muestra un conjunto ordenado de números en el que el usuario puede realizar una selección.
 ProgressBar El componente ProgressBar muestra el progreso del contenido que se está cargando.
 ProgressBarDirection La clase ProgressBarDirection define los valores de la propiedad direction de la clase ProgressBar.
 ProgressBarMode La clase ProgressBarMode define los valores de la propiedad mode de la clase ProgressBar.
 RadioButton El componente RadioButton permite obligar al usuario a seleccionar una opción de un conjunto de opciones.
 RadioButtonGroup La clase RadioButtonGroup define un grupo de componentes RadioButton para que actúen como un solo componente.
 ScrollBar El componente ScrollBar ofrece al usuario final un modo de controlar la parte de datos que se muestra cuando hay demasiada información para ajustar en el área de visualización.
 ScrollBarDirection Define los valores de la propiedad de dirección del componente ScrollBar.
 ScrollPolicy Valores de las propiedades horizontalScrollPolicy y verticalScrollPolicy de la clase BaseScrollPane.
 SelectableList SelectableList es la clase base de todos los componentes basados en listas como, por ejemplo, List, TileList, DataGrid y ComboBox.
 Slider El componente Slider permite al usuario seleccionar un valor moviendo el deslizador entre los extremos de la guía de desplazamiento.
 SliderDirection Orientación del componente Slider.
 TextArea El componente TextArea es un campo de texto de varias líneas con un borde y barras de desplazamiento opcionales.
 TextInput TextInput es un componente de una sola línea de texto que incluye un objeto TextField nativo de ActionScript.
 TileList La clase TileList proporciona una cuadrícula de filas y columnas que suele utilizarse para dar formato y mostrar imágenes en "mosaico".
 UIScrollBar La clase UIScrollBar incluye toda la funcionalidad de la barra de desplazamiento, y añade una propiedad scrollTarget para que se pueda asociar a una instancia del componente TextField o TLFTextField.

[ X ]¿Por qué hay texto en inglés?
Cierto contenido de la Referencia de ActionScript 3.0 se muestra en inglés

No todo el contenido de la Referencia de ActionScript 3.0 se traduce a todos los idiomas. Si un elemento del lenguaje no se traduce, aparecerá en inglés. Por ejemplo, la clase ga.controls.HelpBox no está traducida en ningún idioma. Por lo tanto, en la versión en español de la referencia, la clase ga.controls.HelpBox aparecerá en inglés.
