HerenciaStyleManager Object

Versión del lenguaje: ActionScript 3.0
Versión de producto: Flash CS3
Versiones de motor de ejecución: Flash Player 9.0.28.0, AIR 1.0

La clase StyleManager proporciona métodos estáticos que se pueden utilizar para obtener y definir estilos de una instancia de componente, de un tipo completo de componente o de todos los componentes de la interfaz de usuario en un documento de Flash. Los estilos se definen como valores que influyen en la visualización de un componente, incluido el relleno, los formatos de texto y los aspectos.

Registra una instancia de componente con Style Manager. Tras crear una instancia de un componente, ésta se puede registrar con Style Manager para que se notifiquen los cambios de estilo. Las instancias de componente se pueden registrar para recibir notificaciones sobre los cambios de estilo generales o basados en componentes.

Parámetros

instance:UIComponent — Instancia de componente que se va a registrar en la administración de estilos.

Define un estilo en todas las instancias de un tipo de componente, por ejemplo, en todas las instancias de un componente Button o de un componente ComboBox.

Parámetros

component:Object — Tipo de componente, por ejemplo, Button o ComboBox. Este parámetro también acepta una clase o instancia de componente que se pueda utilizar para identificar todas las instancias de un tipo de componente.
 
name:String — Nombre del estilo que se va a definir.
 
style:Object — Objeto de estilo que describe el estilo que se va a definir.

setStyle

()método 
public static function setStyle(name:String, style:Object):void

Versión del lenguaje: ActionScript 3.0
Versión de producto: Flash CS3
Versiones de motor de ejecución: Flash Player 9.0.28.0, AIR 1.0

Define un estilo global para todos los componentes de la interfaz de usuario de un documento.

Parámetros

name:String — Valor de cadena que asigna un nombre al estilo que se va a definir.
 
style:Object — Objeto de estilo que se va a definir. El valor de esta propiedad depende del estilo que define el usuario. Por ejemplo, si el estilo se define como "textFormat", la propiedad de estilo se debería definir como un objeto TextFormat. Discrepancia entre el nombre del estilo y el valor de la propiedad de estilo puede provocar que el componente actúe de forma incorrecta.

StyleManagerExample.as

Este ejemplo muestra cómo crear de forma dinámica un componente DataGrid y detectar sus eventos.

Para realizar el ejemplo, siga estos pasos:

  1. Añada los componentes DataGrid, Button, Label y TextArea a la biblioteca.
  2. Guarde este código como StyleManagerExample.as en el mismo directorio que el archivo FLA.
  3. Defina la clase Document como StyleManagerExample en el archivo FLA.

        private function setupStyles():void {
            formatForGlobal = new TextFormat();
            formatForGlobal.font = "Georgia";
            formatForGlobal.size = 10;
            formatForInstance = new TextFormat();
            formatForInstance.underline = true;            
        }
        private function setInstanceStyle(e:MouseEvent):void {
            var fm:FocusManager = new FocusManager(this);
            var uiComponent:UIComponent = fm.getFocus() as UIComponent;
            if(uiComponent is TextArea) {
                uiComponent.setStyle("textFormat",formatForInstance);    
                description.text = "The TextArea object with focus is now underlined.";            
            }
        }
        private function clearInstanceStyle(e:MouseEvent):void {
            var fm:FocusManager = new FocusManager(this);
            var uiComponent:UIComponent = fm.getFocus() as UIComponent;
            if(uiComponent is TextArea) {
                uiComponent.clearStyle("textFormat");
                description.text = "The TextArea object with focus is no longer underlined.";
            }
        }
        private function setClassStyle(e:MouseEvent):void {
            var c:Class = TextArea;
            var styleName:String = "textPadding";
            var value:Number = 10;
            StyleManager.setComponentStyle(c,styleName,value);    
            description.text = "Padding for all TextArea instances is now 10.";
        }
        private function clearClassStyle(e:MouseEvent):void {
            var styleName:String = "textPadding";
            var c:Class = TextArea;
            StyleManager.clearComponentStyle(c,styleName);                    
            description.text = "TextArea padding has been reset to the default.";
        }
        private function setGlobalStyle(e:MouseEvent):void {            
            var styleName:String = "textFormat";
            var value:Object = formatForGlobal;
            StyleManager.setStyle(styleName,value);
            description.text = "All components now use font Georgia size 10.";
        }
        private function clearGlobalStyle(e:MouseEvent):void {
            var styleName:String = "textFormat";
            StyleManager.clearStyle(styleName);    
            description.text = "Font style for all components has been cleared.";
        }
    }
