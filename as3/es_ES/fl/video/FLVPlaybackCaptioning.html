Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 1.0, Flash Player 9.0.28.0

El componente FLVPlaybackCaptioning activa el texto de FLVPlayback. NOTA: esta documentación está pensada para su uso con el componente FLVPlaybackCaptioning con accesibilidad.

Actualiza la clase FLVPlaybackCaptioning del manual Referencia del lenguaje y componentes ActionScript 3.0 con las siguientes propiedades y métodos incorporados para mejorar la accesibilidad del componente.

Es importante que incluya la versión "con accesibilidad" del componente en el proyecto antes de intentar acceder a las nuevas propiedades y métodos.

El componente FLVPlaybackCaptioning descarga un archivo XML de texto temporizado (TT) y aplica dicho texto a un componente FLVPlayback con el que éste está asociado.

Para más información sobre el formato TT (texto temporizado), consulte http://www.w3.org/AudioVideo/TT/. El componente FLVPlaybackCaptioning admite un subconjunto de la especificación Timed Text 1.0. Para información detallada sobre el subconjunto admitido, consulte Etiquetas de texto temporizado. Este es un ejemplo breve:

	 <?xml version="1.0" encoding="UTF-8"?>
	 <tt xml:lang="en" xmlns="http://www.w3.org/2006/04/ttaf1"  xmlns:tts="http://www.w3.org/2006/04/ttaf1#styling">
	     <head>
	         <styling>
	             <style id="1" tts:textAlign="right"/>
	             <style id="2" tts:color="transparent"/>
	             <style id="3" style="2" tts:backgroundColor="white"/>
	             <style id="4" style="2 3" tts:fontSize="20"/>
	         </styling>
	     </head>
	     <body>
	          <div xml:lang="en">
	             <p begin="00:00:00.50" dur="500ms">Four score and twenty years ago</p>
	             <p begin="00:00:02.50"><span tts:fontFamily="monospaceSansSerif,proportionalSerif,TheOther"tts:fontSize="+2">our forefathers</span> brought forth<br /> on this continent</p>
	             <p begin="00:00:04.40" dur="10s" style="1">a <span tts:fontSize="12 px">new</span> <span tts:fontSize="300%">nation</span></p>
	             <p begin="00:00:06.50" dur="3">conceived in <span tts:fontWeight="bold" tts:color="#ccc333">liberty</span> <span tts:color="#ccc333">and dedicated to</span> the proposition</p>
	             <p begin="00:00:11.50" tts:textAlign="right">that <span tts:fontStyle="italic">all</span> men are created equal.</p>
	 			<p begin="15s" style="4">The end.</p>
	         </div>    
	     </body>
	 </tt>

autoLayout

propiedad
autoLayout:Boolean

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 1.0, Flash Player 9.0.28.0

Determina si el componente FLVPlaybackCaptioning mueve y cambia automáticamente el tamaño del objeto TextField para texto.

Si la propiedad autoLayout se establece como true, la instancia DisplayObject o el objeto TextField que contiene el texto muestra 10 píxeles desde la parte inferior de la instancia FLVPlayback. El área de texto cubre la anchura de la instancia FLVPlayback, y se mantiene un margen de 10 píxeles a cada lado.

Si esta propiedad está establecida como true, la instancia DisplayObject o el objeto TextField se muestra directamente sobre la instancia FLVPlayback. Si crea un objeto TextField propio, se recomienda que establezca autoLayout como false. Si wordWrap = false, el área de texto se centra sobre la instancia FLVPlayback, pero puede ser más amplia que la instancia FLVPlayback.

Para controlar el diseño, es necesario que detecte el evento captionChange para determinar cuándo se crea la instancia de objeto TextField.

El valor predeterminado es true.

captionTarget

propiedad 
captionTarget:DisplayObject

Versión del lenguaje: ActionScript 3.0
Versiones de motor de ejecución: AIR 1.0, Flash Player 9.0.28.0

Establece la instancia DisplayObject en la que mostrar el texto.

Si establece la instancia como un objeto TextField, será directamente el destino. Si establece la instancia como un contenedor DisplayObjectContainer que contiene uno o varios objetos TextField, el texto se muestra en el objeto TextField con el índice de visualización más bajo.

El método DisplayObjectContainer admite un objeto similar a un clip de película con un fondo de escala 9, que se puede escalar cuando el objeto TextField cambia de tamaño.

Para una escala y dibujo más complejos, escriba código para que el método DisplayObjectContainer detecte un evento captionChange.

Nota: si no se establece la propiedad captionTargetName o captionTarget, la instancia FLVPlaybackCaptioning crea un campo de texto establecido por la propiedad captionTarget con este formato:

  • fondo negro (background = true; backgroundColor = 0x000000;)
  • texto blanco (textColor = 0xFFFFFF)
  • autoSize = TextFieldAutoSize.LEFT
  • multiLine = true
  • wordWrap = true
  • font = "_sans"
  • size = 12

Para personalizar estos valores, detecte el evento captionTargetCreated.

Ejemplos Cómo utilizar este ejemplo
FLVPlaybackCaptioningExample.as

Este ejemplo muestra cómo crear dinámicamente una instancia de FLVPlayback con texto conmutable.

Para realizar el ejemplo, siga estos pasos:

  1. Abra un nuevo archivo de Flash (ActionScript 3.0).
  2. Coloque el componente FLVPlayback en el escenario en la posición 0,0 y asígnele el nombre de instancia player.
  3. Coloque el componente FLVPlaybackCaptioning en el escenario en la posición 0,0 y asígnele el nombre de instancia captioning.
  4. Coloque el componente CaptionButton en el escenario.
  5. Guarde el archivo FLA y asígnele un nombre, por ejemplo, captionEx.
  6. Abra un nuevo archivo de ActionScript.
  7. Pegue el siguiente código de ejemplo de clase en un nuevo archivo de ActionScript y guárdelo como FLVPlaybackCaptioningExample.as en el mismo directorio que su archivo FLA.
  8. En el archivo FLA (por ejemplo, captionEx), defina la entrada de la clase Document en el inspector de propiedades como FLVPlaybackCaptioningExample.
  9. Seleccione Control > Probar película.
package
{
    import fl.video.CaptionChangeEvent;
    import fl.video.FLVPlayback;
    import fl.video.FLVPlaybackCaptioning;
    import flash.display.Sprite;
    import flash.text.TextField;
    
    public class FLVPlaybackCaptioningExample extends Sprite {

        private var videoPath:String = "http://www.helpexamples.com/flash/video/caption_video.flv";
        private var captioningPath:String = "http://www.helpexamples.com/flash/video/caption_video.xml";
        
        public function FLVPlaybackCaptioningExample() {
            player.source = videoPath;
            player.skin = "SkinOverAllNoCaption.swf";
            player.skinBackgroundColor = 0x666666;
            player.skinBackgroundAlpha = 0.5;

            captioning.flvPlayback = player;
            captioning.source = captioningPath;
            captioning.autoLayout = false;            
            captioning.addEventListener(CaptionChangeEvent.CAPTION_CHANGE, onCaptionChange);
        }

        private function onCaptionChange(e:CaptionChangeEvent):void {
            var tf:* = e.target.captionTarget;
            var player:FLVPlayback = e.target.flvPlayback;

            // move the caption below the video
            tf.y = 210;            
        }
    }
