Información sobre constantes

EXACT_FIT

Constante
public static const EXACT_FIT:String = "exactFit"

Versión del lenguaje: ActionScript 3.0
Versión de producto: Flash CS3
Versiones de motor de ejecución: Flash Player 9.0.28.0, AIR 1.0

Especifica que el vídeo se muestre a la altura y anchura especificadas por las propiedades registrationHeight o height y registrationWidth o width.

Por ejemplo, si registrationWidth = width = 100 y registrationHeight = height = 100, si registrationX = 200 yregistrationY = 200, y si un archivo FLV se carga con un valor width de 648 y un valor height de 480 (para una proporción aproximada de 4:3), y si align = VideoAlign.CENTER y scaleMode = EXACT_FIT, se obtiene width = 100, height = 100, x = 200 y y = 200.

MAINTAIN_ASPECT_RATIO

Constante 
public static const MAINTAIN_ASPECT_RATIO:String = "maintainAspectRatio"

Versión del lenguaje: ActionScript 3.0
Versión de producto: Flash CS3
Versiones de motor de ejecución: Flash Player 9.0.28.0, AIR 1.0

Especifica que el vídeo se debe restringir al rectángulo determinado por las propiedades registrationX, registrationY, registrationWidth y registrationHeight, pero que debe conservar su proporción original.

Por ejemplo, si registrationWidth = 100 y registrationHeight = 100, si registrationX = 200 yregistrationY = 200, y si un archivo FLV se carga con un valor width de 648 y un valor height de 480 (para una proporción aproximada de 4:3), y si align = VideoAlign.CENTER y scaleMode = MAINTAIN_ASPECT_RATIO, se obtiene width = 100 y height = (100 * 480 / 648) = 74, x = 10 y y = (10 + ((100 - 74) / 2)) = 23.

NO_SCALE

Constante 
public static const NO_SCALE:String = "noScale"

Versión del lenguaje: ActionScript 3.0
Versión de producto: Flash CS3
Versiones de motor de ejecución: Flash Player 9.0.28.0, AIR 1.0

Especifica que el vídeo se muestre a la altura y anchura exactas del vídeo de origen.

Por ejemplo, si registrationWidth = 100 y registrationHeight = 100, si registrationX = 200 y registrationY = 200, y si un archivo FLV se carga con un valor width de 648 y un valor height de 480 (para una proporción aproximada de 4:3), y si align = VideoAlign.CENTER y scaleMode = NO_SCALE, se obtiene width = 648, height = 480, x = (200 + ((100 - 648) / 2)) = -74 y y = (200 + ((100 - 480) / 2)) = 10.
