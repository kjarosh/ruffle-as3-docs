Çalışma Zamanı Sürümleri: AIR 1.0

HTMLLoader sınıfı, HTML içeriği için bir kap görevi gören bir görüntüleme nesnesi türü tanımlar.

AIR profili desteği: Bu özellik tüm masaüstü işletim sistemlerinde desteklenir ancak mobil aygıtlarda veya AIR for TV aygıtlarında desteklenmez. Desteği çalışma zamanında HTMLLoader.isSupported özelliğini kullanarak test edebilirsiniz. Birden fazla profilde API desteği ile ilgili daha fazla bilgi için bkz. AIR Profil Desteği.

Masaüstü bilgisayarlarda (masaüstü ve genişletilmiş masaüstü profillerinde), HTMLLoader sınıfı dahili AIR WebKit motorunu kullanır. Kullanılabilir özellikler ve görüntü oluşturma görünümü StageWebView sınıfınınkilerin yanı sıra ActionScript ve JavaScript arasında yakın tümleştirme ve komut komut dosyası köprülemeye ait olanlarla aynıdır. StageWebView sınıfı, Flash Player eklentisi tarafından sağlanan sistem web denetimini kullandığından, StageWebView ve HTMLLoader örneklerinin eş zamanlı kullanımı sınıf, tanımlanmayan davranış içerdiği ve uygulamayı sonlandırabileceği için kesinlikle önerilmez.

Bir HTMLLoader öğesinin varsayılan boyutları, 0 x 0 pikseldir. HTMLLoader nesnesini görünür yapmak için width ve height özelliklerini ayarlayın.

Bir HTMLLoader nesnesi için maksimum boyut yükseklikte ve genişlikte 4.096 pikseldir. Bir HTMLLoader nesnesinin width ve height özelliklerini 4.095'i aşan bir sayıya ayarlamak, HTMLLoader nesnesinin bir ArgumentError istisnası atmasına neden olur. (AIR 2'den önce HTMLLoader nesnesinin maksimum boyutunun 2880 piksel olduğunu unutmayın.)

Not: Şeffaf bir pencerede görüntülendiğinde, bir HTML sayfasındaki SWF içeriği transparent veya opaque wmode kullanılarak gömülmelidir. wmode için varsayılan değer window değeri olduğundan, bir değer belirtmezseniz SWF içeriği görüntülenmeyebilir. Windows ve Linux'ta, wmode window veya opaque olarak ayarlandığında SWF içeriği her zaman diğer içeriğin üstünde görünür. Hangi wmode ayarının kullanıldığı önemli olmaksızın, PDF içeriği şeffaf bir pencerede görüntülenemez.

locationChanging

 Olay  
Olay Nesne Türü: flash.events.LocationChangeEvent
özellik LocationChangeEvent.type = flash.events.LocationChangeEvent.LOCATION_CHANGING

Dil Sürümü: ActionScript 3.0

HTMLLoader nesnesinin location özelliğinin değiştirilmek üzere olduğunu söyler.

locationChanging olayı yalnızca konum değişikliği kullanıcı etkileşimi örneğin, kullanıcı bir bağlantıyı tıklattığında veya istemci tarafı yeniden yönlendirmesi gibi HTMLLoader içinde çalışan kod tarafından başlatıldığında gönderilir. Varsayılan olarak yeni konum bu HTMLLoader nesnesinde görüntülenir. Varsayılan davranışı iptal etmek için olay nesnesinin preventDefault() yöntemini çağırabilirsiniz. Örneğin, olay nesnesinin location özelliğine göre sayfayı sistem tarayıcısında açmak için flash.net.navigateToURL() işlevini kullanabilirsiniz.

Aşağıdaki yöntemleri çağırdığınızda locationChanging() olayı gönderilmez:

  • load()
  • loadString()
  • reload()
  • historyBack()
  • historyForward()
  • historyGo()

Ayrıca, navigateInSystemBrowser özelliği true değerindeyken ve HTMLLoader konumu değişmediğinden, hedef sayfa sistem tarayıcısında açıldığında olay gönderilmez.

LOCATION_CHANGING sabiti, sayfa konumunda bir değişiklik yapılmadan önce LocationChangeEvent nesnesinin gönderdiği type özelliğinin değerini tanımlar.
Örnek  ( Bu örnek nasıl kullanılır? )

Aşağıdaki kod, bir HTMLLoader nesnesi başlatır, bu nesnenin genişliğini ve yüksekliğini konfigüre eder, http://www.adobe.com URL'sini yükler ve nesneyi görüntüleme listesine ekler. Kullanıcı (veya sayfadaki JavaScript) taban URL ile başlamayan bir URL'ye gitmeye çalışırsa, gezinme engellenir: 
package{
    import flash.display.Sprite;
    import flash.html.HTMLLoader;
    import flash.net.URLRequest;
    
    public class LocationChanging extends Sprite
    {
        var htmlLoader:HTMLLoader = new HTMLLoader();
        public function LocationChanging()
        {
            htmlLoader.width = stage.stageWidth;
            htmlLoader.height = stage.stageHeight;        
            htmlLoader.addEventListener( LocationChangeEvent.LOCATION_CHANGING, onLocationChanging );
            htmlLoader.load( new URLRequest( "http://www.adobe.com" ) );
            this.addChild( htmlLoader );
        }
    
        private function onLocationChanging( event:LocationChangeEvent ):void
        {
            trace( "Location changing: " + event.location );
        
            if ( event.location.indexOf( "http://www.adobe.com" ) < 0 ) 
            {
                event.preventDefault();
            }
        }
    }
}
