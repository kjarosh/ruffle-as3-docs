Paketflash.html
Sınıfpublic class HTMLWindowCreateOptions
Miras AlmaHTMLWindowCreateOptions Object

Çalışma Zamanı Sürümleri: AIR 1.0

Bu sınıf, HTMLLoader nesnesinde çalışan JavaScript'in window.open() yöntemini çağırarak yeni bir HTML penceresi oluşturması durumunda belirtilebilen seçenekleri tanımlar.

Bu sınıf, JavaScript'teki window.open() yöntemine iletilen features parametresindeki seçeneklere karşılık gelen özellikleri ve yöntemleri tanımlar.

Örneğin, bir HTML belgesindeki (bir HTMLLoader nesnesindeki) JavaScript, window.open() öğesine yapılan şu çağrıyı içerebilir. Bu çağrıda features parametresi (üçüncü parametre) çeşitli seçenekler sunar:

window.open("http://www.adobe.com", "AdobeWindow", "scrollbars=1,menubar=1,location=0,status=0")

HTMLHost sınıfının alt sınıfına yönelik createWindow() yöntemini geçersiz kılmak için HTMLWindowCreateOptions özelliklerini kullanabilirsiniz. HTMLLoader nesnesi bir HTMLWindowCreateOptions nesnesini HTMLHost nesnesinin createWindow() yöntemine yönelik windowCreateOptions parametresi olarak iletir.

fullscreen

özellik
public var fullscreen:Boolean = false

Çalışma Zamanı Sürümleri: AIR 1.0

Pencerenin ekranı kaplayıp kaplamayacağını belirtir. window.open() yöntemine yapılan JavaScript çağrısının features dizesi "fullscreen", "fullscreen=1" veya "fullscreen=y" öğelerini içeriyorsa bu özellik true olarak ayarlanır.

    

height

özellik 
public var height:Number = NaN

Çalışma Zamanı Sürümleri: AIR 1.0

Yeni pencerenin istenilen ilk yüksekliğini belirtir. Bu, window.open() yöntemine yapılan JavaScript çağrısının features dizesindeki height değerine ayarlanır. Değer, features dizesinde belirtilen height değeri olmadığındaki varsayılan değer olan NaN ise, varsayılan bir x penceresi konumu kullanılır.
