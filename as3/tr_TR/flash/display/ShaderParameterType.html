Sınıfpublic final class ShaderParameterType
Miras AlmaShaderParameterType Object

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: Flash Player 10, AIR 1.5

Bu sınıf, ShaderParameter sınıfının type özelliğine yönelik olası değerleri temsil eden sabitleri tanımlar. Her sabit, Pixel Bender gölgelendirici dilindeki parametreler için Flash Player'da kullanılabilen veri türlerinden birini temsil eder.

BOOL

Sabit
public static const BOOL:String = "bool"

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: Flash Player 10, AIR 1.5

shader parametresinin, ActionScript'teki tek bir Boolean örneğine eşdeğer bir bool değeri olarak tanımlandığını belirtir.

Parametre yalnızca bir değer beklese de, ShaderParameter.value özelliğinin bir Array olduğunu, bu nedenle tek değerin aşağıdaki gibi value özelliğine atanan bir Array'in tek öğesi olması gerektiğini unutmayın:

         // assumes the shader has a parameter named "param"
         // whose data type is bool
         myShader.data.param.value = [true];
