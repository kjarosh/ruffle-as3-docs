Sınıfpublic class InteractiveObject
Miras AlmaInteractiveObject DisplayObject EventDispatcher Object
Alt Sınıflar DisplayObjectContainer, SimpleButton, TextField

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: AIR 1.0, Flash Player 9, Flash Lite 4

InteractiveObject sınıfı, kullanıcının fare, klavye veya başka bir kullanıcı girdi aygıtı kullanarak etkileşim kurabildiği tüm görüntüleme nesneleri için soyut bir temel sınıftır.

InteractiveObject sınıfını doğrudan başlatamazsınız. Yeni new InteractiveObject() yapıcısına çağrı yapılması bir ArgumentError istisnası atar.

InteractiveObject sınıfı, ekranda içerik oluşturmaya yönelik API'ler içermez. InteractiveObject sınıfının özel bir alt sınıfını oluşturmak için, Sprite, SimpleButton, TextField veya MovieClip sınıfları gibi ekran içeriği oluşturma API'leri olmayan alt sınıflardan birini genişletin.

Flash Player'da çalışan içerik için, bu özellik bir ContextMenu nesnesidir. AIR çalışma zamanında, ContextMenu sınıfı NativeMenu sınıfını genişletir, ancak Flash Player yalnızca ContextMenu sınıfını destekler, NativeMenu sınıfını desteklemez.

TextField nesneleri için bağlam menüsünden bu komutları kaldıramazsınız. TextField nesneleri için bu komutların (veya klavyedeki eşdeğerlerinin) seçilmesi, clear, copy, cut, paste veya selectAll olaylarını üretmez.



Uygulama
    public function get contextMenu():NativeMenu
    public function set contextMenu(value:NativeMenu):void

Örnek  ( Bu örnek nasıl kullanılır? )
Aşağıdaki örnek, Sprite nesnesinin contextMenu özelliğini bir ContextMenu nesnesine ayarlayarak Sprite nesnesine nasıl özel bir bağlam menüsü öğesi ekleyebileceğinizi gösterir. Örnek, ActionScriptExamples.com tarafından sağlanmıştır. 
var red_cmi:ContextMenuItem = new ContextMenuItem("red");
red_cmi.addEventListener(ContextMenuEvent.MENU_ITEM_SELECT, cmi_menuItemSelect);
 
var cm:ContextMenu = new ContextMenu();
cm.customItems.push(red_cmi);
cm.hideBuiltInItems();
 
var spr:Sprite = new Sprite();
spr.contextMenu = cm;
spr.graphics.beginFill(0x000000);
spr.graphics.drawRect(0, 0, 120, 90);
spr.graphics.endFill();
spr.x = 10;
spr.y = 10;
addChild(spr);
 
function cmi_menuItemSelect(evt:ContextMenuEvent):void {
    spr.graphics.clear();
    spr.graphics.beginFill(0xFF0000);
    spr.graphics.drawRect(0, 0, 120, 90);
    spr.graphics.endFill();
}

tabIndex

özellik 
tabIndex:int

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: AIR 1.0, Flash Player 9

SWF dosyasındaki nesnelerin sekme sıralamasını belirtir. tabIndex özelliği varsayılan olarak -1 olup bu, nesne için herhangi bir sekme dizininin ayarlanmadığı anlamına gelir.

SWF dosyasında geçerli olarak görüntülenen herhangi bir nesne bir tabIndex özelliği içeriyorsa, otomatik sekme sıralaması devre dışı bırakılır ve sekme sıralaması, SWF dosyasındaki nesnelerin tabIndex özelliklerinden hesaplanır. Özel sekme sıralaması yalnızca tabIndex özelliklerine sahip nesneleri içerir.

tabIndex özelliği negatif olmayan bir tam sayı olabilir. Nesneler, tabIndex özelliklerine göre artan sırada sıralanır. tabIndex değeri 1 olan bir nesne, tabIndex değeri 2 olan bir nesneden önce gelir. Birden çok nesne için aynı tabIndex değerini kullanmayın.

tabIndex özelliğinin tanımladığı özel sekme sıralaması flat olur. Bu, SWF dosyasındaki nesnelerin hiyerarşik ilişkilerinin dikkate alınmadığı anlamına gelir. SWF dosyasında tabIndex özellikleri olan tüm nesneler sekme sırasına yerleştirilir ve sekme sırası, tabIndex değerlerinin sırasıyla belirlenir.

Not: TLFTextField örnekleri için sekme sırasını ayarlamak üzere, TLFTextField öğesinin görüntüleme nesnesi alt öğesini InteractiveObject olarak dağıtın, ardından tabIndex özelliğini ayarlayın. Örneğin: 

	 InteractiveObject(tlfInstance.getChildAt(1)).tabIndex = 3;
Sekme sırasını bir TLFTextField nesnesinin üç örneği (tlfInstance1, tlfInstance2 ve tlfInstance3) için varsayılan ayardan ters çevirmek üzere, şunları kullanın: 
	 InteractiveObject(tlfInstance1.getChildAt(1)).tabIndex = 3;
	 InteractiveObject(tlfInstance2.getChildAt(1)).tabIndex = 2;
	 InteractiveObject(tlfInstance3.getChildAt(1)).tabIndex = 1;

var mySprite = new Sprite();
mySprite.addEventListener(TransformGestureEvent.GESTURE_ZOOM , onZoom);

addChild(myTextField);

function onZoom(evt:TransformGestureEvent):void {

    evt.target.scaleX++;

  1. InteractiveObjectExample yapıcısında, şekil çizmek ve şekle mouse olayları eklemek () için ChildSprite constructor yöntemini çağıran child adında ve Sprite türünde yeni bir ChildSprite nesnesi oluşturulur. Görüntüleme listesinin en üstüne x = 0, y = 0koordinatlarında child nesnesi eklenir.
  2. ChildSprite sınıfında, daha sonra draw() yönteminde ve MouseEvent yöntemlerinde kullanılacak size ve overSize özelliklerini bildirin.
  3. Arka plan rengini turuncu, fare üzerinde gezdirme rengini koyu sarı ve fare düğmesi rengini açık mavi olarak ayarlayan özellikleri bildirin.
  4. ChildSprite yapıcısında, Graphics sınıfındaki yöntemler ve draw() yöntemi kullanılarak turuncu bir kare çizilir.
  5. Yapıcı dört MouseEvent olay dinleyicisi yöntemi ekler:
    • mouseOverHandler: orijinal koordinatlarda koyu sarı renkli, daha büyük 60 x 60 piksel bir kareyi yeniden çizer.
    • mouseOutHandler: kareyi orijinal boyut ve rengine döndürür.
    • mouseDownHandler: orijinal koordinatlarda açık mavi renkli, daha büyük 60 x 60 piksel bir kareyi yeniden çizer.
    • mouseUpHandler: mouseOverHandler ile aynıdır.

package {
    import flash.display.Sprite;

    public class InteractiveObjectExample extends Sprite {

import flash.display.Sprite;
import flash.events.MouseEvent;

class ChildSprite extends Sprite {
    private var size:uint = 50;
    private var overSize:uint = 60;
    private var backgroundColor:uint = 0xFFCC00;
    private var overColor:uint = 0xCCFF00;
    private var downColor:uint = 0x00CCFF;

    public function ChildSprite() {
        buttonMode = true;
        draw(size, size, backgroundColor);
        addEventListener(MouseEvent.MOUSE_OVER, mouseOverHandler);
        addEventListener(MouseEvent.MOUSE_OUT, mouseOutHandler);
        addEventListener(MouseEvent.MOUSE_DOWN, mouseDownHandler);
        addEventListener(MouseEvent.MOUSE_UP, mouseUpHandler);
    }

    private function draw(w:uint, h:uint, bgColor:uint):void {
        graphics.clear();
        graphics.beginFill(bgColor);
        graphics.drawRect(0, 0, w, h);
        graphics.endFill();
    }

    public function mouseOverHandler(event:MouseEvent):void {
        trace("mouseOverHandler");
        draw(overSize, overSize, overColor);
    }

    public function mouseOutHandler(event:MouseEvent):void {
        trace("mouseOutHandler");
        draw(size, size, backgroundColor);
    }

    public function mouseDownHandler(event:MouseEvent):void {
        trace("mouseDownHandler");
        draw(overSize, overSize, downColor);
    }

    public function mouseUpHandler(event:MouseEvent):void {
        trace("mouseUpHandler");
        draw(overSize, overSize, overColor);
    }
}
