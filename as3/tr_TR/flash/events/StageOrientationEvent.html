Sınıfpublic class StageOrientationEvent
Miras AlmaStageOrientationEvent Event Object

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: AIR 2

Sahne alanının yönlendirmesi değiştiğinde Stage nesnesi StageOrientationEvent nesnesi gönderir. Cihaz döndürüldüğünde, kullanıcı bir kaydırılan klavye açtığında veya Stage öğesinin setAspectRatio() yöntemi çağrıldığında bu gerçekleşebilir.

İki tür StageOrientationEvent olayı vardır: orientationChanging (StageOrientationEvent.ORIENTATION_CHANGING), ekran yeni bir yönlendirmeye geçiş yapmadan önce gönderilir. orientationChanging için gönderilen olay nesnesinin preventDefault() yönteminin çağrılması, sahne alanının yönlendirmeyi değiştirmesini önler. orientationChange (StageOrientationEvent.ORIENTATION_CHANGE), ekran yeni bir yönlendirmeye geçiş yaptıktan sonra gönderilir.

Not: autoOrients özelliği false ise, cihaz döndürüldüğünde sahne alanı yönlendirmesi değişmez. Böylece, StageOrientationEvents öğeleri cihaz döndürüldüğünde yalnızca autoOrients öğesi true olduğunda gönderilir.

Yapıcı Ayrıntı
    

StageOrientationEvent

()Yapıcı
public function StageOrientationEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, beforeOrientation:String = null, afterOrientation:String = null)

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: AIR 2

Sahne alanı yönlendirmesi olaylarıyla ilgili belirli bilgileri içeren bir StageOrientationEvent nesnesi oluşturur. Event nesneleri, olay dinleyicilerine parametre olarak iletilir. Genellikle bu olayı, yapıcı işlevini kullanarak oluşturmazsınız. Bunun yerine, oluştuğu anda bu olayları algılamak için Stage nesnesine bir olay dinleyici eklersiniz.

Parametreler
type:String — Olayın türü: StageOrientationEvent.ORIENTATION_CHANGE
 
bubbles:Boolean (default = false) — Event nesnesinin olay akışının köpürme aşamasında yer alıp almadığını belirtir.
 
cancelable:Boolean (default = false) — Event nesnesinin iptal edilip edilemeyeceğini belirtir.
 
beforeOrientation:String (default = null) — Değişiklik öncesindeki yönlendirmeyi belirtir.
 
afterOrientation:String (default = null) — Değişiklik sonrasındaki yönlendirmeyi belirtir.
