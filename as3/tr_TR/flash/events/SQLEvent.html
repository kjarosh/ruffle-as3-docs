Yapıcı Ayrıntı
    

SQLEvent

()Yapıcı
public function SQLEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false)

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: AIR 1.0

Olay dinleyicilerine parametre olarak verilecek bir SQLEvent nesnesi oluşturur.

Parametreler
type:String — type özelliğinde bulunan olay türü.
 
bubbles:Boolean (default = false) — Event nesnesinin olay akışının köpürme aşamasında yer alıp almayacağını belirler. Varsayılan değer false şeklindedir.
 
cancelable:Boolean (default = false) — Event nesnesinin iptal edilip edilemeyeceğini belirler. Varsayılan değer false şeklindedir.
Yöntem Ayrıntısı

    clone

()yöntem
override public function clone():Event

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: AIR 1.0

SQLEvent nesnesinin bir kopyasını oluşturur ve her özelliğin değerini, orijinal ile aynı olacak şekilde ayarlar.

Döndürür
Event — Orijinaliyle eşlenen özellik değerlerine sahip yeni bir SQLEvent nesnesi.
