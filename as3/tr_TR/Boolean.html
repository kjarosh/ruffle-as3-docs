Sınıfpublic final class Boolean
Miras AlmaBoolean Object

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: AIR 1.0, Flash Player 9, Flash Lite 4

Boolean nesnesi, mantıksal işlemler için kullanılan, true veya false şeklinde iki değerden birine sahip olabilen bir veri türüdür. Boolean nesnesinin temel veri türünü veya dize olarak temsil edilen halini almak için Boolean sınıfını kullanın.

Bir Boolean nesnesi oluşturmak için yapıcıyı veya global işlevini kullanabilir ya da değişmez bir değer atayabilirsiniz. Hangi tekniği kullandığınız fark etmez; ActionScript 3.0'da üç tekniğin hepsi eşdeğerdir. (Bu, Boolean nesnesinin Boolean temel türünden farklı olduğu JavaScript'ten farklıdır.)

Şu kod satırları birbirine eşdeğerdir:

var flag:Boolean = true;
var flag:Boolean = new Boolean(true);
var flag:Boolean = Boolean(true);

Boolean

()Yapıcı
public function Boolean(expression:Object = false)

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: AIR 1.0, Flash Player 9, Flash Lite 4

Belirtilen değer ile bir Boolean nesnesi oluşturur. expression parametresini atlarsanız, Boolean nesnesi false değeriyle başlatılır. expression parametresi için bir değer belirtirseniz, yöntem bunu değerlendirir ve global Boolean() işlevindeki kurallara göre bir Boolean değerini sonuç olarak döndürür.

Parametreler
expression:Object (default = false) — Herhangi bir ifade.

İlgili API Öğeleri

Boolean() genel işlev

Örnek  ( Bu örnek nasıl kullanılır? )
Şu kod, myBoolean adı verilen bir false değerine başlatılan yeni bir Boolean nesnesi oluşturur: 
 var myBoolean:Boolean = new Boolean();

AS3 function toString():String

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: AIR 1.0, Flash Player 9, Flash Lite 4

Boolean nesnenin dize olarak temsil edilen halini ("true" veya "false") döndürür. Çıktı yerelleştirilmez ve sistem diline bakılmaksızın "true" veya "false" olur.

Döndürür
String — "true" veya "false" dizesi.

Örnek  ( Bu örnek nasıl kullanılır? )
Bu örnek, Boolean türünün bir değişkenini oluşturur ve ardından değeri bir dizeler dizisinde kullanmak üzere bir dizeye dönüştürmek için toString() yöntemini kullanır: 
     var myStringArray:Array = new Array("yes", "could be");
     var myBool:Boolean = 0;
     myBool.toString();
     myStringArray.push(myBool);
     trace(myStringArray); // yes,could be,false

AS3 function valueOf():Boolean

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: AIR 1.0, Flash Player 9, Flash Lite 4

Belirtilen Boolean nesnenin değeri true ise true; değilse false döndürür.

Döndürür
Boolean — Bir Boolean değeri.

Örnek  ( Bu örnek nasıl kullanılır? )
Şu örnek, bu yöntemin nasıl çalıştığını ve ayrıca yeni bir Boolean nesnesinin değerinin false olduğunu gösterşr: 
     var myBool:Boolean = new Boolean();
     trace(myBool.valueOf());   // false
     myBool = (6==3+3);
     trace(myBool.valueOf());   // true
Örnekler Bu örnek nasıl kullanılır?
BooleanExample.as

Şu örnek, Boolean nesnesinin karşılık gelen değerlerinin her biri arasında geçiş yapar ve bu değerleri gösterir. 
 package {
    import flash.display.Sprite;

    public class BooleanExample extends Sprite {
        private var flag:Boolean;

        public function BooleanExample() {
            trace(flag);    // false
            toggle();
            trace(flag);    // true
            toggle();
            trace(flag);    // false
        }
        
        private function toggle():void{
            flag = !flag;
        }
    }
}
