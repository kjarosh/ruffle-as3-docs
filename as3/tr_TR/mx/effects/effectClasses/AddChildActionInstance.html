Dil Sürümü: ActionScript 3.0
Ürün Sürümü: Flex 3
Çalışma Zamanı Sürümleri: Flash Player 9, AIR 1.1

The AddChildActionInstance class implements the instance class for the AddChildAction effect. Flex creates an instance of this class when it plays an AddChildAction effect; you do not create one yourself.
