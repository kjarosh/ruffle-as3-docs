Paketflashx.textLayout.conversion
Arabirimpublic interface ITextExporter

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: Flash Player 10, AIR 1.5

Bir TextFlow örneğinin metin içeriğini belirli bir biçime içe aktarma arabirimi. Örneğin, bu String veya XML biçimi veya kullanıcı tanımlı bir biçim olabilir. Dışa aktarıcılar veri dışa aktarımını denetlemek için özelliklerin alınmasını ve ayarlanmasını destekler. Bu özellikler genel özellikler olarak uygulanır, ancak bir kullanıcı TextConverter kayıt defterindeki converter sınıfını değiştirip alt türe çevirme işleminin başarısız olmasına neden olabileceğinden, bu özelliklere doğrudan erişimden kaçınılmalıdır.

throwOnError özelliği, varsayılan olan false olarak ayarlanmışsa bu özellik, bir exporter yöntemine yapılan çağrıdan sonra dize olarak bir hata mesajı vektörü içerir. Hata yoksa özellik null değerini döndürür. Özellik her yöntem çağrısında sıfırlanır.

throwOnError

özellik 
throwOnError:Boolean

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: Flash Player 10.2, AIR 2.0

throwOnError özelliği, dışa aktarıcının hataları işleme biçimini denetler. true olarak ayarlanırsa, yöntemler hatalarda bir Error örneği atar. Varsayılan olan false olarak ayarlanırsa, hatalar bir dize vektörüne toplanır ve errors özelliğinde saklanır ve dışa aktarıcı bunları atmaz.

useClipboardAnnotations

özellik 
useClipboardAnnotations:Boolean

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: Flash Player 10, AIR 1.5

useClipboardAnnotations özelliği içe aktarıcının pano için gerekli olan ek bilgileri işleyip işlememesi gerektiğini denetler. Veri bir TextFlow öğesinde olduğunda, paragraflar her zaman tamdır ve bir sonlandırıcı karakteri içerir. Bir metin aralığı panoya yapıştırıldığında paragraf oluşturur. Ancak, aralık son sonlandırıcıyı içermeyebilir. Bu durumda, paragraf panoya yönelikse kısmi paragraf olarak işaretlenmelidir. Böylece, daha sonra yapıştırılırsa yeni metnin içine düzgün bir şekilde yerleşir. İçerik panoya yönelikse, useClipboardAnnotations öğesi true olur.

Yöntem Ayrıntısı

export

()yöntem
public function export(source:flashx.textLayout.elements:TextFlow, conversionType:String):Object

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: Flash Player 10, AIR 1.5

String öğesindeki TextFlow örneğinden, XML'den veya kullanıcı tanımlı biçimden metin içeriği dışa aktarır.

conversionType parametresini aşağıdaki değerlerden birine ayarlayın veya kullanıcı tanımlı dışa aktarıcılarda kullanıcı tanımlı biçim kullanın.

  • flashx.textLayout.conversion.ConversionType.STRING_TYPE;
  • flashx.textLayout.conversion.ConversionType.XML_TYPE.

Parametreler

source:flashx.textLayout.elements:TextFlow — Dışa aktarılan TextFlow
 
conversionType:String — Bir String (STRING_TYPE) veya XML (XML_TYPE) veya herhangi bir kullanıcı tanımlı biçim döndürür.

Döndürür
Object — Object Dışa aktarılan içerik

İlgili API Öğeleri

flashx.textLayout.conversion.ConversionType

Örnek  ( Bu örnek nasıl kullanılır? )

Bu kod parçacığı, biçimlendirilmiş metni art arda dışa aktarma işlemlerini gerçekleştirmek için export yönteminin kullanımını gösterir. 


package flashx.textLayout.conversion.examples
{
    import flashx.textLayout.conversion.ConversionType;
    import flashx.textLayout.conversion.ITextExporter;
    import flashx.textLayout.conversion.TextConverter;
    import flashx.textLayout.elements.TextFlow;
    
    public class ITextExporterExample 
    {
        // Serialize the TextFlow into a String
        static public function export(textFlow:TextFlow):String
        {
            var exporter:ITextExporter = TextConverter.getExporter(TextConverter.TEXT_LAYOUT_FORMAT);
            return exporter.export(textFlow, ConversionType.STRING_TYPE) as String;
        }
    }
}
