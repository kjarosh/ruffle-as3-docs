Sınıfpublic class FlowGroupElement
Miras AlmaFlowGroupElement FlowElement Object
Alt Sınıflar ParagraphFormattedElement, SubParagraphGroupElementBase

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: Flash Player 10, AIR 1.5

FlowGroupElement sınıfı, bir alt öğe dizisine sahip olabilen, FlowElement nesnelerinin temel sınıfıdır. Bu sınıflar arasında TextFlow, ParagraphElement, DivElement ve LinkElement yer alır.

Doğrudan bir FlowGroupElement nesnesi oluşturamazsınız. Yeni FlowGroupElement() öğesi çağrıldığında bir hata istisnası atılır.

addChild

()yöntem
public function addChild(child:FlowElement):FlowElement

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: Flash Player 10, AIR 1.5

Bir FlowElement alt nesnesini ekler. Alt öğe listesinin sonuna yeni alt öğe eklenir.

Parametreler

child:FlowElement — Eklenecek alt öğe.

Döndürür
FlowElement — eklenen alt FlowElement öğesi

Örnek  ( Bu örnek nasıl kullanılır? )

Bu örnek bir paragrafa bir aralık eklemek ve paragrafı metin akışına eklemek için addChild() öğesini çağırır. 
package flashx.textLayout.elements.examples {    
    import flash.display.Sprite;
    import flashx.textLayout.container.ContainerController;
    import flashx.textLayout.elements.ParagraphElement;
    import flashx.textLayout.elements.SpanElement;
    import flashx.textLayout.elements.TextFlow;
    
    public class FlowGroupElement_addChildExample extends Sprite
    {        
        public function FlowGroupElement_addChildExample()
        {
            // create string for text flow
            var simpleText:String = "It was a dark and stormy night.";        
                    
            // define TextFlow, ParagraphElement, and SpanElement objects
            var textFlow:TextFlow = new TextFlow();            
            var paragraph:ParagraphElement = new ParagraphElement();
            var span:SpanElement = new SpanElement;
            span.text = simpleText;
            // add span to paragraph and add paragraph to text flow
            paragraph.addChild(span);
            textFlow.addChild(paragraph);
            // assign composer and controller and update controller to display text
            textFlow.flowComposer.addController(new ContainerController(this,200,200));
            textFlow.flowComposer.updateAllControllers();
        }
    }
}

package flashx.textLayout.elements.examples {    
    import flash.display.Sprite;
    import flashx.textLayout.container.ContainerController;
    import flashx.textLayout.elements.ParagraphElement;
    import flashx.textLayout.elements.SpanElement;
    import flashx.textLayout.elements.TextFlow;
    
    public class FlowGroupElement_addChildAtExample extends Sprite
    {        
        public function FlowGroupElement_addChildAtExample()
        {
            // define text strings for the contents of the text flow
            var textA:String = "It was a dark and stormy night. ";
            var textB:String = "The quick red fox jumped over the lazy brown dog. ";
            var textC:String = "Mary had a little lamb. ";
            // define TextFlow, ParagraphElement and SpanElement objects

replaceChildren

()yöntem 
public function replaceChildren(beginChildIndex:int, endChildIndex:int, ... rest):void

Dil Sürümü: ActionScript 3.0
Çalışma Zamanı Sürümleri: Flash Player 10, AIR 1.5

Gruptaki alt öğeleri, belirtilen yeni öğelerle değiştirir. Aşağıdaki gibi, işlemi yönetmek için beginChildIndex ve endChildIndex parametrelerini kullanın:

  • Öğeleri silmek için herhangi bir değiştirme öğesi iletmeyin.
  • Öğe eklemek için, aynı beginChildIndex ve endChildIndex değerini iletin. Yeni öğeler, belirtilen dizinden önce eklenir.
  • Öğe eklemek üzere, beginChildIndex ve endChildIndex için numChildren öğesini iletin.

Aksi takdirde, bu yöntem beginChildIndex öğesinden başlayıp endChildIndex öğesine kadar (bu öğe hariç olmak koşuluyla) belirtilen öğeleri değiştirir.

Parametreler

beginChildIndex:int — Alt öğe dizisindeki değiştirme aralığının başlangıç konumu için dizin değeri.
 
endChildIndex:int — Alt öğe dizisindeki değiştirme aralığının bitiş konumu için dizin değeri.
 
... rest — Belirtilen öğe aralığının değiştirilmesine yönelik öğeler. Akış öğelerini veya akış öğeleri dizilerini ya da vektörlerini içeren bir sıra olabilir.


Atar
RangeError — Belirtilen beginChildIndex veya endChildIndex aralık dışındadır.

Örnek  ( Bu örnek nasıl kullanılır? )

Bu örnek ikinci paragraftaki üç aralığı yenileriyle değiştirir.
