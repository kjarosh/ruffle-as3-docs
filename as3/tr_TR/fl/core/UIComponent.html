Miras AlmaUIComponent Sprite DisplayObjectContainer InteractiveObject DisplayObject EventDispatcher Object
Alt Sınıflar BaseButton, BaseScrollPane, ColorPicker, ComboBox, IndeterminateBar, Label, NumericStepper, ProgressBar, ScrollBar, Slider, TextArea, TextInput, UILoader

Dil Sürümü: ActionScript 3.0
Ürün Sürümü: Flash CS3
Çalışma Zamanı Sürümleri: Flash Player 9.0.28.0, AIR 1.0

UIComponent sınıfı hem etkileşimli hem de etkileşimli olmayan, tüm görsel bileşenler için taban sınıfıdır. Etkileşimli bileşenler, klavye veya fare etkinliği gibi kullanıcı girdisini alan bileşenler olarak tanımlanır. Etkileşimli olmayan bileşenler, veri görüntülemek için kullanılır; kullanıcı etkileşimine yanıt vermez. ProgressBar ve UILoader bileşenleri, etkileşimli olmayan bileşenlere örnektir.

Sekme ve ok tuşları, odağı etkileşimli bileşene ve etkileşimli bileşen üzerine taşımak için kullanılabilir; etkileşimli bileşen, fare ve klavye aygıtlarından gelen girdiler gibi düşük düzeyde olayları kabul edebilir. Etkileşimli bileşen ayrıca, fare ve klavye girdisi alamaması için devre dışı bırakılabilir.

Bu yöntemin döndürdüğü tür, yöntemin aldığı style özelliğine bağlı olarak farklılık gösterir. Olası tür aralığı, Boolean; String; Number; int; RGB rengi için bir birim; bir kaplama için Class veya herhangi bir nesne türü içerir.

Belirli bir style özelliğini almak için bu yöntemi çağırırsanız, aynı türde bir değişkene saklayabileceğiniz bilinen bir türde olur. Tür çevrimine gerek kalmaz. Bunun yerine aşağıdaki gibi basit bir atama ifadesi işe yarayacaktır:

var backgroundColor:uint = getStyle("backgroundColor");

Geçerli bir style değeri hiçbir zaman undefined olmaz. Bir değerin ayarlanıp ayarlanmadığını test etmek için StyleManager.isValidStyleValue() statik yönetimini kullanabilirsiniz.

Bileşen görünürlüğü görünmez durumdan görünür duruma geçtikten sonra gönderilir.

Bir show olayı nesnesinintype özelliğinin değerini tanımlar.

Bu olay şu özelliklere sahiptir:

ÖzellikDeğer
bubblesfalse
cancelablefalse; iptal edilecek varsayılan davranış yoktur.
currentTargetOlay nesnesini olay dinleyicisiyle etkin olarak işleyen nesne.

Örnek  ( Bu örnek nasıl kullanılır? )

Şu örnek bir List bileşeninin görünürlüğünü açıp kapatan bir Button bileşeni oluşturur: 

import fl.controls.Button;
import fl.controls.List;
import fl.events.ComponentEvent;
import fl.events.ListEvent;

var toggleButton:Button = new Button();
toggleButton.label = "toggle visibility";
toggleButton.toggle = true;
toggleButton.selected = true;
toggleButton.emphasized = toggleButton.selected;
toggleButton.addEventListener(Event.CHANGE, changeHandler);
toggleButton.width = 150;
toggleButton.move(10, 10);
addChild(toggleButton);

var myList:List = new List();
myList.addItem({label:"Item 1"});
myList.addItem({label:"Item 2"});
myList.addItem({label:"Item 3"});
myList.addItem({label:"Item 4"});
myList.addItem({label:"Item 5"});
myList.move(10, 40);
myList.addEventListener(ComponentEvent.HIDE, hideHandler);
myList.addEventListener(ComponentEvent.SHOW, showHandler);
myList.addEventListener(ListEvent.ITEM_ROLL_OVER, itemRollOverHandler);
myList.addEventListener(ListEvent.ITEM_ROLL_OUT, itemRollOverHandler);
addChild(myList);


function changeHandler(event:Event):void {
    toggleButton.emphasized = toggleButton.selected;
    myList.visible = toggleButton.selected;
}

function hideHandler(event:ComponentEvent):void {
    trace("hiding list");
    myList.removeEventListener(ListEvent.ITEM_ROLL_OVER, itemRollOverHandler);
    myList.removeEventListener(ListEvent.ITEM_ROLL_OUT, itemRollOverHandler);
}

function showHandler(event:ComponentEvent):void {
    trace("showing list");
    myList.addEventListener(ListEvent.ITEM_ROLL_OVER, itemRollOverHandler);
    myList.addEventListener(ListEvent.ITEM_ROLL_OUT, itemRollOverHandler);
}

function itemRollOverHandler(event:ListEvent):void {
    trace(event.type + ":", event.item.label);
}

function itemRollOutHandler(event:ListEvent):void {
    trace(event.type + ":", event.item.label);
}
