TextInput bileşeni, yerel bir ActionScript TextField nesnesi içeren, tek satırlık bir metin bileşenidir.

Bir TextInput bileşeni, bir uygulamada etkinleştirilebilir veya devre dışıdır. TextInput bileşeni devre dışı bırakıldığında, fare veya klavyeden girdi alamaz. Etkin bir TextInput bileşeni, bir ActionScript TextField nesnesi gibi odak, seçim ve gezinme özelliklerini uygular.

TextInput bileşenini özelleştirmek için, örneğin devre dışı olduğunda, görünüşünü değiştirerek stiller kullanabilirsiniz. Bu bileşen üzerinde uygulayabileceğiniz bazı diğer özelleştirmeler arasında bileşeni HTML ile biçimlendirmek veya metninin gizli olması gereken bir şifre alanı olacak şekilde ayarlamaktır.

Metin alanının odağı olmaması durumunda bir seçimin nasıl yapıldığını görüntülendiğini gösteren bir Boolean değeri alır veya ayarlar.

Bu değer, true olarak ayarlandığında ve metin alanı odakta olmadığında Flash Player, metin alanındaki seçimi gri renkte vurgular. Bu değer, false olarak ayarlandığında ve metin alanı odakta olmadığında, Flash Player metin alanındaki seçimi gri renkte vurgular.

Varsayılan değer şudur false.



Uygulama

Örnek  ( Bu örnek nasıl kullanılır? )
Şu örnek, bir TextInput bileşeni oluşturur ve setSelection() yöntemini kullanarak bazı metinler seçer: 

import fl.controls.TextInput;

var myTextInput:TextInput = new TextInput();
myTextInput.text = "The quick brown fox jumped over the lazy dog.";
myTextInput.alwaysShowSelection = true;
myTextInput.setSelection(16, 19);
myTextInput.move(10, 10);
addChild(myTextInput);

condenseWhite

özellik 
condenseWhite:Boolean

Dil Sürümü: ActionScript 3.0
Ürün Sürümü: Flash CS3
Çalışma Zamanı Sürümleri: Flash Player 9.0.28.0, AIR 1.0

HTML metni içeren bir TextInput bileşeninden ekstra beyaz boşluğun kaldırılıp kaldırılmadığını gösteren bir Boolean değeri alır veya ayarlar. Bileşendeki ekstra beyaz boşluk örnekleri, boşlukları ve satır kesmeleri içerir. true değeri, ekstra beyaz boşluğun kaldırıldığını gösterir; false değeri ise ekstra beyaz boşluğun kaldırılmadığını gösterir.

Bu özellik, yalnızca htmlText kullanılarak ayarlanan metni etkiler; text özelliği kullanılarak oluşturulan metni etkilemez. Metni ayarlamak için text özelliğini kullanırsanız, condenseWhite özelliği yok sayılır.

condenseWhite özelliği, true olarak ayarlanırsa, standart HTML komutlarını kullanmanız gerekir. Örneğin, metin alanına satır kesme yerleştirmek için <br> ve <p>.

Varsayılan değer şudur false.



Uygulama
    public function get condenseWhite():Boolean
    public function set condenseWhite(value:Boolean):void

Bu örnek, TextInput sınıfı kullanılarak nasıl temel bir oturum açma sistemi oluşturulacağını gösterir
  1. TextInput, Label ve Button bileşenlerini kütüphanenize ekleyin
  2. Bu kodu FLA dosyanızla aynı dizine TextInputExample.as olarak kaydedin.
  3. FLA dosyasındaki DocumentClass öğesini TextInputExample olarak ayarlayın.
package 
{
    import flash.display.Sprite;
    import flash.events.MouseEvent;
    import flash.events.Event;
    import fl.controls.TextInput;
    import fl.controls.Label;
    import fl.controls.Button;
    
    public class TextInputExample extends Sprite
    {
        private var ti1:TextInput;
        private var ti2:TextInput;
        private var label1:Label;
        private var label2:Label;
        private var submitBtn:Button;
        
        public function TextInputExample() {
            setupLabels();
            setupInputFields();    
            setupButton();
        }
        private function setupInputFields():void {
            ti1 = new TextInput();
            ti2 = new TextInput();
            ti1.move(50,10);
            ti2.move(50,40);
            ti2.displayAsPassword = true;
            ti1.addEventListener(Event.CHANGE, textEntered);
            ti2.addEventListener(Event.CHANGE, textEntered);
            addChild(ti1);
            addChild(ti2);
        }
        private function setupLabels():void {
            label1 = new Label();
            label2 = new Label();
            label1.move(10,10);
            label2.move(10,40);
            label1.text = "User:";
            label2.text = "Pass:"
            addChild(label1);
            addChild(label2);
        }
        private function setupButton():void {
            submitBtn = new Button();
            submitBtn.move(150,70);
            submitBtn.label = "Go";
            submitBtn.enabled = false;
            submitBtn.setSize(50,20);
            submitBtn.addEventListener(MouseEvent.CLICK, submitLogin);
            addChild(submitBtn);
        }
        private function textEntered(e:Event):void {
            if(ti1.text != "" && ti2.text != "") {
                submitBtn.enabled = true;
            }
            else {
                submitBtn.enabled = false;    
            }
        }
        private function submitLogin(e:MouseEvent):void {
            var submitLabel:Label = new Label();
            submitLabel.move(10,70);
            submitLabel.autoSize = "left";
            submitLabel.text = "Login has been submitted."
            addChild(submitLabel);
            submitBtn.removeEventListener(MouseEvent.CLICK, submitLogin);
        }
    }
}
