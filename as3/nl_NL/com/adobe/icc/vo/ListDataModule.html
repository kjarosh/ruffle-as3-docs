Klassepublic class ListDataModule
OverervingListDataModule DataModule Object

Taalversie: ActionScript 3.0
Productversie: Asset Composer Building Block 9.5
Runtimeversies: AIR (unsupported), Flash Player 9, Flash Player 10

A value object for a list module. Contains a list of LDMAssignment objects. Each LDMAssignment represents the target DataModule assigned to this list. Extends the basic attributes of a DataModule.

custom

eigenschap 
custom:String

Taalversie: ActionScript 3.0
Productversie: Asset Composer Building Block 9.5
Runtimeversies: AIR (unsupported), Flash Player 9, Flash Player 10

Specifies a custom bullet when the list style is STYLE_BULLETED and its type is TYPE_BULLET_CUSTOM. It is expected to be a single character. This character can be an HTML character code in the "&#xyz" format where "xyz" is a number. For example, → for an arrow or ♦ for a diamond.

This value is ignored unless the style is STYLE_BULLETED and type is TYPE_BULLET_CUSTOM. If this property has the style and type for a custom bulleted list, it cannot be an empty string.
