Klassen x


Het pakket flashx.textLayout.elements bevat de basisklassen voor de representatie van tekstinhoud in TLF (Text Layout Framework).


Interfaces
 InterfaceBeschrijving
 IConfiguration Alleen-lezen-interface voor een configuratieobject.
 IFormatResolver Interface van een opmaakoplosser.
Klassen
 KlasseBeschrijving
 BreakElement De BreakElement-klasse definieert een regeleinde. Hierdoor kunt u een regeleinde in de tekst plaatsen zonder een nieuwe alinea te maken.
 Configuration De Configuration-klasse vormt een primair integratiepunt tussen Text Layout Framework en een toepassing.
 ContainerFormattedElement ContainerFormattedElement is de basisklasse voor alle blokelementen op containerniveau, zoals DivElement- en TextFlow-objecten.
 DivElement De DivElement-klasse definieert een element voor het groeperen van alinea's (ParagraphElement-objecten).
 FlowElement De tekst in een flow wordt opgeslagen in een boomstructuur, waarbij de elementen van de boom staan voor logische divisies in de tekst.
 FlowGroupElement De FlowGroupElement-klasse is de basisklasse voor FlowElement-objecten die beschikken over een array met onderliggende items.
 FlowLeafElement Basisklasse voor FlowElements die worden weergegeven op het laagste niveau van de flowhiërarchie.
 GlobalSettings Configuratie die van toepassing is op alle TextFlow-objecten.
 InlineGraphicElement De InlineGraphicElement-klasse verwerkt grafische objecten die inline in de tekst worden weergegeven.
 InlineGraphicElementStatus De InlineGraphicElementStatus-klasse definieert een set constanten om de waarde van InlineGraphicElement.status te controleren.
 LinkElement De LinkElement-klasse definieert een koppeling met een URI (Universal Resource Identifier) die wordt uitgevoerd wanneer de gebruiker er op klikt.
 LinkState De LinkState-klasse definieert een set constanten voor de LinkState-eigenschap van de LinkElement-klasse.
 ListElement De klasse List wordt gebruikt voor het groeperen van items in een genummerde of niet-genummerde lijst.
 ListItemElement ListItemElement is een item in een lijst.
 OverflowPolicy De OverflowPolicy-klasse definieert een set constanten voor de overFlowPolicy-eigenschap van de IConfiguration-klasse.
 ParagraphElement De ParagraphElement-klasse staat voor een alinea in de tekstflowhiërarchie.
 ParagraphFormattedElement De ParagraphFormattedElement-klasse is een abstracte basisklasse voor FlowElement-klassen met alineakenmerken.
 SpanElement De SpanElement-klasse staat voor tekstregels waarop één verzameling opmaakkenmerken is toegepast.
 SpecialCharacterElement De SpecialCharacterElement-klasse is een abstracte basisklasse voor elementen die speciale tekens vertegenwoordigen.
 SubParagraphGroupElement Het SubParagraphGroupElement is een groeperingselement voor FlowLeafElements en andere klassen die een uitbreiding vormen van SubParagraphGroupElementBase.
 SubParagraphGroupElementBase De SubParagraphGroupElementBase-klasse groepeert FlowLeafElements.
 TabElement De TabElement-klasse wordt weergegeven als een <tab/> in de tekstflow.
 TCYElement De TCYElement-klasse (Tatechuuyoko - ta-tae-chu-yo-ko) is een subklasse van SubParagraphGroupElementBase waarmee tekst horizontaal wordt getekend binnen een verticale regel.
 TextFlow De TextFlow-klasse is verantwoordelijk voor het beheer van alle tekstinhoud in een artikel.
 TextRange Een alleen-lezen-klasse waarmee een bereik van opeenvolgende tekst wordt beschreven.

[ X ]Waarom in het Engels?
Sommige inhoud van de Naslaggids voor ActionScript 3.0 staat in het Engels

Niet alle onderdelen van de Naslaggids voor ActionScript 3.0 zijn in alle talen vertaald. Wanneer een taalelement niet is vertaald, wordt het in het Engels weergegeven. Zo is de klasse ga.controls.HelpBox niet in elke taal vertaald. In de Nederlandse versie van de naslaggids wordt de klasse ga.controls.HelpBox bijvoorbeeld in het Engels weergegeven.
