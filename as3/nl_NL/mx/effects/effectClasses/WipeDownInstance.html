Taalversie: ActionScript 3.0
Productversie: Flex 3
Runtimeversies: Flash Player 9, AIR 1.1

The WipeDownInstance class implements the instance class for the WipeDown effect. Flex creates an instance of this class when it plays a WipeDown effect; you do not create one yourself.
