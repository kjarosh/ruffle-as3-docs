Taalversie: ActionScript 3.0
Productversie: Flash CS3
Runtimeversies: Flash Player 9.0.28.0, AIR 1.0

De klasse LabelButton is een abstracte klasse die de klasse BaseButton uitbreidt door een label, een pictogram en schakelfunctionaliteit toe te voegen. De klasse LabelButton bevat de subklassen Button, CheckBox, RadioButton en CellRenderer.

De component LabelButton wordt gebruikt als een eenvoudige knopklasse, maar kan worden gecombineerd met aangepaste skintoestanden die ondersteuning bieden voor ScrollBar-knoppen, NumericStepper-knoppen, ColorPicker-stalen, enzovoort.

selected:Boolean[overschrijven]

Taalversie: ActionScript 3.0

Hiermee wordt een Booleaanse waarde opgehaald of ingesteld die aangeeft of een schakelknop is in- of uitgeschakeld. De waarde true duidt op 'aan', de waarde false duidt op 'uit'. Deze eigenschap kan uitsluitend worden ingesteld wanneer de eigenschap toggle op true is ingesteld.

Voor een component CheckBox geeft deze waarde aan of het vak een vinkje bevat. Geeft voor een component RadioButton aan of de component is geselecteerd.

De gebruiker kan deze eigenschap wijzigen door op de component te klikken, maar u kunt deze eigenschap ook via programmacode instellen.

Wanneer de eigenschap toggle op true is ingesteld, wordt bij het wijzigen van deze eigenschap ook een gebeurtenis change verzonden.

De standaardwaarde is false.



Implementatie

    override public function set selected(value:Boolean):void

Voorbeeld  ( Hoe dit voorbeeld te gebruiken )

import fl.controls.Button;

var myButton:Button = new Button();
myButton.toggle = true;
myButton.move(10, 10);
myButton.addEventListener(Event.CHANGE, changeHandler);
addChild(myButton);

function changeHandler(event:Event):void {
    trace("Button toggled (selected:" + event.currentTarget.selected + ")");
}

toggle

eigenschap 
toggle:Boolean

Taalversie: ActionScript 3.0
Productversie: Flash CS3
Runtimeversies: Flash Player 9.0.28.0, AIR 1.0

Hiermee wordt een Booleaanse waarde opgehaald of ingesteld die aangeeft of een knop schakelbaar is. De waarde true geeft aan dat het mogelijk is, de waarde false geeft aan van niet.

Wanneer deze waarde true is, kunt u op de knop klikken om tussen de geselecteerde en niet-geselecteerde toestand te schakelen. U kunt deze toestand via programmacode ophalen of instellen met de eigenschap selected.

Bij false blijft de knop niet ingedrukt nadat de gebruiker deze loslaat. In dit geval is de waarde selected altijdfalse.

Opmerking: wanneer de schakelknop is ingesteld op false wordt selected op false gezet, omdat alleen schakelknoppen kunnen worden geselecteerd.

De standaardwaarde is false.



Implementatie
    public function get toggle():Boolean
    public function set toggle(value:Boolean):void

Voorbeeld  ( Hoe dit voorbeeld te gebruiken )
In het volgende voorbeeld wordt een schakelbare Button gemaakt door de eigenschap toggle in te stellen op true en te luisteren naar de gebeurtenis change die moet worden verzonden: 

import fl.controls.Button;

var myButton:Button = new Button();
myButton.toggle = true;
myButton.selected = true;
myButton.label = "selected:" + myButton.selected;
myButton.width = 120;
myButton.move(10, 10);
myButton.addEventListener(Event.CHANGE, changeHandler);
addChild(myButton);

function changeHandler(event:Event):void {
    var myBtn:Button = event.currentTarget as Button;
    myBtn.label = "selected:" + myBtn.selected;
}

IconWithToolTip.as

Sla de volgende code op als IconWithToolTip.as in dezelfde map als het FLA-bestand: 
package {
    import fl.controls.LabelButton;
    import flash.events.MouseEvent;
    import flash.text.TextField;
    import flash.text.TextFieldAutoSize;
    import flash.text.TextFormat;
    
    public class IconWithToolTip extends LabelButton
    {
        public var iconDescription:String = "";
        private var toolTip:TextField;
        private var format:TextFormat;
        
        public function IconWithToolTip() {
            addEventListener(MouseEvent.MOUSE_OVER,showToolTip);
            addEventListener(MouseEvent.MOUSE_OUT,hideToolTip);
            format = new TextFormat();
            format.font = "Verdana";
            format.size = 10;
            format.bold = true;
            createToolTip();
        }

        private function createToolTip():void {
            toolTip = new TextField();
            toolTip.background = true;
            toolTip.backgroundColor = 0xFFCC66;
            toolTip.border = true;
            toolTip.borderColor = 0x000000;
            toolTip.multiline = false;
            toolTip.autoSize = TextFieldAutoSize.CENTER;
            toolTip.visible = false;
            addChild(toolTip);            
        }

        private function showToolTip(e:MouseEvent):void {
            toolTip.text = iconDescription;
            toolTip.x = 20;
            toolTip.y = 0;
            toolTip.visible = true;
            toolTip.setTextFormat(format);
        }

        private function hideToolTip(e:MouseEvent):void {
            toolTip.visible = false;    
        }
    }
}
