De component List geeft op lijst gebaseerde informatie weer en is bij uitstek geschikt voor het weergeven van arrays met informatie.

De component List bestaat uit items, rijen en een gegevensaanbieder, die als volgt worden beschreven:

  • Item: Een ActionScript-object dat gewoonlijk een beschrijvende eigenschap label bevat en een eigenschap data waarin de gegevens zijn opgeslagen die aan het item zijn gekoppeld.
  • Rij: Een component die wordt gebruikt om het item weer te geven.
  • Gegevensaanbieder: Een component die de items modelleert die in de component List worden weergegeven.

Standaard gebruikt de component List de klasse CellRenderer om de rijen te leveren waarin de lijstitems worden weergegeven. U kunt deze rijen met programmacode maken. Dit gebeurt gewoonlijk door de klasse CellRenderer met subklassen uit te breiden. De klasse CellRenderer implementeert de interface ICellRenderer, die de reeks eigenschappen en methoden levert waarmee de component List de rijen manipuleert en gegevens en statusinformatie voor weergave naar elke rij verzendt. Deze bevat informatie over de grootte en selectie van gegevens.

De component List bevat methoden die op de gegevensaanbieder ervan werken (zoals addItem() en removeItem()). Met deze en andere methoden kunt u de gegevens manipuleren van een willekeurige array die in hetzelfde frame bestaat als een component List, en de wijzigingen vervolgens uitzenden naar meerdere weergaven. Wanneer een component List niet is voorzien van een externe gegevensaanbieder, maken deze methoden automatisch een instantie van een gegevensaanbieder en stellen deze beschikbaar via de eigenschap List.dataProvider. De component List geeft elke rij weer door middel van een Sprite die de interface ICellRenderer implementeert. Gebruik de eigenschap List.cellRenderer om deze renderer op te geven. U kunt ook zelf een instantie Array maken of van een server downloaden en als gegevensmodel gebruiken voor meerdere lijsten, keuzelijsten, gegevensrasters, enzovoort.

ListExample.as

In dit voorbeeld zien u hoe u een lijst maakt en andere instanties List met de selecties ervan vult.
  1. Voeg de klassen List, Button en Label toe aan de bibliotheek.
  2. Sla deze code op als ListExample.as in dezelfde map als de FLA.
  3. Stel de DocumentClass in de FLA in op ListExample.
package
{
    import flash.display.Sprite
    
    public class ListExample extends Sprite
    {
        import flash.events.*;
        import fl.data.DataProvider;
        import fl.controls.List
        import fl.controls.Label
        import fl.controls.Button
        
        private var clearButton:Button;
        private var availableItems:List;
        private var selectedItemList:List;
        private var selectedItemsList:List;

        public function ListExample() {
            createComponents();
            setupComponents();
        }

        private function setupComponents():void {
            var dp:Array = new Array();
            var i:uint;
            var count:uint = availableItems.rowCount * 2;
            
            for (i = 0; i < count; i++) {
                dp.push({label:"Item " + i});
            }
            
            availableItems.allowMultipleSelection = true;
            availableItems.dataProvider = new DataProvider(dp);
            availableItems.addEventListener(Event.CHANGE, updateLists);
            clearButton.addEventListener(MouseEvent.CLICK, clearHandler);            
        }
        
        private function clearHandler(event:MouseEvent):void {
            availableItems.clearSelection();
            // clear data providers
            selectedItemList.dataProvider = new DataProvider();
            selectedItemsList.dataProvider = new DataProvider();
        }
        
        private function updateLists(e:Event):void {
            selectedItemList.dataProvider = availableItems.selectedItem ? new DataProvider([availableItems.selectedItem]) : new DataProvider();
            selectedItemsList.dataProvider = new DataProvider(availableItems.selectedItems);
        }

        private function createComponents():void {
            clearButton = new Button();
            availableItems = new List();
            selectedItemList = new List();
            selectedItemsList = new List();
            var availableItemsLabel:Label = new Label();
            var selectedItemListLabel:Label = new Label();
            var selectedItemsListLabel:Label = new Label();
            
            clearButton.move(10,142);
            availableItems.move(10,32);
            selectedItemList.move(120,32);
            selectedItemsList.move(230,32);
            availableItemsLabel.move(10,10);
            selectedItemListLabel.move(120,10);
            selectedItemsListLabel.move(230,10);
            
            clearButton.label = "Clear Selection"
            availableItemsLabel.text = "Available Items";
            selectedItemListLabel.text = "Selected Item";
            selectedItemsListLabel.text = "All Selected Items";
            
            addChild(clearButton);
            addChild(availableItems);
            addChild(selectedItemList);
            addChild(selectedItemsList);
            addChild(availableItemsLabel);
            addChild(selectedItemListLabel);
            addChild(selectedItemsListLabel);
        }
    }
}
