Klassepublic class FileMode
OverervingFileMode Object

Taalversie: ActionScript 3.0
Runtimeversies: AIR 1.0

De klasse FileMode definieert reeksconstanten die worden gebruikt in de parameter fileMode van de methode open() en openAsync() van de klasse FileStream. De parameter fileMode van deze methoden bepalen welke mogelijkheden beschikbaar zijn voor het object FileStream als het bestand wordt geopend.

De volgende mogelijkheden zijn in verschillende combinaties beschikbaar, gebaseerd op de waarde van de parameter fileMode die is opgegeven in de methode open:

  • Lezen—Het object FileStream kan gegevens van het bestand lezen.
  • Schrijven—Het object FileStream kan gegevens naar het bestand schrijven.
  • Maken—Het object FileStream maakt na openen een nog niet bestaand bestand.
  • Afkorten na openen—Gegevens in het bestand worden na openen verwijderd (voordat gegevens worden geschreven).
  • Geschreven gegevens toevoegen—Gegevens worden altijd aan het eind van het bestand geschreven (wanneer een schrijfmethode wordt aangeroepen).
