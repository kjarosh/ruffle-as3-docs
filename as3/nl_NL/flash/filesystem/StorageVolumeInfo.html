Pakketflash.filesystem
Klassepublic final class StorageVolumeInfo
OverervingStorageVolumeInfo EventDispatcher Object

Taalversie: ActionScript 3.0
Runtimeversies: AIR 2

Het StorageVolumeInfo-object verstuurt een StorageVolumeChangeEvent-object wanneer een opslagvolume wordt gemonteerd of gedemonteerd. De statische eigenschap StorageVolume.storageVolume verwijst naar het singleton StorageVolumeInfo-object dat de gebeurtenissen verstuurt. De StorageVolumeInfo-klasse definieert bovendien een getStorageVolumes-methode voor maken van een lijst met opslagvolumes die momenteel zijn gemonteerd.

AIR-profielondersteuning: deze functie wordt ondersteund op alle desktopbesturingssystemen, maar niet op alle AIR for TV-apparaten. Deze functie wordt ook niet ondersteund op mobiele apparaten. Met de eigenschap StorageVolumeInfo.isSupported kunt u testen op ondersteuning tijdens runtime. Zie AIR-profielondersteuning voor meer informatie over API-ondersteuning voor meerdere profielen.

Op moderne Linux-distributies verzendt het object StorageVolumeInfo alleen storageVolumeMount- en storageVolumeUnmount-gebeurtenissen voor fysieke apparaten en netwerkstations die op bepaalde locaties zijn gemonteerd.

public function getStorageVolumes():Vector.<StorageVolume>

Taalversie: ActionScript 3.0
Runtimeversies: AIR 2

Geeft vector- of StorageVolume-objecten weer die overeenkomen met de op dit moment gemonteerde opslagvolumes.

Op moderne Linux-distributies retourneert deze methode objecten voor fysieke apparaten en netwerkstations die op bepaalde locaties zijn gemonteerd.

Geretourneerde waarde
Vector.<StorageVolume>

Verwante API-elementen

flash.filesystem.StorageVolume

Voorbeeld  ( Hoe dit voorbeeld te gebruiken )

De volgende code bevat het native pad voor de hoofdmap of elk gemonteerd opslagvolume: 
var volumes:Vector.<StorageVolume> = new Vector.<StorageVolume>;
volumes = StorageVolumeInfo.storageVolumeInfo.getStorageVolumes();
for (var i:int = 0; i < volumes.length; i++)
{
    trace(volumes[i].rootDirectory.nativePath);
}
