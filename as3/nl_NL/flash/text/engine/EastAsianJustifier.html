Klassepublic final class EastAsianJustifier
OverervingEastAsianJustifier TextJustifier Object

Taalversie: ActionScript 3.0
Runtimeversies: Flash Player 10, AIR 1.5, Flash Lite 4

De klasse EastAsianJustifier bevat eigenschappen die nodig zijn om de uitvullingsopties voor tekstregels in een tekstblok te bepalen waarvan de inhoud voornamelijk Aziatische tekst is.

Gebruik de constructor new EastAsianJustifier() om een object EastAsianJustifier te maken voordat u de eigenschappen ervan instelt. Als u de eigenschappen van een EastAsianJustifier-object instelt nadat het op een TextBlock is toegepast, wordt het TextBlock niet ongeldig gemaakt.

justificationStyle

eigenschap 
justificationStyle:String

Taalversie: ActionScript 3.0
Runtimeversies: Flash Player 10, AIR 1.5, Flash Lite 4

Geeft de uitvullingsstijl aan voor de tekst in een tekstblok.

De standaardwaarde is JustificationStyle.PUSH_IN_KINSOKU.

Gebruik een van de constanten in de klasse JustificationStyle om de waarde voor deze eigenschap in te stellen. In de onderstaande tabel worden de mogelijke waarden weergegeven:

TekenreekswaardeBeschrijving
JustificationStyle.PUSH_IN_KINSOKUGeeft uitvulling met induwen aan.
JustificationStyle.PUSH_OUT_ONLYGeeft uitvulling met uitduwen aan.
JustificationStyle.PRIORITIZE_LEAST_ADJUSTMENTGeeft uitvulling aan waarin de minste aanpassing prioriteit krijgt.

public function EastAsianJustifier(locale:String = "ja", lineJustification:String = "allButLast", justificationStyle:String = "pushInKinsoku")

Taalversie: ActionScript 3.0
Runtimeversies: Flash Player 10, AIR 1.5, Flash Lite 4

Maakt een EastAsianJustifier-object.

Parameters
locale:String (default = "ja") — De landinstelling waarmee de uitvullingsregels moeten worden bepaald. De standaardwaarde is 'ja'.
 
lineJustification:String (default = "allButLast") — Het type regeluitvulling voor de alinea. Gebruik constanten van LineJustification voor deze eigenschap. De standaardwaarde is LineJustification.ALL_BUT_LAST.
 
justificationStyle:String (default = "pushInKinsoku") — De uitvullingsstijl voor de tekst in een tekstblok met een Oost-Aziatische justifier. Gebruik constanten van JustificationStyle voor deze eigenschap. De standaardwaarde is JustificationStyle.PUSH_IN_KINSOKU.

Gegenereerde uitzondering
ArgumentError — De opgegeven locale is null of te kort om een geldige landinstelling aan te duiden.
 
ArgumentError — De opgegeven lineJustification is geen lid van LineJustification.
 
ArgumentError — De opgegeven justifictionStyle is geen lid van JustificationStyle.

Voorbeelden Hoe dit voorbeeld te gebruiken
EastAsianJustifierExample.as

Dit is een voorbeeld van een verticaal weergegeven blok Japanse tekst, waarbij EastAsianJustifier-eigenschappen zijn gebruikt om de tekst uit te vullen. 
 
package {
    import flash.text.engine.TextBlock;
    import flash.text.engine.TextLine;
    import flash.text.engine.TextElement;
    import flash.text.engine.TextBaseline;
    import flash.text.engine.EastAsianJustifier;
    import flash.text.engine.LineJustification;
    import flash.text.engine.TextRotation;
    import flash.text.engine.FontDescription;
    import flash.text.engine.ElementFormat;
    import flash.display.Stage;
    import flash.display.Sprite;
    import flash.system.Capabilities;
    
    public class EastAsianJustifierExample extends Sprite {
        
        public function EastAsianJustifierExample():void {
            
            var Japanese_txt:String = String.fromCharCode(
                0x5185, 0x95A3, 0x5E9C, 0x304C, 0x300C, 0x653F, 0x5E9C, 0x30A4,
                0x30F3, 0x30BF, 0x30FC, 0x30CD, 0x30C3, 0x30C8, 0x30C6, 0x30EC, 
                0x30D3, 0x300D, 0x306E, 0x52D5, 0x753B, 0x914D, 0x4FE1, 0x5411, 
                0x3051, 0x306B, 0x30A2, 0x30C9, 0x30D3, 0x30B7, 0x30B9, 0x30C6, 
                0x30E0, 0x30BA, 0x793E, 0x306E
            ) +
            "FMS 2" +
            String.fromCharCode(0x3092, 0x63A1, 0x7528, 0x3059, 0x308B, 0x3068, 
                0x767a, 0x8868, 0x3057, 0x307e, 0x3057, 0x305F, 0x3002);
            
            var textBlock:TextBlock = new TextBlock();
            var font:FontDescription = new FontDescription();
            var format:ElementFormat = new ElementFormat();
            format.fontSize = 12;
            format.locale = "ja";
            format.color = 0xCC0000;
            textBlock.baselineZero = TextBaseline.IDEOGRAPHIC_CENTER;
            textBlock.textJustifier = new EastAsianJustifier("ja", LineJustification.ALL_INCLUDING_LAST);
            textBlock.lineRotation = TextRotation.ROTATE_90;
            var linePosition:Number = this.stage.stageWidth - 75;
            if (Capabilities.os.search("Mac OS") > -1) 
                // set fontName: Kozuka Mincho Pro R
                font.fontName = String.fromCharCode(0x5C0F, 0x585A, 0x660E, 0x671D) + " Pro R";                     
            else 
                font.fontName = "Kozuka Mincho Pro R";
            textBlock.content = new TextElement(Japanese_txt, format);
            var previousLine:TextLine = null;
                
            while (true) 
            {
                var textLine:TextLine = textBlock.createTextLine(previousLine, 320);
                if (textLine == null) 
                    break;
                textLine.y = 20;
                textLine.x = linePosition;
                linePosition -= 25;
                addChild(textLine);                
                previousLine = textLine;
            }
        }
    }
}
