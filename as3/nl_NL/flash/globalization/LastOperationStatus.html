Pakketflash.globalization
Klassepublic final class LastOperationStatus
OverervingLastOperationStatus Object

Taalversie: ActionScript 3.0
Runtimeversies: Flash Player 10.1, AIR 2

De LastOperationStatus-klasse bevat een opsomming van constanten waarmee de status van de meeste recente globalisatieservicebewerking wordt aangegeven. Deze waarden kunnen worden opgehaald aan de hand van de alleen-lezen eigenschap lastOperationStatus, die in de meeste globalisatieklassen beschikbaar is.

Geeft aan dat een ongeldig argument is doorgegeven aan een methode.

        trace(cf.lastOperationStatus); // "illegalArgumentError"

PARSE_ERROR

Constante 
public static const PARSE_ERROR:String = "parseError"

Taalversie: ActionScript 3.0
Runtimeversies: Flash Player 10.1, AIR 2

Geeft aan dat een getal niet kan worden geparseerd. Deze status kan worden geretourneerd door parseermethoden van opmaakklassen, zoals CurrencyFormatter.parse() en NumberFormatter.parseNumber(). Als de waarde '12abc34' bijvoorbeeld als parameter aan de methode CurrencyFormatter.parse() wordt doorgegeven, retourneert de methode 'NaN' en wordt de waarde lastOperationStatus ingesteld op LastOperationStatus.PARSE_ERROR.

PATTERN_SYNTAX_ERROR

Constante 
public static const PATTERN_SYNTAX_ERROR:String = "patternSyntaxError"

Taalversie: ActionScript 3.0
Runtimeversies: Flash Player 10.1, AIR 2

Geeft aan dat het opmaakpatroon voor een getal, datum of tijd ongeldig is. Deze status wordt ingesteld wanneer het besturingssysteem van de gebruiker het opgegeven patroon niet ondersteunt.

In de volgende code wordt bijvoorbeeld de waarde getoond van de eigenschap lastOperationStatus nadat het ongeldige patroon 'xx' is gebruikt voor de datumnotatie:

        var df:DateTimeFormatter = new DateTimeFormatter("en-US");
        df.setDateTimePattern("xx");
        trace(df.lastOperationStatus); // "patternSyntaxError"

        var kav:Object = locale.getKeysAndValues();
        trace(locale.lastOperationStatus); // "unexpectedToken"
