Pakketflash.display
Klassepublic final class ShaderParameterType
OverervingShaderParameterType Object

Taalversie: ActionScript 3.0
Runtimeversies: Flash Player 10, AIR 1.5

Deze klasse definieert de constanten die de mogelijke waarden vertegenwoordigen voor de eigenschap type van de ShaderParameter-klasse. Elke constante vertegenwoordigt een van de gegevenstypen die in Flash Player beschikbaar zijn voor parameters in de Pixel Bender-arceringstaal.

Constantedetails

BOOL

Constante
public static const BOOL:String = "bool"

Taalversie: ActionScript 3.0
Runtimeversies: Flash Player 10, AIR 1.5

Geeft aan dat de arceringsparameters wordt gedefinieerd als een bool-waarde, gelijk aan één Boolean-instantie in ActionScript.

Hoewel de parameter slechts één waarde verwacht, is de eigenschap ShaderParameter.value een Array, dus de ene waarde moet het enige element zijn van een Array die is toegewezen aan de eigenschap value, zoals hier:

         // assumes the shader has a parameter named "param"
         // whose data type is bool
         myShader.data.param.value = [true];
