Constructordetails

ActivityEvent

()Constructor
public function ActivityEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, activating:Boolean = false)

Taalversie: ActionScript 3.0
Runtimeversies: AIR 1.0, Flash Player 9

Maakt een Event-object dat informatie bevat over gebeurtenissen activity. Gebeurtenisobjecten worden als parameters aan gebeurtenislisteners doorgegeven.

Parameters
type:String — Het type gebeurtenis. Gebeurtenislisteners hebben toegang tot deze informatie via de overerfde eigenschap type. Er is slechts één type gebeurtenis activity: ActivityEvent.ACTIVITY.
 
bubbles:Boolean (default = false) — Bepaalt of het Event-object deelneemt aan de terugkoppelfase van de gebeurtenisstroom. Gebeurtenislisteners hebben toegang tot deze informatie via de overerfde eigenschap bubbles.
 
cancelable:Boolean (default = false) — Bepaalt of het Event-object kan worden geannuleerd. Gebeurtenislisteners hebben toegang tot deze informatie via de overerfde eigenschap cancelable.
 
activating:Boolean (default = false) — Geeft aan of het apparaat wordt geactiveerd (true) of niet (false). Gebeurtenislisteners hebben toegang tot deze informatie via de overgenomen eigenschap activating.

ACTIVITY

Taalversie: ActionScript 3.0

De constante ActivityEvent.ACTIVITY definieert de waarde van de eigenschap type van een gebeurtenisobject activity.

Deze gebeurtenis heeft de volgende eigenschappen:

EigenschapWaarde
activatingtrue wanneer het apparaat wordt geactiveerd of false wanneer deze wordt gedeactiveerd.
bubblesfalse
cancelablefalse; er is geen standaardgedrag om te annuleren.
currentTargetHet object dat het gebeurtenisobject actief verwerkt met een gebeurtenislistener.
targetHet object dat een sessie begint of beëindigt, zoals een object Camera of Microfoon.

Voorbeelden Hoe dit voorbeeld te gebruiken
ActivityEventExample.as

In het volgende voorbeeld wordt het gebruik van de klasse ActivityEvent getoond door een methode gebeurtenislistener met de naam activityHandler() aan de microfoon te koppelen en tekstinformatie te genereren zodra de microfoon een gebeurtenis activity genereert. 

package {
    import flash.display.Sprite;
    import flash.events.ActivityEvent;
    import flash.media.Microphone;

    public class ActivityEventExample extends Sprite {
        public function ActivityEventExample() {
            var mic:Microphone = Microphone.getMicrophone();
            mic.addEventListener(ActivityEvent.ACTIVITY, activityHandler);
        }

        private function activityHandler(event:ActivityEvent):void {
            trace("event: " + event);
            trace("event.activating: " + event.activating);
        }
    }
}
