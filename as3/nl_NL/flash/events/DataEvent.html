Klassepublic class DataEvent
OverervingDataEvent TextEvent Event Object

Taalversie: ActionScript 3.0
Runtimeversies: AIR 1.0, Flash Player 9, Flash Lite 4

Een object verzendt een DataEvent-object wanneer het laden van onbewerkte gegevens is voltooid. Er zijn twee soorten gebeurtenissen data:
  • DataEvent.DATA: verzonden voor verzonden of ontvangen gegevens.
  • DataEvent.UPLOAD_COMPLETE_DATA: wordt verzonden wanneer gegevens zijn verzonden en de server heeft gereageerd.

public function DataEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, data:String = "")

Taalversie: ActionScript 3.0
Runtimeversies: AIR 1.0, Flash Player 9, Flash Lite 4

Maakt een Event-object dat informatie bevat over gegevensgebeurtenissen. Gebeurtenisobjecten worden als parameters aan gebeurtenislisteners doorgegeven.

Parameters
type:String — Het type gebeurtenis. Gebeurtenislisteners hebben toegang tot deze informatie via de overerfde eigenschap type. Er is slechts één type gegevensgebeurtenis: DataEvent.DATA.
 
bubbles:Boolean (default = false) — Bepaalt of het Event-object deelneemt aan de terugkoppelfase van de gebeurtenisstroom. Gebeurtenislisteners hebben toegang tot deze informatie via de overerfde eigenschap bubbles.
 
cancelable:Boolean (default = false) — Bepaalt of het Event-object kan worden geannuleerd. Gebeurtenislisteners hebben toegang tot deze informatie via de overerfde eigenschap cancelable.
 
data:String (default = "") — De onbewerkte gegevens worden geladen in Flash Player of Adobe AIR. Gebeurtenislisteners hebben toegang tot deze informatie via de overgenomen eigenschap data.

Voorbeelden Hoe dit voorbeeld te gebruiken
DataEventExample.as

In het volgende voorbeeld wordt een XMLSocket gemaakt en verbonden met een socketserver die wordt uitgevoerd op poort 8080 van yourDomain. Een gebeurtenislistener wordt verbonden met het object XMLSocket dat luistert naar gebeurtenissen data, die worden verzonden wanneer onbewerkte gegevens worden ontvangen.

Opmerkingen:

  • Wanneer u een gebeurtenis securityError wilt genereren in dit voorbeeld, moet u een SWF-bestand compileren met Lokale afspeelbeveiliging ingesteld op Alleen netwerk benaderen.
  • U hebt een server nodig die [yourDomain] uitvoert op poort 8080.

package {
    import flash.display.Sprite;
    import flash.events.DataEvent;
    import flash.net.XMLSocket;

    public class DataEventExample extends Sprite {
        private var hostName:String = "[yourDomain]";
        private var port:uint = 8080;
        private var socket:XMLSocket;

        public function DataEventExample() {
            socket = new XMLSocket();
            socket.addEventListener(DataEvent.DATA, dataHandler);
            socket.connect(hostName, port);
        }

        private function dataHandler(event:DataEvent):void {
            trace("dataHandler: " + event.data);
        }
    }
}
