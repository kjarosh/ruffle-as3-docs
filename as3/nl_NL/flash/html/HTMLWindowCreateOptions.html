Pakketflash.html
Klassepublic class HTMLWindowCreateOptions
OverervingHTMLWindowCreateOptions Object

Runtimeversies: AIR 1.0

Deze klasse definieert welke opties kunnen worden opgegeven wanneer JavaScript dat in een HTMLLoader-object wordt uitgevoerd, een nieuw HTML-venster probeert te maken door de methode window.open() aan te roepen.

Deze klasse definieert de eigenschappen en methoden die overeenkomen met opties in de parameter features die aan de JavaScript-methode window.open() wordt doorgegeven.

JavaScript in een HTML-document (in een HTMLLoader-object) kan bijvoorbeeld de volgende aanroep window.open() bevatten, waarin de parameter features (de derde parameter) een aantal opties vermeldt:

window.open("http://www.adobe.com", "AdobeWindow", "scrollbars=1,menubar=1,location=0,status=0")

U gebruikt de klasse HTMLWindowCreateOptions om de methode createWindow() van een subklasse van de klasse HTMLHost te overschrijven. Het object HTMLLoader stuurt een HTMLWindowCreateOptions-object door als de parameter windowCreateOptions van de methode createWindow() van het object HTMLHost.

Eigenschapdetails
    

fullscreen

eigenschap
public var fullscreen:Boolean = false

Runtimeversies: AIR 1.0

Geeft aan of het het venster schermvullend moet zijn. Deze eigenschap is ingesteld op true als de reeks features van de JavaScript-aanroep naar de methode window.open() "fullscreen", "fullscreen=1" of "fullscreen=y" bevat.

    

height

eigenschap 
public var height:Number = NaN

Runtimeversies: AIR 1.0

Geeft de gewenste uitgangshoogte van het nieuwe venster. Deze wordt ingesteld op de waarde height in de features-tekenreeks van de JavaScript-aanroep naar de methode window.open(). Als de waarde NaN is - de standaard wanneer geen weight-waarde is opgegeven in de features-tekenreeks - wordt een standaardvensterhoogte gebruikt.
