Klassepublic class URLMonitor
OverervingURLMonitor ServiceMonitor EventDispatcher Object

Runtimeversies: AIR 1.0

De klasse URLMonitor bewaakt de beschikbaarheid van een op HTTP of HTTPS gebaseerde service.

Deze klasse is opgenomen in het bestand aircore.swc. Adobe® Flash™ Builder laadt deze klasse automatisch wanneer u een project maakt voor Adobe® AIR™. De Adobe® Flex™ SDK bevat eveneens het bestand aircore.swc, dat u mee moet insluiten wanneer u de toepassing compileert als u de Flex SDK gebruikt.

In Adobe ® Flash ® CS3 Professional, is deze klasse opgenomen in het bestand ServiceMonitorShim.swc. Sleep eerst de component ServiceMonitorShim vanuit het deelvenster Componenten naar de Bibliotheek en voeg vervolgens de volgende instructie import aan uw ActionScript 3.0-code toe als u klassen in het air.net-pakket wilt gebruiken: 

import air.net.*;

Zo gebruikt u het pakket air.net in Adobe® Flash® Professional (CS4 of hoger):

  1. Kies de opdracht Bestand > Instellingen publiceren.
  2. Klik in het Flash-deelvenster op de knop Instellingen voor ActionScript 3.0. Selecteer Bibliotheekpad.
  3. Klik op de knop Bladeren naar SWC-bestand. Blader naar het bestand Adobe Flash CSn/AIKn.n/framework/libs/air/aircore.swc in de installatiemap van Adobe Flash Professional.
  4. Klik op de knop OK.
  5. Voeg de volgende import-instructie toe aan de ActionScript 3.0-code: import air.net.*;

URLMonitor

()Constructor
public function URLMonitor(urlRequest:URLRequest, acceptableStatusCodes:Array = null)

Runtimeversies: AIR 1.0

Creëert een object URLMonitor voor een opgegeven op HTTP of HTTPS gebaseerde service.

Nadat de aanroeper een URLMonitor heeft gecreëerd, moet hij de methode start() aanroepen om de status van de service te controleren.

Net zoals bij het object Timer moet de aanroeper een referentie naar het object URLMonitor behouden. Anders zou de runtime het object kunnen verwijderen, waardoor de controle wordt beëindigd.

Een parameter URLRequest geeft de detectieaanvraag om de server te pollen op. Doorgaans zal de aanvraagmethode ofwel "GET", ofwel "HEAD" zijn.

Parameters
urlRequest:URLRequest — Het object URLRequest dat een detectieaanvraag om de server te pollen vertegenwoordigt.
 
acceptableStatusCodes:Array (default = null) — Een reeks numerieke statuscodes die de codes opsommen die een succesvol resultaat vertegenwoordigen.

De volgende statuscodes worden als succesvolle antwoorden herkend als u geen waarde opgeeft voor de eigenschap acceptableStatusCodes:

  • 200 (OK)
  • 202 (Geaccepteerd)
  • 204 (Geen inhoud)
  • 205 (Inhoud herstellen)
  • 206 (Gedeeltelijke inhoud, als antwoord op een aanvraag met een Range-kop)
Methodedetails

    checkStatus

()methode
override protected function checkStatus():void

Runtimeversies: AIR 1.0

Tracht inhoud te laden van een URL op de achtergrond om te controleren op een geretourneerde HTTP-statuscode.

De eigenschap available wordt ingesteld op true als het een statuscode ontvangt die wordt opgesomd in de eigenschap acceptableStatusCodes. De eigenschap available wordt ingesteld op false als het een statuscode ontvangt die niet wordt opgesomd in de lijst acceptableStatusCodes, of als er een beveiligingsfout of I/O-fout is opgetreden.
