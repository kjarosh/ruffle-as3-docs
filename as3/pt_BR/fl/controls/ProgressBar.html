Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

O componente ProgressBar exibe o progresso do conteúdo que está sendo carregado. ProgressBar é normalmente usado para exibir o status de imagens, bem como partes de aplicativos, enquanto estão sendo carregados. O processo de carregamento pode ser determinado ou indeterminado. Uma barra de progresso determinada é uma representação linear do progresso de uma tarefa ao longo do tempo, e é utilizada quando a quantidade de conteúdo a ser carregada é conhecida. Uma barra de progresso indeterminada tem um preenchimento listrado e uma fonte de carregamento de tamanho desconhecido.

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

Obtém ou define o método a ser usado para atualizar a barra de progresso.

Os valores a seguir são válidos para essa propriedade:

  • ProgressBarMode.EVENT
  • ProgressBarMode.POLLED
  • ProgressBarMode.MANUAL

O modo de evento e o modo em poll são os modos mais comuns. No modo de evento, a propriedade source especifica o conteúdo de carregamento que gera os eventos de progresso e conclusão; é recomendável usar um objeto UILoader nesse modo. No modo em poll, a propriedade source especifica o conteúdo de carregamento que expõe as propriedades bytesLoaded e bytesTotal. Qualquer objeto que expõe essas propriedades pode ser utilizado como uma origem no modo em poll.

É possível também usar o componente ProgressBar em modo manual definindo manualmente as propriedades maximum e minimum e fazendo chamadas ao método ProgressBar.setProgress().

O valor padrão é ProgressBarMode.EVENT.

O exemplo a seguir cria ocorrências dos componentes ProgressBar e Label para exibir o status da barra de progresso: 

import fl.controls.Label;

var t:Timer = new Timer(150);
t.addEventListener(TimerEvent.TIMER, timerHandler);
t.start();

function timerHandler(event:TimerEvent):void {
    myProgressBar.setProgress(myProgressBar.value + 1, myProgressBar.maximum);
    if (myProgressBar.percentComplete == 100) {
        myProgressBar.setProgress(0, myProgressBar.maximum);
    }
    myLabel.text = int(myProgressBar.value) + " of " + int(myProgressBar.maximum) + " (" + int(myProgressBar.percentComplete) + "%)";
}

ProgressBarExample.as

Este exemplo demonstra como incrementar manualmente uma ProgressBar
  1. Adicione os componentes ProgressBar, Label e Button à biblioteca.
  2. Salve esse código como ProgressBarExample.as no mesmo diretório de FLA.
  3. Defina DocumentClass em FLA como ProgressBarExample.
package
{
    import flash.display.Sprite;
    import flash.events.MouseEvent;
    import fl.controls.ProgressBar;
    import fl.controls.ProgressBarMode;
    import fl.controls.Button;
    
    public class ProgressBarExample extends Sprite
    {
        private var j:uint;
        private var pb:ProgressBar;
        private var fakeProgressCount:uint;
        
        public function ProgressBarExample() {
            setupProgressBar();    
            setupButton();
        }
        private function advancePreloader(e:MouseEvent):void {
            fakeProgressCount = fakeProgressCount == 100 ? 0 : fakeProgressCount + 10;
            pb.setProgress(fakeProgressCount, 100);
        }
        private function setupButton():void {
            var b:Button = new Button();
            b.move(10,30);
            b.setSize(120,20);
            b.label = "Increment Progress";
            b.addEventListener(MouseEvent.CLICK,advancePreloader);
            addChild(b);
        }
        private function setupProgressBar():void {
            fakeProgressCount = 0;
            pb = new ProgressBar();
            pb.mode = ProgressBarMode.MANUAL;
            pb.move(10,10);
            addChild(pb);
        }
    }
}
