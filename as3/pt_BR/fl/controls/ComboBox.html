Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

O componente ComboBox contém uma lista suspensa em que o usuário pode selecionar um valor. Sua funcionalidade é semelhante à do elemento de formulário SELECT em HTML. O componente ComboBox pode ser editado e, nesse caso, o usuário pode digitar entradas que não estão na lista na parte de TextInput do componente ComboBox.

Obtém ou define o modelo de dados da lista de itens a serem visualizados. Um provedor de dados pode ser compartilhado por vários componentes com base em lista. As alterações no provedor de dados estão imediatamente disponíveis a todos os componentes que o usam como uma fonte de dados.

DataProvider

Exemplo  ( Como usar este exemplo )
O exemplo a seguir demonstra como definir um provedor de dados para um ComboBox. Quando o provedor de dados é atualizado, o ComboBox é atualizado: 

import fl.controls.ComboBox;
import fl.data.DataProvider;

var dp:DataProvider = new DataProvider();

var cb:ComboBox = new ComboBox();
cb.dataProvider = dp;
addChild(cb);

var counter = 0;
var t:Timer = new Timer(500);
t.addEventListener(TimerEvent.TIMER,addAnotherItem);
t.start();

function addAnotherItem(e:TimerEvent = null):void {
    dp.addItem( { label: "Item " + counter++ } );
}
