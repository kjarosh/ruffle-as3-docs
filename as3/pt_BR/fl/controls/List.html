O componente List exibe informações com base em lista e é ideal para a exibição de matrizes de informações.

O componente List consiste em itens, linhas e um provedor de dados, descritos da seguinte maneira:

  • Item: um objeto ActionScript que geralmente contém uma propriedade label descritiva e uma propriedade data, que armazena os dados associados a esse item.
  • Linha: um componente utilizado para exibir o item.
  • Provedor de dados: um componente que modela os itens exibidos pelo componente List.

Por padrão, o componente List usa a classe CellRenderer para fornecer as linhas nas quais os itens da lista são exibidos. É possível criar essas linhas de forma programática; isso é feito geralmente subclassificando a classe CellRenderer. A classe CellRenderer implementa a interface ICellRenderer, que fornece um conjunto de propriedades e métodos, que são utilizados pelo componente List para manipular suas linhas e enviar dados e informações de estado para cada linha para exibição. Isso inclui informações sobre o dimensionamento e a seleção de dados.

O componente List fornece métodos que agem em seu provedor de dados -- por exemplo, os métodos addItem() e removeItem(). É possível usar esses e outros métodos para manipular os dados de qualquer matriz existente no mesmo quadro de um componente List e, em seguida, transmitir as alterações para várias exibições. Se um componente List não for fornecido com um provedor de dados externo, esses métodos criam automaticamente uma ocorrência de um provedor de dados e a expõe por meio da propriedade List.dataProvider. O componente List renderiza cada linha usando um Sprite que implementa a interface ICellRenderer. Para especificar esse renderizador, use a propriedade List.cellRenderer. É possível também criar uma ocorrência de Array ou obter uma de um servidor e usá-la como um modelo de dados para várias listas, caixas de combinação, grades de dados, e assim por diante.

iconField

propriedade
iconField:String

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

Obtém ou define o campo de item que fornece o ícone para o item.

Observação: iconField não é usado, se a propriedade iconFunction estiver definida como uma função de retorno.

O valor padrão é "icon".



Implementação
    public function get iconField():String
    public function set iconField(value:String):void

Exemplo  ( Como usar este exemplo )
O exemplo a seguir cria uma lista com um ícone para cada item. É necessário que exista um símbolo com nome RedBox na biblioteca e que "Exportar para o ActionScript" esteja alternado em suas propriedades de símbolo: 
import fl.data.DataProvider;
import fl.controls.List;

var dp:DataProvider = new DataProvider();
dp.addItem( { iconSource:RedBox, label:"Item 1" } );
dp.addItem( { iconSource:RedBox, label:"Item 2" } );
dp.addItem( { iconSource:RedBox, label:"Item 3" } );

var list:List = new List();
list.iconField = "iconSource";
list.dataProvider = dp;
addChild(list);

iconFunction

propriedade 
iconFunction:Function

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

Obtém ou define a função a ser usada para obter o ícone do item.

Observação: iconField não é usado, se a propriedade iconFunction estiver definida como uma função de retorno.

O valor padrão é null.



Implementação
    public function get iconFunction():Function
    public function set iconFunction(value:Function):void

Exemplo  ( Como usar este exemplo )
O exemplo a seguir usa a propriedade iconFunction para fornecer um ícone a um número limitado de itens em uma lista. Para que esse exemplo funcione corretamente, é necessário que exista um símbolo com nome RedBox na biblioteca e que "Exportar para o ActionScript" esteja alternado em suas propriedades de símbolo: 
import fl.data.DataProvider;
import fl.controls.List;

var dp:DataProvider = new DataProvider();
dp.addItem( { label:"Item 1" } );
dp.addItem( { label:"Item 2" } );
dp.addItem( { label:"Item 3" } );

var list:List = new List();
list.iconFunction = determineIcon;
list.dataProvider = dp;
addChild(list);

function determineIcon(item:Object):String {
    if(item.label == "Item 2") {
        return "RedBox";
    }
    else {
        return null;
    }
}

labelField

propriedade 
labelField:String

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

Recupera a string exibida pelo renderizador para o objeto de dados determinado com base nas propriedades labelField e labelFunction.

Exemplo  ( Como usar este exemplo )

O exemplo a seguir demonstra como determinar o rótulo de um item da lista, quando o mouse flutuar sobre esse item:

    return item.label + " - Current price: ($" + item.price.toFixed(2) + ")";

function announceLabel(e:ListEvent):void {
    var list:List = e.target as List;
    var item:Object = e.item;
    trace("Label: " + item.label);
    trace("Label displayed: " + list.itemToLabel(item));
}

ListExample.as

Esse exemplo demonstra como criar uma ocorrência e preencher outras ocorrências de List com suas seleções.
  1. Adicione as classes List, Button e Label à biblioteca.
  2. Salve esse código como ListExample.as no mesmo diretório de FLA.
  3. Defina DocumentClass em FLA como ListExample.
package
{
    import flash.display.Sprite
    
    public class ListExample extends Sprite
    {
        import flash.events.*;
        import fl.data.DataProvider;
        import fl.controls.List
        import fl.controls.Label
        import fl.controls.Button
        
        private var clearButton:Button;
        private var availableItems:List;
        private var selectedItemList:List;
        private var selectedItemsList:List;

        public function ListExample() {
            createComponents();
            setupComponents();
        }

        private function setupComponents():void {
            var dp:Array = new Array();
            var i:uint;
            var count:uint = availableItems.rowCount * 2;
            
            for (i = 0; i < count; i++) {
                dp.push({label:"Item " + i});
            }
            
            availableItems.allowMultipleSelection = true;
            availableItems.dataProvider = new DataProvider(dp);
            availableItems.addEventListener(Event.CHANGE, updateLists);
            clearButton.addEventListener(MouseEvent.CLICK, clearHandler);            
        }
        
        private function clearHandler(event:MouseEvent):void {
            availableItems.clearSelection();
            // clear data providers
            selectedItemList.dataProvider = new DataProvider();
            selectedItemsList.dataProvider = new DataProvider();
        }
        
        private function updateLists(e:Event):void {
            selectedItemList.dataProvider = availableItems.selectedItem ? new DataProvider([availableItems.selectedItem]) : new DataProvider();
            selectedItemsList.dataProvider = new DataProvider(availableItems.selectedItems);
        }

        private function createComponents():void {
            clearButton = new Button();
            availableItems = new List();
            selectedItemList = new List();
            selectedItemsList = new List();
            var availableItemsLabel:Label = new Label();
            var selectedItemListLabel:Label = new Label();
            var selectedItemsListLabel:Label = new Label();
            
            clearButton.move(10,142);
            availableItems.move(10,32);
            selectedItemList.move(120,32);
            selectedItemsList.move(230,32);
            availableItemsLabel.move(10,10);
            selectedItemListLabel.move(120,10);
            selectedItemsListLabel.move(230,10);
            
            clearButton.label = "Clear Selection"
            availableItemsLabel.text = "Available Items";
            selectedItemListLabel.text = "Selected Item";
            selectedItemsListLabel.text = "All Selected Items";
            
            addChild(clearButton);
            addChild(availableItems);
            addChild(selectedItemList);
            addChild(selectedItemsList);
            addChild(availableItemsLabel);
            addChild(selectedItemListLabel);
            addChild(selectedItemsListLabel);
        }
    }
}
