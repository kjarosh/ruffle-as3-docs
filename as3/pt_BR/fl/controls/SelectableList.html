SelectableList é a classe base de todos os componentes com base em lista; por exemplo, os componentes List, TileList, DataGrid e ComboBox. Essa classe fornece métodos e propriedades que são usados para a renderização e o layout de linhas, e para definir os estilos da barra de rolagem e os provedores de dados.

Observação: essa classe não cria um componente; ela é apenas exposta para que possa ser estendida.
