Pacotefl.motion
Classepublic class MotionEvent
HerançaMotionEvent Event Object

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

A classe MotionEvent representa eventos que são transmitidos pela classe fl.motion.Animator. Use esses eventos com um listener de eventos para iniciar funções personalizadas. Por exemplo, se você tiver uma ocorrência Animator nomeada abox_animator, o Flash Player executará a seguinte instrução trace quando a animação estiver concluída: 
 import fl.motion.MotionEvent;
 abox_animator.addEventListener(MotionEvent.MOTION_END,afterMotion);
 function afterMotion(e:MotionEvent) {
    trace("animation complete!");
 }

Construtor.

Parâmetros
type:String — O tipo de evento; indica a ação que causou o evento.
 
bubbles:Boolean (default = false) — Especifica se o evento pode colocar a hierarquia da lista de exibição em um balão.
 
cancelable:Boolean (default = false) — Especifica se o comportamento associado ao evento pode ser evitado.
Detalhes da constante

MOTION_END

Constante
public static const MOTION_END:String = "motionEnd"

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

Indica que o movimento parou, seja por uma chamada explícita para Animator.stop() ou Animator.end(), ou por ter atingido o final da ocorrência Motion.

As propriedades do objeto do evento têm os seguintes valores:

PropriedadeValor
bubblesfalse
cancelablefalse
currentTargetO objeto que define o listener de evento que manipula o evento. Por exemplo, se você usa myButton.addEventListener() para registrar um listener de evento, myButton é o valor da currentTarget.
targetO objeto que enviou o evento; nem sempre é o objeto que ouve o evento. Use a propriedade currentTarget para acessar sempre o objeto que ouve o evento.
