Pacotefl.events
Classepublic class SliderEvent
HerançaSliderEvent Event Object

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

A classe SliderEvent define eventos associados ao componente Slider. Incluindo o seguinte:
  • SliderEvent.CHANGE: despachado depois que ocorre uma alteração na posição do controle deslizante.
  • SliderEvent.THUMB_DRAG: despachado quando o usuário arrasta a miniatura do controle deslizante.
  • SliderEvent.THUMB_PRESS: despachado quando o usuário pressiona a miniatura do controle deslizante.
  • SliderEvent.THUMB_RELEASE: despachado quando o usuário solta a miniatura do controle deslizante.

SliderEvent

()Construtor
public function SliderEvent(type:String, value:Number, clickTarget:String, triggerEvent:String, keyCode:int = 0)

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

Cria um novo objeto SliderEvent com os parâmetros especificados.

Parâmetros
type:String — O tipo de evento; este valor identifica a ação que disparou o evento.
 
value:Number — O novo valor do controle deslizante.
 
clickTarget:String — Indica se uma miniatura de controle deslizante ou se o controle deslizante foi pressionado. O valor SliderEventClickTarget.THUMB indica que a miniatura do controle deslizante foi pressionada; o valor SliderEventClickTarget.TRACK indica que o controle deslizante foi pressionado.
 
triggerEvent:String — Uma string que indica a fonte de entrada. O valor InteractionInputType.MOUSE indica que o mouse foi a fonte de entrada; o valorInteractionInputType.KEYBOARD indica que o teclado foi a fonte de entrada.
 
keyCode:int (default = 0) — Se o evento tiver sido disparado por uma tecla pressionada, este valor será o código de tecla que identifica aquela tecla.

PropriedadeValor
bubblesfalse
cancelablefalse; não há comportamento padrão a ser cancelado.
