BUFFERING

Constante
public static const BUFFERING:String = "buffering"

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

O player de vídeo está no estado em buffer. Ele entra neste estado imediatamente depois que é feita uma chamada ao método play() ou load(). Use a propriedade FLVPlayback.state para obter o estado atual do player de vídeo.

Este é um estado responsivo.

CONNECTION_ERROR

Constante 
public static const CONNECTION_ERROR:String = "connectionError"

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

O player de vídeo está no estado de erro de conexão. Ele entra neste estado quando ocorreu uma tentativa de carregar um fluxo de vídeo, mas ela não foi bem-sucedida. Há duas razões possíveis para o erro: não há conexão com o servidor ou o fluxo não foi encontrado. Use a propriedade FLVPlayback.state para obter o estado atual do player de vídeo.

Este é um estado não responsivo.

Elementos da API relacionados

FLVPlayback.state
FLVPlayback.stateResponsive

DISCONNECTED

Constante 
public static const DISCONNECTED:String = "disconnected"

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

O player de vídeo está no estado desconectado. Ele entra neste estado quando o fluxo é fechado por uma chamada ao método closeVideoPlayer() ou foi encerrado por ociosidade. Use a propriedade FLVPlayback.state para obter o estado atual do player de vídeo.

A ocorrência FLVPlayback encontra-se em um estado desconectado até você configurar a propriedade FLVPlayback.source.

REWINDING

Constante 
public static const REWINDING:String = "rewinding"

Versão da linguagem: ActionScript 3.0
Versão de produto: Flash CS3
Versões de runtime: Flash Player 9.0.28.0, AIR 1.0

O player de vídeo está no estado de retrocesso. Ele entra neste estado quando o vídeo é retrocedido automaticamente. O vídeo é retrocedido automaticamente depois que é parado, ou clicando-se no botão parar ou com a reprodução do vídeo até o final. Depois que o retrocesso é concluído, o estado é interrompido. Use a propriedade FLVPlayback.state para obter o estado atual do player de vídeo.

Este é um estado não responsivo.
