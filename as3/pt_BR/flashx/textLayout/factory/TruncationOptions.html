A classe TruncationOptions especifica as opções para limitar o número de linhas de texto criadas por uma fábrica de linhas de texto e para indicar quando linhas foram deixadas de fora.

Detalhes do construtor

TruncationOptions

()Construtor
public function TruncationOptions(truncationIndicator:String, lineCountLimit:int, truncationIndicatorFormat:flashx.textLayout.formats:ITextLayoutFormat = null)

Versão da linguagem: ActionScript 3.0
Versões de runtime: Flash Player 10, AIR 1.5

Cria um objeto TruncationOptions.

Parâmetros
truncationIndicator:String (default = NaN) — a string usada para indicar que o texto foi truncado. Aparece no final do texto composto. O valor padrão são as reticências (U+2026).
 
lineCountLimit:int (default = NaN) — especifica um critério de truncação na forma do número máximo permitido de linhas. O valor padrão de NO_LINE_COUNT_LIMIT indica que não há limite para o número de linhas.
 
truncationIndicatorFormat:flashx.textLayout.formats:ITextLayoutFormat (default = null) — especifica o formato do indicador de truncação. Um formato nulo (o valor padrão) especifica que o indicador de truncação assume o formato do conteúdo pouco antes do ponto de truncação. Os métodos TextLineFactory que utilizam uma string simples também ignoram este parâmetro e implementam o comportamento padrão.
