Detalhes do construtor

SelectionFormat

()Construtor
public function SelectionFormat(rangeColor:uint = 0xffffff, rangeAlpha:Number = 1.0, rangeBlendMode:String = "difference", pointColor:uint = 0xffffff, pointAlpha:Number = 1.0, pointBlendMode:String = "difference", pointBlinkRate:Number = 500)

Versão da linguagem: ActionScript 3.0
Versões de runtime: Flash Player 10, AIR 1.5

Cria um objeto SelectionFormat com as propriedades especificadas.

Um SelectionFormat criado com os valores padrão utilizará preto para as cores de destaque, 1.0 para os alfas, e BlendMode.DIFFERENCE para os modos de mesclagem. A velocidade na qual o cursor pisca é 500 milissegundos.

Definir as propriedades pointAlpha e rangeAlpha em zero desativa o realce da seleção.

A velocidade de piscar diferente de zero só é usada quando um EditManager é anexado ao TextFlow.

Parâmetros
rangeColor:uint (default = 0xffffff) — A cor para desenhar o destaque.
 
rangeAlpha:Number (default = 1.0) — O valor de transparência para desenhar o destaque. Os valores válidos estão entre 0 (completamente transparente) e 1 (completamente opaco, que é o padrão).
 
rangeBlendMode:String (default = "difference") — O modo de mesclagem para desenhar o destaque. Use as constantes definidas na classe BlendMode para definir este parâmetro.
 
pointColor:uint (default = 0xffffff) — A cor para desenhar o cursor.
 
pointAlpha:Number (default = 1.0) — O valor de transparência para desenhar o cursor. Os valores válidos estão entre 0 (completamente transparente) e 1 (completamente opaco, que é o padrão).
 
pointBlendMode:String (default = "difference") — O modo de mesclagem para desenhar o cursor. Use as constantes definidas na classe BlendMode para definir este parâmetro.
 
pointBlinkRate:Number (default = 500) — Velocidade na qual o cursor pisca, em milissegundos.
