Classepublic class SecureStreamingHTTPChannel
HerançaSecureStreamingHTTPChannel StreamingHTTPChannel HTTPChannel PollingChannel Channel EventDispatcher Object

Versão da linguagem: ActionScript 3.0
Versões de produto: BlazeDS 4, Adobe Digital Enterprise Platform Data Services for Java EE 3
Versões de runtime: Flash Player 9, AIR 1.1

The SecureStreamingHTTPChannel class is identical to the StreamingHTTPChannel class except that it uses a secure protocol, HTTPS, to send messages to an HTTP endpoint.
