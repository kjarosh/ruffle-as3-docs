O pacote flash.desktop contém classes usadas nas operações de copiar e colar e arrastar e soltar, bem como a classe Ícone, usada para definir ícones de sistema usados por um arquivo. Este pacote só está disponível para conteúdo em execução no Adobe AIR.


Interfaces
 InterfaceDescrição
 IFilePromise     A interface de IFilePromise define a interface usada pelo tempo de execução de AIR para ler dados de uma promessa de arquivo.

 Clipboard A classe Clipboard fornece um contêiner para transferir dados e objetos pela área de transferência.
 ClipboardFormats A classe ClipboardFormats define constantes dos nomes dos formatos de dados padrão usados com a classe Clipboard.
 ClipboardTransferMode A classe ClipboardTransferMode define as constantes para os modos usados como valores do parâmetro transferMode do método Clipboard.getData().
 DockIcon     A classe DockIcon representa o ícone de encaixe de estilo Mac OS X®.

 InvokeEventReason     A classe InvokeEventReason enumera valores retornados pela propriedade do motivo de um objeto InvokeEvent.
 NativeApplication     A classe NativeApplication representa esta aplicação AIR.

 NativeDragOptions     A classe NativeDragOptions define constantes dos nomes de ações de arrastar e soltar permitidas em uma operação de arrastar e soltar.

 SystemIdleMode     A classe SystemIdleMode fornece valores constantes para comportamentos ociosos do sistema.
 SystemTrayIcon     A classe SystemTrayIcon representa o ícone do estilo da área de notificação (bandeja do sistema) da barra de tarefas do Windows®.
 Updater     A classe Updater é usada para atualizar o aplicativo em execução atual com uma versão diferente.

[ X ]Por que inglês?
O conteúdo na Referência do ActionScript 3.0 aparece em inglês

Nem todas as partes da Referência do ActionScript 3.0 estão traduzidas em todos os idiomas. Quando um elemento de um idioma não está traduzido, ele é exibido em inglês. Por exemplo, a classe ga.controls.HelpBox não está traduzida em nenhum idioma. Por isso, na versão da referência em português do Brasil, a classe ga.controls.HelpBox é exibida em inglês.
