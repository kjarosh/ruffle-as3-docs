Classepublic final class GesturePhase
HerançaGesturePhase Object

Versão da linguagem: ActionScript 3.0
Versões de runtime: Flash Player 10.1, AIR 2, Flash Lite 4

A classe GesturePhase é uma classe de lista de valores constantes para uso com as classes GestureEvent, PressAndTapGestureEvent e TransformGestureEvent. Use estes valores para acompanhar o início, o progresso e o fim de um gesto de toque (como a movimentação de vários dedos através de uma tela de toque ativada) para que o seu aplicativo possa responder às etapas individuais do contato de usuário. Alguns gestos (gestos de batida e de torneira de dois dedos) não têm múltiplas fases e definem a propriedade de objeto de evento phase como all.

Um valor único que engloba todas as fases dos gestos simples como toque com dois dedos ou batida. Para gestos que estabelecem o objeto de evento fase propriedade a todos (batem e gestos de torneira de dois dedos), o fase o valor é sempre todos uma vez que o evento é despachado.

Exemplos Como usar este exemplo
TransformGestureEventExample.as

O seguinte exemplo mostra o controle de eventos para os eventos GESTURE_ROTATE. Enquanto o usuário realiza um gesto de rotação no dispositivo ativado pelo toque, o mySprite gira e myTextField preenche com a fase atual. 
Multitouch.inputMode = MultitouchInputMode.GESTURE;

var mySprite = new Sprite();
mySprite.addEventListener(TransformGestureEvent.GESTURE_ROTATE , onRotate );
mySprite.graphics.beginFill(0x336699);
mySprite.graphics.drawRect(0, 0, 100, 80);
var myTextField = new TextField();
myTextField.y = 200;
addChild(mySprite);
addChild(myTextField);

function onRotate(evt:TransformGestureEvent):void {

    evt.target.rotation -= 45;

    if (evt.phase==GesturePhase.BEGIN) {
        myTextField.text = "Begin";
    }
    if (evt.phase==GesturePhase.UPDATE) {
        myTextField.text = "Update";
    }
    if (evt.phase==GesturePhase.END) {
        myTextField.text = "End";
    }
}
