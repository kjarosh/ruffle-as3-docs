public function DatagramSocketDataEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, srcAddress:String = "", srcPort:int = 0, dstAddress:String = "", dstPort:int = 0, data:ByteArray = null)

Versão da linguagem: ActionScript 3.0
Versões de runtime: AIR 2

Cria um objeto Event que contém informações sobre eventos de datagrama. Objetos Event são transmitidos como parâmetros para ouvintes de eventos.

Parâmetros
type:String — O tipo de evento. Os valores possíveis são:DatagramSocketDataEvent.DATA
 
bubbles:Boolean (default = false) — Determina se o objeto Event participa do estágio de bubbling do fluxo de eventos.
 
cancelable:Boolean (default = false) — Determina se o objeto Event pode ser cancelado.
 
srcAddress:String (default = "") — O endereço IP da máquina que enviou o pacote.
 
srcPort:int (default = 0) — A porta da máquina que enviou o pacote.
 
dstAddress:String (default = "") — O endereço IP ao qual o pacote é dirigido.
 
dstPort:int (default = 0) — A porta de conexão à qual o pacote é dirigido.
 
data:ByteArray (default = null) — Os dados do pacote de datagrama.
