Versão da linguagem: ActionScript 3.0

Este evento é despachado pelo objeto NativeProcess quando o processo sair. É possível que este evento nunca seja despachado. Por exemplo, se o processo filho durar mais que o aplicativo AIR que o criou, o evento não será despachado.

Detalhes da propriedade
    

exitCode

propriedade
exitCode:Number

Versão da linguagem: ActionScript 3.0
Versões de runtime: AIR 2

O código de saída que o processo nativo retornou ao sistema operacional de host ao sair. Se o aplicativo de AIR termina o processo chamando o método exit() do objeto NativeProcess, a propriedade exitCode é definida para NaN. OBSERVAÇÃO: em sistemas operacionais Windows, se o processo não tenha saído, mas o tempo de execução está saindo ou um erro ocorreu, este valor poderá ser definido para 259 (STILL_ACTIVE). Para evitar esse tipo confusão, não use 259 como um código de retorno em um processo nativo.

public function NativeProcessExitEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, exitCode:Number = NaN)

Versão da linguagem: ActionScript 3.0
Versões de runtime: AIR 2

Cria um NativeProcessExitEvent que contém informações específicas quanto ao código de saída de um NativeProcess

Parâmetros
type:String — O tipo do evento, acessível como Event.type.
 
bubbles:Boolean (default = false) — Determina se o objeto Event participa do estágio de bubbling do fluxo de eventos. O valor padrão é false.
 
cancelable:Boolean (default = false) — Determina se o objeto Event pode ser cancelado. O valor padrão é false.
 
exitCode:Number (default = NaN) — O número que o processo retornou ao sistema operacional durante a saída.
