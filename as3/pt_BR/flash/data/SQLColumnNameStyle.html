HerançaSQLColumnNameStyle Object

Versão da linguagem: ActionScript 3.0
Versões de runtime: AIR 1.0

Essa classe contém as constantes que representam os valores possíveis para a propriedade SQLConnection.columnNameStyle. Esses valores indicam as opções diferentes que controlam como os nomes de colunas (nomes de propriedades) são formatados nos objetos retornados como uma resultado de uma instrução SELECT SQL.

DEFAULT

Constante
public static const DEFAULT:String = "default"

Versão da linguagem: ActionScript 3.0
Versões de runtime: AIR 1.0

Indica que os nomes de colunas retornados de uma instrução SELECT usam o formato padrão. No formato padrão, os nomes de colunas têm forma [table-name]_[column-name] quando várias tabelas são incluídas na instrução SELECT, ou [column-name] quando a instrução SELECT inclui uma única tabela.

SHORT

Constante 
public static const SHORT:String = "short"

Versão da linguagem: ActionScript 3.0
Versões de runtime: AIR 1.0

Indica que os nomes de colunas retornados de uma instrução SELECT usam o formato de nome de coluna curto. Nesse formato, os nomes de colunas usam a forma [column-name] independentemente de quantas tabelas estejam inclusas na instrução SELECT.

Se o conjunto de resultados contiver várias colunas com o mesmo nome, somente uma propriedade com esse nome será adicionada ao objeto resultante. O valor atribuído a essa propriedade é tirado da última coluna com esse nome na linha de resultado. Por exemplo, considere a seguinte instrução SELECT:

	 SELECT customers.customerId, addresses.customerId
	 FROM customers INNER JOIN addresses
	    ON customers.customerId = addresses.customerId

Quando essa instrução for executada em uma instância SQLConnection com o formato de nome de coluna curto, cada objeto resultante terá uma propriedade denominada customerId, que contém o valor da coluna customerId da tabela addresses.
