繼承FLVPlaybackCaptioning Sprite DisplayObjectContainer InteractiveObject DisplayObject EventDispatcher Object

語言版本: ActionScript 3.0
執行階段版本: AIR 1.0, Flash Player 9.0.28.0

FLVPlaybackCaptioning 組件可啟用 FLVPlayback 組件的註解功能。 注意：本文件需搭配具輔助功能的 FLVPlaybackCaptioning 組件使用。

本文件更新了「Adobe® Flash® Professional CS5 的 ActionScript® 3.0 參考」的 FLVPlaybackCaptioning 類別，以包含下列為了改進組件的輔助功能而新增的屬性和方法。

在您嘗試存取新的屬性或方法之前，務必確定專案中包含組件的「具輔助功能」版本。

FLVPlaybackCaptioning 組件會下載 Timed Text (TT) XML 檔，然後將這些註解套用至與此組件搭配運作的 FLVPlayback 組件。

如需有關 Timed Text 格式的詳細資訊，請參閱 http://www.w3.org/AudioVideo/TT/。 FLVPlaybackCaptioning 組件支援 Timed Text 1.0 規格的子集。 如需有關受支援子集的詳細資訊，請參閱 Timed Text 標籤。 以下為簡要的範例：

	 <?xml version="1.0" encoding="UTF-8"?>
	 <tt xml:lang="en" xmlns="http://www.w3.org/2006/04/ttaf1"  xmlns:tts="http://www.w3.org/2006/04/ttaf1#styling">
	     <head>
	         <styling>
	             <style id="1" tts:textAlign="right"/>
	             <style id="2" tts:color="transparent"/>
	             <style id="3" style="2" tts:backgroundColor="white"/>
	             <style id="4" style="2 3" tts:fontSize="20"/>
	         </styling>
	     </head>
	     <body>
	          <div xml:lang="en">
	             <p begin="00:00:00.50" dur="500ms">Four score and twenty years ago</p>
	             <p begin="00:00:02.50"><span tts:fontFamily="monospaceSansSerif,proportionalSerif,TheOther"tts:fontSize="+2">our forefathers</span> brought forth<br /> on this continent</p>
	             <p begin="00:00:04.40" dur="10s" style="1">a <span tts:fontSize="12 px">new</span> <span tts:fontSize="300%">nation</span></p>
	             <p begin="00:00:06.50" dur="3">conceived in <span tts:fontWeight="bold" tts:color="#ccc333">liberty</span> <span tts:color="#ccc333">and dedicated to</span> the proposition</p>
	             <p begin="00:00:11.50" tts:textAlign="right">that <span tts:fontStyle="italic">all</span> men are created equal.</p>
	 			<p begin="15s" style="4">The end.</p>
	         </div>    
	     </body>
	 </tt>

autoLayout

屬性
autoLayout:Boolean

語言版本: ActionScript 3.0
執行階段版本: AIR 1.0, Flash Player 9.0.28.0

判斷 FLVPlaybackCaptioning 組件是否會自動移動 TextField 物件並調整其大小，以便做為註解。

如果 autoLayout 屬性設定為 true，則包含註解的 DisplayObject 實體或 TextField 物件會從 FLVPlayback 實體底端起顯示 10 個像素。 註解區域會涵蓋 FLVPlayback 實體的寬度，並在每一邊都會維持 10 個像素寬的邊界。

當此屬性設定為 true 時，DisplayObject 實體或 TextField 物件會直接顯示在 FLVPlayback 實體上。 如果您要建立自己的 TextField 物件，則應該將 autoLayout 設定為 false。 如果 wordWrap = false，註解區會置中出現在 FLVPlayback 實體上，但是可能會比 FLVPlayback 實體還要寬。

若要控制配置，您必須偵聽 captionChange 事件，以判斷 TextField 物件實體是何時建立的。

預設值為 true。

captionTarget

屬性 
captionTarget:DisplayObject

語言版本: ActionScript 3.0
執行階段版本: AIR 1.0, Flash Player 9.0.28.0

會設定要在其中顯示註解的 DisplayObject 實體。

如果您將實體設定為 TextField 物件，即是直接以它為目標。 如果您將實體設定為包含一個或多個 TextField 物件的 DisplayObjectContainer，則註解會顯示在具有最小顯示索引的 TextField 物件中。

DisplayObjectContainer 方法支援像是有九宮格縮放 (Scale-9) 背景之物件的影片片段，這可以在 TextField 物件大小變更時進行縮放。

如需較複雜的縮放和繪圖功能，請撰寫程式碼讓 DisplayObjectContainer 方法偵聽 captionChange 事件。

注意：如果未設定 captionTargetName 或 captionTarget 屬性，則 FLVPlaybackCaptioning 實體會建立由 captionTarget 屬性以下列格式設定的文字欄位：

  • 黑色背景 (background = true ; backgroundColor = 0x000000 ;)
  • 白色文字 (textColor = 0xFFFFFF)
  • autoSize = TextFieldAutoSize.LEFT
  • multiLine = true
  • wordWrap = true
  • font = "_sans"
  • size = 12

若要自訂這些值，請偵聽 captionTargetCreated 事件。

您必須遵循 FLVPlaybackCaptioning 提示點的支援格式。

如果 track 屬性設定為 0 以外的值，FLVPlaybackCaptioning 組件便會在提示點上搜尋 text<n> 屬性，其中 n 即是 track 值。

範例 如何使用本範例
FLVPlaybackCaptioningExample.as

此範例將示範如何以動態方式，建立具有可切換註解功能的 FLVPlayback 實體。

若要執行範例，請遵循下列步驟：

  1. 開啟新的 Flash 檔案 (ActionScript 3.0)。
  2. 將 FLVPlayback 組件放在「舞台」上 0,0 的位置，並提供實體名稱「player」。
  3. 將 FLVPlaybackCaptioning 組件放在舞台上 0,0 的位置，並提供實體名稱「captioning」。
  4. 將 CaptionButton 組件放在「舞台」上。
  5. 儲存 FLA 檔並指定名稱，例如 captionEx。
  6. 開啟新的 ActionScript 檔案。
  7. 請將下列類別範例程式碼貼至新的 ActionScript 檔案，並且在您 FLA 檔所在的目錄中將它儲存為 FLVPlaybackCaptioningExample.as。
  8. 在 FLA 檔 (例如 captionEx) 中，使用「屬性」檢測器將 Document 類別項目設定為 FLVPlaybackCaptioningExample。
  9. 選取「控制 > 測試影片」。
package
{
    import fl.video.CaptionChangeEvent;
    import fl.video.FLVPlayback;
    import fl.video.FLVPlaybackCaptioning;
    import flash.display.Sprite;
    import flash.text.TextField;
    
    public class FLVPlaybackCaptioningExample extends Sprite {

        private var videoPath:String = "http://www.helpexamples.com/flash/video/caption_video.flv";
        private var captioningPath:String = "http://www.helpexamples.com/flash/video/caption_video.xml";
        
        public function FLVPlaybackCaptioningExample() {
            player.source = videoPath;
            player.skin = "SkinOverAllNoCaption.swf";
            player.skinBackgroundColor = 0x666666;
            player.skinBackgroundAlpha = 0.5;

            captioning.flvPlayback = player;
            captioning.source = captioningPath;
            captioning.autoLayout = false;            
            captioning.addEventListener(CaptionChangeEvent.CAPTION_CHANGE, onCaptionChange);
        }

        private function onCaptionChange(e:CaptionChangeEvent):void {
            var tf:* = e.target.captionTarget;
            var player:FLVPlayback = e.target.flvPlayback;

            // move the caption below the video
            tf.y = 210;            
        }
    }
}
