BUFFERING

常數
public static const BUFFERING:String = "buffering"

語言版本: ActionScript 3.0
產品版本: Flash CS3
執行階段版本: Flash Player 9.0.28.0, AIR 1.0

視訊播放程式處於緩衝狀態。 在呼叫 play() 或 load() 方法之後，視訊播放程式就會立即進入這個狀態。 您可以使用 FLVPlayback.state 屬性，取得視訊播放程式目前的狀態。

這是回應狀態。

相關 API 元素

FLVPlayback.buffering
FLVPlayback.state
FLVPlayback.stateResponsive

CONNECTION_ERROR

常數 
public static const CONNECTION_ERROR:String = "connectionError"

語言版本: ActionScript 3.0
產品版本: Flash CS3
執行階段版本: Flash Player 9.0.28.0, AIR 1.0

視訊播放程式處於連線錯誤狀態。 當視訊串流嘗試載入但是失敗時，視訊播放程式就會進入這個狀態。 發生這項錯誤有兩個可能的原因：尚未建立與伺服器的連線，或是找不到串流。 您可以使用 FLVPlayback.state 屬性，取得視訊播放程式目前的狀態。

這是無回應狀態。

相關 API 元素

FLVPlayback.state
FLVPlayback.stateResponsive

DISCONNECTED

常數 
public static const DISCONNECTED:String = "disconnected"

語言版本: ActionScript 3.0
產品版本: Flash CS3
執行階段版本: Flash Player 9.0.28.0, AIR 1.0

視訊播放程式處於斷線狀態。 當串流透過呼叫 closeVideoPlayer() 方法或因為閒置逾時而關閉時，視訊播放程式就會進入這個狀態。 您可以使用 FLVPlayback.state 屬性，取得視訊播放程式目前的狀態。

在您設定 FLVPlayback.source 屬性之前，FLVPlayback 實體會一直處於斷線狀態。

相關 API 元素

FLVPlayback.closeVideoPlayer()
FLVPlayback.idleTimeout
FLVPlayback.source
FLVPlayback.state
FLVPlayback.stateResponsive

視訊播放程式處於調整大小的狀態。 視訊播放程式會在自動配置期間進入這種狀態。 FLVPlayback 實體永遠不會有這個常數的狀態值，只有 VideoPlayer 實體才有。 您可以使用 VideoPlayer.state 屬性，取得視訊播放程式目前的狀態。

這是無回應狀態。

REWINDING

常數 
public static const REWINDING:String = "rewinding"

語言版本: ActionScript 3.0
產品版本: Flash CS3
執行階段版本: Flash Player 9.0.28.0, AIR 1.0

視訊播放程式處於倒帶狀態。 當視訊自動倒帶時，視訊播放程式就會進入這個狀態。 在按一下停止按鈕或因為視訊已播放到結尾而停止播放之後，視訊就會自動倒帶。 在倒帶完成之後，狀態就會成為停止狀態。 您可以使用 FLVPlayback.state 屬性，取得視訊播放程式目前的狀態。

這是無回應狀態。

相關 API 元素

FLVPlayback.autoRewind
FLVPlayback.state
FLVPlayback.stateResponsive
