套件mx.controls.advancedDataGridClasses
類別public class AdvancedDataGridListData
繼承AdvancedDataGridListData DataGridListData BaseListData EventDispatcher Object

語言版本: ActionScript 3.0
產品版本: Flex 3
執行階段版本: Flash Player 9, AIR 1.1

The AdvancedDataGridListData class defines the data type of the listData property implemented by drop-in item renderers or drop-in item editors for the AdvancedDataGrid control. All drop-in item renderers and drop-in item editors must implement the IDropInListItemRenderer interface, which defines the listData property.

While the properties of this class are writable, you should consider them to be read only. They are initialized by the AdvancedDataGrid class, and read by an item renderer or item editor. Changing these values can lead to unexpected results.
