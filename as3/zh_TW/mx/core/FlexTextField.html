子類別 StyleableTextField, TextFieldAsset, UITextField

語言版本: ActionScript 3.0
產品版本: Flex 3
執行階段版本: Flash Player 9, AIR 1.1

FlexTextField is a subclass of the Player's TextField class and the superclass of UITextField. It overrides the toString() method to return a string indicating the location of the object within the hierarchy of DisplayObjects in the application.
