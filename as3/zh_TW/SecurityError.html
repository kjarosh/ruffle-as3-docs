類別public dynamic class SecurityError
繼承SecurityError Error Object

語言版本: ActionScript 3.0
執行階段版本: Flash Player 9, AIR 1.0, Flash Lite 4

發生某些類型的安全性違規時，便會擲回 SecurityError 例外。

安全性錯誤的範例：

  • 跨越安全執行程序邊界，並進行未授權的屬性存取或方法呼叫。
  • 嘗試存取安全執行程序不允許的 URL。
  • 嘗試對未授權的連接埠號進行通訊端連線，例如 65535 以上的連接埠。
  • 嘗試存取使用者的攝影機或麥克風，而且存取裝置的要求已由使用者所拒絕。

範例 如何使用本範例
SecurityErrorExample.as

下列範例會示範 SecurityError 錯誤如何產生，並在 try...catch 陳述式中加以處理，錯誤的產生是因為嘗試連接比允許的最大埠號還要多一的埠號。 
package {
    import flash.display.Sprite;
    import flash.net.Socket;

    public class SecurityErrorExample extends Sprite {

        public function SecurityErrorExample() {
            try {
                var host:String = "www.[yourDomain].com";
                var socket:Socket = new Socket();
                socket.connect(host, 65536);
            }
            catch(e:SecurityError) {
                trace(e);
            }
        }
    }
}
