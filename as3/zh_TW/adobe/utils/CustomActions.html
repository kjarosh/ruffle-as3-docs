類別public final class CustomActions
繼承CustomActions Object

語言版本: ActionScript 3.0
產品版本: Flash CS3
執行階段版本: AIR 1.0, Flash Player 9

CustomActions 類別的方法可讓在 Flash 編寫工具中播放的 SWF 檔，管理以編寫工具註冊的任何自訂動作。 SWF 檔可以安裝並解除安裝自訂動作、擷取某個自訂動作的 XML 定義，並擷取已註冊之自訂動作的清單。

您可以使用這些方法，建立做為 Flash 編寫工具之擴充功能使用的 SWF 檔。 例如，這類擴充功能可以使用 Flash 應用程式協定來導覽「通用描述、探索與整合」(UDDI) 貯存庫，並將網路服務下載到「動作工具箱」中。

actionsList

屬性
actionsList:Array  [唯讀]

語言版本: ActionScript 3.0
產品版本: Flash CS3
執行階段版本: AIR 1.0, Flash Player 9

傳回 Array 物件，其中包含所有註冊到 Flash 編寫工具中的自訂動作名稱。 陣列的元素具有簡單的名稱，不含 .xml 副檔名，也不含任何目錄分隔符號 (例如，":"、"/" 或 "\")。 如果沒有已註冊的自訂動作，actionsList() 會傳回長度為零的陣列。 如果發生錯誤，actionsList() 會傳回 undefined 值。

getActions

()方法
public static function getActions(name:String):String

語言版本: ActionScript 3.0
產品版本: Flash CS3
執行階段版本: AIR 1.0, Flash Player 9

讀取名為 name 的自訂動作 XML 定義檔案的內容。

定義檔案的名稱必須是簡單的檔案名稱，不含 .xml 副檔名，也不含任何目錄分隔符號 (":"、"/" 或 "\")。

如果找不到 name 所指定的定義檔案，便會傳回 undefined 值。 如果找到 name 參數所指定的自訂動作 XML 定義，便會讀取它的完整內容並當做字串傳回。

參數

name:String — 要擷取的自訂動作定義名稱。

傳回值
String — 如果找到自訂動作 XML 定義，便會傳回字串；否則會傳回 undefined。

installActions

()方法 
public static function installActions(name:String, data:String):void

語言版本: ActionScript 3.0
產品版本: Flash CS3
執行階段版本: AIR 1.0, Flash Player 9

安裝 name 參數所指定的新自訂動作 XML 定義檔案。 這個檔案的內容是由字串 data 所指定。

定義檔案的名稱必須是簡單的檔案名稱，不含 .xml 副檔名，也不含任何目錄分隔符號 (":"、"/" 或 "\")。

如果名為 name 的自訂動作檔案已經存在，就會將它覆寫。

叫用這個方法時，如果 Configuration/ActionsPanel/CustomActions 目錄不存在，便會建立這個目錄。

參數

name:String — 要安裝的自訂動作定義名稱。
 
data:String — 要安裝的 XML 定義文字。

語言版本: ActionScript 3.0
產品版本: Flash CS3
執行階段版本: AIR 1.0, Flash Player 9

移除名為 name 的自訂動作 XML 定義檔案。

定義檔案的名稱必須是簡單的檔案名稱，不含 .xml 副檔名，也不含任何目錄分隔符號 (":"、"/" 或 "\")。

參數

name:String — 要解除安裝的自訂動作定義名稱。
