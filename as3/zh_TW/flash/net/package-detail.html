類別 x


flash.net 套件包含可用來從網路傳送和接收的類別。


函數
 函數說明
 getClassByAlias 查閱之前已經透過呼叫 registerClassAlias() 方法註冊別名的類別。
 navigateToURL 會開啟或取代應用程式中的視窗，而該應用程式包含 Flash Player 容器 (通常是指瀏覽器)。
 registerClassAlias 當物件以 Action Message Format (AMF) 編碼時，會保留該物件的類別 (類型)。
 sendToURL 會將 URL 要求傳送至伺服器，但忽略任何回應。
類別
 類別說明
 DatagramSocket     DatagramSocket 類別可以讓程式碼傳送及接收通用資料包通訊協定 (UDP) 封包。
 FileFilter FileFilter 類別是用來指出使用者系統中的哪些檔案會在檔案瀏覽對話方塊顯示，該對話方塊是在呼叫 FileReference.browse() 方法或 FileReferenceList.browse() 方法時顯示，或是在呼叫 File、FileReference 或 FileReferenceList 物件的 browse 方法時顯示。
 FileReference FileReference 類別提供一個可以在使用者電腦與伺服器之間上傳和下載檔案的工具。
 FileReferenceList FileReferenceList 類別會提供一個工具，讓使用者選取一或多個上傳的檔案。
 GroupSpecifier GroupSpecifier 類別用於建構不透明 groupspec 字串，以傳送到 NetStream 及 NetGroup 建構函式。
 InterfaceAddress     InterfaceAddress 類別會報告網路介面位址的屬性。
 IPVersion     IPVersion 類別定義代表不同 IP 位址家族的常數。
 LocalConnection LocalConnection 類別可讓您建立一個 LocalConnection 物件，以在另一個 LocalConnection 物件中叫用方法。
 NetConnection NetConnection 類別會在用戶端與伺服器之間建立雙向連線。
 NetGroup NetGroup 類別的實體代表 RTMFP 群組的成員資格。
 NetGroupInfo NetGroupInfo 類別指定在 RTMFP 對等式資料傳輸下，相關 NetGroup 物件的各種不同服務品質 (QOS) 統計資料。
 NetGroupReceiveMode NetGroupReceiveMode 類別是用於 NetGroup 類別 receiveMode 屬性的常數值列舉。
 NetGroupReplicationStrategy NetGroupReplicationStrategy 類別是用於設定 NetGroup 類別之 replicationStrategy 屬性的常數值列舉。
 NetGroupSendMode NetGroupSendMode 類別是用於 NetGroup.sendToNeighbor() 方法 sendMode 參數的常數值列舉。
 NetGroupSendResult NetGroupSendResult 類別是用於傳回與 NetGroup 實體關聯之 Directed Routing 方法值的常數值列舉。
 NetMonitor NetMonitor 類別會維護一份 NetStream 物件清單。
 NetStream NetStream 類別會在 NetConnection 上開啟單向串流通道。
 NetStreamAppendBytesAction NetStreamAppendBytesAction 類別是您可以傳遞到 NetStream.appendBytesAction() 方法的常數列舉。
 NetStreamInfo NetStreamInfo 類別會指定與 NetStream 物件以及音效、視訊和資料的基礎串流緩衝區相關的各種服務品質 (QOS) 統計資料和其他資訊。
 NetStreamMulticastInfo NetStreamMulticastInfo 類別指定在 RTMFP 對等式及 IP 多點傳送串流傳輸下，相關 NetStream 物件的各種不同服務品質 (QOS) 統計資料。
 NetStreamPlayOptions NetStreamPlayOptions 類別會指定可傳遞給 NetStream.play2() 方法的各種選項。
 NetStreamPlayTransitions NetStreamPlayTransitions 類別會指定可搭配 NetStreamPlayOptions.transition 屬性使用的有效字串。
 NetworkInfo     NetworkInfo 類別提供在電腦上關於網路介面的資訊。
 NetworkInterface     NetworkInterface 類別描述網路介面。
 ObjectEncoding ObjectEncoding 類別用於定義可序列化物件 (如 FileStream、NetStream、NetConnection、SharedObject 和 ByteArray) 之類別中的序列化設定，以使用舊版 ActionScript。
 Responder Responder 類別會提供用於 NetConnection.call() 中的物件，以處理伺服器的傳回值 (與特定作業成功或失敗相關)。
 SecureSocket SecureSocket 類別讓程式碼可以使用通訊端連線，以運用安全通訊端階層 (SSL) 及傳輸層安全性 (TLS) 通訊協定。
 ServerSocket     ServerSocket 類別允許程式碼做為傳輸控制通訊協定 (TCP) 連線的伺服器。
 SharedObject 您可以使用 SharedObject 類別，在使用者電腦或伺服器上讀取並儲存有限的資料量。
 SharedObjectFlushStatus SharedObjectFlushStatus 類別會提供呼叫 SharedObject.flush() 方法所傳回之程式碼的值。
 Socket Socket 類別讓程式碼可以建立傳輸控制通訊協定 (TCP) 通訊端連線以傳送及接收二進位資料。
 URLLoader URLLoader 類別能以文字、二進位資料或 URL 編碼之變數的形式，從 URL 下載資料。
 URLLoaderDataFormat URLLoaderDataFormat 類別會提供一些值，以指定接收下載資料的方式。
 URLRequest URLRequest 類別會在單一 HTTP 要求中捕捉所有資訊。
 URLRequestDefaults     URLRequestDefaults 類別包含靜態屬性，可用來定義 URLRequest 類別之屬性的預設值。
 URLRequestHeader URLRequestHeader 物件會封裝單一 HTTP 要求檔頭，而且包含名稱/值的配對。
 URLRequestMethod URLRequestMethod 類別會提供一些值，可指定在傳送資料至伺服器時，URLRequest 物件應該使用 POST 方法或 GET 方法。
 URLStream URLStream 類別會提供對於下載 URL 的低階存取。
 URLVariables URLVariables 類別可讓您在應用程式和伺服器之間傳輸變數。
 XMLSocket XMLSocket 類別會實作用戶端通訊端，讓 Flash Player 或 AIR 應用程式能與以 IP 位址或網域名稱做為識別的伺服器電腦進行通訊。
