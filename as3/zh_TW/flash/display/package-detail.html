類別 x


flash.display 套件包含 Flash Player 用來建立視覺顯示的核心類別。


介面
 介面說明
 IBitmapDrawable IBitmapDrawable 介面是由物件來實作，這些物件都可以當做 BitmapData 類別之 draw() 方法的 source 參數來傳遞。
 IGraphicsData 這個介面用來定義可用做 flash.display.Graphics 方法中之參數的物件，包括填色、筆畫和路徑。
 IGraphicsFill 這個介面用來定義可用做 flash.display.Graphics 方法和繪圖類別中之填色參數的物件。
 IGraphicsPath 這個介面用來定義可用做 flash.display.Graphics 方法和繪圖類別中之路徑參數的物件。
 IGraphicsStroke 這個介面用來定義可用做 flash.display.Graphics 方法和繪圖類別中之筆畫參數的物件。
類別
 類別說明
 ActionScriptVersion ActionScriptVersion 類別是常數值的列舉項目，可用來指出載入之 SWF 檔案的語言版本。
 AVLoader 
 AVM1Movie AVM1Movie 是一種代表 AVM1 影片片段 (採用 ActionScript 1.0 或 2.0 格式) 的簡單類別。
 Bitmap Bitmap 類別代表了用來代表點陣圖影像的顯示物件。
 BitmapData BitmapData 類別可讓您使用 Bitmap 物件點陣圖影像的資料 (像素)。
 BitmapDataChannel BitmapDataChannel 類別是常數值的列舉項目，可用來指出要使用的色版：紅色、藍色、綠色或 Alpha 透明度。
 BitmapEncodingColorSpace BitmapEncodingColorSpace 類別會定義常數，這些常數會在您將壓縮程式指定為 flash.display.JPEGXREncoderOptions 時，指定 flash.display.BitmapData.encode() 方法取樣顏色色版的方式。
 BlendMode 可提供視覺混合模式特效所需之常數值的類別。
 CapsStyle CapsStyle 類別是常數值的列舉項目，可用來指定繪製線條時所使用的端點樣式。
 ColorCorrection ColorCorrection 類別可提供 flash.display.Stage.colorCorrection 屬性的值。
 ColorCorrectionSupport ColorCorrectionSupport 類別可提供 flash.display.Stage.colorCorrectionSupport 屬性的值。
 DisplayObject DisplayObject 類別是所有可以放置在顯示清單上的物件的基底類別。
 DisplayObjectContainer DisplayObjectContainer 類別是顯示清單上可以當作顯示物件容器的所有物件的基底類別。
 FocusDirection     FocusDirection 類別會列舉供 direction 參數 (屬於 Stage 物件的 assignFocus() 方法) 與 FocusEvent 物件之 direction 屬性使用的數值。
 FrameLabel FrameLabel 物件包含可指定影格編號與對應標籤名稱的屬性。
 GradientType GradientType 類別會為位於 flash.display.Graphics 類別的 beginGradientFill() 和 lineGradientStyle() 方法中的 type 參數提供值。
 Graphics Graphics 類別包含一組方法，可讓您用來建立向量形狀。
 GraphicsBitmapFill 定義點陣圖填色。
 GraphicsEndFill 指出圖像填色的結尾。
 GraphicsGradientFill 定義漸層填色。
 GraphicsPath 繪圖命令的集合以及這些命令的座標參數。
 GraphicsPathCommand 定義可用來指定繪製路徑之命令的值。
 GraphicsPathWinding GraphicsPathWinding 類別會提供 flash.display.GraphicsPath.winding 屬性和 flash.display.Graphics.drawPath() 方法的值，以決定繪製路徑的方向。
 GraphicsShaderFill 定義著色器填色。
 GraphicsSolidFill 定義基本色填色。
 GraphicsStroke 定義線段樣式或筆畫。
 GraphicsTrianglePath 定義一組依順序排列，並可使用 (u,v) 填色座標或是正常填色來顯示的三角形。
 InteractiveObject InteractiveObject 類別是使用者可以使用滑鼠、鍵盤或其他使用者輸入裝置，與其進行互動之所有顯示物件的抽象基底類別。
 InterpolationMethod InterpolationMethod 類別可提供 Graphics.beginGradientFill() 和 Graphics.lineGradientStyle() 方法中之 interpolationMethod 參數的值。
 JointStyle JointStyle 類別是常數值的列舉項目，可用來指定繪製線條時所用的接合樣式。
 JPEGEncoderOptions JPEGEncoderOptions 類別會為 flash.display.BitmapData.encode() 方法定義壓縮演算法。
 JPEGXREncoderOptions JPEGXREncoderOptions 類別會為 flash.display.BitmapData.encode() 方法定義壓縮演算法。
 LineScaleMode LineScaleMode 類別可提供 Graphics.lineStyle() 方法中之 scaleMode 參數的值。
 Loader Loader 類別可用來載入 SWF 檔案或影像檔 (JPG、PNG 或 GIF)。
 LoaderInfo LoaderInfo 類別會提供有關載入的 SWF 檔案或影像檔 (JPEG、GIF 或 PNG) 的資訊。
 MorphShape MorphShape 類別代表顯示清單上的 MorphShape 物件。
 MovieClip MovieClip 類別會繼承下列類別：Sprite、DisplayObjectContainer、InteractiveObject、DisplayObject 與 EventDispatcher。
 NativeMenu     NativeMenu 類別包含定義原生選單所需的方法與屬性。
 NativeMenuItem     NativeMenuItem 類別代表選單中的單一項目。
 NativeWindow     NativeWindow 類別可提供用來建立與控制原生桌面視窗的介面。
 NativeWindowDisplayState     NativeWindowDisplayState 類別會針對視窗顯示狀態的名稱定義常數。
 NativeWindowInitOptions     NativeWindowInitOptions 類別會定義用來建構新 NativeWindow 實體的初始化選項。
 NativeWindowRenderMode     NativeWindowRenderMode 類別會針對用來建立原生視窗之 NativeWindowInitOptions 物件的 renderMode 屬性定義常數。
 NativeWindowResize     NativeWindowResize 類別會針對 NativeWindow startResize() 方法之 edgeOrCorner 參數可能的值定義其常數。
 NativeWindowSystemChrome     NativeWindowSystemChrome 類別會針對用來建立原生視窗之 NativeWindowInitOptions 物件的 systemChrome 屬性定義常數。
 NativeWindowType     NativeWindowType 類別會針對用來建立原生視窗之 NativeWindowInitOptions 物件的 type 屬性定義常數。
 PixelSnapping PixelSnapping 類別是常數值的列舉項目，可使用 Bitmap 物件的 pixelSnapping 屬性來設定像素貼齊選項。
 PNGEncoderOptions PNGEncoderOptions 類別會為 flash.display.BitmapData.encode() 方法定義壓縮演算法。
 Scene Scene 類別包含用來辨識場景中影格名稱、標籤與數目的屬性。
 Screen     Screen 類別會提供此應用程式可用之顯示螢幕的相關資訊。
 ScreenMode     ScreenMode 物件提供有關螢幕寬度、高度和重新整理速率的資訊。
 Shader Shader 實體代表了 ActionScript 中的 Pixel Bender 著色器核心。
 ShaderData ShaderData 物件包含可代表任何參數以及著色器核心輸入的屬性，以及內含任何中繼資料 (針對著色器所指定) 的屬性。
 ShaderInput ShaderInput 實體代表著色器核心的單一輸入影像。
 ShaderJob ShaderJob 實體可用來在獨立模式下執行著色器作業。
 ShaderParameter ShaderParameter 實體代表著色器核心的單一輸入參數。
 ShaderParameterType 此類別會針對 ShaderParameter 類別的 type 屬性定義代表其可能數值的常數。
 ShaderPrecision 此類別會針對 Shader 類別的 precisionHint 屬性定義代表其可能值的常數。
 Shape 此類別會使用 ActionScript 繪製應用程式設計介面 (API) 來建立小型形狀。
 SimpleButton SimpleButton 類別可讓您控制 SWF 檔案中所有的按鈕元件實體。
 SpreadMethod SpreadMethod 類別可提供 Graphics 類別之 beginGradientFill() 與 lineGradientStyle() 方法中參數的值。
 Sprite Sprite 類別是基本的顯示清單建構區塊：可以顯示圖像，也可以包含子系的顯示清單節點。
 Stage Stage 類別代表主要繪圖區。
 Stage3D Stage3D 類別提供一個顯示區域和可程式化的顯示內容，以供繪製 2D 和 3D 圖形。
 StageAlign StageAlign 類別可提供要用於 Stage.align 屬性的常數值。
 StageAspectRatio     StageAspectRatio 類別提供 Stage.setAspectRatio() 方法的值。
 StageDisplayState StageDisplayState 類別可提供 Stage.displayState 屬性的值。
 StageOrientation     StageOrientation 類別會定義常數，以列舉舞台和裝置的可能方向。
 StageQuality StageQuality 類別可提供 Stage.quality 屬性值，並為 BitmapData.drawWithQuality() 方法的 quality 參數提供值。
 StageScaleMode StageScaleMode 類別可提供 Stage.scaleMode 屬性的值。
 SWFVersion SWFVersion 類別是常數值的列舉項目，可用來指出載入的 SWF 檔案之檔案格式版本。
 TriangleCulling 定義剔除演算法的程式碼，此程式碼可在繪製三角形路徑時決定不要顯示哪個三角形。
