套件flash.data
類別public class SQLTriggerSchema
繼承SQLTriggerSchema SQLSchema Object

語言版本: ActionScript 3.0
執行階段版本: AIR 1.0

SQLTriggerSchema 實體會提供用來描述資料庫中特定觸發器的資訊。它包含觸發器名稱 (name 屬性)、相關聯表格的名稱 (table 屬性)，以及用來建立觸發器的 SQL 陳述式 (sql 屬性)。

若要取得資料庫的觸發器結構資訊，請使用 SQLConnection.loadSchema() 方法載入結構資訊 (請務必使用 null 或 SQLTriggerSchema 做為 type 引數的值)。在產生的 SQLSchemaResult 實體中，triggers 屬性包含 SQLTriggerSchema 實體的陣列，這些實體表示資料庫中的觸發器。

一般而言，開發人員程式碼不會直接建構 SQLTriggerSchema 實體。

語言版本: ActionScript 3.0

建立 SQLTriggerSchema 實體。一般而言，開發人員程式碼不會直接呼叫 SQLTriggerSchema 建構函式。若要取得資料庫的結構資訊，請呼叫 SQLConnection.loadSchema() 方法。

參數
database:String — 相關聯的資料庫名稱。
 
name:String — 觸發器名稱。
 
sql:String — 用來建立觸發器的 SQL。
 
table:String — 與該觸發器相關聯的表格名稱。
