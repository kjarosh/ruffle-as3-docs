套件flash.events
類別public class DRMStatusEvent
繼承DRMStatusEvent Event Object

語言版本: ActionScript 3.0
執行階段版本: AIR 1.0, Flash Player 10.1

當使用數位版權管理 (DRM) 保護的內容成功開始播放 (當憑證已經過確認，而且使用者已通過驗證並經過授權得以檢視內容) 時，NetStream 物件就會傳送 DRMStatusEvent 物件。DRMStatusEvent 物件包含憑證相關資訊，例如是否可以離線檢視內容，或者憑證的有效期限，過期之後便無法再繼續檢視內容。應用程式可以使用此資料，告知使用者其原則和權限的狀態。

相關 API 元素

isAvailableOffline

屬性 
isAvailableOffline:Boolean  [唯讀]

語言版本: ActionScript 3.0
執行階段版本: AIR 1.0

指出受保護 (利用數位版權管理 (DRM) 加密) 的內容是否可供離線檢視。如果是的話，值為 true。否則，值為 false。

如果要讓數位保護的內容可供離線檢視，其憑證必須快取至使用者的本機電腦 (應用程式會決定要在本機何處儲存內容，才能讓它可供離線檢視)。

DRMStatusEvent

()建構函式
public function DRMStatusEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, inMetadata:DRMContentData = null, inVoucher:DRMVoucher = null, inLocal:Boolean = false)

語言版本: ActionScript 3.0
執行階段版本: AIR 1.0, Flash Player 10.1

建立包含 DRM 狀態事件特定資訊的 Event 物件。Event 物件會當做參數傳遞至事件偵聽程式。

參數
type:String (default = NaN) — 事件類型。事件偵聽程式可以透過繼承的 type 屬性來存取此資訊。DRMAuthenticate 事件只有一種類型：DRMAuthenticateEvent.DRM_AUTHENTICATE。
 
bubbles:Boolean (default = false) — 判斷 Event 物件是否參與事件流程的反昇階段。事件偵聽程式可以透過繼承的 bubbles 屬性來存取此資訊。
 
cancelable:Boolean (default = false) — 判斷是否可以取消 Event 物件。事件偵聽程式可以透過繼承的 cancelable 屬性來存取此資訊。
 
inMetadata:DRMContentData (default = null) — 包含自訂 DRM 屬性的自訂物件。
 
inVoucher:DRMVoucher (default = null) — Event 的內容。
 
inLocal:Boolean (default = false) — 指出是否可以離線檢視內容。
