content

Eigenschaft 
content:*  [schreibgeschützt]

Steuert den Inhalt der Markierung.

Zulässige Werte für diesen String sind:

  • none - Keine Markierung.
  • counter(ordered) - Markierung anzeigen.
  • counter(ordered,ListStyleType) - Markierung anzeigen, aber listStyleType in angegebenen Wert ändern.
  • counters(ordered) - Beginnend mit dem übergeordneten ListElement auf der höchsten Ebene wird eine Folge von Werten für die geordnete Zählung erstellt, wobei die angegebenen listStyleType-Werte für die einzelnen Zähler verwendet und durch die jeweiligen Suffixe getrennt werden. Dies wird für Nummerierungen in Inhaltsübersichten verwendet, z. B. I.1., I.2. usw.
  • counters(ordered,"<string>") - Ähnlich wie der vorherige Wert, wobei jedoch anstelle des Suffixes für jeden einzelnen Zähler ein <string> steht.
  • counters(ordered,"<string>",ListStyleType) - Ähnlich wie der vorherige Wert, wobei jedoch der listStyleType der einzelnen Zähler durch den angegebenen Wert ersetzt wird.

Wird kein Wert angegeben, wird für diese Eigenschaft der Standardwert "counter(ordered)" verwendet.

counterIncrement

Eigenschaft 
counterIncrement:*  [schreibgeschützt]

Steuert das Erhöhen des Zählerwerts.

Zulässige Werte für diesen String sind:

  • none - Keine Erhöhung.
  • ordered - Zählerwert wird jeweils um eins erhöht.
  • ordered integer - Zählerwert wird jeweils um integer erhöht.

Wird kein Wert angegeben, wird für diese Eigenschaft der Standardwert "ordered 1" verwendet.

Hinweis: Die counterIncrement-Eigenschaft wird vor der counterReset-Eigenschaft angewendet.

counterReset

Eigenschaft 
counterReset:*  [schreibgeschützt]

Steuert das Zurücksetzen des Zählerwerts.

Zulässige Werte für diese Eigenschaft sind:

  • none - Keine Zurücksetzung.
  • ordered - Zählerwert wird auf null zurückgesetzt.
  • ordered integer - Zählerwert wird auf integer zurückgesetzt.

Bei undefined wird für diese Eigenschaft der Standardwert „none“ verwendet.

Hinweis: Die counterReset-Eigenschaft wird vor der counterIncrement-Eigenschaft angewendet.
