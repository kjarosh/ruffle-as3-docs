VererbungMultiTriggerValidator Validator EventDispatcher Object
Unterklassen EndDateOrderValidator, StartDateOrderValidator

Sprachversion: ActionScript 3.0
Produktversion: Baustein „Review, Commenting, and Approval“ 10.0
Laufzeitversionen: Flash Player 10.2, AIR (unsupported)

An extension of Flex Validator class that triggers the validation on more than one configurable event names. The default Flex Validator class allows you to configure the triggerEvent. User can specify only one event name. This class allows you to provide an array of event names each of which triggers the validation.

Nicht alle Teile des ActionScript 3.0-Referenzhandbuchs wurden in alle Sprachen übersetzt. Wenn der Text zu einem Sprachelement nicht übersetzt wurde, wird er auf Englisch angezeigt. Zum Beispiel wurden die Informationen zur ga.controls.HelpBox-Klasse nicht in andere Sprachen übersetzt.
