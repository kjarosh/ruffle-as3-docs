Klassepublic class AbstractEntityMetadata
VererbungAbstractEntityMetadata EventDispatcher Object
Implementiert IModelType, IModelInstance

Sprachversion: ActionScript 3.0
Produktversion: Adobe Digital Enterprise Platform Data Services für Java EE 3
Laufzeitversionen: Flash Player 9, AIR 1.1

Diese Klasse definiert standardmäßige Fehler ausgebende Implementierungen aller Funktionen, die in IModelType und IModelInstance-Schnittstellen deklariert werden. Zusätzlich stellen alle generierten Erweiterungen dieser Klasse die folgenden generierten APIs für einzelne Entitätseigenschaften bereit. 
      function get property-nameStyle():IStyle

Bindbare Get-Funktionen für die com.adobe.fiber.styles.IStyle-Instanz dieser Eigenschaft. Gibt für Eigenschaften, die keinen verknüpften Stil besitzen, „null“ zurück.

      function get isproperty-nameAvailable():Boolean

Bindbarer Get-Funktionen für den Verfügbarkeitsstatus der Eigenschaft im Hinblick auf Varianten. Gibt für Eigenschaften, die sich nicht innerhalb einer Variante befinden, immer „true“ zurück.

      function invalidateDependentproperty-name():void

Funktionen für diese Eigenschaft, die eine erneute Berechnung für alle von dieser Eigenschaft abhängigen abgeleiteten Eigenschaften auslösen. PropertyChanged-Ereignisse werden ausgelöst, wenn die Berechnung jeder einzelnen abgeleiteten Eigenschaft abgeschlossen ist. Beachten Sie, dass diese Funktionen nur für Dateneigenschaften generiert werden, die abhängige abgeleitete Eigenschaften besitzen.

isAvailable

()Methode 
public function isAvailable(propertyName:String):Boolean

Sprachversion: ActionScript 3.0
Produktversion: Adobe Digital Enterprise Platform Data Services für Java EE 3
Laufzeitversionen: Flash Player 9, AIR 1.1

Ermittelt, ob eine Eigenschaft verfügbar ist wie durch Varianten bestimmt. Es wird ein Fehler ausgegeben, wenn der übergebene Eigenschaftsname keine gültige Eigenschaft des value-Objekts ist. Um zum Verfügbarkeits-Flag für eine bestimmte Eigenschaft Bindungen hinzuzufügen, verwenden Sie die ispropertyNameAvailable()-Funktion. Diese Funktion existiert für jede Eigenschaft und unterstützt Bindungen.

Parameter

propertyName:String — Eigenschaft, die auf Verfügbarkeit geprüft werden soll

Rückgabewerte
Boolean — „true“, wenn diese Eigenschaft aktuell verfügbar ist, wie von Varianten bestimmt
