dateAdd

()Methode 
public static function dateAdd(part:String, units:int, date:Date):Date

Sprachversion: ActionScript 3.0
Produktversion: Adobe Digital Enterprise Platform Data Services für Java EE 3
Laufzeitversionen: Flash Player 9, AIR 1.1

Fügt einem Datum Zeiteinheiten hinzu. Einheiten sind:

  • yyyy: Jahr
  • q: Quartal
  • m: Monat
  • y: Tag des Jahres
  • d: Tag
  • w: Wochentag
  • ww: Woche
  • h: Stunde
  • n: Minute
  • s: Sekunde
  • l: Millisekunde

Parameter

part:String — Typ der hinzuzufügenden Einheiten
 
units:int — Anzahl der Einheiten
 
date:Date — Datum

Rückgabewerte
Date — Neues Datum; null, wenn das Datum null ist

dateCompare

()Methode 
public static function dateCompare(date1:Date, date2:Date):int

Sprachversion: ActionScript 3.0
Produktversion: Adobe Digital Enterprise Platform Data Services für Java EE 3
Laufzeitversionen: Flash Player 9, AIR 1.1

Benötigt zwei Datumsangaben und gibt -1, 0 oder 1 zurück, je nachdem, ob der Wert kleiner, gleich oder größer als der zweite ist.

Parameter

date1:Date — Erstes Datum für den Vergleich
 
date2:Date — Zweites Datum für den Vergleich

Rückgabewerte
int — Ganzzahl; -1, wenn Datum1 < Datum2, 0, wenn Datum1 = Datum2, 1, wenn Datum1 > Datum2. Gibt -0 zurück, wenn eine der beiden Datumsangaben den Wert „null“ hat;
