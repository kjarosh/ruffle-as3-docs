Produktversion: Flash CS3
Laufzeitversionen: Flash Player 9.0.28.0, AIR 1.0

Die ColorPickerEvent-Klasse definiert Ereignisse, die mit der ColorPicker-Komponente verknüpft sind. Zu diesen Ereignissen gehören:
  • ColorPickerEvent.CHANGE: Wird ausgelöst, wenn ein Benutzer auf eine andere Farbe in der ColorPicker-Komponente klickt.
  • ColorPickerEvent.ENTER: Wird ausgelöst, wenn die Eingabetaste gedrückt wird, nachdem ein Wert in die ColorPicker-Komponente eingegeben wurde.
  • ColorPickerEvent.ITEM_ROLL_OUT: Wird ausgelöst, wenn das Zeigegerät von einer Farbzelle in der ColorPicker-Komponente abgezogen wird.
  • ColorPickerEvent.ITEM_ROLL_OVER: Wird ausgelöst, wenn das Zeigegerät über eine Farbzelle in der ColorPicker-Komponente bewegt wird.

EigenschaftWert

cancelabletrue
colorDer aktuelle Farbwert der ColorPicker-Komponente.
currentTargetDas Objekt, welches das Ereignisobjekt aktiv mit einem Ereignis-Listener verarbeitet.
targetDas Objekt, von dem das Ereignis ausgelöst wurde. Das Ziel ist nicht immer das Objekt, das auf das Ereignis wartet. Verwenden Sie die Eigenschaft currentTarget, um auf das Objekt zuzugreifen, das auf das Ereignis wartet.


Beispiel  ( Verwendung dieses Beispiels )

Das folgende Beispiel zeigt den Hexadezimalwert der ausgewählten Farbe bei einer Farbänderung: 

import fl.controls.ColorPicker;
import fl.events.ColorPickerEvent;

var myColorPicker:ColorPicker = new ColorPicker();
myColorPicker.addEventListener(ColorPickerEvent.CHANGE, changeHandler);
myColorPicker.move(10, 10);
addChild(myColorPicker);

function changeHandler(event:ColorPickerEvent):void {
    trace("color changed:", event.color, "(#" + event.target.hexValue + ")");
}
