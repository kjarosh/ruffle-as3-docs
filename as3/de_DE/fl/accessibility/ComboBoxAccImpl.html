Mithilfe der ComboBoxAccImpl-Klasse, auch ComboBox Accessibility Implementation-Klasse genannt, wird eine ComboBox-Komponente zugänglich gemacht.

Die ComboBoxAccImpl-Klasse unterstützt Systemrollen, objektbasierte Ereignisse und Zustände.

Eine ComboBox-Komponente meldet die Rolle ROLE_SYSTEM_COMBOBOX (0x2E) einem Bildschirmleseprogramm. Elemente einer ComboBox-Komponente melden die Rolle ROLE_SYSTEM_LISTITEM (0x22).

Eine ComboBox-Komponente meldet die folgenden Zustände einem Bildschirmleseprogramm:

  • STATE_SYSTEM_NORMAL (0x00000000)
  • STATE_SYSTEM_UNAVAILABLE (0x00000001)
  • STATE_SYSTEM_FOCUSED (0x00000004)
  • STATE_SYSTEM_PRESSED (0x00000008)
  • STATE_SYSTEM_CHECKED (0x00000010)
  • STATE_SYSTEM_FOCUSABLE (0x00100000)

Darüber hinaus melden Elemente einer ComboBox-Komponente folgende Zustände:

  • STATE_SYSTEM_SELECTED (0x00000002)
  • STATE_SYSTEM_SELECTABLE (0x00200000)

Eine ComboBox-Komponente sendet die folgenden Ereignisse einem Bildschirmleseprogramm:

  • EVENT_OBJECT_SELECTION (0x8006)
  • EVENT_OBJECT_STATECHANGE (0x800A)
  • EVENT_OBJECT_NAMECHANGE (0x800C)
  • EVENT_OBJECT_VALUECHANGE (0x800E)

public static function enableAccessibility():void

Sprachversion: ActionScript 3.0
Produktversion: Flash CS3

Aktiviert Eingabehilfen für eine ComboBox-Komponente. Diese Methode ist erforderlich, damit der Compiler die Accessibility-Klassen für eine Komponente aktivieren kann.
