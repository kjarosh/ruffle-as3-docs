Klassen x


Das mx.charts-Paket enthält die Flex-Charting-Klassen.


Klassen
 KlasseBeschreibung
 AreaChart Das AreaChart-Steuerelement stellt die Daten als Fläche dar, die durch eine Linie begrenzt wird, welche die Datenwerte verbindet.
 AxisLabel Ein AxisLabel-Objekt stellt eine einzelne Beschriftung auf der Diagrammachse dar.
 AxisRenderer Sie verwenden die AxisRenderer-Klasse, um die horizontale und vertikale Achse eines Diagramms zu beschreiben.
 BarChart Das BarChart-Steuerelement stellt die Daten als Serie horizontaler Balken dar, deren Länge durch Datenwerte bestimmt wird.
 BubbleChart Das BubbleChart-Steuerelement stellt Daten mit drei Werten für jeden Datenpunkt dar.
 CandlestickChart Das CandlestickChart-Steuerelement stellt Finanzdaten als Kerzenserie dar, die Höchst-, Tiefst-, Anfangs- und Schlusswerte einer Datenreihe anzeigt.
 CategoryAxis Durch die CategoryAxis-Klasse werden die im Diagramm dargestellten Daten nach bestimmten Werten entlang einer Achse gruppiert.
 ChartItem Ein ChartItem stellt ein einzelnes Element in einer ChartSeries dar.
 ColumnChart Das ColumnChart-Steuerelement stellt die Daten als eine Serie vertikaler Säulen dar, deren Höhe durch Datenwerte bestimmt wird.
 DateTimeAxis Die DateTimeAxis-Klasse ordnet Zeitwerte gleichmäßig zwischen Mindest- und Höchstwert entlang der Diagrammachse zu.
 GridLines Die GridLines-Klasse zeichnet Rasterlinien in den Datenbereich des Diagramms.
 HitData Die HitData-Klasse stellt Informationen zu dem Datenelement an einer bestimmten Position auf dem Bildschirm dar.
 HLOCChart Das HLOCChart-Steuerelement (High Low Open Close) stellt finanzielle Daten durch Elemente dar, die Höchst-, Tiefst-, Schluss- und optional die Anfangswerte einer Serie darstellen.
 Legend Das Legendensteuerelement fügt den Diagrammen eine Legende hinzu. Diese zeigt die Beschriftung für jede Datenserie im Diagramm und einen Schlüssel mit dem Diagrammelement für die Datenserie an.
 LegendItem Steuert die einzelnen Legendenelemente in einer Diagrammlegende.
 LinearAxis Die LinearAxis-Klasse ordnet numerische Werte gleichmäßig zwischen einem Mindest- und einem Höchstwert entlang einer Diagrammachse zu.
 LineChart Das LineChart-Steuerelement stellt eine Datenserie als Punkte dar, die mit einer fortlaufenden Linie verbunden sind.
 LogAxis Die LogAxis-Klasse ordnet numerische Werte logarithmisch Werten zwischen dem Mindest- und Höchstwert auf einer Diagrammachse zu.
 PieChart Das PieChart-Steuerelement stellt eine Serie als Standardkreisdiagramm dar.
 PlotChart Das PlotChart-Steuerelement stellt die Daten mit zwei Werten für jeden Datenpunkt dar.

[ X ]Warum auf Englisch?
Inhalt des ActionScript 3.0-Referenzhandbuchs wird in englischer Sprache angezeigt

Nicht alle Teile des ActionScript 3.0-Referenzhandbuchs wurden in alle Sprachen übersetzt. Wenn der Text zu einem Sprachelement nicht übersetzt wurde, wird er auf Englisch angezeigt. Zum Beispiel wurden die Informationen zur ga.controls.HelpBox-Klasse nicht in andere Sprachen übersetzt. In der deutschen Version des Referenzhandbuchs erscheint der Abschnitt zur ga.controls.HelpBox-Klasse deshalb auf Englisch.
