kerning

Eigenschaft 
kerning:Boolean

Sprachversion: ActionScript 3.0
Produktversion: Flex 3
Laufzeitversionen: Flash Player 9, AIR 1.1

Ein boolescher Wert, der angibt, ob Kerning aktiviert (true) oder deaktiviert (false) ist. Der Begriff „Kerning“ bezeichnet die Anpassung der Pixel zwischen bestimmten Zeichenpaaren zur Verbesserung der Lesbarkeit. Diese Funktion sollte nur verwendet werden, wenn sie unbedingt erforderlich ist, beispielsweise bei Überschriften in großen Schriftarten. Kerning wird nur für eingebettete Schriftarten unterstützt. Bestimmte Schriftarten wie Verdana und Schriftarten mit fester Breite wie Courier New unterstützen Kerning nicht

Der Standardwert ist false.

Konstruktordetails

TextRange

()Konstruktor
public function TextRange(owner:UIComponent, modifiesSelection:Boolean = false, beginIndex:int = -1, endIndex:int = -1)

Sprachversion: ActionScript 3.0
Produktversion: Flex 3
Laufzeitversionen: Flash Player 9, AIR 1.1

Erstellt ein neues TextRange-Objekt, das eine Untergruppe des Inhalts eines Text-Steuerelements einschließlich Formatierungsinformationen darstellt.

Parameter
owner:UIComponent — Das Steuerelement, das den Text enthält. Das Steuerelement muss über eine textField-Eigenschaft oder, bei einem RichTextEditor-Steuerelement, über eine textArea-Eigenschaft verfügen.
 
modifiesSelection:Boolean (default = false) — Ob der Text im Texbereich ausgewählt wird. Wenn Sie diesen Parameter auf true setzen und keinen Beginn- oder Endeindex angeben, der mit Text im Steuerelement übereinstimmt, verwendet Flex den Beginn- oder Endeindex der aktuellen Textauswahl. Wenn dieser Parameter true ist, Sie die Parameter beginIndex und endIndex weglassen und keine Auswahl vorgenommen wurde, ist das TextRange-Objekt leer.
 
beginIndex:int (default = -1) — Auf null basierende Indexposition des ersten Zeichens im Bereich. Wenn der modifiesSelection-Parameter false ist und Sie diesen Parameter weglassen oder einen negativen Wert angeben, beginnt der Bereich mit dem ersten Textzeichen.
 
endIndex:int (default = -1) — Auf null basierender Index der Position nach dem letzten Zeichen im Bereich. Wenn der modifiesSelection-Parameter false ist und Sie diesen Parameter weglassen, einen negativen Wert angeben oder einen Wert nach dem Ende des Texts angeben, endet der Bereich mit dem letzten Textzeichen.
