AccessPrivileges

()Konstruktor
public function AccessPrivileges(privileges:uint = 14)

Sprachversion: ActionScript 3.0
Produktversion: Adobe Digital Enterprise Platform Data Services für Java EE 3
Laufzeitversionen: Flash Player 9, AIR 1.1

Erstellt eine Instanz mit den angegebenen Berechtigungen.

Parameter
privileges:uint (default = 14) — Bitmaske, die die Zugriffsberechtigungen darstellt. Angefangen beim Bit mit der niedrigsten Position in der Reihenfolge: Erstes Bit - nicht verwendet - sollte immer 0 sein; schreibgeschützt, wenn der Wert 0 ist. Zweites Bit - ein, wenn das Hinzufügen zulässig ist Drittes Bit - ein, wenn das Aktualisieren zulässig ist Viertes Bit - ein, wenn das Entfernen zulässig ist

Inhalt des ActionScript 3.0-Referenzhandbuchs wird in englischer Sprache angezeigt

Nicht alle Teile des ActionScript 3.0-Referenzhandbuchs wurden in alle Sprachen übersetzt. Wenn der Text zu einem Sprachelement nicht übersetzt wurde, wird er auf Englisch angezeigt. Zum Beispiel wurden die Informationen zur ga.controls.HelpBox-Klasse nicht in andere Sprachen übersetzt. In der deutschen Version des Referenzhandbuchs erscheint der Abschnitt zur ga.controls.HelpBox-Klasse deshalb auf Englisch.
