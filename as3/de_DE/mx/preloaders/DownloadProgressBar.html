Produktversion: Flex 3
Laufzeitversionen: Flash Player 9, AIR 1.1

Die DownloadProgressBar-Klasse zeigt den Downloadfortschritt an. Es wird vom Preloader-Steuerelement benutzt, um Benutzerfeedback während des Downloads und des Ladens bereitzustellen.

Die Downloadfortschrittsleiste zeigt Informationen zu zwei unterschiedlichen Phasen der Anwendung an: die Downloadphase und die Initialisierungsphase.

Verwenden Sie im Application-Container die preloader-Eigenschaft, um den Namen der Unterklasse festzulegen.

Sie können eine benutzerdefinierte Downloadfortschrittsleiste implementieren, indem Sie eine Unterklasse der DownloadProgressBar-Klasse erzeugen. Implementieren Sie keine Downloadfortschrittsleiste als MXML-Komponente, da diese zu langsam lädt.

setProgress

()Methode 
protected function setProgress(completed:Number, total:Number):void

Sprachversion: ActionScript 3.0
Produktversion: Flex 3
Laufzeitversionen: Flash Player 9, AIR 1.1

Aktualisiert die Anzeige der Downloadfortschrittsleiste mit den aktuellen Downloadinformationen. Eine typische Implementierung teilt den geladenen Wert durch den Gesamtwert und zeigt eine Prozentangabe an. Wenn Sie diese Methode nicht implementieren, sollten Sie eine Fortschrittsleiste mit einer Animation erstellen, die dem Benutzer anzeigt, dass ein Download stattfindet.

Die setProgress()-Methode wird nur aufgerufen, wenn die Anwendung von einem Remoteserver heruntergeladen wird und die Anwendung nicht im Browsercache vorliegt.

Parameter

completed:Number — Anzahl der Byte der SWF-Anwendungsdatei, die heruntergeladen wurden.
 
total:Number — Größe der SWF-Anwendungsdatei in Byte.

showDisplayForDownloading

()Methode 
protected function showDisplayForDownloading(elapsedTime:int, event:ProgressEvent):Boolean

Sprachversion: ActionScript 3.0
Produktversion: Flex 3
Laufzeitversionen: Flash Player 9, AIR 1.1

Definiert den Algorithmus zur Bestimmung, ob die Downloadfortschrittsleiste während der Downloadphase angezeigt werden soll.

Parameter

elapsedTime:int — Anzahl der Millisekunden, die seit dem Start der Downloadphase vergangen sind.
 
event:ProgressEvent — Das ProgressEvent-Objekt, welches die bytesLoaded- und die bytesTotal-Eigenschaft enthält.

Rückgabewerte
Boolean — Wenn der zurückgegebene Wert true ist, ist die Downloadfortschrittsleiste anzuzeigen. Standardmäßig wird die Downloadfortschrittsleiste angezeigt, wenn: mehr als 700 Millisekunden vergangen sind und Flex weniger als die Hälfte der Byte einer SWF-Datei heruntergeladen hat.
