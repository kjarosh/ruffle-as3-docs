Einige Tags stehen nur in MXML zur Verfügung. Der Flex-Compiler implementiert diese Tags; deshalb sind sie in ActionScript nicht verfügbar.

TagBeschreibung
fx:BindingMit dem <fx:Binding>-Tag können Sie die Daten in einem Objekt mit denen in einem anderen Objekt verbinden.
fx:ComponentMit dem <fx:Component>-Tag definieren Sie einen Inline-Zellen-Renderer oder Zellen-Editor in einer MXML-Datei.
fx:DeclarationsMit dem <fx:Declarations>-Tag können Sie nicht standardmäßige, nicht visuelle Eigenschaften der aktuellen Klasse deklarieren.
fx:DefinitionSie verwenden das <fx:Definition>-Tag in einem <fx:Library>-Tag, um untergeordnete grafische Elemente zu definieren, die Sie anschließend in anderen Teilen der Anwendung verwenden können.
fx:DesignLayerSie verwenden das <fx:DesignLayer>-Tag nur zur internen Verwendung.
fx:LibraryMit dem <fx:Library>-Tag können Sie nicht standardmäßige, nicht visuelle Eigenschaften der aktuellen Klasse deklarieren.
fx:MetadataMit dem <fx:Metadata&>-Tag können Sie Metadaten-Tags in eine MXML-Datei einfügen.
fx:ModelMit dem <fx:Model>-Tag können Sie ein Datenmodell in MXML deklarieren.
fx:PrivateMit dem <fx:Private>-Tag können Sie Metainformationen über das MXML- oder FXG-Dokument angeben.
fx:ReparentMit dem <fx:Reparent>-Tag können Sie den übergeordneten Container einer Komponente als Teil der Änderung eines Anzeigezustands ändern.
fx:ScriptMit dem <fx:Script>-Tag können Sie Blöcke von ActionScript-Code definieren.
fx:StyleMit dem <fx:Style>-Tag können Sie Stile für das aktuelle Dokument und dessen untergeordnete Dokumente definieren.
fx:XMLSie können ein Datenmodell im XML-Format in MXML in einem <fx:mx:XML>-Tag deklarieren.
fx:XMLListSie können ein E4X XMLList-Objekt aus einem Modell im Textformat in MXML in einem <fx:XMLList>-Tag erstellen.

[ X ]Warum auf Englisch?
Inhalt des ActionScript 3.0-Referenzhandbuchs wird in englischer Sprache angezeigt

Nicht alle Teile des ActionScript 3.0-Referenzhandbuchs wurden in alle Sprachen übersetzt. Wenn der Text zu einem Sprachelement nicht übersetzt wurde, wird er auf Englisch angezeigt. Zum Beispiel wurden die Informationen zur ga.controls.HelpBox-Klasse nicht in andere Sprachen übersetzt. In der deutschen Version des Referenzhandbuchs erscheint der Abschnitt zur ga.controls.HelpBox-Klasse deshalb auf Englisch.
