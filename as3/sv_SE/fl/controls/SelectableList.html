Språkversion: ActionScript 3.0
Produktversion: Flash CS3
Körningsmiljöversioner: Flash Player 9.0.28.0, AIR 1.0

SelectableList är basklassen för alla listbaserade komponenter, t.ex. List, TileList, DataGrid och ComboBox. Den här klassen innehåller metoder och egenskaper som används för återgivning och layout av rader, och för att ställa in rullningslistformat och DataProvider.

Obs! Den här klassen skapar inte en komponent. Den visas bara så att den kan utökas.

import fl.controls.List;
import fl.data.DataProvider;

var items:XML = <items>
        <item label="Item A" />
        <item label="Item B" />
        <item label="Item C" />
    </items>;

var dp:DataProvider = new DataProvider(items);

var myList:List = new List();
myList.dataProvider = dp;
myList.allowMultipleSelection = true;
myList.move(10, 10);
myList.setSize(320, 240);
myList.addEventListener(Event.CHANGE, changeHandler);
addChild(myList);

    trace("change:");
    trace("\t" + "selectedIndex:", lst.selectedIndex);
    trace("\t" + "selectedIndices:", lst.selectedIndices);
    trace("\t" + "Number selected items:", lst.selectedIndices.length);
}
