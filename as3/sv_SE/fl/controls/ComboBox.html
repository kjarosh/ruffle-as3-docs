Språkversion: ActionScript 3.0

Körningsmiljöversioner: Flash Player 9.0.28.0, AIR 1.0

ComboBox-komponenten innehåller en listruta från vilken användaren kan välja ett värde. Dess funktion liknar den för SELECT-formulärelementet i HTML. ComboBox-komponenten kan redigeras, och i så fall kan användaren ange poster i TextInput-området i ComboBox-komponenten som inte finns med i listan.

Hämtar eller ställer in datamodellen för listobjekten som ska visas. En DataProvider kan delas med flera listbaserade komponenter. Ändringar i DataProvider är tillgängliga direkt för alla komponenter som använder den som en datakälla.

    public function set dataProvider(value:DataProvider):void

Relaterade API-element

Exempel  ( Så här använder du exemplet )
I följande exempel visas hur du ställer in DataProvider för en ComboBox. När DataProvider uppdateras, uppdateras även ComboBox: 

import fl.controls.ComboBox;
import fl.data.DataProvider;

var dp:DataProvider = new DataProvider();

var cb:ComboBox = new ComboBox();
cb.dataProvider = dp;
addChild(cb);

var counter = 0;
