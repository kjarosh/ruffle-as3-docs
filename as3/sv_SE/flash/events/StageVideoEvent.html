Klasspublic class StageVideoEvent
ArvStageVideoEvent Event Object

Språkversion: ActionScript 3.0
Körningsmiljöversioner: AIR 2.5, Flash Player 10.2

Ett StageVideo-objekt skickar ett StageVideoEvent-objekt när metoden attachNetStream() för StageVideo-objektet och metoden play() för det anslutna NetStream-objektet båda har anropats. Beroende på plattformen kan ändringar i uppspelningsstatusen även ge upphov till att händelsen skickas. Typen för StageVideoEvent är StageVideoEvent.RENDER_STATE.

()Konstruktor
public function StageVideoEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, status:String = null, colorSpace:String = null)

Språkversion: ActionScript 3.0
Körningsmiljöversioner: AIR 2.5, Flash Player 10.2

Skapar ett Event-objekt som innehåller information om StageVideo-händelser. Event-objekt skickas som parametrar till händelseavlyssnare.

Parametrar
type:String — Händelsens typ. Händelseavlyssnare kan hämta informationen genom den ärvda egenskapen type. Typen för StageVideoEvent är StageVideoEvent.RENDER_STATE.
 
bubbles:Boolean (default = false) — Bestämmer om Event-objektet deltar i bubblingssteget av händelseflödet. Händelseavlyssnare kan hämta informationen genom den ärvda egenskapen bubbles.
 
cancelable:Boolean (default = false) — Avgör om Event-objektet kan avbrytas. Händelseavlyssnare kan hämta informationen genom den ärvda egenskapen cancelable.
 
status:String (default = null) — Anger statusen för StageVideo-målobjektet.
 
colorSpace:String (default = null) — Den färgmodell som används av den video som visas.

RENDER_STATUS_UNAVAILABLE

Konstant 
public static const RENDER_STATUS_UNAVAILABLE:String = "unavailable"
Borttagen sedan Flash Player 10.2, AIR 3: Använd flash.media.VideoStatus.UNAVAILABLE

Språkversion: ActionScript 3.0
Körningsmiljöversioner: AIR 2.5

Anger att det inte går att visa videon med StageVideo-objektet.

Det här värdet är ett av de möjliga värdena för egenskapen status i StageVideoEvent-objektet.

Om plattformen till exempel saknar stöd för avkodning och visning av videons ljud- eller videokodek både med maskinvara och med programvara. I så fall har StageVideoEvent-objektet detta statusvärde.

Detta statusvärde används också om det inte finns några tillgängliga maskinvaruavkodare. Den här situationen kan inträffa i AIR for TV. För bakåtkompatibilitet med tidigare versioner kan du på enheter med AIR for TV använda ett Video-objekt för avkodning och visning med maskinvara. När du använder ett Video-objekt använder du den underliggande maskinvaruavkodaren, och därför har du ett StageVideo-objekt mindre tillgängligt. Tänk på att du bör använda ett StageVideo-objekt för videokodning och -visning med maskinvara.
