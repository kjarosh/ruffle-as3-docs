Klasspublic class ScreenMouseEvent
ArvScreenMouseEvent MouseEvent Event Object

Språkversion: ActionScript 3.0
Körningsmiljöversioner: AIR 1.0

SystemTrayIcon-objektet skickar händelser av typen ScreenMouseEvent som svar på musinteraktion.

ScreenMouseEvent-objektet ger MouseEvent-klassen ytterligare två egenskaper, screenX och screenY, som rapporterar muskoordinaterna i relation till den primära skärmen snarare än till programfönstret eller scenen.

Konstruktordetaljer
    

ScreenMouseEvent

()Konstruktor
public function ScreenMouseEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, screenX:Number = NaN, screenY:Number = NaN, ctrlKey:Boolean = false, altKey:Boolean = false, shiftKey:Boolean = false, buttonDown:Boolean = false, commandKey:Boolean = false, controlKey:Boolean = false)

Språkversion: ActionScript 3.0
Körningsmiljöversioner: AIR 1.0

Skapar ett ScreenMouseEvent-objekt som innehåller musplaceringen i skärmkoordinater.

Parametrar
type:String — Händelsens typ. Händelseavlyssnare kan hämta informationen genom den ärvda egenskapen type.
 
bubbles:Boolean (default = false) — X-positionen på klicket visas med skärmkoordinater.
 
cancelable:Boolean (default = false) — Y-positionen på klicket visas med skärmkoordinater.
 
screenX:Number (default = NaN) — Anges till false eftersom skärmmushändelser aldrig bubblar.
 
screenY:Number (default = NaN) — Anges till false eftersom det inte finns ett standardbeteende att avbryta.
 
ctrlKey:Boolean (default = false) — I Windows och Linux: anger om Ctrl-tangenten var nedtryckt när den här händelsen inträffade. I Mac: anger om Ctrl-tangenten eller Kommando-tangenten var nedtryckt.
 
altKey:Boolean (default = false) — Anges till true för att indikera att Alt-tangenten var nedtryckt när händelsen inträffade.
 
shiftKey:Boolean (default = false) — Anges till true för att indikera att Shift-tangenten var nedtryckt när händelsen inträffade.
 
buttonDown:Boolean (default = false) — Anges till true för att indikera att musknappen var nedtryckt när händelsen inträffade.
 
commandKey:Boolean (default = false) — Anger om Kommando-tangenten var nedtryckt (endast Mac).
 
controlKey:Boolean (default = false) — Anger om Ctrl-tangenten var nedtryckt.

ScreenMouseEvent.CLICK-konstanten definierar värdet på type-egenskapen i ett click-händelseobjekt.

Den här händelsen har följande relevanta egenskaper:

EgenskapVärde
buttonDowntrue om den primära musknappen trycks ned, annars false.
ctrlKeytrue i Windows eller Linux om Ctrl-tangenten är aktiv. true om antingen Ctrl-tangenten eller Kommando-tangenten är aktiv i Macintosh. Annars false.
currentTargetDet objekt som aktivt behandlar Event-objektet med en händelseavlyssnare.
shiftKeytrue om skifttangenten är aktiverad eller false om den är inaktiverad.
commandKeytrue om Kommando-tangenten är aktiv i Mac och false om den är inaktiv. Alltid false i Windows.
controlKeytrue om Ctrl-tangenten är aktiv och false om den är inaktiv.
screenXDen vågräta koordinat som händelsen inträffade vid i skärmkoordinater.
screenYDen lodräta koordinat som händelsen inträffade vid i skärmkoordinater.
