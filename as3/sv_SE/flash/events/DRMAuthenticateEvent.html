Klasspublic class DRMAuthenticateEvent
ArvDRMAuthenticateEvent Event Object

Språkversion: ActionScript 3.0
Körningsmiljöversioner: AIR 1.0

Ett NetStream-objekt skickar ett DRMAuthenticateEvent-objekt vid ett försök att spela upp krypterat DRM-innehåll (digital rights management) som kräver inloggningsuppgifter för autentisering.

DRMAuthenticateEvent-hanteraren samlar nödvändiga inloggningsuppgifter (som användarnamn, lösenord och typ) och skickar värdena till NetStream.setDRMAuthenticationCredentials()-metoden för autentisering. Varje AIR-program måste ha en funktion som erhåller inloggningsuppgifter. Programmet kan till exempel tillhandahålla en användare med ett enkelt användargränssnitt för att ange användarnamn- och lösenordsvärden, och eventuellt även typvärdet.

Om användarautentiseringen misslyckas gör programmet ett nytt autentiseringsförsök och skickar en ny DRMAuthenticateEvent-händelse för NetStream-objektet.

authenticationType

egenskap
authenticationType:String  [skrivskyddad]

Språkversion: ActionScript 3.0
Körningsmiljöversioner: AIR 1.0

Indikerar om inloggningsuppgifterna gäller för autentisering mot Flash Media Rights Management Server (FMRMS) eller en proxyserver. "Proxy"-alternativet innebär till exempel att programmet autentiserar mot en proxyserver, om ett företag kräver det, innan användaren kan börja använda Internet. Om inte anonym autentisering används efter proxy-autentiseringen, behöver användaren fortfarande autentisera mot FMRMS för att erhålla vouchern och spela upp innehållet. Du kan använda setDRMAuthenticationcredentials() en andra gång med "drm"-alternativet för att autentisera mot FMRMS.

Språkversion: ActionScript 3.0

Skapar ett Event-objekt som innehåller specifik information om DRM-autentiseringshändelser. Event-objekt skickas som parametrar till händelseavlyssnare.

Parametrar
type:String — Händelsens typ. Händelseavlyssnare kan hämta informationen genom den ärvda egenskapen type. Det finns bara ett slags DRMAuthenticate-händelse: DRMAuthenticateEvent.DRM_AUTHENTICATE.
 
bubbles:Boolean (default = false) — Bestämmer om Event-objektet deltar i bubblingssteget av händelseflödet. Händelseavlyssnare kan hämta informationen genom den ärvda egenskapen bubbles.
 
cancelable:Boolean (default = false) — Avgör om Event-objektet kan avbrytas. Händelseavlyssnare kan hämta informationen genom den ärvda egenskapen cancelable.
 
header:String (default = "") — Den krypterade innehållsfilens rubrik tillhandahålls av servern.
 
userPrompt:String (default = "") — Ett kommando för ett inloggningsanvändarnamn tillhandahålls av servern.
 
passPrompt:String (default = "") — Ett kommando för ett inloggningslösenord tillhandahålls av servern.
 
urlPrompt:String (default = "") — Ett kommando för en URL som ska visas tillhandahålls av servern.
 
authenticationType:String (default = "") — Indikerar om inloggningsuppgifterna gäller för autentisering mot Flash Media Rights Management Server (FMRMS) eller en proxyserver.
 
netstream:NetStream (default = null) — NetStream-objektet som initierade den här händelsen.

Exempel Så här använder du exemplet
DRMAuthenticateEventExample.as

package
{
    import flash.display.Sprite;
    import flash.events.AsyncErrorEvent;
    import flash.events.NetStatusEvent;
    import flash.events.DRMAuthenticateEvent;
    import flash.media.Video;
    import flash.net.NetConnection;
    import flash.net.NetStream;

    public class DRMAuthenticateEventExample extends Sprite
    {
        var videoURL:String = "Video.flv";
        var videoConnection:NetConnection;
        var videoStream:NetStream;
        var video:Video = new Video();            

        public function DRMAuthenticateEventExample()
        {
            videoConnection = new NetConnection();
            videoConnection.addEventListener(NetStatusEvent.NET_STATUS, netStatusHandler);
            videoConnection.connect(null);
        }

        private function connectStream():void {
            videoStream = new NetStream(videoConnection);
            videoStream.addEventListener(NetStatusEvent.NET_STATUS, netStatusHandler);
            videoStream.addEventListener(AsyncErrorEvent.ASYNC_ERROR, asyncErrorHandler);
            videoStream.addEventListener(DRMAuthenticateEvent.DRM_AUTHENTICATE, drmAuthenticateEventHandler);
            video.attachNetStream(videoStream);
            videoStream.play(videoURL);
            addChild(video);
        }

        private function netStatusHandler(event:NetStatusEvent):void {
            switch (event.info.code) {
                case "NetConnection.Connect.Success":
                    connectStream();
                    break;
                case "NetStream.Play.StreamNotFound":
                    trace("Unable to locate video: " + videoURL);
                    break;
            }
        }

        private function asyncErrorHandler(event:AsyncErrorEvent):void {
            // ignore AsyncErrorEvent events.
        }

         private function drmAuthenticateEventHandler(event:DRMAuthenticateEvent):void {
            videoStream.setDRMAuthenticationCredentials("User", "password", "drm");
        }
     }
}
