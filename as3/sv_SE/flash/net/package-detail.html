Paketet flash.net innehåller klasser för att skicka och ta emot över ett nätverk.


Funktioner
 FunktionBeskrivning
 getClassByAlias Letar upp en klass som tidigare hade ett alias registrerat via ett anrop till metoden registerClassAlias().
 navigateToURL Öppnar eller ersätter ett fönster i programmet som innehåller Flash Player-behållaren (vanligtvis en webbläsare).
 registerClassAlias Bevarar klassen (typen) för ett objekt när objektet är kodat i AMF-format (Action Message Format).
 sendToURL Skickar en URL-begäran till en server, men ignorerar eventuella svar.
Klasser
 KlassBeskrivning
 DatagramSocket     Med klassen DatagramSocket kan kod skicka och ta emot UDP-paket (Universal Datagram Protocol).
 FileFilter Klassen FileFilter används för att ange vilka filer i användarens system som visas i den dialogruta för filbläddring som visas när metoden FileReference.browse() eller metoden FileReferenceList.browse() anropas, eller när en bläddringsmetod för ett File-, FileReference- eller FileReferenceList-objekt anropas.
 FileReference Med klassen FileReference kan du överföra och ladda ned filer mellan en användares dator och en server.
 FileReferenceList Klassen FileReferenceList gör att användaren kan markera en eller flera filer att överföra.
 GroupSpecifier Klassen GroupSpecifier används för att skapa de ogenomskinliga groupspec-strängar som kan skickas till NetStream- och NetGroup-konstruktorerna.
 InterfaceAddress     Klassen InterfaceAddress rapporterar egenskaperna för en nätverksgränssnittadress.
 IPVersion     Klassen IPVersion definierar konstanter som representerar de olika typerna av IP-adresser.
 LocalConnection Med klassen LocalConnection kan du skapa ett LocalConnection-objekt som kan anropa en metod i ett annat LocalConnection-objekt.
 NetConnection Med klassen NetConnection skapas en dubbelriktad anslutning mellan en klient och en server.
 NetGroup Instanser av NetGroup-klassen representerar ett medlemskap i en RTMFP-grupp.
 NetGroupInfo Klassen NetGroupInfo anger diverse QoS-statistik (Quality of Service) som rör ett NetGroup-objekts underliggande RTMFP Peer-to-Peer-datatransport.
 NetGroupReceiveMode Klassen NetGroupReceiveMode är en uppräkning av konstanta värden som används för egenskapen receiveMode för klassen NetGroup.
 NetGroupReplicationStrategy Klassen NetGroupReplicationStrategy är en uppräkning av konstanta värden som används för att ställa in egenskapen replicationStrategy för klassen NetGroup.
 NetGroupSendMode Klassen NetGroupSendMode är en uppräkning av konstanta värden som används för parametern sendMode för metoden NetGroup.sendToNeighbor().
 NetGroupSendResult Klassen NetGroupSendResult är en uppräkning av konstanta värden, som används för returvärdet på de riktade routningsmetoder som är kopplade till en NetGroup-instans.
 NetMonitor Klassen NetMonitor innehåller en lista över NetStream-objekt.
 NetStream Klassen NetStream öppnar en enkelriktad direktuppspelningskanal via en NetConnection.
 NetStreamAppendBytesAction Klassen NetStreamAppendBytesAction är en uppräkning av konstanter som du kan skicka till metoden NetStream.appendBytesAction().
 NetStreamInfo NetStreamInfo-klassen anger den blandade Quality of Service-statistik (QOS) och annan information som är relaterad till ett NetStream-objekt och den underliggande strömningsbufferten för ljud, video och data.
 NetStreamMulticastInfo Klassen NetStreamMulticastInfo anger diverse QoS-statistik (Quality of Service) som rör ett NetStream-objekts underliggande RTMFP Peer-to-Peer- och IP Multicast-strömtransport.
 NetStreamPlayOptions Klassen NetStreamPlayOptions anger de olika alternativ som kan skickas till NetStream.play2-metoden.
 NetStreamPlayTransitions Klassen NetStreamPlayTransitions anger de giltiga strängar som kan användas med egenskapen NetStreamPlayOptions.transition.
 NetworkInfo     Klassen NetworkInfo innehåller information om nätverksgränssnitt på en dator.
 NetworkInterface     Klassen NetworkInterface beskriver ett nätverksgränssnitt.
 ObjectEncoding Klassen ObjectEncoding används för att definiera serialiseringsinställningar i klasser som serialiserar objekt (till exempel FileStream, NetStream, NetConnection, SharedObject och ByteArray) så att de fungerar med tidigare versioner av ActionScript.
 Responder Klassen Responder ger ett objekt som används i NetConnection.call() för att hantera returvärden från servern relaterat till om vissa åtgärder fungerar eller inte.
 SecureSocket Med klassen SecureSocket kan koden skapa socket-anslutningar med hjälp av protokollen SSL (Secure Sockets Layer) och TLS (Transport Layer Security).
 ServerSocket     Klassen ServerSocket medger att koden fungerar som en server för TCP-anslutningar (Transport Control Protocol).
 SharedObject Klassen SharedObject används för att läsa och spara begränsade mängder data på en användares dator eller på en server.
 SharedObjectFlushStatus Klassen SharedObjectFlushStatus ger värden för koden som returneras från ett anrop till metoden SharedObject.flush().
 Socket Med klassen Socket kan koden skapa TCP-socketanslutningar (Transport Control Protocol) för att skicka och ta emot binära data.
 URLLoader Klassen URLLoader laddar ned data från en URL som text, binära data eller URL-kodade variabler.
 URLLoaderDataFormat Klassen URLLoaderDataFormat innehåller värden som anger hur laddade ned data tas emot.
 URLRequest Klassen URLRequest samlar all information i en enda HTTP-begäran.
 URLRequestDefaults     Klassen URLRequestDefaults innehåller statiska egenskaper som du kan ange för att definiera standardvärden för egenskaperna i klassen URLRequest.
 URLRequestHeader Ett URLRequestHeader-objekt kapslar in en enda rubrik för HTTP-begäran och består av ett namnvärdespar.
 URLRequestMethod Klassen URLRequestMethod innehåller värden som anger om URLRequest-objektet ska använda metoden POST eller metoden GET när data skickas till en server.
 URLStream Klassen URLStream ger åtkomst till URL-nedladdning på låg nivå.
 URLVariables Med klassen URLVariables kan du överföra variabler mellan ett program och en server.
 XMLSocket Klassen XMLSocket implementerar klientsocketar som gör att Flash Player eller AIR-programmet kan kommunicera med en server som identifieras med en IP-adress eller ett domännamn.

[ X ]Varför på engelska?
Innehåll i Referenshandbok för ActionScript 3.0 i Adobe Flash-plattformen är på engelska

Det är inte alla delar av Referenshandbok för ActionScript 3.0 i Adobe Flash-plattformen som översätts till alla språk. Om ett språkelement inte översätts visas det på engelska. Klassen ga.controls.HelpBox översätts till exempel inte till något språk. I den svenska versionen av referenshandboken visas därför klassen ga.controls.HelpBox på engelska.
