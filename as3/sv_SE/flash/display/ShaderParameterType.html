Paketflash.display
Klasspublic final class ShaderParameterType
ArvShaderParameterType Object

Språkversion: ActionScript 3.0
Körningsmiljöversioner: Flash Player 10, AIR 1.5

Den här klassen definierar de konstanter som representerar möjliga värden för ShaderParameter-klassens type-egenskap. Varje konstant representerar en av de datatyper som är tillgängliga i Flash Player för parametrar i skuggningsspråket Pixel Bender.

BOOL

Konstant
public static const BOOL:String = "bool"

Språkversion: ActionScript 3.0
Körningsmiljöversioner: Flash Player 10, AIR 1.5

Visar att skuggningsparametern är definierad som ett bool-värde, vilket motsvarar en enda Boolean-instans i ActionScript.

Observera, att även om parametern bara förväntar sig ett enda värde så är ShaderParameter.value-egenskapen en array. Därför måste det enda värdet vara det enda elementet i en array som är tilldelad value-egenskapen. Exempel:

         // assumes the shader has a parameter named "param"
         // whose data type is bool
         myShader.data.param.value = [true];
