Klasspublic class CallActionInstance
ArvCallActionInstance ActionEffectInstance EffectInstance EventDispatcher Object

Språkversion: ActionScript 3.0
Produktversion: Flex 4
Körningsmiljöversioner: Flash Player 10, AIR 1.5

The CallActionInstance class implements the instance class for the CallAction effect. Flex creates an instance of this class when it plays a CallAction effect; you do not create one yourself.
