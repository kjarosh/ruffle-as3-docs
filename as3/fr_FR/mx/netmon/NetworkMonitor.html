Classepublic class NetworkMonitor
HéritageNetworkMonitor Object

Version du langage: ActionScript 3.0
Version du produit: Flex 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

NetworkMonitor est un utilitaire utilisé par les classes de service d’entreprise et de structure pour envoyer des informations à la fonctionnalité de surveillance du réseau de Flash Builder. Le NetworkMonitor déclare des fonctions Impl à spécifier par une classe mélangée fournie par Flash Builder. Si ces fonctions ne sont pas assignées, l’utilitaire ne fait rien, ou renvoie false ou null selon le cas. Pour l’instant, nous n’utilisons pas de singleton ou d’objet d’instance pour simplifier le code qui appelle le moniteur.

adjustURLRequest

()méthode 
public static function adjustURLRequest(urlRequest:URLRequest, rootURL:String, correlationID:String):void

Version du langage: ActionScript 3.0
Version du produit: Flex 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Ajustez la requête URLRequest de sorte qu’elle passe par le moniteur. L’objet URLRequest envoyé au moniteur possède deux en-têtes :

  • Actualhostport : le véritable host:port de l’URL.
  • Correlationid : correlationID de la requête au cas où elles proviennent de la messagerie (les requêtes Image/Loader doivent créer leurs propres correlationID)

Paramètres

urlRequest:URLRequest — Requête URLRequest à ajuster (les adresses URL relatives sont prises en charge).
 
rootURL:String — Calcule une URL absolue à partir de l’URL relative. Si nécessaire, transmettez l’URL du fichier SWF sous la forme de cette valeur.
 
correlationID:String — ID correlationID pouvant être utilisé pour les événements associés.

monitorEvent

()méthode 
public static function monitorEvent(event:Event, correlationID:String):void

Version du langage: ActionScript 3.0
Version du produit: Flex 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Indique au moniteur qu’un événement s’est produit. Cette méthode peut être utilisée par l’objet Loader pour surveiller la sécurité et les erreurs IO. Cette méthode ne doit pas être utilisée pour l’événement complète de l’objet Loader.

Paramètres

event:Event — Evénement sur le point de se produire (ou qui s’est produit).
 
correlationID:String — ID correlationID à associer aux autres événements ou à la demande.
