Détails de la propriété

addItemToCollectionOperation

propriété
public var addItemToCollectionOperation:String

Version du langage: ActionScript 3.0
Version du produit: Adobe Digital Enterprise Platform Data Services for Java EE 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Méthodes addItemToCollectionOperation() et removeItemFromCollectionOperation() facultatives permettant de définir une fonction à appeler lorsqu’un élément est ajouté ou supprimé dans une collection. Dans de nombreux cas, l’appartenance des éléments à la requête est déterminée implicitement (en fonction des valeurs de propriété), de sorte que le serveur n’a pas besoin de traiter ces opérations. Dans cette situation, les éléments sont ignorés. Utilisez ces méthodes lorsque vous gérez explicitement une table d’associations liant l’élément à une collection spécifique.

countOperation

propriété 
public var countOperation:String

Version du langage: ActionScript 3.0
Version du produit: Adobe Digital Enterprise Platform Data Services for Java EE 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Nom d’une opération réalisée sur le service utilisant les mêmes paramètres que cette opération et renvoyant le nombre d’éléments dans la requête. Utilisez cette propriété si vous activez la pagination, de manière à obtenir une taille de collection correcte. Si vous ne définissez pas d’opération de comptage, le système utilise un mode de dimensionnement dynamique qui récupère page par page les éléments de la collection, en laissant un emplacement vide à la fin. Si vous n’accédez pas à cet emplacement vide, la page suivante n’est pas récupérée. Lorsque vous accédez à cet emplacement, la page suivante est récupérée et la taille de la collection est augmentée, jusqu’à ce qu’il n’y ait plus d’élément dans la requête.

pagingEnabled

propriété 
public var pagingEnabled:Boolean

Version du langage: ActionScript 3.0
Version du produit: Adobe Digital Enterprise Platform Data Services for Java EE 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Définissez cette propriété sur true pour activer la pagination. Lorsque la pagination est activée, l’opération réalisée sur le serveur doit utiliser des paramètres de début et de comptage pour définir le sous-ensemble d’éléments à renvoyer. Il n’est pas nécessaire de fournir ces éléments au service lorsque vous appelez l’opération : ils sont ajoutés automatiquement au fur et à mesure que le système récupère les pages à la demande auprès du serveur, en réponse aux appels getItemAt(). Ces appels acceptent un paramètre prefetch permettant de définir le nombre d’éléments récupérés lors d’un appel.

positionalPagingParameters

propriété 
public var positionalPagingParameters:Boolean = false

Version du langage: ActionScript 3.0
Version du produit: Adobe Digital Enterprise Platform Data Services for Java EE 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Lorsque la pagination est activée, l’opération réalisée sur le serveur doit utiliser des paramètres de début et de comptage pour définir le sous-ensemble d’éléments à renvoyer. Il n’est pas nécessaire de fournir ces éléments au service lorsque vous appelez l’opération : ils sont ajoutés automatiquement au fur et à mesure que le système récupère les pages à la demande auprès du serveur, en réponse aux appels getItemAt(). Ces appels acceptent un paramètre prefetch permettant de définir le nombre d’éléments récupérés lors d’un appel. Par défaut, les paramètres de début et de décompte, qui doivent être appelés « startIndex » and « numItems », peuvent apparaître n’importe où dans la signature de votre opération. La définition de cet indicateur sur true lève la restriction de nommage sur ces deux paramètres, mais les oblige à devenir les deux derniers paramètres de l’opération. L’avant-dernier paramètre sera interprété comme le paramètre de début et le dernier comme paramètre de décompte.

removeItemFromCollectionOperation

propriété 
public var removeItemFromCollectionOperation:String

Version du langage: ActionScript 3.0
Version du produit: Adobe Digital Enterprise Platform Data Services for Java EE 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Méthodes addItemToCollectionOperation() et removeItemFromCollectionOperation() facultatives permettant de définir une fonction à appeler lorsqu’un élément est ajouté ou supprimé dans une collection. Dans de nombreux cas, l’appartenance des éléments à la requête est déterminée implicitement (en fonction des valeurs de propriété), de sorte que le serveur n’a pas besoin de traiter ces opérations. Dans cette situation, les éléments sont ignorés. Utilisez ces méthodes lorsque vous gérez explicitement une table d’associations liant l’élément à une collection spécifique.

Lorsqu’un élément est supprimé d’une collection, vous n’êtes pas obligé d’utiliser cette opération pour supprimer cet élément. En effet, vous pouvez définir la propriété deleteItemOnRemoveFromFill pour que l’opération deleteItem du serveur soit appelée automatiquement lorsque l’élément est supprimé de la collection.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
