createUpdateEvent

()méthode
public static function createUpdateEvent(obj:IManaged, property:Object, event:PropertyChangeEvent):PropertyChangeEvent

Version du langage: ActionScript 3.0
Version du produit: Adobe Digital Enterprise Platform Data Services for Java EE 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Crée un nouvel objet PropertyChangeEvent à partir d’un objet PropertyChangeEvent existant. La valeur de la propriété property du nouvel objet correspond à la valeur de l’argument property ajouté devant event.property.

Paramètres

obj:IManaged — Référence IManaged à l’objet dont la propriété souhaitée doit être extraite.
 
property:Object — Chaîne contenant le nom de la propriété souhaitée.
 
event:PropertyChangeEvent — Objet PropertyChangeEvent à copier.

Valeur renvoyée
PropertyChangeEvent — Nouvel objet PropertyChangeEvent, auquel la valeur de l’argument property est ajoutée devant event.property.

getProperty

()méthode 
public static function getProperty(obj:IManaged, property:String, value:*, useHierarchicalValues:Boolean = true):*

Version du langage: ActionScript 3.0
Version du produit: Adobe Digital Enterprise Platform Data Services for Java EE 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Cette méthode permet d’extraire la propriété d’un objet géré. Cette méthode permet non seulement de renvoyer la valeur de la propriété spécifiée, mais également de maintenir une parfaite synchronisation entre les artefacts associés, de manière à permettre la distribution des événements profondément imbriqués et les propriétés chargées à la demande.

Paramètres

obj:IManaged — Référence IManaged à l’objet dont la propriété souhaitée doit être extraite.
 
property:String — Chaîne contenant le nom de la propriété souhaitée.
 
value:* — Référence à la valeur actuelle de la propriété.
 
useHierarchicalValues:Boolean (default = true) — Si une valeur complexe (hiérarchique) est gérée.

Valeur renvoyée
* — Nouvelle valeur normalisée de la propriété. Cette valeur doit être affectée à la référence d’origine.

Eléments de l’API associés

setProperty()

Exemple  ( Comment utiliser cet exemple ) 
      public class Customer implements IManaged
      {
         private var _addresses:ArrayCollection;
         public function get addresses():ArrayCollection
         {
             _addresses = Managed.getProperty(this, "addresses", _addresses);
         }
     
         public function set addresses(value:ArrayCollection):void
         {
             Managed.setProperty(this, "addresses", _addresses, _addresses = value);
         }
      }

setProperty

()méthode 
public static function setProperty(obj:IManaged, property:Object, oldValue:*, newValue:*):void

Version du langage: ActionScript 3.0
Version du produit: Adobe Digital Enterprise Platform Data Services for Java EE 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Cette méthode permet de définir la propriété dun objet géré. Cette méthode permet non seulement de définir la valeur de la propriété spécifiée, mais également de maintenir une parfaite synchronisation entre les artefacts associés et de nettoyer les éventuelles distributions d’événements profondément imbriqués et les propriétés chargées à la demande.

Paramètres

obj:IManaged — Référence IManaged à l’objet géré dont la propriété doit être définie.
 
property:Object — Objet contenant le nom de la propriété à définir. Il s’agit en général d’une valeur String, mais il peut s’agir d’une valeur QName.
 
oldValue:* — Référence à la valeur précédente de la propriété.
 
newValue:* — Référence à la nouvelle valeur de la propriété.

Eléments de l’API associés

getProperty()

Exemple  ( Comment utiliser cet exemple ) 
      public class Customer implements IManaged
      {
         private var _addresses:ArrayCollection;
         public function get addresses():ArrayCollection
         {
             _addresses = Managed.getProperty(this, "addresses", _addresses);
         }
     
         public function set addresses(value:ArrayCollection):void
         {
             Managed.setProperty(this, "addresses", _addresses, _addresses = value);
         }
      }

toString

()méthode 
public static function toString(value:Object, namespaceURIs:Array = null, exclude:Array = null, indent:int = 0, printTypes:Boolean = false, refs:Dictionary = null):String

Version du langage: ActionScript 3.0
Version du produit: Adobe Digital Enterprise Platform Data Services for Java EE 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Renvoie une représentation de chaîne de cet objet géré.

Paramètres

value:Object — instance d’objet géré.
 
namespaceURIs:Array (default = null) — tableau de chaînes représentant tous les espaces de noms qui doivent être inclus dans la sortie.
 
exclude:Array (default = null) — tableau de chaînes représentant tous les espaces de noms qui doivent être exclus de la sortie
 
indent:int (default = 0) — nombre d’espaces utilisé pour mettre en retrait chaque ligne de la sortie.
 
printTypes:Boolean (default = false) — les noms de type sont inclus dans la sortie lorsque leur valeur est true.
 
refs:Dictionary (default = null) — dictionnaire utilisée pour éviter la lecture en boucle infinie en présence de références circulaires.

Valeur renvoyée
String — Représentation de chaîne de cet objet géré.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
