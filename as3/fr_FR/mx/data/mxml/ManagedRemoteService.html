Versions du moteur d’exécution: Flash Player 10.1, AIR 2.0

La classe ManagedRemoteService offre une fonctionnalité de niveau supérieur côté client pour le service distant géré ADEP Data Services. Il s’agit d’une sous-classe MXML de la classe mx.data.ManagedRemoteService permettant de définir des liaisons avec les propriétés d’une instance ManagedRemoteService.

Remarque : pour que vous puissiez utiliser un objet ManagedRemoteService, la propriété destination doit être définie.

public function ManagedRemoteService(dest:String = null)

Version du langage: ActionScript 3.0
Version du produit: Adobe Digital Enterprise Platform Data Services for Java EE 4.5
Versions du moteur d’exécution: Flash Player 10.1, AIR 2.0

Crée une instance de l’objet ManagedRemoteService en utilisant la destination spécifiée. La destination doit être une référence à une destination configurée dans le fichier services-config.xml.

Paramètres
dest:String (default = null) — Chaîne contenant le nom de la destination distante à laquelle ce service se connectera.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
