internalLabelFunction

propriété 
public var internalLabelFunction:Function

Version du langage: ActionScript 3.0
Version du produit: Flex 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Fonction qui est créée si le regroupement des colonnes nécessite l’extraction de données des objets imbriqués.

Par exemple, si chaque rangée de données apparaît sous cette forme : 

row = {.., .., Q1: { y2005: 241, y2006:353}};

et si vous définissez un groupe de colonne ainsi :

     <mx:AdvancedDataGridColumnGroup dataField="Q1">
        <mx:AdvancedDataGridColumn dataField="y2005">
        <mx:AdvancedDataGridColumn dataField="y2006">
     </mx:AdvancedDataGridColumnGroup>

La fonction de la colonne correspondant à y2005 est définie comme :

     function foo():String
     {
        return row["Q1"]["2005"];
     }

La fonction gère également ce cas lorsqu’une des colonnes ou l’un des groupes de colonnes utilise une fonction d’étiquette au lieu d’un champ de données.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
