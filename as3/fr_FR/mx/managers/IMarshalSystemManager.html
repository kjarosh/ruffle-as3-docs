Interfacepublic interface IMarshalSystemManager

Version du langage: ActionScript 3.0
Version du produit: Flex 4
Versions du moteur d’exécution: Flash Player 10, AIR 1.5

L’interface IMarshalSystemManager définit les méthodes et les propriétés que les classes doivent implémenter pour accéder, ajouter et supprimer des ponts à d’autres applications dans une configuration inter-versionnée.

dispatchEventFromSWFBridges

()méthode 
public function dispatchEventFromSWFBridges(event:Event, skip:IEventDispatcher = null, trackClones:Boolean = false, toOtherSystemManagers:Boolean = false):void

Version du langage: ActionScript 3.0
Version du produit: Flex 4
Versions du moteur d’exécution: Flash Player 10, AIR 1.5

Distribuez un message à toutes les applications parent et enfant du groupe de ponts SWF de ce SystemManager, qu’elles soient dans le même objet SecurityDomain ou non. Vous pouvez exclure une application avec les paramètres de cette méthode.

Paramètres

event:Event — Evénement à distribuer.
 
skip:IEventDispatcher (default = null) — Indique un objet IEventDispatcher auquel vous ne voulez pas distribuer de message. Ceci est principalement utilisé pour ignorer l’objet IEventDispatcher qui est à l’origine de l’événement.
 
trackClones:Boolean (default = false) — Indique si une référence aux événements doit être conservée lors de leur distribution.
 
toOtherSystemManagers:Boolean (default = false) — Indique si l’événement doit être distribué à d’autres objets SystemManager de niveau supérieur dans AIR.

useSWFBridge

()méthode 
public function useSWFBridge():Boolean

Version du langage: ActionScript 3.0
Version du produit: Flex 4
Versions du moteur d’exécution: Flash Player 10, AIR 1.5

Détermine si l’appelant qui utilise le gestionnaire système doit communiquer directement avec d’autres gestionnaires ou s’il doit communiquer avec un pont.

Valeur renvoyée
Boolean — La valeur est true si l’appelant qui utilise ce gestionnaire système doit communiquer à l’aide des ponts sandbox. La valeur est false si le gestionnaire système peut appeler directement d’autres gestionnaires via des références.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
