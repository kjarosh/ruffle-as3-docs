Classes x


Le package mx.effects contient des classes prenant en charge les comportements.


Interfaces
 InterfaceDescription
 IAbstractEffect L’interface IAbstractEffect permet d’indiquer qu’une propriété ou un paramètre doit être de type Effect, mais n’implémente pas réellement les API de l’interface IEffect.
 IEffect L’interface IEffect définit l’interface de base de tous les effets Flex.
 IEffectInstance L’interface IEffectInstance représente une instance d’un effet exécuté sur une cible.
 IEffectTargetHost L’interface IEffectTargetHost définit l’interface permettant d’accéder au contrôle basé sur la liste cible d’un effet data.
Classes
 ClasseDescription
 AddChildAction La classe AddChildAction définit un effet d’action qui correspond à la propriété AddChild de la définition d’un état d’affichage.
 AddItemAction La classe AddItemAction définit un effet d’action qui détermine lorsque le rendu d’élément s’affiche dans le contrôle pour élément ajouté à un contrôle à base de liste, tel que List ou TileList, ou pour un élément qui remplace un élément existant dans le contrôle.
 AnimateProperty L’effet AnimateProperty anime une propriété ou un style d’un composant.
 Blur L’effet Blur permet d’appliquer un effet visuel flou à un composant.
 CompositeEffect La classe CompositeEffect est une classe parent des classes Parallel et Sequence, qui définit les balises MXML des propriétés <mx:Parallel> ; et <mx:Sequence>.
 DefaultListEffectLa classe DefaultListEffect définit l’effet par défaut appliqué aux fonctionnalités de rendu d’élément d’un contrôle List lorsque les données du fournisseur de données changent.
 DefaultTileListEffectLa classe DefaultTileListEffect définit l’effet par défaut appliqué à la fonctionnalité de rendu d’élément d’un contrôle TileList contrôle lorsque les données du fournisseur de données changent.
 Dissolve Vous pouvez animer le composant de transparent à opaque, ou d’opaque à transparent.
 Effect La classe d’effets est une classe de base abstraite qui définit la fonctionnalité de base de tous les effets Flex.
 EffectInstance La classe EffectInstance représente l’instance d’un effet en cours de lecture sur une cible.
 EffectManager La classe EffectManager écoute les événements, tels que les événements show et move envoyés par des objets dans une application Flex.
 EffectTargetFilter La classe EffectTargetFilter définit un filtre personnalisé qui est exécuté par chaque effet de transition sur chaque cible de l’effet.
 Fade L’effet Fade anime la propriété alpha d’un composant, de transparent à opaque, ou d’opaque à transparent.
 Glow L’effet Glow permet d’appliquer un effet de rayonnement visuel à un composant.
 Iris L’effet Iris anime la cible de l’effet en développant ou en contractant un masque rectangulaire centré sur la cible.
 MaskEffect La classe MaskEffect est une classe de base abstraite de tous les effets qui animent un masque, telles que les effets wipe et l’effet Iris.
 Move L’effet Move modifie la position d’un composant pendant un intervalle de temps spécifiée.
 Parallel L’effet Parallel exécute plusieurs effets enfant au même moment.
 Pause L’effet Pause est très utile lors de la mise en séquence des effets.
 RemoveChildAction La classe RemoveChildAction définit un effet d’action qui correspond à la propriété RemoveChild d’une définition de l’état d’affichage.
 RemoveItemAction La classe RemoveItemAction définit un effet d’action qui détermine lorsque le rendu d’élément disparaît de la commande de la fonctionnalité de rendu d’élément d’un élément supprimé d’une contrôle à base de liste, tel que List ou TileList, ou d’un élément remplacé par un nouvel élément ajouté dans le contrôle.
 Resize L’effet Resize modifie la largeur, la hauteur ou les deux dimensions d’un composant au cours d’un laps de temps spécifié.
 Rotate L’effet Rotate fait pivoter un composant autour d’un point donné.
 Sequence L’effet Sequence exécute plusieurs effets enfant l’un après l’autre, dans l’ordre auquel ils sont ajoutés.
 SetPropertyAction La classe SetpropertyAction définit un effet action qui correspond à la propriété SetProperty d’une définition de l’état d’affichage.
 SetStyleAction La classe SetStyleAction définit un effet action qui correspond à la propriété SetStyle de la définition d’un état d’affichage.
 SoundEffect La classe SoundEffect lit un fichier audio MP3.
 Tween Cette classe définit une interpolation, une animation de propriété effectuée sur un objet cible pendant une période de temps.
 TweenEffect La classe TweenEffect est la super-classe de tous les effets basées sur l’objet Tween.
 UnconstrainItemAction La classe UnconstrainItemAction définit un effet d’action qui est utilisé dans une définition d’effet-données pour arrêter temporairement le positionnement des rendus d’élément par l’algorithme de mise en forme du contrôle parent.
 WipeDown La classe WipeDown définit un effet de balayage vers le bas.
 WipeLeft La classe WipeLeft définit un effet de balayage vers la gauche.
 WipeRight La classe WipeRight définit un effet de balayage vers la droite.
 WipeUp La classe WipeUp définit un effet de balayage vers le haut.
 Zoom L’effet Zoom pour effectuer un zoom avant ou un zoom arrière de l’objet sur un point central.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
