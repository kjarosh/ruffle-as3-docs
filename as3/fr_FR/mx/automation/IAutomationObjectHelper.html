addSynchronization

()méthode
public function addSynchronization(isComplete:Function, target:Object = null):void

Version du langage: ActionScript 3.0
Version du produit: Flex 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Ajoute un objet de synchronisation au gestionnaire d’automatisation. Le gestionnaire d’automatisation attend que la méthode isComplete renvoie true avant de passer à l’événement de réexécution suivant.

Paramètres

isComplete:Function — Fonction qui indique si l’opération synchronisée est terminée.
 
target:Object (default = null) — Si nul, la réexécution entière est différée jusqu’à ce que la méthode isComplete renvoie true. Dans le cas contraire, le gestionnaire d’automatisation attend uniquement si l’opération suivante porte sur la cible.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
