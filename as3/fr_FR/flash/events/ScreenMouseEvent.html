Packageflash.events
Classepublic class ScreenMouseEvent
HéritageScreenMouseEvent MouseEvent Event Object

Version du langage: ActionScript 3.0
Versions du moteur d’exécution: AIR 1.0

L’objet SystemTrayIcon déclenche des événements de type ScreenMouseEvent en réponse à l’interaction de la souris.

L’objet ScreenMouseEvent étend la classe MouseEvent pour fournir deux propriétés supplémentaires, screenX et screenY, qui signalent les coordonnées de la souris par rapport à l’écran du bureau principal au lieu d’une fenêtre d’application ou scène.

ScreenMouseEvent

()Constructeur
public function ScreenMouseEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, screenX:Number = NaN, screenY:Number = NaN, ctrlKey:Boolean = false, altKey:Boolean = false, shiftKey:Boolean = false, buttonDown:Boolean = false, commandKey:Boolean = false, controlKey:Boolean = false)

Version du langage: ActionScript 3.0
Versions du moteur d’exécution: AIR 1.0

Crée un objet ScreenMouseEvent contenant l’emplacement de la souris en coordonnées d’écran.

Paramètres
type:String — Type de l’événement. Les écouteurs d’événement peuvent accéder à cette information par le biais de la propriété type héritée.
 
bubbles:Boolean (default = false) — Position X du clic en coordonnées d’écran.
 
cancelable:Boolean (default = false) — Position Y du clic en coordonnées d’écran.
 
screenX:Number (default = NaN) — Définissez cette propriété sur false car les événements de souris d’écran ne se propagent jamais vers le haut.
 
screenY:Number (default = NaN) — Définissez sur false car il n’existe aucun comportement par défaut à annuler.
 
ctrlKey:Boolean (default = false) — Sous Windows ou Linux, indique si la touche Ctrl était enfoncée lorsque cet événement a eu lieu. Sous Mac, indique si la touche Ctrl ou Commande était enfoncée.
 
altKey:Boolean (default = false) — Définissez sur true pour indiquer que la touche alt était enfoncée lorsque cet événement a eu lieu.
 
shiftKey:Boolean (default = false) — Définissez sur true pour indiquer que la touche Maj était enfoncée lorsque cet événement a eu lieu.
 
buttonDown:Boolean (default = false) — Définissez sur true pour indiquer qu’un bouton de la souris était enfoncé lorsque cet événement a eu lieu.
 
commandKey:Boolean (default = false) — Indique si la touche Commande était enfoncée (Mac uniquement).
 
controlKey:Boolean (default = false) — Indique si la touche Ctrl ou Contrôle était enfoncée.

La constante ScreenMouseEvent.CLICK définit la valeur de la propriété type d’un objet d’événement click.

Les principales propriétés de cet événement sont les suivantes :

PropriétéValeur
buttonDowntrue si le bouton principal de la souris est enfoncé ; false dans tous les autres cas.
ctrlKeytrue sous Windows ou Linux si la touche Ctrl est enfoncée. true sur Mac si la touche Ctrl ou la touche Commande est activée. Sinon, la valeur false est renvoyée.
currentTargetL’objet qui traite activement l’objet Event avec un écouteur d’événements.
shiftKeytrue si la touche Maj est activée ; false si elle ne l’est pas.
commandKeytrue sur le Mac si la touche Commande est activée ; false si elle est désactivée. Toujours false sur Windows.
controlKeytrue si la touche Ctrl ou Contrôle est activée ; false si elle est désactivée.
screenXCoordonnée horizontale à laquelle l’événement s’est produit, par rapport aux coordonnées de l’écran.
screenYCoordonnée verticale à laquelle l’événement s’est produit, par rapport aux coordonnées de l’écran.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
