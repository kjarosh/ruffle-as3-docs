Le Guide de référence ActionScript® 3.0 pour la plate-forme Adobe® Flash® contient les éléments, les bibliothèques principales, ainsi que les packages de composants et de classes du langage ActionScript pour les outils, moteurs d’exécution, services et serveurs de la plate-forme Flash.

Filtrer par produit à l’aide des commandes de filtre

  • Cliquez sur le lien Afficher la liste des packages et des classes pour afficher le panneau Packages et classes si cela n’est pas déjà fait.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
