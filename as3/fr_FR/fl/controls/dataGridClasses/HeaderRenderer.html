Classepublic class HeaderRenderer
HéritageHeaderRenderer LabelButton BaseButton UIComponent Sprite DisplayObjectContainer InteractiveObject DisplayObject EventDispatcher Object

Version du langage: ActionScript 3.0
Version du produit: Flash CS3
Versions du moteur d’exécution: Flash Player 9.0.28.0, AIR 1.0

La classe HeaderRenderer affiche l’en-tête de la colonne DataGrid active. Cette classe étend la classe LabelButton et ajoute une propriété column qui associe l’en-tête actif à sa colonne DataGrid.

Récupère la texture de style par défaut du composant actif. La texture de style contient le type adapté au composant, qui dépend du style utilisé par ce dernier. Par exemple, le style disabledTextFormat contient la valeur null ou un objet TextFormat. Vous pouvez utiliser ces styles et appeler setStyle() pour le composant actif. Le code suivant ignore le style par défaut disabledTextFormat dans le composant spécifié : 

componentInstance.setStyle("disabledTextFormat", new TextFormat());

Valeur renvoyée
Object — Objet de styles par défaut.

function showStyleDefinition(e:Event):void {
    var componentClass:Class = e.target.selectedItem.data as Class;
    var styles:Object = componentClass["getStyleDefinition"].call(this);
    trace(styles.toString());
    var styleData:DataProvider = new DataProvider();
    for(var i:* in styles) {
        trace(i + " : " + styles[i]);
        styleData.addItem( { StyleName:i, DefaultValue:styles[i] } );
    }
    styleData.sortOn("StyleName");
    dg.dataProvider = styleData;
}
Exemples Comment utiliser cet exemple
HeaderRendererExample.as

Cet exemple montre comment obtenir des informations sur l’en-tête d’un objet DataGrid.

Pour exécuter l’exemple, suivez ces étapes :

  1. Ajoutez les composants DataGrid, Button, Label et TextArea à la bibliothèque.
  2. Enregistrez ce code sous le nom HeaderRendererExample.as dans le même répertoire que votre fichier FLA.
  3. Dans le fichier FLA, définissez la classe Document sur HeaderRendererExample.
package
{
    import fl.controls.DataGrid;
    import fl.controls.dataGridClasses.DataGridColumn;
    import fl.controls.dataGridClasses.HeaderRenderer;
    import fl.data.DataProvider;
    import fl.events.DataGridEvent;
    import flash.display.Sprite;
    
    public class HeaderRendererExample extends Sprite 
    {
        private var dg:DataGrid;
        
        public function HeaderRendererExample() {
            createDataGrid();
        }
        private function createDataGrid():void {
            dg = new DataGrid();
            addChild(dg);
            dg.setSize(500,300);
            dg.sortableColumns = true;
            
            var dp:DataProvider = new DataProvider();
            var i:uint;
            var totalRows:uint = 42;
            for(i=0; i<totalRows; i++) {
                dp.addItem( {     col1:getRandomNumber(),
                                   col2:getRandomNumber(),
                                   col3:getRandomNumber(),
                                   col4:getRandomNumber(),
                                   col5:getRandomNumber(),
                                   col6:getRandomNumber(),
                                   col7:getRandomNumber(),
                                col8:getRandomNumber() } );
            }
            dg.dataProvider = dp;
            dg.addEventListener(DataGridEvent.COLUMN_STRETCH, announceColumnStretch);
            dg.addEventListener(DataGridEvent.HEADER_RELEASE, announceHeaderClick);
        }
        private function announceColumnStretch(e:DataGridEvent):void {
            var hr:HeaderRenderer = e.itemRenderer as HeaderRenderer;
            var column:DataGridColumn = dg.columns[hr.column];
            trace(column.headerText + " has been stretched");
        }
        private function announceHeaderClick(e:DataGridEvent):void {
            var hr:HeaderRenderer = e.itemRenderer as HeaderRenderer;
            var column:DataGridColumn = dg.columns[hr.column];
            trace(column.headerText + " header has been clicked");
        }
        private function getRandomNumber():uint {
            return Math.round(Math.random()*100);
        }
    }
}
