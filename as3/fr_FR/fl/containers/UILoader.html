Version du langage: ActionScript 3.0
Version du produit: Flash CS3
Versions du moteur d’exécution: Flash Player 9.0.28.0, AIR 1.0

La classe UILoader permet de définir le contenu à charger puis de contrôler l’opération de chargement au moment de l’exécution. Cette classe gère également le redimensionnement du contenu chargé. Si le contenu est chargé à partir d’un autre domaine (sandbox), des mesures de sécurité peuvent rendre ses propriétés inaccessibles. Pour plus d’informations, voir la classe ProLoader.

La définition d’une propriété de la classe UILoader à l’aide d’ActionScript a pour effet d’ignorer le paramètre du même nom défini dans l’inspecteur des propriétés ou l’inspecteur des composants.

Ce composant englobe flash.display.ProLoader. La classe ProLoader gère le chargement dans son ensemble ; le composant UILoader fournit uniquement un affichage visuel à l’objet ProLoader.

Remarque : si le contenu est chargé à partir d’un domaine ou d’un sandbox différent, les propriétés du contenu risquent de ne pas être accessibles pour des raisons de sécurité. Pour plus d’informations sur la façon dont la sécurité du domaine affecte le processus de chargement, voir la classe ProLoader.

Exemple  ( Comment utiliser cet exemple )
L'exemple suivant crée une occurrence de composant UILoader qui affiche une image chargée, et une occurrence des composants ProgressBar et Label qui affichent le statut de téléchargement actuel de l'image : 

import fl.containers.UILoader;
import fl.controls.Label;
import fl.controls.ProgressBar;
import fl.controls.ProgressBarMode;

var url:String = "http://www.helpexamples.com/flash/images/image3.jpg";

var myLabel:Label = new Label();
myLabel.autoSize = TextFieldAutoSize.LEFT;
myLabel.text = "";
myLabel.move(10, 10);
addChild(myLabel);

var myProgressBar:ProgressBar = new ProgressBar();
myProgressBar.mode = ProgressBarMode.MANUAL;
myProgressBar.move(10, 30);
addChild(myProgressBar);

var myUILoader:UILoader = new UILoader();
myUILoader.load(new URLRequest(url));
myUILoader.addEventListener(ProgressEvent.PROGRESS, progressHandler);
myUILoader.addEventListener(Event.COMPLETE, completeHandler);
myUILoader.move(10, 35);
addChild(myUILoader);

function progressHandler(event:ProgressEvent):void {
    var uiLdr:UILoader = event.currentTarget as UILoader;
    var kbLoaded:String = Number(uiLdr.bytesLoaded / 1024).toFixed(1);
    var kbTotal:String = Number(uiLdr.bytesTotal / 1024).toFixed(1);
    myLabel.text = kbLoaded + " of " + kbTotal + " KB" + " (" + Math.round(uiLdr.percentLoaded) + "%)";
    myProgressBar.setProgress(event.bytesLoaded, event.bytesTotal);
}

function completeHandler(event:Event):void {
    myLabel.visible = false;
    myProgressBar.visible = false;
    myUILoader.removeEventListener(ProgressEvent.PROGRESS, progressHandler);
    myUILoader.removeEventListener(Event.COMPLETE, completeHandler);
}

progress

 Evénement  
Type d’objet d’événement: flash.events.ProgressEvent
propriété ProgressEvent.type = flash.events.ProgressEvent.PROGRESS

Version du langage: ActionScript 3.0
Version du produit: Flash CS3
Versions du moteur d’exécution: Flash Player 9.0.28.0, AIR 1.0

Distribué lors du chargement de contenu. Cet événement est distribué, que l’opération de chargement ait été déclenchée par un processus de chargement automatique ou par un appel explicite à la méthode load().

Définit la valeur de la propriété type d’un objet événement progress.

Les propriétés de cet événement sont les suivantes :

PropriétéValeur
bubblesfalse
bytesLoadedNombre d’éléments ou d’octets chargés lors du traitement de l’événement par l’écouteur.
bytesTotalNombre total d’éléments ou d’octets qui seront chargés si le processus de chargement aboutit.
cancelablefalse ; il n’existe aucun comportement par défaut à annuler.
currentTargetL’objet qui traite activement l’objet Event avec un écouteur d’événements.
targetObjet indiquant la progression.

Exemple  ( Comment utiliser cet exemple )

L'exemple suivant crée une occurrence de composant UILoader qui affiche une image chargée, et une occurrence des composants ProgressBar et Label qui affichent le statut de téléchargement actuel de l'image : 

import fl.containers.UILoader;
import fl.controls.Label;
import fl.controls.ProgressBar;
import fl.controls.ProgressBarMode;

var url:String = "http://www.helpexamples.com/flash/images/image3.jpg";

var myLabel:Label = new Label();
myLabel.autoSize = TextFieldAutoSize.LEFT;
myLabel.text = "";
myLabel.move(10, 10);
addChild(myLabel);

var myProgressBar:ProgressBar = new ProgressBar();
myProgressBar.mode = ProgressBarMode.MANUAL;
myProgressBar.move(10, 30);
addChild(myProgressBar);

var myUILoader:UILoader = new UILoader();
myUILoader.load(new URLRequest(url));
myUILoader.addEventListener(ProgressEvent.PROGRESS, progressHandler);
myUILoader.addEventListener(Event.COMPLETE, completeHandler);
myUILoader.move(10, 35);
addChild(myUILoader);

function progressHandler(event:ProgressEvent):void {
    var uiLdr:UILoader = event.currentTarget as UILoader;
    var kbLoaded:String = Number(uiLdr.bytesLoaded / 1024).toFixed(1);
    var kbTotal:String = Number(uiLdr.bytesTotal / 1024).toFixed(1);
    myLabel.text = kbLoaded + " of " + kbTotal + " KB" + " (" + Math.round(uiLdr.percentLoaded) + "%)";
    myProgressBar.setProgress(event.bytesLoaded, event.bytesTotal);
}

function completeHandler(event:Event):void {
    myLabel.visible = false;
    myProgressBar.visible = false;
    myUILoader.removeEventListener(ProgressEvent.PROGRESS, progressHandler);
    myUILoader.removeEventListener(Event.COMPLETE, completeHandler);
}

Exemple  ( Comment utiliser cet exemple )

L’exemple suivant crée une occurrence de composant UILoader, écoute l’événement resize et centre l’image sur la scène à la fin de son chargement : 

import fl.containers.UILoader;
import fl.events.ComponentEvent;

var myUILoader:UILoader = new UILoader();
myUILoader.scaleContent = false;
myUILoader.source = "http://www.helpexamples.com/flash/images/image3.jpg";
myUILoader.addEventListener(ComponentEvent.RESIZE, resizeHandler);
addChild(myUILoader);

function resizeHandler(event:ComponentEvent):void {
    var uiLdr:UILoader = event.currentTarget as UILoader;
    trace(uiLdr.width, uiLdr.height); // 400 267
    uiLdr.move((stage.stageWidth - uiLdr.width) / 2, (stage.stageHeight - uiLdr.height) / 2);
}

Exemples Comment utiliser cet exemple
UILoaderExample.as

Cet exemple illustre une interaction simple avec la classe UILoader.
  1. Placez une occurrence de UILoader sur la scène et nommez-la « uiLoader ».
  2. Placez une occurrence de Label au-dessus de l’occurrence de UILoader et nommez-la « uiLoaderLabel ».
  3. Enregistrez ce code sous le nom UILoaderExample.as dans le même répertoire que votre fichier FLA.
  4. Dans le fichier FLA, définissez DocumentClass sur UILoaderExample.
package
{
    import flash.display.Sprite;
    import flash.net.URLRequest;
    import flash.events.Event;
    import flash.events.ProgressEvent;

    public class UILoaderExample extends Sprite
    {
        private var sampleImagePath:String = "test.jpg";
        
        public function UILoaderExample() {
            var request:URLRequest = new URLRequest(sampleImagePath);
            uiLoader.scaleContent = false;
            uiLoader.addEventListener(Event.COMPLETE,loadComplete);
            uiLoader.addEventListener(ProgressEvent.PROGRESS,loadProgress);            
            uiLoader.load(request);
        }
        private function loadProgress(e:ProgressEvent):void {
            uiLoaderLabel.text = String(e.target.percentLoaded);
        }
        private function loadComplete(e:Event):void {
            uiLoaderLabel.text = "Load Complete";    
        }
    }
}
