Packagefl.events
Classepublic class RSLEvent
HéritageRSLEvent Event Object

Version du langage: ActionScript 3.0
Version du produit: Flash CS5
Versions du moteur d’exécution: Flash Player 10.1, AIR 2

La classe RSLEvent définit les événements distribués par RSLPreloader. Ces événements indiquent la progression du téléchargement des fichiers RSL, et fournissent des informations sur le nombre de fichiers et le nombre d’octets chargés. Lorsqu’une boucle de préchargement d’événement personnalisé est utilisée et que le code RSLPreloader est généré automatiquement par l’environnement de création de Flash, ces événements sont distribués par le scénario principal. Dans ce cas, utilisez addEventListener sans avoir à référencer RSLPreloader.

public function RSLEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, rslsLoaded:int = 0, rslsFailed:int = 0, rslsTotal:int = 0, bytesLoaded:int = 0, bytesTotal:int = 0)

Version du langage: ActionScript 3.0
Version du produit: Flash CS5
Versions du moteur d’exécution: Flash Player 10.1, AIR 2

Crée un objet Event à l’aide d’informations spécifiques relatives aux événements RSL.

Paramètres
type:String — Type de l’événement. Les valeurs possibles sont les suivantes : RSLEvent.RSL_LOAD_COMPLETE et RSLEvent.RSL_PROGRESS.
 
bubbles:Boolean (default = false) — Détermine si l’objet Event prend part à la phase de propagation vers le haut du flux d’événements.
 
cancelable:Boolean (default = false) — Détermine si l’objet d’événement peut être annulé.
 
rslsLoaded:int (default = 0) — Nombre de fichiers qui ont été téléchargés correctement.
 
rslsFailed:int (default = 0) — Nombre de fichiers dont le téléchargement a échoué.
 
rslsTotal:int (default = 0) — Nombre total de fichiers correctement téléchargés ou non.
 
bytesLoaded:int (default = 0) — Nombre total d’octets téléchargés dans les fichiers correctement téléchargés.
 
bytesTotal:int (default = 0) — Nombre total d’octets téléchargés dans les fichiers téléchargés, correctement ou non.

RSL_LOAD_COMPLETE

Constante
public static const RSL_LOAD_COMPLETE:String = "rslLoadComplete"

Version du langage: ActionScript 3.0
Version du produit: Flash CS5
Versions du moteur d’exécution: Flash Player 10.1, AIR 2

Evénement distribué par RSLPreloader lorsque le téléchargement de tous les fichiers RSL est terminé. Les propriétés bytesLoaded et bytesTotal ont toujours pour valeur 0 pour ce type d’événement. En cas de préchargement avec une boucle d’événement personnalisé, le contenu de l’utilisateur DOIT écouter cet événement pour savoir quand quitter la boucle d’animation de chargement.

Les propriétés de cet événement sont les suivantes :

PropriétéValeur
bubblestrue
cancelablefalse ; il n’existe aucun comportement par défaut à annuler.
currentTargetL’objet qui traite activement l’objet Event avec un écouteur d’événements.
targetOccurrence de DisplayObject ajoutée à la liste d’affichage. La propriété target ne correspond pas toujours à l’objet de la liste d’affichage qui est associé à l’écouteur d’événement. Utilisez la propriété currentTarget pour accéder à l’objet de la liste d’affichage qui traite actuellement l’événement.

         myPreloader.addEventListener(RSLEvent.RSL_LOAD_COMPLETE, loadComplete);

         gotoAndPlay('startOfContent');
         }

         import fl.events.RSLEvent;
         myPreloader.addEventListener(RSLEvent.RSL_PROGRESS, progressHandler);
         function progressHandler(e:RSLEvent) {
         trace('rslsLoaded = ' + e.rslsLoaded);
         trace('rslsFailed = ' + e.rslsFailed);
