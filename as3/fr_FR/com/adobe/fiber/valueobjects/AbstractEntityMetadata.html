Classepublic class AbstractEntityMetadata
HéritageAbstractEntityMetadata EventDispatcher Object
Implémente IModelType, IModelInstance

Version du langage: ActionScript 3.0
Version du produit: Adobe Digital Enterprise Platform Data Services for Java EE 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Cette classe définit les implémentations par défaut de génération d’erreur pour toutes les fonctions déclarées dans les interfaces IModelType et IModelInstance. En outre, toutes les extensions générées de cette classe exposent les API suivantes, générées par propriété d’entité : 
      function get property-nameStyle():IStyle

Accesseur Get liable de l’instance com.adobe.fiber.styles.IStyle de la propriété. Renvoie null pour les propriétés sans style associé.

      function get isproperty-nameAvailable():Boolean

accesseur Get liable de l’état disponible de la propriété relativement aux variantes. Renvoie toujours la valeur true pour les propriétés non contenues dans une variante.

      function invalidateDependentproperty-name():void

Fonctions par propriété qui entraînent le recalcul de toutes les propriétés dérivées de cette propriété. Les événements PropertyChanged sont supprimés à la fin du calcul de chaque propriété dérivée. Notez que ces fonctions sont générées uniquement pour les propriétés de données dont dépendent des propriétés dérivées.

isAvailable

()méthode 
public function isAvailable(propertyName:String):Boolean

Version du langage: ActionScript 3.0
Version du produit: Adobe Digital Enterprise Platform Data Services for Java EE 3
Versions du moteur d’exécution: Flash Player 9, AIR 1.1

Détermine si une propriété est disponible comme le déterminent les variantes. Une erreur est émise si le nom de la propriété transmise n’est pas une propriété valide de l’objet de valeur. Pour ajouter des liaisons à l’indicateur de disponibilité pour une propriété précise, utilisez la fonction ispropertyNameAvailable(). Cette fonction existe pour chaque propriété et prend en charge la liaison.

Paramètres

propertyName:String — Propriété dont la disponibilité est à vérifier.

Valeur renvoyée
Boolean — true si cette propriété est actuellement disponible comme les variantes le déterminent.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
