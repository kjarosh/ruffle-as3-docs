Le package org.osmf.net.metrics contient les classes qui définissent les objets de mesure OSMF utilisés dans la commande de commutation Flux continu adaptatif.


Classes
 ClasseDescription
 ActualBitrateMetric Mesure calculant le débit actuel du niveau de qualité en cours de téléchargement. Cette mesure est de type MetricType.ACTUAL_BITRATE (org.osmf.net.abr.metrics.actualBitrate)
 AvailableQualityLevelsMetric Mesure qui répertorie tous les niveaux de qualité disponibles sous la forme de paires (index, débit)
 BandwidthMetric Mesure de la bande passante Unité de mesure : octets / secondes
 BufferFragmentsMetric BufferFragmentsMetric calcule le nombre de fragments dans la mémoire tampon
 BufferLengthMetric Mesure indiquant la longueur de la mémoire tampon. Cette mesure est de type MetricType.BUFFER_LENGTH (org.osmf.net.abr.metrics.bufferLength)
 BufferOccupationRatioMetric Mesure calculant l’occupation de la mémoire tampon (longueur de la mémoire tampon de / durée de mise en mémoire tampon). Cette mesure est de type MetricType.BUFFER_OCCUPATION_RATIO (org.osmf.net.abr.metrics.bufferOccupationRatio)
 CurrentStatusMetric Mesure de l’état actuel
 DefaultMetricFactory DefaultMetricFactory est l’implémentation par défaut de MetricFactory.
 DroppedFPSMetric Unité de mesure du nombre d’images perdues par seconde : images / seconde
 EmptyBufferMetric Mesure du tampon vide.
 FPSMetric Unité de la mesure du nombre d’images par seconde : images / seconde
 FragmentCountMetric Mesure chargée de déterminer le nombre de fragments dans l’historique
 MetricBase MetricBase est une classe de base pour les mesures utilisées pour le débit adaptatif.
 MetricFactory MetricFactory représente une classe de fabrique pour les mesures.
 MetricFactoryItem MediaFactoryItem est l’encapsulation de toutes les informations nécessaires pour créer et initialiser dynamiquement un MetricBase à partir d’un MetricFactory.
 MetricRepository MetricRepository est chargé des mesures de stockage. Il répond aux requêtes contenant le type de mesure et ses paramètres. MetricRepository utilisera un MetricFactory pour créer de nouvelles mesures.
 MetricType MetricType identifie les divers types de mesures qu’Open Source Media Framework peut gérer en dehors de la zone.
 MetricValue MetricValue représente une valeur de MetricBase.
 RecentSwitchMetric Mesure indiquant si une commutation s’est produite récemment (vérifie si le dernier fragment téléchargé est d’une qualité différente de celle que le précédent). Cette mesure spécifie le type de commutation (vers le haut ou vers le bas). Cette mesure est de type MetricType.RECENT_SWITCH (org.osmf.net.abr.metrics.recentSwitch)

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
