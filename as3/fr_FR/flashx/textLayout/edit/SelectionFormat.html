public function SelectionFormat(rangeColor:uint = 0xffffff, rangeAlpha:Number = 1.0, rangeBlendMode:String = "difference", pointColor:uint = 0xffffff, pointAlpha:Number = 1.0, pointBlendMode:String = "difference", pointBlinkRate:Number = 500)

Version du langage: ActionScript 3.0
Versions du moteur d’exécution: Flash Player 10, AIR 1.5

Crée un objet SelectionFormat avec les propriétés spécifiées.

Un objet SelectionFormat créé avec les valeurs par défaut utilise du noir pour les couleurs de surbrillance, 1.0 pour les alphas et BlendMode.DIFFERENCE pour les modes de fusion. La cadence de clignotement du curseur est de 500 millisecondes.

La définition des propriétés pointAlpha et rangeAlpha sur zéro désactive la mise en surbrillance de la sélection.

Une cadence de clignotement non nulle est utilisée uniquement lorsqu’un objet EditManager est associé à l’objet TextFlow.

Paramètres
rangeColor:uint (default = 0xffffff) — La couleur de la mise en surbrillance.
 
rangeAlpha:Number (default = 1.0) — La valeur de la transparence de la mise en surbrillance. Les valeurs disponibles sont comprises entre 0 (entièrement transparent) et 1 (entièrement opaque, qui est la valeur par défaut).
 
rangeBlendMode:String (default = "difference") — Le mode de fondu pour la mise en surbrillance. Utilisez les valeurs constantes définies dans la classe BlendMode pour définir ce paramètre.
 
pointColor:uint (default = 0xffffff) — La couleur à utiliser pour le curseur de tracé.
 
pointAlpha:Number (default = 1.0) — La valeur de la transparence pour le dessin du curseur. Les valeurs disponibles sont comprises entre 0 (entièrement transparent) et 1 (entièrement opaque, qui est la valeur par défaut).
 
pointBlendMode:String (default = "difference") — Le mode de fondu pour le dessin du curseur. Utilisez les valeurs constantes définies dans la classe BlendMode pour définir ce paramètre.
 
pointBlinkRate:Number (default = 500) — La cadence à laquelle le curseur clignote, en millisecondes.

Certaines parties du Guide de référence ActionScript 3.0 sont en anglais

Tout le contenu du Guide de référence ActionScript 3.0 n’a pas été traduit dans toutes les langues. Si un élément de langage n’est pas traduit, il s’affiche en anglais. Par exemple, la classe ga.controls.HelpBox n’est traduite dans aucune langue. Ainsi, dans la version française du guide de référence, la classe ga.controls.HelpBox apparaît en anglais.
