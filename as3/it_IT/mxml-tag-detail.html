Classi x

Alcuni tag sono disponibili solo in MXML. Questi tag vengono implementati dal compilatore Flex e quindi non sono disponibili in ActionScript.

TagDescrizione
fx:BindingUtilizzate il tag <fx:Binding> per associare i dati di un oggetto a un altro oggetto.
fx:ComponentUtilizzate il tag <fx:Component> per definire un renderer di celle o editor di celle inline in un file MXML.
fx:DeclarationsUtilizzate il tag <fx:Declarations> per dichiarare proprietà non predefinite e non visive della classe corrente.
fx:DefinitionUtilizzate il tag <fx:Definition> all’interno di un tag <fx:Library> per definire elementi grafici secondari che potrete quindi utilizzare in altre parti dell’applicazione.
fx:DesignLayerIl tag <fx:DesignLayer> è solo per uso interno.
fx:LibraryUtilizzate il tag <fx:Library> per dichiarare proprietà non predefinite e non visive della classe corrente.
fx:MetadataUtilizzate il tag <fx:Metadata> per inserire tag di metadati in un file MXML.
fx:ModelUtilizzate il tag <fx:Model> per dichiarare un modello dati in MXML.
fx:PrivateUtilizzate il tag <fx:Private> per fornire meta informazioni sul documento MXML o FXG.
fx:ReparentUtilizzate il tag <fx:Reparent> per modificare il contenitore principale di un componente come parte di una modifica dello stato di visualizzazione.
fx:ScriptUtilizzate il tag <fx:Script> per definire blocchi di codice ActionScript.
fx:StyleUtilizzate il tag <fx:Style> per definire stili applicabili al documento corrente e ai relativi elementi secondari.
fx:XMLIn MXML potete dichiarare un modello dati in formato XML in un tag <fx:XML>.
fx:XMLListIn MXML potete creare un oggetto XMLList E4X da un modello in formato di testo in un tag <fx:XMLList>.

[ X ]Perché in inglese?
Il contenuto della Guida di riferimento di ActionScript 3.0 appare in inglese

Non tutte le parti della Guida di riferimento di ActionScript 3.0 sono tradotte in tutte le lingue. Quando un elemento del linguaggio non è tradotto, viene riportato in inglese. Ad esempio, la classe ga.controls.HelpBox non è tradotta in nessuna lingua. Pertanto, nella versione italiana della guida di riferimento, la descrizione della classe ga.controls.HelpBox è riportata in inglese.
