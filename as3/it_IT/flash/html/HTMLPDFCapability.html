Classepublic final class HTMLPDFCapability
EreditarietàHTMLPDFCapability Object

Versione linguaggio: ActionScript 3.0
Versioni runtime: AIR 1.0

La classe HTMLPDFCapability contiene possibili valori della proprietà pdfCapability di un oggetto HTMLLoader. Essa definisce inoltre i valori della proprietà errorID di un oggetto ErrorEvent inviato quando del contenuto PDF non viene caricato correttamente.

ERROR_CANNOT_LOAD_READER

Costante
public static const ERROR_CANNOT_LOAD_READER:int = 3204

Versioni runtime: AIR 1.0

È stato restituito un errore dal sistema operativo durante il caricamento dell'applicazione Adobe Reader o Acrobat o una delle relative librerie necessarie.

Nota: l'errore non viene restituito da HTMLLoader.pdfCapability, ma viene inviato come proprietà errorID di un oggetto ErrorEvent quando un oggetto HTMLLoader tenta di caricare contenuto PDF e il sistema operativo restituisce un errore. HTMLLoader.pdfCapability può restituire PDFCapability.STATUS_OK, poiché esamina esclusivamente la configurazione e non tenta di caricare alcuna libreria.

    

ERROR_INSTALLED_READER_NOT_FOUND

Costante 
public static const ERROR_INSTALLED_READER_NOT_FOUND:int = 3201

Versioni runtime: AIR 1.0

Non è stata rilevata nessuna versione di Adobe Reader. Un oggetto HTMLLoader non può visualizzare contenuto PDF.

ERROR_PREFERRED_READER_TOO_OLD

Costante 
public static const ERROR_PREFERRED_READER_TOO_OLD:int = 3203

Versioni runtime: AIR 1.0

Una versione sufficiente (8.1 o successiva) di Adobe Reader o Acrobat viene rilevata, ma la versione di Adobe Reader impostata per gestire il contenuto PDF è precedente ad Adobe Reader o Acrobat 8.1. Un oggetto HTMLLoader non può visualizzare contenuto PDF.
