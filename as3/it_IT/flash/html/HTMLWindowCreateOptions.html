HTMLWindowCreateOptions  - AS3

Pacchettoflash.html
Classepublic class HTMLWindowCreateOptions
EreditarietàHTMLWindowCreateOptions Object

Versioni runtime: AIR 1.0

Questa classe definisce le opzioni che possono essere specificate quando JavaScript in esecuzione in un oggetto HTMLLoader tenta di creare una nuova finestra HTML chiamando il metodo window.open().

Questa classe definisce le proprietà e i metodi che possono che corrispondono alle opzioni nel parametro features passato al metodo window.open() in JavaScript.

Ad esempio, in un documento HTML (in un oggetto HTMLLoader), JavaScript può includere la seguente chiamata al metodo window.open(), in cui il parametro features (il terzo parametro) elenca una serie di opzioni:

window.open("http://www.adobe.com", "AdobeWindow", "scrollbars=1,menubar=1,location=0,status=0")

Potete utilizzare la classe HTMLWindowCreateOptions nella sostituzione del metodo createWindow() di una sottoclasse della classe HTMLHost. L'oggetto HTMLLoader passa un oggetto HTMLWindowCreateOptions come il parametro windowCreateOptions del metodo createWindow() dell'oggetto HTMLHost.

Descrizione delle proprietà
    

fullscreen

proprietà
public var fullscreen:Boolean = false

Versioni runtime: AIR 1.0

Specifica se la finestra deve essere a tutto schermo. Questa proprietà è impostata su true se la stringa features della chiamata JavaScript al metodo window.open() include "fullscreen", "fullscreen=1" o "fullscreen=y".

    

height

proprietà 
public var height:Number = NaN

Versioni runtime: AIR 1.0

Specifica l'altezza iniziale richiesta della nuova finestra sullo schermo. Tale posizione è impostata sul valore specificato per height nella stringa features della chiamata JavaScript al metodo window.open(). Se il valore è NaN, impostazione predefinita quando non viene specificato alcun valore height nella stringa features, viene utilizzata un'altezza finestra predefinita.
