Classepublic class FormItemLayout
EreditarietàFormItemLayout ConstraintLayout LayoutBase OnDemandEventDispatcher Object

Versione linguaggio: ActionScript 3.0
Versione prodotto: Flex 4.5
Versioni runtime: Flash Player 10, AIR 2.5

The FormItemLayout is used by FormItems to provide a constraint based layout. Elements using FormItemLayout within a FormLayout are aligned along columns.
