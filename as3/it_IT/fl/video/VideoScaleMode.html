Descrizione delle costanti

EXACT_FIT

Costante
public static const EXACT_FIT:String = "exactFit"

Versione linguaggio: ActionScript 3.0
Versione prodotto: Flash CS3
Versioni runtime: Flash Player 9.0.28.0, AIR 1.0

Specifica che il video venga visualizzato con la stessa altezza e larghezza specificate dalle proprietà registrationHeight o height e registrationWidth o width.

Ad esempio, se registrationWidth = width = 100 e registrationHeight = height = 100, se registrationX = 200 e registrationY = 200, e se caricate un file FLV con width = 648 e height = 480 (per un rapporto proporzionale approssimativo di 4:3) e align = VideoAlign.CENTER e scaleMode = EXACT_FIT, otterrete width = 100, height = 100, x = 200 e y = 200.

MAINTAIN_ASPECT_RATIO

Costante 
public static const MAINTAIN_ASPECT_RATIO:String = "maintainAspectRatio"

Versione linguaggio: ActionScript 3.0
Versione prodotto: Flash CS3
Versioni runtime: Flash Player 9.0.28.0, AIR 1.0

Specifica che il video venga vincolato all'interno del rettangolo determinato dalle proprietà registrationX, registrationY, registrationWidth e registrationHeight, ma conservi le proporzioni originali.

Ad esempio, se registrationWidth = 100 e registrationHeight = 100, se registrationX = 200 e registrationY = 200, e se caricate un file FLV con width = 648 e height = 480 (per un rapporto proporzionale approssimativo di 4:3) e align = VideoAlign.CENTER e scaleMode = MAINTAIN_ASPECT_RATIO, otterrete width = 100, height = (100 * 480 / 648) = 74, x = 10 e y = (10 + ((100 - 74) / 2)) = 23.

NO_SCALE

Costante 
public static const NO_SCALE:String = "noScale"

Versione linguaggio: ActionScript 3.0
Versione prodotto: Flash CS3
Versioni runtime: Flash Player 9.0.28.0, AIR 1.0

Specifica che il video deve essere visualizzato con i valori height e width corrispondenti a quelli del video di origine.

Ad esempio, se registrationWidth = 100 e registrationHeight = 100, se registrationX = 200 e registrationY = 200, e se caricate un file FLV con width = 648 e height = 480 (per un rapporto proporzionale approssimativo di 4:3) e align = VideoAlign.CENTER e scaleMode = NO_SCALE, otterrete width = 648, height = 480, x = (200 + ((100 - 648) / 2)) = -74 e y = (200 + ((100 - 480) / 2)) = 10.
