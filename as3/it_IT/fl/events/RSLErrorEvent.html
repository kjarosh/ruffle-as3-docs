Classepublic class RSLErrorEvent
EreditarietàRSLErrorEvent ErrorEvent TextEvent Event Object

Versione linguaggio: ActionScript 3.0
Versione prodotto: Flash CS5
Versioni runtime: Flash Player 10.1, AIR 2

La classe RSLErrorEvent definisce un evento di errore inviato da RSLPreloader. Questo evento indica che lo scaricamento delle RSL non è riuscito e fornisce informazioni sui file che hanno dato esito negativo, oltre al numero di file che si è tentato di scaricare e quelli con esito negativo. Quando si utilizza un ciclo di precaricamento di eventi personalizzati e il codice RSLPreloader viene generato automaticamente dallo strumento di creazione Flash, questo evento viene inviato dalla linea temporale principale. In questo caso, utilizzate addEventListener senza l'esigenza di fare riferimento al codice RSLPreloader.

public function RSLErrorEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, rslsLoaded:int = 0, rslsFailed:int = 0, rslsTotal:int = 0, failedURLs:Array = null)

Versione linguaggio: ActionScript 3.0
Versione prodotto: Flash CS5
Versioni runtime: Flash Player 10.1, AIR 2

Crea un oggetto RSLErrorEvent contenente informazioni relative agli eventi RSLError. Gli oggetti RSLErrorEvent vengono passati ai listener di eventi come parametri.

Parametri
type:String — Il tipo di evento. I possibili valori sono: RSLEvent.RSL_LOAD_COMPLETE e RSLEvent.RSL_PROGRESS.
 
bubbles:Boolean (default = false) — Determina se l'oggetto Event partecipa alla fase di bubbling del flusso di eventi.
 
cancelable:Boolean (default = false) — Determina se l'oggetto Event può essere annullato.
 
rslsLoaded:int (default = 0) — Il numero di file che sono stati scaricati correttamente.
 
rslsFailed:int (default = 0) — Il numero di file che non è stato possibile scaricare.
 
rslsTotal:int (default = 0) — Il numero di file che scaricati correttamente o non scaricati.
 
failedURLs:Array (default = null) — Un array di file che non è stato possibile scaricare.
Descrizione delle costanti

RSL_LOAD_FAILED

Costante
public static const RSL_LOAD_FAILED:String = "rslLoadFailed"

Versione linguaggio: ActionScript 3.0
Versione prodotto: Flash CS5
Versioni runtime: Flash Player 10.1, AIR 2

Errore inviato da RSLPreloader al termine dello scaricamento di tutte le RSL, qualora una o più di esse non sia stata scaricata. Quando il precaricamento avviene con un ciclo di eventi personalizzati, è NECESSARIO che il contenuto utente intercetti questo evento per determinare quando uscire dal ciclo di caricamento dell'animazione.

Questo evento ha le seguenti proprietà:

ProprietàValore
bubblestrue
cancelablefalse; non è presente alcun comportamento predefinito da annullare.
currentTargetL'oggetto che elabora attivamente l'oggetto Event con un listener di eventi.
targetL'istanza DisplayObject che viene aggiunta all'elenco di visualizzazione. Il target non è sempre l'oggetto nell'elenco di visualizzazione che ha registrato il listener di eventi. Utilizzate la proprietà currentTarget per accedere all'oggetto dell'elenco di visualizzazione che sta elaborando l'evento.
rslsLoadedIl numero di file che sono stati scaricati correttamente.
rslsFailedIl numero di file che non è stato possibile scaricare.
