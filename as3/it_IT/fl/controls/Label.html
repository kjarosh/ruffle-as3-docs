Classepublic class Label
EreditarietàLabel UIComponent Sprite DisplayObjectContainer InteractiveObject DisplayObject EventDispatcher Object

Versione linguaggio: ActionScript 3.0
Versione prodotto: Flash CS3
Versioni runtime: Flash Player 9.0.28.0, AIR 1.0

Un componente Label visualizza una o più righe di testo semplice o in formato HTML e di cui è possibile formattare l'allineamento e la dimensione. I componenti Label non hanno bordi e non possono essere attivati.

Nell'anteprima dal vivo di ogni istanza del componente Label vengono visualizzate le modifiche apportate durante la creazione ai parametri nella finestra di ispezione Proprietà o nella finestra di ispezione dei componenti. Poiché un componente Label non ha bordi, l'unico modo per visualizzare l'anteprima di un'istanza Label consiste nell'impostarne la proprietà text. La proprietà autoSize non è supportata nell'anteprima dal vivo.

autoSize

proprietà
autoSize:String

Versione linguaggio: ActionScript 3.0
Versione prodotto: Flash CS3
Versioni runtime: Flash Player 9.0.28.0, AIR 1.0

Ottiene o imposta una stringa che indica in che modo avvengono il ridimensionamento e l'allineamento dell'etichetta per adattarvi il valore della relativa proprietà text. I valori validi sono i seguenti:

  • TextFieldAutoSize.NONE: l'etichetta non viene ridimensionata o allineata per adattarvi il testo.
  • TextFieldAutoSize.LEFT: vengono ridimensionati i lati destro e inferiore dell'etichetta per adattarvi il testo. I lati sinistro e superiore non vengono ridimensionati.
  • TextFieldAutoSize.CENTER: vengono ridimensionati i lati sinistro e destro dell'etichetta per adattarvi il testo. Il centro orizzontale dell'etichetta rimane ancorato alla posizione originale.
  • TextFieldAutoSize.RIGHT: vengono ridimensionati i lati sinistro e inferiore dell'etichetta per adattarvi il testo. Il lato superiore e quello destro non vengono ridimensionati.

Il valore predefinito è TextFieldAutoSize.NONE.

function showStyleDefinition(e:Event):void {
    var componentClass:Class = e.target.selectedItem.data as Class;
    var styles:Object = componentClass["getStyleDefinition"].call(this);
    trace(styles.toString());
    var styleData:DataProvider = new DataProvider();
    for(var i:* in styles) {
        trace(i + " : " + styles[i]);
        styleData.addItem( { StyleName:i, DefaultValue:styles[i] } );
    }
    styleData.sortOn("StyleName");
    dg.dataProvider = styleData;
}

Esempi Come utilizzare questo esempio
LabelExample.as

Questo esempio mostra come impostare il testo di un'etichetta, con e senza contenuto HTML.
  1. Aggiungete i componenti Label e Button alla libreria.
  2. Salvate questo codice come LabelExample.as nella stessa directory del FLA.
  3. Impostate DocumentClass nel FLA su LabelExample.
package
{
    import flash.events.MouseEvent;
    import flash.text.TextField;
    import flash.text.TextFieldAutoSize;
    import flash.display.Sprite;
    import fl.controls.Label;
    import fl.controls.Button;
    
    public class LabelExample extends Sprite
    {
        private var simpleLabel:Label;
        private var b1:Button;
        private var b2:Button;        

        private var htmlData:String = "<br>This label displays <b>bold</b> and <i>italic</i> HTML-formatted text.";
        
        public function LabelExample() {
            setupLabel();
            setupButtons();
        }
        private function setupButtons():void {
            b1 = new Button();
            b2 = new Button();

            b1.move(10,50);
            b2.move(10,80);

            b1.setSize(120,22);
            b2.setSize(120,22);
            
            b1.label = "Make Normal Label";
            b2.label = "Make HTML Label";

            b1.addEventListener(MouseEvent.CLICK,buttonClick);
            b2.addEventListener(MouseEvent.CLICK,buttonClick);

            addChild(b1);
            addChild(b2);
        }
        private function setupLabel():void {
            simpleLabel = new Label();
            simpleLabel.move(10,10);
            simpleLabel.autoSize = TextFieldAutoSize.LEFT;
            addChild(simpleLabel);
        }
                               
        // Event handler function to change the image size.
        private function displayHTML():void {
            simpleLabel.htmlText = htmlData;
        }
      
        // Event handler function to change the image size.
        private function displayText():void {
            simpleLabel.text = "This Label displays plain text.";
        }
        private function buttonClick(e:MouseEvent):void {
            switch(e.target) {
                case b1:
                    displayText();
                    break;
                case b2:
                    displayHTML();
                    break;
                default:
                    break;
            }
        }
    }
}
