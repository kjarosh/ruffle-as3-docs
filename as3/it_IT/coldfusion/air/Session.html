beginTransaction

()metodo 
public function beginTransaction(option:String = null):coldfusion.air:SessionToken

Versione linguaggio: ActionScript 3.0
Versione prodotto: ColdFusion 9
Versioni runtime: Flash Player 9, AIR 1.0

Begins a transaction on the session.

Parametri

option:String (default = null) — Indicates the locking strategy that is used by the transaction. The value can be one of the constants defined in the SQLTransactionLockType class: DEFERRED (the default) indicates that a lock is not acquired until the first read or write operation. EXCLUSIVE indicates that a lock is acquired as soon as possible, and no other SQLConnection instance can read from or write to the database. IMMEDIATE indicates that a lock is acquired as soon as possible, in which other SQLConnection instances can read from but can't write to the database.

Restituisce
coldfusion.air:SessionToken — The session token associated with the call

Parametri
