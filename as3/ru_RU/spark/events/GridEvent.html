Классpublic class GridEvent
НаследованиеGridEvent MouseEvent Event Object

Язык версии: ActionScript 3.0
Версия продукта: Flex 4.5
Версии среды выполнения: Flash Player 10, AIR 2.5

Класс GridEvent расширяет класс MouseEvent и добавляет информацию о сетке на основе местоположения события по отношению к ячейке сетки. Эта информация включает следующие данные:
  • Индексы строки и столбца ячейки.
  • Объект GridColumn ячейки.
  • Элемент поставщика данных, который соответствует строке ячейки.
  • Средство визуализации элементов.

События сетки точно соответствуют событиям мыши. Они отправляются в ответ на события мыши, которые представляют собой восходящую цепочку от нижестоящего элемента сетки непосредственно к сетке. Существенная разница состоит в том, что прослушиватели событий сетки обязательно распознают полное действие мыши "вниз-перетаскивание-вверх", даже если перетаскивание и перемещение вверх выполнялись вне пределов сетки. Событие gridMouseDrag соответствует событию перемещения мыши с удерживаемой кнопкой.

Язык версии: ActionScript 3.0
Версия продукта: Flex 4.5
Версии среды выполнения: Flash Player 10, AIR 2.5

События GridEvent, отправленные классом Grid в ответ на события мыши, создаются со свойствами входящих событий мыши. Положение координат x и y события сетки, т. е. значение свойств localX и localY, определяется относительно всей сетки, а не только части сетки, которая была прокручена в область видимости. Аналогично, индексы строки и столбца события могут соответствовать ячейке, которая не была прокручена в область видимости.

Параметры
type:String — Определяет действие мыши, в результате которого было отправлено это событие.
 
bubbles:Boolean (default = false) — Указывает, может ли событие передаваться вверх по иерархии списка отображения.
 
cancelable:Boolean (default = false) — Указывает, существует ли возможность предотвращения поведения, связанного с событием.
 
localX:Number (default = NaN) — Координата X события относительно сетки.
 
localY:Number (default = NaN) — Координата Y события относительно сетки.
 
relatedObject:InteractiveObject (default = null) — Индекс строки, где произошло событие, или-1.
 
ctrlKey:Boolean (default = false) — Индекс столбца, где произошло событие, или-1.
 
altKey:Boolean (default = false) — Столбец, где произошло событие, или null.
 
shiftKey:Boolean (default = false) — Элемент поставщика данных в rowIndex.
 
buttonDown:Boolean (default = false) — Свойство relatedObject события MouseEvent, которое вызвало этот GridEvent.
 
delta:int (default = 0) — Видимое средство визуализации элементов, где произошло событие, или null.
 
rowIndex:int (default = -1) — Указывает, нажата ли клавиша Control.
 
columnIndex:int (default = -1) — Указывает, нажата ли клавиша Alt.
 
column:GridColumn (default = null) — Указывает, нажата ли клавиша Shift.
 
item:Object (default = null) — Указывает, нажата ли клавиша Control.
 
itemRenderer:IGridItemRenderer (default = null) — Не используется.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
