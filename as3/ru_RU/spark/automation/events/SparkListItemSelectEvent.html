Сведения о конструкторе

SparkListItemSelectEvent

()Конструктор
public function SparkListItemSelectEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, itemRenderer:IItemRenderer = null, triggerEvent:Event = null, ctrlKey:Boolean = false, altKey:Boolean = false, shiftKey:Boolean = false)

Язык версии: ActionScript 3.0
Версия продукта: Flex 4
Версии среды выполнения: Flash Player 9, AIR 1.5

Конструктор. Как правило, вызывается управляющим элементом Flex и не используется в коде приложения.

Параметры
type:String — Тип события; указывает действие, вызвавшее событие.
 
bubbles:Boolean (default = false) — Указывает, может ли событие передаваться вверх по иерархии списка отображения.
 
cancelable:Boolean (default = false) — Указывает, существует ли возможность предотвращения поведения, связанного с событием.
 
itemRenderer:IItemRenderer (default = null) — Объект средства визуализации элементов для элемента.
 
triggerEvent:Event (default = null) — Событие, например событие мыши или клавиатуры, инициировавшее операцию выбора.
 
ctrlKey:Boolean (default = false) — Указывает, была ли нажата клавиша Alt во время события.
 
altKey:Boolean (default = false) — Указывает, была ли нажата клавиша Ctrl во время события.
 
shiftKey:Boolean (default = false) — Указывает, была ли нажата клавиша Shift во время события.

Константа SparkListItemSelectEvent.SELECT определяет значение свойства type объекта события, которое передается при выборе отдельного элемента.

Свойства объекта события имеют следующие значения. Не все свойства являются значимыми для каждого из видов событий. Дополнительную информацию см. в подробных описаниях свойств.

СвойствоЗначение
altKeyЛогическое значение, указывающее на то, была ли нажата клавиша Alt во время события.
bubblesfalse
cancelablefalse
ctrlKeyЛогическое значение, указывающее на то, была ли нажата клавиша Ctrl во время события.
currentTargetОбъект, определяющий прослушиватель события, который выполняет обработку события. Например, если используется myButton.addEventListener() для регистрации прослушивателя события, значением currentTarget является myButton.
itemRendererОбъект средства визуализации элементов для элемента.
shiftKeyЛогическое значение, указывающее на то, была ли нажата клавиша Shift во время события.
targetОбъект, отправивший событие. Это не всегда тот объект, который прослушивает событие. Свойство currentTarget всегда используется для получения доступа к объекту, прослушивающему событие.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
