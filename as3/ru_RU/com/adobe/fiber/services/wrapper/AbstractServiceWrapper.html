Классpublic class AbstractServiceWrapper
НаследованиеAbstractServiceWrapper EventDispatcher Object
Реализует IMXMLObject
Подклассы HTTPServiceWrapper, ManagedRemoteServiceWrapper, RemoteObjectServiceWrapper, WebServiceWrapper

Язык версии: ActionScript 3.0
Версия продукта: Adobe Digital Enterprise Platform Data Services for Java EE 3
Версии среды выполнения: Flash Player 9, AIR 1.1

Класс AbstractServiceWrapper является суперклассом иерархии для созданных оболочек службы. В этом качестве он объявляет функции и переменные, являющиеся общими для всех созданных оболочек службы. Подклассы включают в себя HTTPServiceWrapper, RemoteObjectServiceWrapper, WebServiceWrapper и ManagedRemoteServiceWrapper. Оболочки служб создаются для соответствующих аннотированных служб, объявленных в моделях, созданных с использованием технологии моделирования приложений Adobe. Они обеспечивают удобство использования включаемых основных служб.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
