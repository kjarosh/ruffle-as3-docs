Классы x


В пакете flashx.textLayout.elements содержатся основные классы для представления содержимого текста в Text Layout Framework.


Интерфейсы
 ИнтерфейсОписание
 IConfiguration Доступный только для чтения интерфейс объекта configuration.
 IFormatResolver Интерфейс преобразователя формата.
Классы
 КлассОписание
 BreakElement Класс BreakElement определяет разрыв строки, что обеспечивает создание разрыва строки в тексте без создания нового абзаца.
 Configuration Класс Configuration – основной пункт интеграции между Text Layout Framework и приложением.
 ContainerFormattedElement ContainerFormattedElement является корневым классом для всех элементов блока уровня контейнера, например для объектов DivElement и TextFlow.
 DivElement Класс DivElement определяет элемент для группировки абзацев (объекты ParagraphElement).
 FlowElement Текст в потоке хранится в виде дерева с элементами, представляющими логические деления в тексте.
 FlowGroupElement Класс FlowGroupElement — это базовый класс для объектов FlowElement, которые могут содержать массив дочерних элементов.
 FlowLeafElement Базовый класс для FlowElements, расположенных на самом низком уровне иерархии потока.
 GlobalSettings Конфигурация, применяемая ко всем объектам TextFlow.
 InlineGraphicElement Класс InlineGraphicElement обрабатывает графические объекты, отображающие встроенные в текст элементы.
 InlineGraphicElementStatus Класс InlineGraphicElementStatus определяет набор констант для проверки значения InLineGraphicElement.status.
 LinkElement Класс LinkElement определяет ссылку на URI (Universal Resource Identifier), которая запускается, когда пользователь нажимает на нее.
 LinkState Класс LinkState определяет ряд констант для свойства linkState класса LinkElement.
 ListElement Класс List используется для группировки элементов в списке с нумерацией или без нумерации.
 ListItemElement ListItemElement - это элемент списка.
 OverflowPolicy Класс OverflowPolicy определяет набор констант для свойства overflowPolicy класса IConfiguration.
 ParagraphElement Класс ParagraphElement представляет абзац в иерархии текстового потока.
 ParagraphFormattedElement Класс ParagraphFormattedElement представляет собой абстрактный базовый класс для классов FlowElement, имеющих свойства абзаца.
 SpanElement Класс SpanElement представляет непрерывную последовательность текста, к которой применен отдельный набор атрибутов форматирования.
 SpecialCharacterElement Класс SpecialCharacterElement является абстрактным базовым классом для элементов, представляющих специальные символы.
 SubParagraphGroupElement SubParagraphGroupElement — это элемент группировки для FlowLeafElements и других классов, расширяющих SubParagraphGroupElementBase.
 SubParagraphGroupElementBase Класс SubParagraphGroupElementBase группирует элементы FlowLeafElement.
 TabElement Класс TabElement представляет <tab/> в текстовом потоке.
 TCYElement Класс TCYElement (Tatechuuyoko — та-тей-чу-йо-ко) является подклассом SubParagraphGroupElementBase, позволяющим отображать текст в горизонтальном направлении внутри вертикальной строки.
 TextFlow Класс TextFlow используется для управления всем текстовым содержимым статьи.
 TextRange Доступный только для чтения класс, описывающий диапазон смежного текста.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
