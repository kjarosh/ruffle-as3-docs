Классы x


В пакете flashx.textLayout.formats содержатся классы для представления форматирования текста в Text Layout Framework.


Интерфейсы
 ИнтерфейсОписание
 IListMarkerFormat Этот интерфейс предоставляет доступ для чтения к свойствам ListMarkerFormat.
 ITabStopFormat Данный интерфейс предоставляет доступ для чтения свойств, связанных с позициями табуляции.
 ITextLayoutFormat Данный интерфейс предоставляет доступ для чтения свойств, связанных с FlowElements.
Классы
 КлассОписание
 BackgroundColor Указывает константу для определения того, что значение свойства backgroundColor класса TextLayoutFormat равно transparent.
 BaselineOffset Определяет значения для свойства firstBaselineOffset классов TextLayoutFormat и ContainerFormattedElement.
 BaselineShift Определяет константы для подстрочного или надстрочного индекса в свойстве baselineShift класса TextLayoutFormat.
 BlockProgression Определяет значения для свойства blockProgression класса TextLayouFormat.
 ClearFloats Определяет значения для установки свойства clear класса TextLayoutFormat.
 Direction Определяет значения для установки свойства direction класса TextLayoutFormat.
 Float Определяет значения для свойства float класса InlineGraphicElement.
 FormatValue Указывает значения для определения того, что свойство форматирования наследует родительское значение или генерирует это значение автоматически.
 JustificationRule Определяет значения для установки свойства justificationRule класса TextLayoutFormat.
 LeadingModel Определяет значения для установки свойства leadingModel класса TextLayoutFormat, составленные из допустимых комбинаций основы межстрочного интервала и направления межстрочного интервала.
 LineBreak Определяет значения для настройки свойства lineBreak в TextLayoutFormat для указания способа разбивки строк при расстановке переносов в тексте.
 ListMarkerFormat Определяет формат маркера в ListItemElement.
 ListStylePosition Определяет значения для установки свойства listStylePosition.
 ListStyleType Определяет значения свойства listStyleType объекта TextLayoutFormat.
 Suffix Определяет значения для установки свойства suffix для ListMarkerFormat.
 TabStopFormat Класс TabStopFormat представляет свойства позиции табуляции в абзаце.
 TextAlign Определяет значения для установки свойств textAlign и textAlignLast класса TextLayoutFormat.
 TextDecoration Определяет значения для свойства textDecoration класса TextLayoutFormat.
 TextJustify Определяет значения для установки свойства textJustify класса TextLayoutFormat.
 TextLayoutFormat Класс TextLayoutFormat содержит все свойства форматирования текста.
 TLFTypographicCase Определяет значения для свойства typographicCase класса TextLayoutFormat.
 VerticalAlign Определяет значения для свойства verticalAlign класса TextLayoutFormat.
 WhiteSpaceCollapse Определяет значения для установки свойства whiteSpaceCollapse класса TextLayoutFormat.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
