content

свойство 
content:*

Язык версии: ActionScript 3.0
Версии среды выполнения: Flash Player 10, AIR 1.5

Определяет содержимое маркера.

Допустимые строковые значения:

  • none: без маркера.
  • counter(ordered): маркер отображается.
  • counter(ordered,ListStyleType): маркер отображается, но listStyleType присваивается указанное значение.
  • counters(ordered): начиная с верхнего вышестоящего элемента ListElement создает строку значений упорядоченного счетчика в каждом счетчике, указанном в listStyleType, с разделителем-суффиксом. Используется для выделения числа, например I.1., I.2. и т. п.
  • counters(ordered,"<string>"): аналогично предыдущему значению, но суффикс для каждого упорядоченного счетчика заменяется на <string>.
  • counters(ordered,"<string>",ListStyleType): аналогично предыдущему значению, но listStyleType каждого счетчика замещается указанным значением.

Если undefined, то значением по умолчанию для этого свойства будет "counter(ordered)".

counterIncrement

свойство 
counterIncrement:*

Язык версии: ActionScript 3.0
Версии среды выполнения: Flash Player 10, AIR 1.5

Управляет возрастанием значения счетчика.

Допустимые строковые значения:

  • none: без приращения.
  • ordered: увеличение счетчика на единицу.
  • ordered integer: увеличение счетчика на integer.

Если undefined, то значением по умолчанию для этого свойства будет "ordered 1".

Примечание. Свойство counterIncrement применяется до свойства counterReset.

createListMarkerFormat

()метод
public static function createListMarkerFormat(initialValues:Object):ListMarkerFormat

Язык версии: ActionScript 3.0
Версии среды выполнения: Flash Player 10, AIR 1.5

Создает новый объект ListMarkerFormat. Все параметры настройки имеют пустые значения или, дополнительно, инициализируются из предоставляемого объекта initialValues.

Параметры

initialValues:Object — Дополнительный экземпляр, из которого копируются начальные значения. Если тип объекта равен IListMarkerFormat или ITextLayoutFormat, то значения скопированы. В противном случае параметр initialValues рассматривается как Dictionary или Object и для него выполняется итерация.

Возвращает
ListMarkerFormat — Новый объект ListMarkerFormat.

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
