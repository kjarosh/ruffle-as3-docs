Пакетflashx.textLayout.container
Классpublic class ColumnState
НаследованиеColumnState Object

Язык версии: ActionScript 3.0
Версии среды выполнения: Flash Player 10, AIR 1.5

Класс ColumnState вычисляет размеры и местоположение колонок с использованием ширины контейнера и атрибутов контейнера. Для вычисления значений колонок можно либо создать экземпляры этого класса независимо, либо получить значения колонок, использованных для текста после составления или обновления (изменения) контейнера.

Сведения о конструкторе

ColumnState

()Конструктор
public function ColumnState(blockProgression:String, columnDirection:String, controller:ContainerController, compositionWidth:Number, compositionHeight:Number)

Язык версии: ActionScript 3.0
Версии среды выполнения: Flash Player 10, AIR 1.5

Функция конструктора — создает объект ColumnState. Если значения controller.compositionWidth и controller.compositionHeight равны NaN (не числу), то конструктор измеряет содержимое контейнера для определения фактической ширины и высоты компоновки, поступающей в ColumnState. Для установки значения параметра blockProgression используются константы, определяемые классом flashx.textLayout.formats.BlockProgression. Для установки значения параметра columnDirection используются константы, определяемые классом flashx.textLayout.formats.Direction.

Параметры
blockProgression:String — Направление строк для текстового потока — либо BlockProgression.TB (сверху вниз), либо BlockProgression.RL (справа налево).
 
columnDirection:String — Направление макета колонки для текстового потока — либо Direction.RTL (справа налево), либо Direction.LTR (слева направо).
 
controller:ContainerController — Экземпляр ContainerController, атрибуты которого используются для вычисления значений колонок.
 
compositionWidth:Number — Допустимый горизонтальный размер текста в контейнере в пикселях.
 
compositionHeight:Number — Допустимый вертикальный размер текста в контейнере в пикселях.

public function getColumnAt(index:int):Rectangle

Язык версии: ActionScript 3.0
Версии среды выполнения: Flash Player 10, AIR 1.5

Возвращает область, которую колонка принимает внутри контейнера. Позволяет обращаться к области конкретной колонки.

Параметры

index:int — Относительная позиция колонки по отношению ко всем колонкам в контейнере, если первая колонка имеет позицию 0.

Возвращает
Rectangle — Область указанной колонки.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
