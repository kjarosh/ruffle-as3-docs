Сведения о конструкторе

SelectionFormat

()Конструктор
public function SelectionFormat(rangeColor:uint = 0xffffff, rangeAlpha:Number = 1.0, rangeBlendMode:String = "difference", pointColor:uint = 0xffffff, pointAlpha:Number = 1.0, pointBlendMode:String = "difference", pointBlinkRate:Number = 500)

Язык версии: ActionScript 3.0
Версии среды выполнения: Flash Player 10, AIR 1.5

Создает объект SelectionFormat с указанными свойствами.

SelectionFormat, созданный на основе значений по умолчанию, использует для подсветки черный цвет, значение 1,0 — для альфа-каналов и BlendMode.DIFFERENCE — для режимов наложения. Частота мигания курсора — 500 миллисекунд.

Установка нулевого значения для свойств pointAlpha и rangeAlpha отключает выделение выбранных элементов.

Ненулевое значение частоты мигания используется только при присоединении EditManager к TextFlow.

Параметры
rangeColor:uint (default = 0xffffff) — Цвет для отображения подсветки.
 
rangeAlpha:Number (default = 1.0) — Значение прозрачности для отображения подсветки. Допустимые значения принадлежат диапазону от 0 (полностью прозрачный) до 1 (полностью непрозрачный, используется по умолчанию).
 
rangeBlendMode:String (default = "difference") — Режим наложения для отображения подсветки. Для настройки этого параметра используются константы, определенные в классе BlendMode.
 
pointColor:uint (default = 0xffffff) — Цвет для отображения курсора.
 
pointAlpha:Number (default = 1.0) — Значение прозрачности для отображения курсора. Допустимые значения принадлежат диапазону от 0 (полностью прозрачный) до 1 (полностью непрозрачный, используется по умолчанию).
 
pointBlendMode:String (default = "difference") — Режим наложения для отображения курсора. Для настройки этого параметра используются константы, определенные в классе BlendMode.
 
pointBlinkRate:Number (default = 500) — Скорость мигания курсора (в миллисекундах).

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
