Пакетmx.netmon
Классpublic class NetworkMonitor
НаследованиеNetworkMonitor Object

Язык версии: ActionScript 3.0
Версия продукта: Flex 3
Версии среды выполнения: Flash Player 9, AIR 1.1

NetworkMonitor представляет собой заглушку, используемую инфраструктурой и корпоративными служебными классами для передачи информации для функции монитора сети Flash Builder. NetworkMonitor объявляет число функций Impl, которые должны быть заполнены смешанным классом, предоставленным Flash Builder. Если данные функции не присвоены, заглушка не выполняет каких-либо действий, возвращая значение false или null соответственно. Отдельный экземпляр или объект экземпляра в настоящее время не используется для упрощения кода, вызывающего монитор.

adjustURLRequest

()метод 
public static function adjustURLRequest(urlRequest:URLRequest, rootURL:String, correlationID:String):void

Язык версии: ActionScript 3.0
Версия продукта: Flex 3
Версии среды выполнения: Flash Player 9, AIR 1.1

Корректировка URLRequest для прохождения через монитор. URLRequest, переданный монитору, будет иметь два заголовка:

  • actualHostPort: фактический host:port для URL.
  • correlationID: correlationID для запроса в случае, если он поступил из системы передачи сообщений (запросы Image/Loader должны создавать собственные correlationID).

Параметры

urlRequest:URLRequest — URLRequest, который требуется откорректировать. (Поддерживаются относительные URL-адреса.)
 
rootURL:String — Вычисляет абсолютный URL из относительного URL. При необходимости передайте URL SWF-файла в качестве этого значения.
 
correlationID:String — correlationID, который можно использовать для связанных событий.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
