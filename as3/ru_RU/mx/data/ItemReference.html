Реализует IItemReference

Язык версии: ActionScript 3.0
Версия продукта: Adobe Digital Enterprise Platform Data Services for Java EE 3
Версии среды выполнения: Flash Player 9, AIR 1.1

Экземпляр ItemReference возвращается из методов DataService.getItem() и DataService.createItem() для создания способа управления ссылкой на удаленный объект. Реализация AsyncToken, расширяющая этот класс, предоставляет список респондентов, которым сообщается результат и события отказа, происходящие в операции по извлечению элемента (получение или создание). Связывание может выполняться для значения свойства result, всегда поддерживающего ссылку на управляемый экземпляр. При удалении элемента в другом клиенте свойство result имеет значение null. Если происходит сбой и экземпляр DataService не может извлечь элемент, свойство invalid имеет значение true. Прослушивание события отказа для получения дополнительной информации о причине ошибки.

Для отмены ссылки на этот элемент вызывается метод ItemReference.releaseItem()(). Аналогично выполнению вызова методом DataService.releaseItem(), однако в отличие от метода DataService.releaseItem() этот метод удалят ссылку, а не первый обнаруженный экземпляр указанного типа. Это происходит, если для извлечения одного элемента из различных частей приложения выполняется несколько вызовов методов getItem().

releaseItem

()метод
public function releaseItem(copyStillManagedItems:Boolean = true, enableStillManagedCheck:Boolean = true):void

Язык версии: ActionScript 3.0
Версия продукта: Adobe Digital Enterprise Platform Data Services for Java EE 3
Версии среды выполнения: Flash Player 9, AIR 1.1

Отменяет ссылку на элемент, полученную в результате вызова метода getItem() или createItem(), возвращающего эту ссылку ItemReference. Возможно несколько ссылок ItemReference на один элемент в одном клиенте. Этот метод используется для удаления соответствующей ссылки.

Параметры

copyStillManagedItems:Boolean (default = true) — Логическое значение, указывающее на то, что копия этого элемента должна быть помещена в эту ссылку ItemReference, при этом ссылка на текущий элемент сохраняется. Такая ситуация возможна при нескольких вызовах DataService.getItem() для одного и того же элемента или в случае, если на этот элемент ссылается коллекция, полученная в результате вызова DataService.fill().
 
enableStillManagedCheck:Boolean (default = true) — Логическое значение, определяющее необходимость проверки дополнительных ссылок в элементе.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
