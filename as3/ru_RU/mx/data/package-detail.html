Классы x


В пакете mx.data содержатся классы для работы со службами данных Adobe Digital Enterprise Platform Data Services для Java EE.


Интерфейсы
 ИнтерфейсОписание
 IAccessPrivileges Ограничивающие доступ объекты должны реализовать интерфейс IAccessPrivileges.
 IChangeObject Предоставляет методы и свойства, необходимые при обнаружении локальных конфликтов.
 IItemReference Интерфейс IItemReference предоставляет соглашение для ItemReference.
 IManaged Интерфейс IManaged предоставляет соглашение для управляемого объекта.
 IPagedList IPagedList представляет собой интерфейс маркера для реализаций списков, организующих собственную подкачку страниц.
 ITokenResponder Расширение mx.rpc.IResponder с дополнительным маркером результата, полем AsyncToken.
Классы
 КлассОписание
 AccessPrivileges Инкапсулирует разрешенные права и обеспечивает доступ к простым описаниям разрешенных операций.
 AsyncTokenResponder Подкласс AsyncResponder, отображающий маркер результата.
 CacheDataDescriptor Класс CacheDataDescriptor предоставляет информацию об атрибутах кэшированных данных.
 ChangedItems Описывает данные, измененные с указанного времени.
 Conflict Представляет ошибку, которая произошла вследствие операции обновления.
 ConflictDetector Экземпляр DataService использует экземпляр ConflictDetector в целях определения возникновения конфликта для операции, инициированной из удаленного адреса назначения.
 Conflicts Класс Conflicts представляет список конфликтов.
 DataManager Класс DataManager предоставляет функциональные возможности синхронизации объектов на сервере для всех экземпляров с определенным типом данных.
 DataService Класс DataService предоставляет функциональные возможности верхнего уровня для службы управления данными.
 DataServiceAdapter Класс DataServiceAdapter является абстрактным базовым классом, который используется для реализации нового типа DataService.
 DataServiceOfflineAdapter Обеспечивает сохраняемость автономных кэшированных элементов DataManager.
 DataStore Операции пакетов DataStore для одного или нескольких экземпляров DataService.
 DataStoreEventDispatcher Отправляет события, связанные с хранилищем данных.
 DynamicManagedItem Типизированный управляемый элемент, который является динамическим, должен расширять этот класс, чтобы при изменении динамических свойств изменялись выполняемые события.
 DynamicProperty Описывает имена и тип динамического свойства.
 ItemReference Экземпляр ItemReference возвращается из методов DataService.getItem() и DataService.createItem() для создания способа управления ссылкой на удаленный объект.
 ManagedAssociation Этот класс используется для представления ассоциации как части метаданных управления данными.
 ManagedEntity Этот класс представляет управляемую сущность как часть метаданных управления данными.
 ManagedObjectProxy Класс ManagedObjectProxy предоставляет возможность отслеживания изменений для анонимного элемента, управляемого с помощью DataService.
 ManagedOperation Класс ManagedOperation используется для представления управляемой операции службы для службы RPC, которая использует управление данными на стороне клиента.
 ManagedQuery Экземпляр этого класса используется для определения запроса, управляемого службой управления данными.
 ManagedRemoteService Класс ManagedRemoteService предоставляет доступ к классам Remoting на удаленном сервере приложений с функциями управления данными LCDS.
 ManagedRemoteServiceOperation Операция, специально используемая RemoteObjects.
 ManualSyncConfiguration Этот класс используется при необходимости более четкого контроля совместного использования данных клиентами, чем предоставляет функция autoSyncEnabled.
 MessageBatch Представляет группу изменений: создает, обновляет, удаляет и отправляет сообщения об обновлении коллекции.
 MessageCacheItem Представляет элемент свойства элементов MessageBatch.
 MessagingDataServiceAdapter Реализация DataServiceAdapter для хранилища данных LiveCycle Data Services.
 PageInformation Класс PageInformation предоставляет подробные данные о том, какие страницы коллекции загружены, о количестве страниц и размере текущей страницы.
 PropertySpecifier Этот класс определяет свойства для включения в указанный вызов findItem или fill.
 RPCDataManager RPCDataManager является компонентом, который можно использовать для управления объектами, извлеченными с помощью службы, например RemoteObject, WebService и HTTPMultiService.
 RPCDataServiceAdapter Абстрактный базовый класс, который используется для реализации нового типа DataService.
 SQLiteOfflineAdapter Обеспечивает формат сохраняемости на основе SQLite для автономно кэшируемых элементов DataManager.
 UpdateCollectionRange Класс UpdateCollectionRange предоставляет определенную информацию о диапазоне операций добавления и удаления, выполненные в коллекции.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
