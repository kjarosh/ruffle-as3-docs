AdvancedDataGridItemSelectEvent

()Конструктор
public function AdvancedDataGridItemSelectEvent(type:String, columnIndex:int = -1, dataField:String = "", headerPart:String = "", bubbles:Boolean = false, cancelable:Boolean = false, itemRenderer:IListItemRenderer = null, triggerEvent:Event = null, ctrlKey:Boolean = false, altKey:Boolean = false, shiftKey:Boolean = false)

Язык версии: ActionScript 3.0
Версия продукта: Flex 3
Версии среды выполнения: Flash Player 9, AIR 1.1

Конструктор. Как правило, вызывается управляющим элементом Flex и не используется в коде приложения.

Параметры
type:String — Тип события; указывает действие, вызвавшее событие.
 
columnIndex:int (default = -1) — Индекс текущего выбранного столбца.
 
dataField:String (default = "") — Поле данных текущего выбранного столбца.
 
headerPart:String (default = "") — Поле данных текущего выбранного столбца, если щелчком мыши было инициировано событие HEADER_RELEASE, относящееся к заголовку.
 
bubbles:Boolean (default = false) — Указывает, может ли событие передаваться вверх по иерархии списка отображения.
 
cancelable:Boolean (default = false) — Указывает, существует ли возможность предотвращения поведения, связанного с событием.
 
itemRenderer:IListItemRenderer (default = null) — Объект средства визуализации элементов для элемента.
 
triggerEvent:Event (default = null) — Событие, например событие мыши или клавиатуры, инициировавшее операцию выбора.
 
ctrlKey:Boolean (default = false) — Указывает, была ли нажата клавиша Ctrl во время события.
 
altKey:Boolean (default = false) — Указывает, была ли нажата клавиша Alt во время события.
 
shiftKey:Boolean (default = false) — Указывает, была ли нажата клавиша Shift во время события.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
