Сведения о конструкторе

Responder

()Конструктор
public function Responder(result:Function, fault:Function)

Язык версии: ActionScript 3.0
Версия продукта: Flex 3
Версии среды выполнения: Flash Player 9, AIR 1.1

Создает экземпляр ответчика с указанными обработчиками.

Параметры
result:Function — Функция, которую требуется вызывать при успешном выполнении запроса.
 
fault:Function — Функция, которую требуется вызывать при выполнении запроса с ошибками.
Сведения о методе

fault

()метод
public function fault(info:Object):void

Этот метод вызывается службой при получении ошибки.

Параметры

info:Object — Объект, содержащий информацию, возвращенную из запроса. Несмотря на то что info имеет тип Object, он часто (но не всегда) может являться mx.rpc.events.FaultEvent.

result

()метод 
public function result(data:Object):void

Этот метод вызывается удаленной службой при получении возвращаемого значения.

Параметры

data:Object — Объект, содержащий информацию об ошибке. Несмотря на то что data имеет тип Object, он часто (но не всегда) может являться mx.rpc.events.ResultEvent.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
