Функция, создаваемая в том случае, если при группировке столбцов требуется извлечение данных из вложенных объектов.

Например, если каждая строка выглядит как: 

row = {.., .., Q1: { y2005: 241, y2006:353}};

и группа столбцов определяется как:

     <mx:AdvancedDataGridColumnGroup dataField="Q1">
        <mx:AdvancedDataGridColumn dataField="y2005">
        <mx:AdvancedDataGridColumn dataField="y2006">
     </mx:AdvancedDataGridColumnGroup>

Функция для столбца, соответствующая y2005, определена как:

     function foo():String
     {
        return row["Q1"]["2005"];
     }

Функция также обеспечивает обработку в случае, если в любом столбце или группе столбцов вместо поля данных используется функция метки.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
