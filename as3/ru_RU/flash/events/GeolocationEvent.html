Сведения о конструкторе

GeolocationEvent

()Конструктор
public function GeolocationEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, latitude:Number = 0, longitude:Number = 0, altitude:Number = 0, hAccuracy:Number = 0, vAccuracy:Number = 0, speed:Number = 0, heading:Number = 0, timestamp:Number = 0)

Язык версии: ActionScript 3.0
Версии среды выполнения: AIR 2, Flash Lite 4

Создает объект GeolocationEvent, который содержит информацию о местоположении устройства. Объекты Event передаются прослушивателям событий в качестве параметров.

Параметры
type:String — Указывает тип события. Прослушиватели событий могут получать эту информацию через унаследованное свойство type. Существует только один тип события обновления: GeolocationEvent.UPDATE.
 
bubbles:Boolean (default = false) — Определяет, может ли событие восходить по цепочке. Прослушиватели событий могут получать эту информацию через унаследованное свойство bubbles.
 
cancelable:Boolean (default = false) — Определяет, можно ли отменить объект события. Прослушиватели событий могут получать эту информацию через унаследованное свойство cancelable.
 
latitude:Number (default = 0) — Возвращает широту в градусах. Значения находятся в следующем диапазоне: (-90 <= широта <= +90).
 
longitude:Number (default = 0) — Возвращает долготу в градусах. Значения находятся в следующем диапазоне: (-180 <= долгота <= +180).
 
altitude:Number (default = 0) — Возвращает высотную отметку в метрах.
 
hAccuracy:Number (default = 0) — Возвращает точность контуров в метрах.
 
vAccuracy:Number (default = 0) — Возвращает точность нивелирования в метрах.
 
speed:Number (default = 0) — Возвращает скорость в м/с.
 
heading:Number (default = 0) — Возвращает направление движения (относительно истинного севера) в целочисленных градусах.
 
timestamp:Number (default = 0) — Определяет метку времени обновления географического местоположения.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
