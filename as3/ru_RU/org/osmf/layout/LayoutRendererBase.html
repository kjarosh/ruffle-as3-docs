Определяет контейнер, в соответствии с которым средство визуализации вычисляет значения размера и местоположения целевых объектов. Кроме того, средство визуализации управляет целевыми объектами, добавляемыми или удаляемыми в качестве нижестоящих элементов списка отображения установленного контейнера.

addTarget

()метод
public final function addTarget(target:org.osmf.layout:ILayoutTarget):org.osmf.layout:ILayoutTarget

Язык версии: ActionScript 3.0
Версия продукта: OSMF 1.0
Версии среды выполнения: Flash Player 10, AIR 1.5

Метод для добавления целевого объекта в список объектов средства визуализации макетов, для которого вычисляется размер и местоположение. При добавлении целевого объекта связанный экранный объект помещается в список отображения контейнера средства визуализации.

Параметры

target:org.osmf.layout:ILayoutTarget — Добавляемый целевой объект.

Возвращает
org.osmf.layout:ILayoutTarget

Выдает
IllegalOperationError — когда указанный целевой объект имеет значение null или уже является целевым объектом средства визуализации.

Метод, запускающий непосредственное повторное вычисление местоположения и размера всех целевых объектов, назначенных в средстве визуализации. Реализующий класс может не выполнять повторное вычисление, если средство визуализации действительно со времени последнего выполнения визуализации.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
