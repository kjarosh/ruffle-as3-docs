Пакетorg.osmf.display
Классpublic final class ScaleMode
НаследованиеScaleMode Object

Язык версии: ActionScript 3.0
Версия продукта: OSMF for Flex 4.0
Версии среды выполнения: Flash Player 10, AIR 1.5

Примечание. Этот класс обеспечивает обратную совместимость с Flex 4 и предназначен для использования только с Flex 4. Вместо инфраструктуры OSMF 1.0 рекомендуется использовать org.osmf.layout.ScaleMode.

Класс ScaleMode управляет макетом из одной части содержимого в контейнере. Четыре значения перечисления включают NONE, STRETCH, LETTERBOX и ZOOM.

LETTERBOX

Константа
public static const LETTERBOX:String = "letterbox"

Язык версии: ActionScript 3.0
Версия продукта: OSMF for Flex 4.0
Версии среды выполнения: Flash Player 10, AIR 1.5

LETTERBOX устанавливает ширину и высоту содержимого с максимальным соответствием ширине и высоте контейнера и выдерживанием пропорций. Содержимое растягивается до максимума границ контейнера, при этом в случае необходимости в контейнер добавляются интервалы для поддержания пропорций.

ZOOM

ZOOMподобен LETTERBOX за исключением того, что ZOOM растягивает содержимое за пределы границ контейнера для удаления интервалов, требуемых для поддержания пропорций. В результате используются все границы контейнера, но также возможно некоторое усечение содержимого.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
