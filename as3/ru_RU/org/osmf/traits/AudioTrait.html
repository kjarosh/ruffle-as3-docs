Классpublic class AudioTrait
НаследованиеAudioTrait MediaTraitBase EventDispatcher Object

Язык версии: ActionScript 3.0
Версия продукта: OSMF 1.0
Версии среды выполнения: Flash Player 10, AIR 1.5

AudioTrait определяет интерфейс характеристик для мультимедийного содержимого с аудиозаписями. Также он может использоваться в качестве базового класса для специфичного класса AudioTrait.

Используйте метод MediaElement.hasTrait(MediaTraitType.AUDIO_TRAIT) для запросов о том, имеет ли элемент мультимедийного содержимого эту характеристику. Если hasTrait(MediaTraitType.AUDIO_TRAIT) возвращает true, используйте метод MediaElement.getTrait(MediaTraitType.AUDIO_TRAIT) для получения объекта этого типа.

Версия продукта: OSMF 1.0
Версии среды выполнения: Flash Player 10, AIR 1.5

Панорамирование звука слева направо. Значения в диапазоне от -1 (полное панорамирование слева) до 1 (полное панорамирование справа).

При передаче значения выше 1 устанавливается значение 1. При передаче значения ниже -1 устанавливается значение -1.

Значение по умолчанию – нуль.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
