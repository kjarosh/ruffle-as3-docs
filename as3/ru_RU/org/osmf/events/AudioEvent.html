public function AudioEvent(type:String, bubbles:Boolean, cancelable:Boolean, muted:Boolean = false, volume:Number = NaN, pan:Number = NaN)

Язык версии: ActionScript 3.0
Версия продукта: OSMF 1.0
Версии среды выполнения: Flash Player 10, AIR 1.5

Конструктор.

Параметры
type:String — Тип события.
 
bubbles:Boolean — Указывает, может ли событие передаваться вверх по иерархии списка отображения.
 
cancelable:Boolean — Указывает, существует ли возможность предотвращения поведения, связанного с событием.
 
muted:Boolean (default = false) — Указывает, выполняется ли в настоящий момент буферизация характеристики.
 
volume:Number (default = NaN) — Новый bufferTime для характеристики.
 
pan:Number (default = NaN) — Предыдущая громкость.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
