类public class VideoElement
继承VideoElement LightweightVideoElement LoadableElementBase MediaElement EventDispatcher Object

语言版本: ActionScript 3.0
产品版本: OSMF 1.0
运行时版本: Flash Player 10, AIR 1.5

VideoElement 是专门为视频播放创建的媒体元素。它支持流格式和渐进式格式以及 HTTP 流和 MBR 流（对于 RTMP 和 HTTP）。

相对于 LightweightVideoElement，VideoElement 功能更为齐全。LightweightVideoElement 仅支持视频传送模式的一个子集（具体来说是指渐进式和简单的 RTMP 流模式），而 VideoElement 支持所有视频传送模式。

VideoElement 使用 NetLoader 类加载和卸载其媒体。需要自定义视频加载逻辑的开发人员可将自己的加载器传递到 LightweightVideoElement 构造函数。这些加载器应是 NetLoader 的子类。

创建和使用 LightweightVideoElement 的基本步骤：

  1. 创建新的 URLResource，该 URLResource 指向视频流的 URL 或包含要加载的视频的文件。
  2. 创建新的 NetLoader。
  3. 创建新的 VideoElement，将 NetLoader 和 URLResource 作为参数传递。
  4. 创建新的 MediaPlayer。
  5. 将 VideoElement 指定给 MediaPlayer 的 media 属性。
  6. 使用 MediaPlayer 的方法、属性和事件控制媒体。
  7. 完成有关 VideoElement 的操作后，将 MediaPlayer 的 media 属性设置为 null。这将卸载 VideoElement。

VideoElement 支持 Flash Media 标记身份验证，以通过 NetConnection 传递身份验证标记。

VideoElement 支持 Flash Player 的 DRM 实现。请注意，在开始播放之前，此元素中 DRMTrait 的 startDate、endDate 和 period 属性对应于凭证有效性。开始播放后，这些属性对应于播放时间窗口（可在 flash.net.drm.DRMVoucher 中找到）。

public function VideoElement(resource:MediaResourceBase = null, loader:NetLoader = null)

语言版本: ActionScript 3.0
产品版本: OSMF 1.0
运行时版本: Flash Player 10, AIR 1.5

构造函数。

参数
resource:MediaResourceBase (default = null) — 指向 VideoElement 将使用的视频源的 URLResource。对于动态流内容，请使用 DynamicStreamingResource。
 
loader:NetLoader (default = null) — 用于加载视频的 NetLoader。如果为 null，将基于资源类型创建适当的 NetLoader。

引发
ArgumentError — 如果资源不是 URLResource。
示例 如何使用本示例
VideoElementExample.as

package
{
    import flash.display.Sprite;
    import flash.display.StageAlign;
    import flash.display.StageScaleMode;
    
    import org.osmf.elements.VideoElement;
    import org.osmf.media.MediaPlayer;
    import org.osmf.media.MediaPlayerSprite;
    import org.osmf.media.URLResource;
    import org.osmf.net.DynamicStreamingItem;
    import org.osmf.net.DynamicStreamingResource;
    
    public class VideoElementExample extends Sprite
    {
        public function VideoElementExample()
        {
            super();
            
            stage.scaleMode = StageScaleMode.NO_SCALE;
            stage.align = StageAlign.TOP_LEFT;
            
            var mediaPlayerSprite:MediaPlayerSprite = new MediaPlayerSprite();
            var videoElement:VideoElement = new VideoElement();
            var dynResource:DynamicStreamingResource = new DynamicStreamingResource("rtmp://cp67126.edgefcs.net/ondemand");
            
            dynResource.streamItems = Vector.<DynamicStreamingItem>(
                [     new DynamicStreamingItem("mp4:mediapm/ovp/content/demo/video/elephants_dream/elephants_dream_768x428_24.0fps_408kbps.mp4", 408, 768, 428)
                    , new DynamicStreamingItem("mp4:mediapm/ovp/content/demo/video/elephants_dream/elephants_dream_768x428_24.0fps_608kbps.mp4", 608, 768, 428)
                    , new DynamicStreamingItem("mp4:mediapm/ovp/content/demo/video/elephants_dream/elephants_dream_1024x522_24.0fps_908kbps.mp4", 908, 1024, 522)
                    , new DynamicStreamingItem("mp4:mediapm/ovp/content/demo/video/elephants_dream/elephants_dream_1024x522_24.0fps_1308kbps.mp4", 1308, 1024, 522)
                    , new DynamicStreamingItem("mp4:mediapm/ovp/content/demo/video/elephants_dream/elephants_dream_1280x720_24.0fps_1708kbps.mp4", 1708, 1280, 720)
                ]);
            
            videoElement.resource = dynResource;
            
            addChild(mediaPlayerSprite);
            mediaPlayerSprite.media = videoElement;    
        }    
    }
}
