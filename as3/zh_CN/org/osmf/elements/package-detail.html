org.osmf.elements 包中包含所有 MediaElement 实现。存在核心媒体类型（视频、音频、图像和 SWF）的 MediaElement。此外，还存在用作更复杂的媒体演示文稿的构造块的 MediaElement 实现。

 类说明
 AudioElement AudioElement 是专门为音频播放创建的媒体元素。
 BeaconElement BeaconElement 是一个媒体元素，该元素将“播放”操作映射到 URL 的请求（通过 HTTP GET）。
 CompositeElement CompositeElement 是一个媒体元素，可封装其它更精细的媒体元素的集合。

 F4MElement F4MElement 是用于从 F4M 文件加载媒体的媒体元素。
 F4MLoader F4MLoader 是能够加载 F4M 文件的加载器。
 HTMLElement HTMLElement 是表示位于 HTML 区域内独立于 Flash SWF 的一部分媒体的媒体元素。
 ImageElement ImageElement 是专门为显示静止图像创建的媒体元素。
 ImageLoader ImageLoader 是能够加载和显示图像文件的加载器。
 LightweightVideoElement LightweightVideoElement 是专门为视频播放创建的媒体元素。

 ManifestLoaderBase ManifestLoader 是基本加载器类，能够从 F4M 文件或直接字符串表示形式的清单加载 Flash Media 清单的对象。
 ParallelElement ParallelElement 是一个媒体组合，其元素平行（并行）显示。

[ X ]为什么显示为英语？
《ActionScript 3.0 参考》中的内容以英语显示

《ActionScript 3.0 参考》中的部分内容未翻译成所有语言。当某个语言元素未翻译时，将显示为英语。例如，ga.controls.HelpBox 类未翻译成任何语言。因此在简体中文版的参考中，ga.controls.HelpBox 类显示为英语。
