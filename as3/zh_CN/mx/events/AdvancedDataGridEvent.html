ITEM_EDIT_BEGIN

常量 
public static const ITEM_EDIT_BEGIN:String = "itemEditBegin"

语言版本: ActionScript 3.0
产品版本: Flex 3
运行时版本: Flash Player 9, AIR 1.1

AdvancedDataGridEvent.ITEM_EDIT_BEGIN 常量可为 itemEditBegin 事件定义事件对象的 type 属性的值，指示项目已准备就绪，可进行编辑。

此事件的默认侦听器执行下列操作：

  • 通过调用 createItemEditor() 方法创建项目编辑器对象。
  • 将 data 属性从项目复制到编辑器。默认情况下，项目编辑器对象是 TextInput 控件的实例。您可以使用 list 控件的 itemEditor 属性指定自定义项目编辑器类。
  • 设置 list 控件的 itemEditorInstance 属性来引用项目编辑器实例。

您可以为此事件编写事件侦听器以修改传递到项目编辑器的数据。例如，您可以修改项目编辑器使用的数据、数据格式或其他信息。
