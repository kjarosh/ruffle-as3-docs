类public class SWFBridgeEvent
继承SWFBridgeEvent Event Object

语言版本: ActionScript 3.0
产品版本: Flex 3
运行时版本: Flash Player 9, AIR 1.1

这是在不同安全沙箱中的应用程序之间发送的事件。该事件让其他沙箱中的对象了解此沙箱中发生的情况。这些事件实际只是用于说明用途，这一点与 SWFBridgeRequest 不同，后者请求对象代表自身进行一些操作。

此类定义引用“activation”的事件常量。在此示例中，activation 表示应用程序中可以接收键盘输入焦点的组件之一。激活应用程序时，最后具有焦点的组件将再次获得焦点。

BRIDGE_WINDOW_ACTIVATE

常量

语言版本: ActionScript 3.0
产品版本: Flex 3
运行时版本: Flash Player 9, AIR 1.1

向父桥或沙箱根调度，用于向其通知已激活了一个窗口。对于兼容的应用程序，data 属性是具有 window 和 notifier 这两个属性的对象。data.window 属性是已激活的 SystemManager 代理。对于不受信任的应用程序，data.window 属性是窗口的字符串 ID。data.notifier 属性是调度事件的应用程序的桥。可能不通过桥而向沙箱根直接调度事件，因此 event.target 可能不是调度事件的应用程序的桥。
