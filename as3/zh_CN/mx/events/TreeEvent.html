类public class TreeEvent
继承TreeEvent Event Object

语言版本: ActionScript 3.0
产品版本: Flex 3
运行时版本: Flash Player 9, AIR 1.1

TreeEvent 类代表传递到与树中活动（如打开或关闭树分支时）相关联的事件的事件侦听器的事件对象。

构造函数详细信息

TreeEvent

()构造函数
public function TreeEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, item:Object = null, itemRenderer:IListItemRenderer = null, triggerEvent:Event = null)

语言版本: ActionScript 3.0
产品版本: Flex 3
运行时版本: Flash Player 9, AIR 1.1

构造函数。通常由 Flex Tree 控件调用；但不在应用程序代码中使用。

参数
type:String — 事件类型；指示引发事件的动作。
 
bubbles:Boolean (default = false) — 指定该事件是否可以在显示列表层次结构得到冒泡处理。
 
cancelable:Boolean (default = false) — 指定是否可以防止与事件相关联的行为。
 
item:Object (default = null) — 应用此事件的 Tree 节点（项目）。
 
itemRenderer:IListItemRenderer (default = null) — 单元格的项呈示器对象。
 
triggerEvent:Event (default = null) — 如果打开或关闭节点以响应用户操作，则指示输入操作的类型。

ITEM_CLOSE

常量
public static const ITEM_CLOSE:String = "itemClose"

语言版本: ActionScript 3.0
产品版本: Flex 3
运行时版本: Flash Player 9, AIR 1.1

TreeEvent.ITEM_CLOSE 事件类型常量指示树分支已关闭或折叠。

此事件类型的事件对象属性包含以下值。并非所有属性对于各种事件都有意义。有关详细信息，请参阅详细的属性说明。

属性值
itemRenderer关闭的节点的 ListItemRenderer
bubblesfalse
cancelablefalse
currentTarget用于定义处理该事件的事件侦听器的对象。例如，如果您使用 myButton.addEventListener() 注册某个事件侦听器，则 myButton 为 currentTarget 的值。
triggerEvent如果节点关闭以响应用户操作，则确定此操作为键盘操作还是鼠标操作。
item关闭的 Tree 项目（节点）
target调度该事件的 Object；它不一定是侦听该事件的 Object。使用 currentTarget 属性始终可以访问侦听事件的 Object。
