语言版本: ActionScript 3.0

运行时版本: Flash Player 9, AIR 1.1

此消息用于传输在托管对象或集合上发生的操作。此类消息在订阅远程目标的客户端之间和群集内的多个服务器节点之间进行传递。此消息的负载描述操作的所有相关详细信息。此信息用于复制更新和检测冲突。消息的主体基于操作设置将包含不同的数据。

此常量用于访问已更改的属性名称列表。在 operation 为 UPDATE_OPERATION 的情况下使用此属性时，请对消息的 body 属性使用方括号。

相关 API 元素

UPDATE_OPERATION

示例  ( 如何使用本示例 ) 
         var msg:DataMessage = DataMessage(event.message);
         if (msg.operation == DataMessage.UPDATE_OPERATION)
         {
            // look at each change and display the old and new values
            var propertyNames:Array = msg.body[DataMessage.UPDATE_BODY_CHANGES];
            var newValues:Object = msg.body[DataMessage.UPDATE_BODY_NEW];
            var oldValues:Object = msg.body[DataMessage.UPDATE_BODY_PREV];
            var propertyName:String;
            for (var i:int=0; i<propertyNames.length; i++)
            {
                propertyName = propertyNames[i];
                trace("Property '"+ propertyName + "' was changed from '" +
                      oldValues[propertyName] + "' to '"+ newValues[propertyName] +
                      "'.");
            }
         }

         var msg:DataMessage = DataMessage(event.message);
         if (msg.operation == DataMessage.UPDATE_OPERATION)

            var newValues:Object = msg.body[DataMessage.UPDATE_BODY_NEW];

UPDATE_OPERATION

常量 
public static const UPDATE_OPERATION:uint = 3

语言版本: ActionScript 3.0
产品版本: Adobe Digital Enterprise Platform Data Services for Java EE 3
运行时版本: Flash Player 9, AIR 1.1

此操作指示已执行对数据对象的更新。更新项时，客户端将接收并发送此操作。

消息的 body 属性是包含有关所发生更新的特定信息的数组。该数组内的 DataMessage.UPDATE_BODY_CHANGES 位置包含在更新期间更改的属性名称的数组。

该数组内的 DataMessage.UPDATE_BODY_NEW 位置包含处于新修改状态的对象。该数组内的 DataMessage.UPDATE_BODY_PREV 位置包含处于更新前的状态的对象。
