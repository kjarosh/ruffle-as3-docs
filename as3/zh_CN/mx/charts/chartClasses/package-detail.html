mx.charts.chartClasses 包中包含 Flex 制图组件的基类。


接口
 接口说明
 IAxis IAxis 类是用于定义图表轴的标签、刻度线和数据定位属性的抽象接口。
 IAxisRenderer IAxis 类是用于定义图表轴的标签、刻度线和数据定位属性的抽象接口。
 IBar 任何可以垂直构建群集的系列都可以实现 IBar 接口，如 BarSeries。
 IChartElement IChartElement 定义在图表的数据空间中显示 UIComponent 所需的一组基本属性和方法。
 IChartElement2 IChartElement2 定义为了使 UIComponent 可在图表的数据空间中表示所需的基本属性和方法组。
 IColumn 可以沿垂直轴进行群集化的 ChartElement 都应实现此接口。
 IDataCanvas 此接口仅供内部使用。
 IStackable 可以进行堆叠的任何系列都可以实现 IStackable 接口。
 IStackable2 IStackable2 接口由可以堆栈的任意系列实现。
类
 类说明
 AxisBase AxisBase 类可作为 Flex 支持的各种轴类型的基类。
 AxisLabelSet AxisLabelSet 给出实现 IAxis 而生成的标签和刻度数据。
 BoundedValue 一个绑定值，用于代表图表元素要在屏幕上显示的数据点。
 CartesianCanvasValue 定义对象在数据画布上的位置。
 CartesianChart CartesianChart 类是公用图表类型的基类。
 CartesianDataCanvas 此类允许您使用图形 API 添加图形元素，如线条、椭圆和其他形状。
 CartesianTransform CartesianTransform 对象表示一组轴，用于实现数据值与 x/y 笛卡尔屏幕坐标之间的双向映射。
 ChartBase ChartBase 类是 Flex 组件体系结构和 DualStyleObject 体系结构之间的链接。
 ChartElement ChartElement 类是可见图表元素的基类。
 ChartItemDragProxy 从图表控件中拖动项目时使用的默认拖动代理。
 ChartLabel 在图表控件上绘制数据标签。
 ChartState 描述图表的当前状态。
 DataDescription ChartElement 将使用 DataDescription 结构来描述它们向 Axis 对象显示的数据特征，这些对象将根据图表中显示的数据自动生成值。
 DataTip DataTip 控件为图表用户提供有关数据点的信息。
 DataTransform DataTransform 对象表示包含字型的图表的部分，可以实现值与屏幕坐标之间的相互转换。
 DateRangeUtilities 由 DateTimeAxis 使用的一组已禁用的日期范围实用程序
 DualStyleObject 对于需要在客户端开发人员控制之外分配类选择器的组件，DualStyleObject 类可以充当这些组件的基类。
 GraphicsUtilities 各种图表类使用的一组内部图形呈示实用工具。
 HLOCSeriesBase HLOCSeriesBase 是 HLOCSeries 和 CandlestickSeries 这两种财务系列类型的基类。
 InstanceCache InstanceCache 是一个实用程序，用于控制一组（n 个）对象实例的创建和管理任务，其中 n 可以随意更改。
 LegendData LegendData 结构，供图表用于描述应显示在自动生成的图例中的项目。
 NumericAxis NumericAxis 类作为轴类型的通用基类，此轴类型表示定义的最小值和最大值之间的连续范围。
 PolarChart PolarChart 控件可用作基于极坐标的 circular 图表的基类。
 PolarDataCanvas 通过 PolarDataCanvas 类，用户可以相对于数据坐标而不是屏幕坐标来使用图形 API。
 PolarTransform PolarTransform 对象给出一组轴，用于在数据值与角度/距离极坐标之间进行双向映射。
 RenderData RenderData 结构，供图表元素用于存储完全呈示图表所需的所有相关的值和数据。
 Series Series 类是允许为图表控件指定数据系列的类的基类。
 StackedSeries 堆叠的系列用作所有堆叠组系列（BarSet、ColumnSet 和 AreaSet）的公用基类。
