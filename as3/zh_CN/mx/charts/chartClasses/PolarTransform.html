类public class PolarTransform
继承PolarTransform DataTransform EventDispatcher Object

语言版本: ActionScript 3.0
产品版本: Flex 3
运行时版本: Flash Player 9, AIR 1.1

PolarTransform 对象给出一组轴，用于在数据值与角度/距离极坐标之间进行双向映射。

通常不需要与 PolarTransform 对象进行交互。转换由内置的图表类型自动创建，由其中包含的系列用于将数据转换为显示的坐标。

transformCache

()方法 
override public function transformCache(cache:Array, aField:String, aConvertedField:String, rField:String, rConvertedField:String):void

语言版本: ActionScript 3.0
产品版本: Flex 3
运行时版本: Flash Player 9, AIR 1.1

将一组表示数据的数值映射到屏幕坐标。此方法假设值都为数字，因此，必须使用 mapCache() 方法预先转换所有非数字的值。

参数

cache:Array — 字段中包含数据值的对象数组。此函数还使用此数组来存储转换后的数值。
 
aField:String — 用于存储 x 轴的数据值的字段。
 
aConvertedField:String — 用于存储映射的 x 屏幕坐标的字段。
 
rField:String — 用于存储 y 轴的数据值的字段。
 
rConvertedField:String — 用于存储映射的 y 屏幕坐标的字段。
