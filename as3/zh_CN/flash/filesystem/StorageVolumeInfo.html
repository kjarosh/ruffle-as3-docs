包flash.filesystem
类public final class StorageVolumeInfo
继承StorageVolumeInfo EventDispatcher Object

语言版本: ActionScript 3.0
运行时版本: AIR 2

装载或卸载存储卷时，StorageVolumeInfo 对象将调度 StorageVolumeChangeEvent 对象。StorageVolume.storageVolume 静态属性引用调度事件的单个 singleton StorageVolumeInfo 对象。StorageVolumeInfo 类也定义 getStorageVolumes 方法，用于列出当前安装的存储卷。

AIR 配置文件支持：所有桌面操作系统均支持此功能，但所有 AIR for TV 设备均不支持此功能。移动设备也不支持此功能。您可以使用 StorageVolumeInfo.isSupported 属性在运行时测试是否受支持。有关在多个配置文件之间支持 API 的详细信息，请参阅 AIR 配置文件支持。

在现代的 Linux 发行版中，StorageVolumeInfo 对象仅对在特定位置装载的物理设备和网络驱动器调度 storageVolumeMount 和 storageVolumeUnmount 事件。

返回与当前装载的存储卷对应的 StorageVolume 对象的矢量。

在现代的 Linux 发行版中，此方法返回与在特定位置装载的物理设备和网络驱动器对应的对象。

示例  ( 如何使用本示例 )

下列代码列出了每个装载的存储卷的根目录的本机路径： 
var volumes:Vector.<StorageVolume> = new Vector.<StorageVolume>;
volumes = StorageVolumeInfo.storageVolumeInfo.getStorageVolumes();
for (var i:int = 0; i < volumes.length; i++)
{
    trace(volumes[i].rootDirectory.nativePath);
}
