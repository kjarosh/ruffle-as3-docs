flash.system 包中包含用于访问系统级功能（例如安全、垃圾回收等）的类。


函数
 函数说明
 fscommand 使 SWF 文件能够与 Flash Player 或承载 Flash Player 的程序（如 Web 浏览器）进行通讯。
类
 类说明
 ApplicationDomain ApplicationDomain 类是分散的类定义组的一个容器。
 Capabilities Capabilities 类提供一些属性，这些属性描述了承载应用程序的系统和运行时。
 ImageDecodingPolicy     ImageDecodingPolicy 类为 LoaderContext 类中的 imageDecodingPolicy 提供值。
 IME 使用 IME 类，您可以在客户端计算机上运行的 Flash 运行时应用程序中直接对操作系统的输入法编辑器 (IME) 进行操作。
 IMEConversionMode 这个类包含与 IME.conversionMode 属性配合使用的常量。
 JPEGLoaderContext JPEGLoaderContext 类包含一个属性，可以在加载 JPEG 图像时启用消除马赛克的滤镜。
 LoaderContext LoaderContext 类提供多种选项，以使用 Loader 类来加载 SWF 文件和其他媒体。
 MessageChannel MessageChannel 类提供了供 worker 用来与其他 worker 进行通信的机制。
 MessageChannelState 此类定义了一些常量，它们表示 MessageChannel 类的 state 属性的可能值。
 Security 通过使用 Security 类，可以指定不同域中的内容相互通信的方式。
 SecurityDomain SecurityDomain 类代表当前安全性“沙箱”，也称为“安全域”。
 SecurityPanel SecurityPanel 类提供一些值，用来指定您希望显示的“安全设置”面板。
 System System 类包含与本地设置和操作相关的属性。
 SystemUpdater 通过 SystemUpdater 类，您可以更新 Flash Player 的模块（如 Adobe Access 的 DRM 模块）以及 Flash Player 本身。
 SystemUpdaterType SystemUpdaterType 类为系统更新提供常量。
 TouchscreenType TouchscreenType 类是枚举类，为不同类型的触摸屏提供值。
 Worker Worker 对象代表一个 worker，这是 Flash 运行时的虚拟实例。
 WorkerDomain WorkerDomain 类提供了一种创建和访问 Worker 对象的方法。
 WorkerState 此类定义了一些常量，它们表示 Worker 类的 state 属性的可能值。
