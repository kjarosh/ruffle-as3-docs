包flash.security
类public class X509Certificate
继承X509Certificate Object

语言版本: ActionScript 3.0
运行时版本: Flash Player 11, AIR 3

X509Certificate 类表示 X.509 证书。此类定义 RFC2459 中指定的 X.509 属性。在成功调用 SecureSocket.connect() 之后，服务器的证书数据将作为 X509Certificate 实例存储在 SecureSocket.serverCertificate 属性中。

在建立安全套接字连接之后，可使用此类检查服务器证书。此类中的属性提供对 X.509 证书最常用属性的访问。如果必须访问服务器证书的其他部分（例如，其扩展名），则 encoded 属性中提供完整证书。encoded 属性中存储的证书是使用 DER 编码的。

version

属性 
version:uint  [只读]

语言版本: ActionScript 3.0
运行时版本: Flash Player 11, AIR 3

提供证书格式的版本号。此属性指示证书有扩展名、唯一标识符还是仅有基本字段。

  • version = 2：指示 X.509 版本 3 - 有扩展名
  • version = 1：指示 X.509 版本 2 - 没有扩展名，但有唯一标识符。
  • version = null：指示 X.509 版本 1 - 仅有基本证书字段。
