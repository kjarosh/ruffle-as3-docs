SoftKeyboardEvent

()构造函数
public function SoftKeyboardEvent(type:String, bubbles:Boolean, cancelable:Boolean, relatedObjectVal:InteractiveObject, triggerTypeVal:String)

语言版本: ActionScript 3.0
运行时版本: Flash Player 10.2, AIR 2.6

创建包含有关软键盘激活和取消激活事件的信息的事件对象。将 Event 对象作为参数传递给事件侦听器。

参数
type:String — 事件的类型为常量值（如 SOFT_KEYBOARD_ACTIVATE）。事件侦听器可以通过继承的 type 属性访问此信息。
 
bubbles:Boolean — 确定 Event 对象是否参与事件流的冒泡阶段。事件侦听器可以通过继承的 bubbles 属性访问此信息。
 
cancelable:Boolean — 确定是否可以取消 Event 对象。事件侦听器可以通过继承的 cancelable 属性访问此信息。
 
relatedObjectVal:InteractiveObject — 对与事件相关的显示列表对象的引用。
 
triggerTypeVal:String — 表示键盘事件由应用程序还是用户触发。

常量

语言版本: ActionScript 3.0

SoftKeyboardEvent.SOFT_KEYBOARD_ACTIVATE 常量定义软键盘显示时 SoftKeyboardEvent 对象的 type 属性值。对于此事件，cancellable = false。

此事件具有以下属性：

属性值
typeSOFT_KEYBOARD_ACTIVATE
bubblestrue
cancelablefalse；没有要取消的默认行为。
relatedObjectVal对与事件相关的显示列表对象的引用。
triggerTypeVal表示键盘事件由应用程序还是用户触发。
currentTarget当前正在使用某个事件侦听器处理 Event 对象的对象。
