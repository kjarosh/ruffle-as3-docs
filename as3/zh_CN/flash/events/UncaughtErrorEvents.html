类public class UncaughtErrorEvents
继承UncaughtErrorEvents EventDispatcher Object

语言版本: ActionScript 3.0
运行时版本: Flash Player 10.1, AIR 2

UncaughtErrorEvents 类提供了一种接收未被捕获的错误事件的方法。如果发生运行时错误，且未在代码中检测到该错误并对其进行处理，则此类的实例调度 uncaughtError 事件。

使用下列属性访问 UncaughtErrorEvents 实例：

  • LoaderInfo.uncaughtErrorEvents：检测同一 SWF 中定义的代码中的未被捕获的错误。
  • Loader.uncaughtErrorEvents：检测 Loader 对象加载的 SWF 中定义的代码中的未被捕获的错误。

要直接捕获错误并防止发生未被捕获的错误事件，请执行以下操作：

  • 使用 try..catch 块来隔离可能会引发同步错误的代码
  • 当执行在发生错误时调度某事件的操作时，为该错误事件注册一个侦听器

如果 Loader 对象加载的内容是 AVM1 (ActionScript 2) SWF 文件，则 AVM1 SWF 文件中未被捕获的错误不会导致出现 uncaughtError 事件。此外，在 HTMLLoader 对象（包括 Flex HTML 控件）中加载的 HTML 内容中的 JavaScript 错误不会导致出现 uncaughtError 事件。
