Pakietflashx.textLayout.edit
Klasapublic class ElementRange
DziedziczenieElementRange Object

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: Flash Player 10, AIR 1.5

Klasa ElementRange reprezentuje zakres obiektów wybranych w przepływie tekstu.

Elementy początkowe (takie jak firstLeaf) są zawsze mniejsze lub równe elementom końcowym (w tym przypadku lastLeaf) dla każdej pary wartości w zakresie elementów.

Atrybuty formatu znaków w zakresie.

Jeśli zakres obejmuje więcej niż jeden element FlowElement (co oznacza, że w zakresie może istnieć więcej niż jeden format znaków), wówczas zwracany jest format pierwszego obiektu FlowElement.

getCommonCharacterFormat

public function getCommonCharacterFormat():flashx.textLayout.formats:TextLayoutFormat

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: Flash Player 10, AIR 1.5

Pobiera atrybuty formatu znaków, które są wspólne dla wszystkich znaków w zakresie tekstu lub bieżącym zaznaczeniu.

W przypadku atrybutów formatu, które nie mają tej samej wartości dla wszystkich znaków w zakresie elementów, ustawiana jest wartość null w zwróconym wystąpieniu klasy TextLayoutFormat.

Zwraca
flashx.textLayout.formats:TextLayoutFormat — Wspólne ustawienia stylu znakowego.
