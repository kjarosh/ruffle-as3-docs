Konstruktor Szczegół

SelectionFormat

()Konstruktor
public function SelectionFormat(rangeColor:uint = 0xffffff, rangeAlpha:Number = 1.0, rangeBlendMode:String = "difference", pointColor:uint = 0xffffff, pointAlpha:Number = 1.0, pointBlendMode:String = "difference", pointBlinkRate:Number = 500)

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: Flash Player 10, AIR 1.5

Tworzy obiekt SelectionFormat o określonych właściwościach.

Format SelectionFormat utworzony z wykorzystaniem wartości domyślnych użyje koloru czarnego dla kolorów podświetlenia, 1,0 dla kanałów alfa oraz BlendMode.DIFFERENCE dla trybów mieszania. Szybkość migania kursora wynosi 500 milisekund.

Ustawienie wartości właściwości pointAlpha i rangeAlpha na zero wyłącza podświetlenie zaznaczenia.

Niezerowa szybkość migania jest używana tylko wtedy, gdy obiekt EditManager jest dołączony do obiektu TextFlow.

Parametry
rangeColor:uint (default = 0xffffff) — Kolor podświetlenia.
 
rangeAlpha:Number (default = 1.0) — Wartość przezroczystości obowiązująca przy rysowaniu podświetlenia. Poprawne wartości należą do zakresu od 0 (całkowicie przezroczyste) do 1 (całkowicie nieprzezroczyste, co jest ustawieniem domyślnym).
 
rangeBlendMode:String (default = "difference") — Tryb mieszania przy rysowaniu podświetlenia. W celu ustawienia tego parametru należy używać stałych zdefiniowanych w klasie BlendMode.
 
pointColor:uint (default = 0xffffff) — Kolor dla rysowania kursora.
 
pointAlpha:Number (default = 1.0) — Wartość przezroczystości dla rysowania kursora. Poprawne wartości należą do zakresu od 0 (całkowicie przezroczyste) do 1 (całkowicie nieprzezroczyste, co jest ustawieniem domyślnym).
 
pointBlendMode:String (default = "difference") — Tryb mieszania dla rysowania kursora. W celu ustawienia tego parametru należy używać stałych zdefiniowanych w klasie BlendMode.
 
pointBlinkRate:Number (default = 500) — Szybkość, z jaką miga kursor, w milisekundach.
