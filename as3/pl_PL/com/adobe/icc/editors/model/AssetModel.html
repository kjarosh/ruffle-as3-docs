Podklasy FragmentLayoutModel, LayoutModel, LetterModel, ModuleModel

Wersja języka: ActionScript 3.0
Wersja produktu: Asset Composer Building Block 9.5
Wersje środowiska wykonawczego: AIR (unsupported), Flash Player 9, Flash Player 10

The base domain class for all the assets.

This class is supposedly abstract and should not be instantiated as is. To work on a particular asset, its corresponding domain model class must be used e.g. TextModel, LayoutModel etc.

It contains several properties and methods pulled up from the concrete classes. In case, a new asset is introduced in the system and a domain class needs to be written for it, this class should be extended even though it's not mandatory.
