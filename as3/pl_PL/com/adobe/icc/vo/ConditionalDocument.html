Wersja języka: ActionScript 3.0
Wersja produktu: Asset Composer Building Block 9.5
Wersje środowiska wykonawczego: AIR (unsupported), Flash Player 9, Flash Player 10

Value object for a conditional document. Contains a list of DocumentAssignment objects. Each DocumentAssignment object represents the target document that is selected when the corresponding expression is evaluated. Extends the basic attributes of Document.

assignmentList:ArrayCollection

Wersja języka: ActionScript 3.0
Wersja produktu: Asset Composer Building Block 9.5
Wersje środowiska wykonawczego: AIR (unsupported), Flash Player 9, Flash Player 10

Specifies the list of document assignments for this condition document. This is an ArrayCollection of DocumentAssignment objects.

Ta właściwość może być używana jako źródło dla tworzenia powiązań danych. Jeśli ta właściwość zostanie zmodyfikowana, zostanie wywołane zdarzenie propertyChange .
