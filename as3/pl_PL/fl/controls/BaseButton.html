Wersja języka: ActionScript 3.0
Wersja produktu: Flash CS3
Wersje środowiska wykonawczego: Flash Player 9.0.28.0, AIR 1.0

Klasa BaseButton jest klasą podstawową dla wszystkich składników typu przycisk, a także definiowania właściwości i metod wspólnych dla wszystkich przycisków. Ta klasa obsługuje stany rysowania i wywołanie zdarzeń dla przycisków.

Pobiera lub ustawia wartość typu Boolean wskazującą, czy zdarzenie buttonDown zostało wywołane więcej niż raz, gdy użytkownik trzyma wciśnięty przycisk myszy nad składnikiem. Wartość true wskazuje, czy zdarzenie buttonDown powinno być wywołane wielokrotnie, gdy użytkownik przytrzymuje wciśnięty przycisk myszy nad składnikiem; wartość false wskazuje, że ma być wywołane tylko raz.

Jeśli ta wartość to true, z opóźnieniem określonym stylem repeatDelay zdarzenie buttonDown jest wywoływane w przedziałach czasowych określonych stylem repeatInterval.

Wartością domyślną jest false.

Przykład  ( Sposób korzystania z tego przykładu )
Poniższy przykład ilustruje sposób używania właściwości autoRepeat do powtórnego wywołania zdarzeń buttonDown tak długo, jak przycisk jest naciskany: 

import fl.controls.Button;
import fl.controls.Label;
import fl.events.ComponentEvent;

var myLabel:Label = new Label();
myLabel.text = "0";
myLabel.move(10, 10);
addChild(myLabel);

var downButton:Button = new Button();
downButton.label = "-";
downButton.autoRepeat = true;
downButton.setSize(20, 20);
downButton.move(10, 30);
downButton.addEventListener(ComponentEvent.BUTTON_DOWN, buttonDownHandler);
addChild(downButton)

var upButton:Button = new Button();
upButton.label = "+";
upButton.autoRepeat = true;
upButton.setSize(20, 20);
upButton.move(40, 30);
upButton.addEventListener(ComponentEvent.BUTTON_DOWN, buttonDownHandler);
addChild(upButton);

function buttonDownHandler(event:ComponentEvent):void {
    var value:Number = Number(myLabel.text);
    switch (event.currentTarget) {
        case downButton:
            value--;
            break;
        case upButton:
            value++;
            break;
    }
    myLabel.text = value.toString();
}

W przypadku ustawienia tej właściwości na wartość false przycisk jest wyłączony. Oznacza to, że mimo że jest on widoczny, nie można go kliknąć. Ta właściwość jest użyteczna w przypadku konieczności wyłączenia pewnych fragmentów interfejsu użytkownika. Na przykład przycisk służący do wyzwalania ponownego ładowania strony sieci Web można wyłączyć, korzystając z tej techniki.

function showStyleDefinition(e:Event):void {
    var componentClass:Class = e.target.selectedItem.data as Class;

Wywoływane, gdy użytkownik naciska składnik Button. Jeśli właściwość autoRepeat ma wartość true, to zdarzenie jest wywoływane z określonymi odstępami czasowymi, aż do chwili zwolnienia przycisku.

Styl repeatDelay służy do określania opóźnienia przed wywołaniem zdarzenia buttonDown po raz drugi. Styl repeatInterval określa przedział czasowy, po którym zdarzenie jest wywoływane dalej, aż do zwolnienia przycisku przez użytkownika.

Definiuje wartość właściwości type obiektu zdarzenia buttonDown.

To zdarzenie ma następujące właściwości:

WłaściwośćWartość
bubblesfalse
cancelablefalse; nie ma domyślnego zachowania, które można byłoby anulować.
currentTargetObiekt przetwarzający aktywnie obiekt Event za pomocą detektora zdarzeń.
targetObiekt, który wywołał zdarzenie. Celem nie zawsze jest obiekt wykrywający zdarzenie. Aby uzyskać dostęp do obiektu, który wykrywa zdarzenie, należy skorzystać z właściwości currentTarget.

Przykład  ( Sposób korzystania z tego przykładu )

Poniższy przykład ilustruje sposób używania właściwości autoRepeat do powtórnego wywoływania zdarzeń buttonDown tak długo, jak przycisk jest naciskany. 

import fl.controls.Button;
import fl.controls.Label;
import fl.events.ComponentEvent;

var myLabel:Label = new Label();
myLabel.text = "0";
myLabel.move(10, 10);
addChild(myLabel);

var downButton:Button = new Button();
downButton.label = "-";
downButton.autoRepeat = true;
downButton.setSize(20, 20);
downButton.move(10, 30);
downButton.addEventListener(ComponentEvent.BUTTON_DOWN, buttonDownHandler);
addChild(downButton)

var upButton:Button = new Button();
upButton.label = "+";
upButton.autoRepeat = true;
upButton.setSize(20, 20);
upButton.move(40, 30);
upButton.addEventListener(ComponentEvent.BUTTON_DOWN, buttonDownHandler);
addChild(upButton);

function buttonDownHandler(event:ComponentEvent):void {
    var value:Number = Number(myLabel.text);
    switch (event.currentTarget) {
        case downButton:
            value--;
            break;
        case upButton:
            value++;
            break;
    }
    myLabel.text = value.toString();
}
