Podklasy ParallelInstance, SequenceInstance

Wersja języka: ActionScript 3.0
Wersja produktu: Flex 3
Wersje środowiska wykonawczego: Flash Player 9, AIR 1.1

The CompositeEffectInstance class implements the instance class for the CompositeEffect class. Flex creates an instance of this class when it plays a CompositeEffect; you do not create one yourself.

addChildSet

()metoda
public function addChildSet(childSet:Array):void

Wersja języka: ActionScript 3.0
Wersja produktu: Flex 3
Wersje środowiska wykonawczego: Flash Player 9, AIR 1.1

Adds a new set of child effects to this Composite effect. A Sequence effect plays each child effect set one at a time, in the order that it is added. A Parallel effect plays all child effect sets simultaneously; the order in which they are added doesn't matter.

Parametry

childSet:Array — Array of child effects to be added to the CompositeEffect.
