Interfejspublic interface IPagedList extends IList , IEventDispatcher

Wersja języka: ActionScript 3.0
Wersja produktu: Adobe Digital Enterprise Platform Data Services for Java EE 3
Wersje środowiska wykonawczego: Flash Player 9, AIR 1.1

The IPagedList is a marker interface for list implementations that implement their own paging.
