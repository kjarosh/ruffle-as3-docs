Klasapublic final class TextBaseline
DziedziczenieTextBaseline Object

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: Flash Player 10, AIR 1.5, Flash Lite 4

Klasa TextBaseline stanowi wyliczenie stałych, za pomocą których można ustawiać właściwości dominantBaseline oraz alignmentBaselineklasy ElementFormat. Stałych tych można także używać we właściwości baselineZero klasy TextBlock. Przeanalizuj następującą sytuację:

Wiersz składa się z czterech obiektów TextElement zawierających odpowiednio „a”, „b”, „cccccccc” oraz „X”. Element zwierający „X” określa linie bazowe wiersza ponieważ jest to największy element w wierszu. Linia bazowa alfabetu łacińskiego elementu „X” jest wyrównana z linią bazową alfabetu łacińskiego wiersza. Górna granica ideograficzna elementu „a” jest wyrównana z górną granicą ideograficzną wiersza. Dolna granica ideograficzna elementu „b” jest wyrównana z dolną granicą ideograficzną wiersza. Środek ideograficzny elementu „cccccccc” jest wyrównany ze środkiem ideograficznym wiersza.

Powiązane elementy interfejsu API

ElementFormat.dominantBaseline
ElementFormat.alignmentBaseline
TextBlock.baselineZero
