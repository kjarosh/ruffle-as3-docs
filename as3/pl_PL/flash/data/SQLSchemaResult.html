Klasapublic class SQLSchemaResult
DziedziczenieSQLSchemaResult Object

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 1.0

Instancja klasy SQLSchemaResult zawiera informacje wynikające z wywołania metody SQLConnection.loadSchema(). Zawiera cztery właściwości typu Array, które przechowują żądane dane schematu na podstawie wartości argumentów, które zostały użyte w momencie wywołania metody SQLConnection.loadSchema().

Aby pobrać instancję klasy SQLSchemaResult do wywołania metody SQLConnection.loadSchema(), należy wywołać metodę getSchemaResult() instancji klasy SQLConnection. Zazwyczaj instancja klasy SQLSchemaResult nie jest tworzona bezpośrednio w kodzie pisanym przez programistę.

Powiązane elementy interfejsu API

flash.data.SQLConnection.loadSchema()
flash.data.SQLConnection.getSchemaResult()

SQLSchemaResult

()Konstruktor
public function SQLSchemaResult(tables:Array, views:Array, indices:Array, triggers:Array)

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 1.0

Tworzy instancję klasy SQLSchemaResult. Zazwyczaj konstruktor klasy SQLSchemaResult nie jest wywoływany bezpośrednio w kodzie pisanym przez programistę. Aby uzyskać informacje o schemacie dla bazy danych, należy wywołać metodę SQLConnection.loadSchema().

Parametry
tables:Array — Tablica instancji klasy SQLTableSchema określonych w wywołaniu metody loadSchema().
 
views:Array — Tablica instancji klasy SQLViewSchema określonych w wywołaniu metody loadSchema().
 
indices:Array — Tablica instancji klasy SQLIndexSchema określonych w wywołaniu metody loadSchema().
 
triggers:Array — Tablica instancji klasy SQLTriggerSchema określonych w wywołaniu metody loadSchema().

Powiązane elementy interfejsu API

flash.data.SQLConnection.loadSchema()
