Konstruktor Szczegół

GeolocationEvent

()Konstruktor
public function GeolocationEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, latitude:Number = 0, longitude:Number = 0, altitude:Number = 0, hAccuracy:Number = 0, vAccuracy:Number = 0, speed:Number = 0, heading:Number = 0, timestamp:Number = 0)

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 2, Flash Lite 4

Tworzy obiekt GeolocationEvent zawierający informacje o położeniu urządzenia. Obiekty Event są przekazywane jako parametry do detektorów zdarzeń.

Parametry
type:String — Określa typ zdarzenia. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości type. Istnieje tylko jeden typ zdarzenia aktualizacji: GeolocationEvent.UPDATE.
 
bubbles:Boolean (default = false) — Określa, czy zdarzenie może przechodzić fazę propagacji. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości bubbles.
 
cancelable:Boolean (default = false) — Określa, czy obiekt zdarzenia może być anulowany. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości cancelable.
 
latitude:Number (default = 0) — Zwraca szerokość geograficzną w stopniach. Przyjmuje wartości z zakresu: (-90 <= Lat <= +90).
 
longitude:Number (default = 0) — Zwraca długość geograficzną w stopniach. Przyjmuje wartości z zakresu: (-180 <= Long <= +180).
 
altitude:Number (default = 0) — Zwraca wysokość nad poziomem morza w metrach.
 
hAccuracy:Number (default = 0) — Zwraca dokładność ustalania położenia poziomego w metrach.
 
vAccuracy:Number (default = 0) — Zwraca dokładność ustalania położenia pionowego w metrach.
 
speed:Number (default = 0) — Zwraca szybkość w metrach/sekundę.
 
heading:Number (default = 0) — Zwraca kierunek ruchu (względem prawdziwej północy) w stopniach całkowitych.
 
timestamp:Number (default = 0) — Określa znacznik czasu aktualizacji informacji geolokalizacyjnych.
