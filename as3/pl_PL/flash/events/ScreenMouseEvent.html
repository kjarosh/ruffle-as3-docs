Klasapublic class ScreenMouseEvent
DziedziczenieScreenMouseEvent MouseEvent Event Object

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 1.0

Obiekt SystemTrayIcon wywołuje zdarzenia typu ScreenMouseEvent w odpowiedzi na interakcję z myszą.

Obiekt ScreenMouseEvent rozszerza klasę MouseEvent o dwie dodatkowe właściwości, screenX i screenY, które informują o współrzędnych myszy względem głównego ekranu z pulpitem, a nie względem okna aplikacji lub stołu montażowego.

Konstruktor Szczegół
    

ScreenMouseEvent

()Konstruktor
public function ScreenMouseEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, screenX:Number = NaN, screenY:Number = NaN, ctrlKey:Boolean = false, altKey:Boolean = false, shiftKey:Boolean = false, buttonDown:Boolean = false, commandKey:Boolean = false, controlKey:Boolean = false)

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 1.0

Tworzy obiekt ScreenMouseEvent zawierający informacje o lokalizacji myszy w postaci współrzędnych ekranowych.

Parametry
type:String — Typ zdarzenia. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości type.
 
bubbles:Boolean (default = false) — Pozycja X kliknięcia we współrzędnych ekranowych.
 
cancelable:Boolean (default = false) — Pozycja Y kliknięcia we współrzędnych ekranowych.
 
screenX:Number (default = NaN) — Ma zawsze wartość false, ponieważ zdarzenia myszy nigdy nie uczestniczą w fazie propagacji.
 
screenY:Number (default = NaN) — Wartość false, ponieważ nie ma zachowania domyślnego, które można byłoby anulować.
 
ctrlKey:Boolean (default = false) — W systemach Windows i Linux określa, czy w momencie wywołania zdarzenia był naciśnięty klawisz Ctrl. W systemach Mac określa, czy naciśnięty był klawisz Ctrl lub Command.
 
altKey:Boolean (default = false) — Wartość true oznacza, że w momencie wystąpienia zdarzenia naciśnięty był klawisz Alt.
 
shiftKey:Boolean (default = false) — Wartość true oznacza, że w momencie wystąpienia zdarzenia naciśnięty był klawisz Shift.
 
buttonDown:Boolean (default = false) — Wartość true oznacza, że w momencie wystąpienia zdarzenia naciśnięty był przycisk myszy.
 
commandKey:Boolean (default = false) — Określa, czy był naciśnięty klawisz Command (tylko w systemach Mac).
 
controlKey:Boolean (default = false) — Określa, czy był naciśnięty klawisz Ctrl lub Control.
