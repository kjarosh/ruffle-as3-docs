Klasapublic class DeviceRotationEvent
DziedziczenieDeviceRotationEvent Event Object

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 26.0

Klasa DeviceRotation wywołuje zdarzenie DeviceRotationEvent i zwraca dane na temat obrotu, odchyłu, skoku i kwaternionu po uzyskaniu aktualizacji klasy DeviceRotation z połączonych odczytów z czujników przyspieszeniomierza i żyroskopu zainstalowanych w urządzeniu.

Konstruktor Szczegół
    

DeviceRotationEvent

()Konstruktor
public function DeviceRotationEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, timestamp:Number = 0, roll:Number = 0, pitch:Number = 0, yaw:Number = 0, quaternion:Array = null)

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 26.0

Tworzy obiekt DeviceRotationEvent, który zawiera informacje na temat obrotu, odchyłu i skoku wzdłuż trzech osi wymiarowych. Obiekty Event są przekazywane jako parametry do detektorów zdarzeń.

Parametry
type:String — Typ zdarzenia. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości type. Istnieje tylko jeden typ zdarzenia aktualizacji: AccelerometerEvent.UPDATE.
 
bubbles:Boolean (default = false) — Określa, czy obiekt Event jest w fazie propagacji. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości bubbles.
 
cancelable:Boolean (default = false) — Określa, czy obiekt Event może zostać anulowany. Detektory zdarzeń mogą uzyskiwać dostęp do tej informacji za pośrednictwem dziedziczonej właściwości cancelable.
 
timestamp:Number (default = 0) — Znacznik czasowy aktualizacji klasy DeviceRotation.
 
roll:Number (default = 0) — Wartość obrotu wyrażona w stopniach wzdłuż osi y.
 
pitch:Number (default = 0) — Wartość skoku wyrażona w stopniach wzdłuż osi x.
 
yaw:Number (default = 0) — Wartość odchyłu wyrażona w stopniach wzdłuż osi z.
 
quaternion:Array (default = null) — Kwaterniony dla obrotu urządzenia w formacie [w, x, y, z].
