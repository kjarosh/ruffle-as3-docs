Klasapublic final class NativeWindowRenderMode
DziedziczenieNativeWindowRenderMode Object

Wersja języka: ActionScript 3.0
Wersje środowiska wykonawczego: AIR 3.0

Klasa NativeWindowRenderMode definiuje stałe dla właściwości renderMode obiektu NativeWindowInitOptions używanego do tworzenia okna macierzystego.

Uwaga: Wartość renderMode jest określona w momencie tworzenia okna i nie może ulec zmianie.

Szczegół stałej
    

AUTO

Stała
public static const AUTO:String = "auto"

Wersje środowiska wykonawczego: AIR 3.0

Okno typowe. Zostanie ustawiony taki tryb renderowania, jaki jest najlepszy dla danego urządzenia. Łączenie z obiektami StageVideo czy Stage3D może, ale nie musi być obsługiwane. To jest wartość domyślna.

    

CPU

Stała 
public static const CPU:String = "cpu"

Wersje środowiska wykonawczego: AIR 3.0

Okno trybu procesora. Wektory są renderowane za pomocą programowego mechanizmu renderowania, a kopiowanie bitowe odbywa się za pomocą procesora, dlatego łączenie z obiektami StageVideo i Stage3D nie jest obsługiwane.

Uwaga: Obecnie w celu określenia tego trybu dla nowego obiektu NativeWindow znacznik <renderMode> w deskryptorze aplikacji musi mieć wartość „cpu”, „auto” lub „direct”, gdyż w przeciwnym razie zostanie wygenerowany wyjątek IllegalOperationError.

    

DIRECT

Stała 
public static const DIRECT:String = "direct"

Wersje środowiska wykonawczego: AIR 3.0

Okno trybu bezpośredniego. W oknach trybu bezpośredniego wektory są renderowane za pomocą programowego mechanizmu renderowania, ale kopiowanie bitowe jest przeprowadzane za pomocą GPU, co umożliwia łączenie z obiektami StageVideo czy Stage3D.

Uwaga: Obecnie w celu określenia tego trybu dla nowego obiektu NativeWindow znacznik <renderMode> w deskryptorze aplikacji musi mieć wartość „cpu”, „auto” lub „direct”, gdyż w przeciwnym razie zostanie wygenerowany wyjątek IllegalOperationError.

    

GPU

Stała 
public static const GPU:String = "gpu"

Wersje środowiska wykonawczego: AIR 3.0

Okno trybu GPU. Renderowanie i kopiowanie bitowe odbywa się przy użyciu GPU, dzięki czemu jest obsługiwane tworzenie kompozycji przy użyciu obiektów StageVideo i Stage3D.

Uwaga: Obecnie w celu określenia tego trybu dla obiektu NativeWindow znacznik <renderMode> w deskryptorze aplikacji również musi mieć wartość „gpu”. W przeciwnym razie zostanie wygenerowany wyjątek IllegalOperationError.
