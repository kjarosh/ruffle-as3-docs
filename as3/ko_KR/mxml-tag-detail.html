클래스 x

일부 태그는 MXML에서만 사용할 수 있습니다. Flex 컴파일러는 이처럼 전용 태그를 구현하므로 ActionScript에서는 사용할 수 없습니다.

태그설명
fx:Binding<fx:Binding> 태그는 한 객체의 데이터를 다른 객체와 연결하는 데 사용합니다.
fx:Component<fx:Component> 태그는 MXML 파일에서 인라인 셀 렌더러 또는 셀 편집기를 정의하는 데 사용합니다.
fx:Declarations<fx:Declarations> 태그는 현재 클래스의 기본 속성이 아닌 속성과 시각적 속성이 아닌 속성을 선언하는 데 사용합니다.
fx:Definition<fx:Definition> 태그는 <fx:Library> 태그 내에서 사용되며, 응용 프로그램의 다른 부분에서 사용할 수 있는 그래픽 자식을 정의하는 데 사용합니다.
fx:DesignLayer<fx:DesignLayer> 태그는 내부에서만 사용할 수 있습니다.
fx:Library<fx:Library> 태그는 현재 클래스의 기본 속성이 아닌 속성과 시각적 속성이 아닌 속성을 선언하는 데 사용합니다.
fx:Metadata<fx:Metadata> 태그는 메타데이터 태그를 MXML 파일에 삽입하는 데 사용합니다.
fx:Model<fx:Model> 태그는 MXML에서 데이터 모델을 선언하는 데 사용합니다.
fx:Private<fx:Private> 태그는 MXML 또는 FXG 문서에 대한 메타 정보를 제공하는 데 사용합니다.
fx:Reparent<fx:Reparent> 태그는 보기 상태를 변경할 때 구성 요소의 상위 컨테이너를 변경하는 데 사용합니다.
fx:Script<fx:Script> 태그는 ActionScript 코드 블록을 정의하는 데 사용합니다.
fx:Style<fx:Style> 태그는 현재 문서와 해당 자식에 적용할 스타일을 정의하는 데 사용합니다.
fx:XML<fx:XML> 태그는 MXML에서 XML 형식 데이터 모델을 선언하는 데 사용할 수 있습니다.
fx:XMLList<fx:XML> 태그는 MXML에서 텍스트 형식 모델로부터 E4X XMLList 객체를 만드는 데 사용할 수 있습니다.
