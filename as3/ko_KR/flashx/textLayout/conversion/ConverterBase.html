MERGE_TO_NEXT_ON_PASTE

상수
public static const MERGE_TO_NEXT_ON_PASTE:String = "mergeToNextOnPaste"

언어 버전: ActionScript 3.0
런타임 버전: Flash Player 10, AIR 1.5

클립보드 데이터를 TextFlow로 변환하는 변환기는 MERGE_TO_NEXT_ON_PASTE 속성을 사용하여 붙여넣기로 요소를 기존 TextFlow에 연결할 때 요소를 처리하는 방식을 제어해야 합니다. 이 방법은 요소의 일부만 복사하는 특수 처리를 원할 때 유용합니다. 예를 들어 목록을 복사할 때 목록의 일부만 복사하여 다른 목록에 붙여 넣을 경우 대상 목록에 추가 항목으로 연결합니다. 전체 목록을 복사할 경우에는 중첩된 목록으로 나타납니다. TLF가 클립보드에 사용할 TextFlow를 만들 경우 부분 요소에 대해 모든 요소의 끝에 다른 요소를 연결할지 여부를 제어하는 사용자 속성을 설정합니다. 이 사용자 속성은 최종 TextFlow에 붙여 넣을 수 없지만 TextScrap.textFlow의 요소에는 있을 수 있습니다. 변환기에는 텍스트를 복사할 때 이 속성을 찾아 클립보드에 게시된 형식으로 전달하는 옵션이 있습니다. 예를 들어 일반 텍스트 내보내기 도구는 단락의 "mergeToNextOnPaste" 속성을 확인하여 속성이 true로 설정되어 있으면 단락 종결자를 무시합니다. 마찬가지로 가져오기의 경우, 들어오는 문자열에 종결자가 없고 useClipboardAnnotations가 true이면 해당 단락의 setStyle(MERGE_TO_NEXT_ON_PASTE, "true")을 호출하여 붙여 넣을 때 단락의 붙여 넣을 위치에 연결되도록 합니다. 이 속성은 TextScrap의 요소와 스크랩의 마지막 요소에만 설정할 수 있습니다.
