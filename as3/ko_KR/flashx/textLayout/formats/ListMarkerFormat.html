content

속성 
content:*

언어 버전: ActionScript 3.0
런타임 버전: Flash Player 10, AIR 1.5

표시자의 내용을 제어합니다.

이 문자열의 올바른 값은 다음과 같습니다.

  • none - 표시자가 없습니다.
  • counter(ordered) - 표시자를 표시합니다.
  • counter(ordered,ListStyleType) - 표시자를 표시하지만 listStyleType을 지정된 값으로 변경합니다.
  • counters(ordered) - 최상위 부모 ListElement부터 시작하여, 일련의 번호 매겨진 카운터 값을 각 카운터에 생성합니다. 지정된 listStyleType은 각각에 대한 접미어로 구분됩니다. 이는 아웃라인 번호에 사용됩니다(예: I.1., I.2. 등).
  • counters(ordered,"<string>") - 앞의 값과 유사하지만 번호가 매겨진 각 카운터의 접미어가 <string>으로 바뀝니다.
  • counters(ordered,"<string>",ListStyleType) - 앞의 값과 유사하지만 각 카운터의 listStyleType이 지정된 값으로 바뀝니다.

undefined인 경우 이 속성의 기본값은 "counter(ordered)"입니다.



구현
    public function get content():*
    public function set content(value:any):void

런타임 버전: Flash Player 10, AIR 1.5

카운터 값 증가를 제어합니다.

이 문자열의 올바른 값은 다음과 같습니다.

  • none - 증가분이 없습니다.
  • ordered - 카운터를 1씩 늘립니다.
  • ordered integer - 카운터를 integer만큼 늘립니다.

undefined인 경우 이 속성의 기본값은 "ordered 1"입니다.

참고: counterIncrement 속성이 counterReset 속성보다 먼저 적용됩니다.

    public function set counterIncrement(value:any):void
