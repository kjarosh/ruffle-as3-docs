클래스public class ColumnState
상속ColumnState Object

언어 버전: ActionScript 3.0
런타임 버전: Flash Player 10, AIR 1.5

ColumnState 클래스는 컨테이너의 폭 및 컨테이너 속성을 사용하여 열의 크기와 위치를 계산합니다. 이 클래스의 인스턴스를 독립적으로 만들어 열 값을 계산하거나, 컨테이너를 컴포지션 또는 업데이트한 후(다시 그린 후) 텍스트에 사용된 열 값을 가져올 수 있습니다.

생성자 세부 정보

ColumnState

()생성자
public function ColumnState(blockProgression:String, columnDirection:String, controller:ContainerController, compositionWidth:Number, compositionHeight:Number)

언어 버전: ActionScript 3.0
런타임 버전: Flash Player 10, AIR 1.5

생성자 함수 - ColumnState 객체를 만듭니다. controller.compositionWidth 및 controller.compositionHeight 값이 NaN(숫자 아님)과 같으면 생성자가 컨테이너의 내용을 평가하여 ColumnState에 제공할 실제 컴포지션 폭과 높이를 결정합니다. flashx.textLayout.formats.BlockProgression 클래스에서 정의된 상수를 사용하여 blockProgression 매개 변수의 값을 지정합니다. flashx.textLayout.formats.Direction에서 정의된 상수를 사용하여 columnDirection 매개 변수의 값을 지정합니다.

매개 변수
blockProgression:String — 텍스트 흐름의 줄 방향으로서, BlockProgression.TB(위쪽에서 아래쪽) 또는 BlockProgression.RL(오른쪽에서 왼쪽)일 수 있습니다.
 
columnDirection:String — 텍스트 흐름의 열 레이아웃 방향으로서, Direction.RTL(오른쪽에서 왼쪽) 또는 Direction.LTR(왼쪽에서 오른쪽)일 수 있습니다.
 
controller:ContainerController — ContainerController 인스턴스로서, 해당 속성을 사용하여 열 값을 계산합니다.
 
compositionWidth:Number — 컨테이너 내 텍스트에 허용할 가로쓰기 범위(픽셀 단위)입니다.
 
compositionHeight:Number — 컨테이너 내 텍스트에 허용할 세로쓰기 범위(픽셀 단위)입니다.

관련 API 요소

BlockProgression
Direction

public function getColumnAt(index:int):Rectangle

언어 버전: ActionScript 3.0
런타임 버전: Flash Player 10, AIR 1.5

컨테이너 내에서 열이 차지하는 영역을 반환합니다. 특정 열의 영역에 액세스할 수 있도록 합니다.

매개 변수

index:int — 컨테이너 내 모든 열 사이에서 해당 열의 상대적 위치(첫 번째 열은 위치 0에서 시작)입니다.

반환값
Rectangle — 지정된 열의 영역입니다.
