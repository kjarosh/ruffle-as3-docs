생성자 세부 정보

SelectionFormat

()생성자
public function SelectionFormat(rangeColor:uint = 0xffffff, rangeAlpha:Number = 1.0, rangeBlendMode:String = "difference", pointColor:uint = 0xffffff, pointAlpha:Number = 1.0, pointBlendMode:String = "difference", pointBlinkRate:Number = 500)

언어 버전: ActionScript 3.0
런타임 버전: Flash Player 10, AIR 1.5

지정된 속성으로 SelectionFormat 객체를 만듭니다.

기본값으로 만들어진 SelectionFormat은 강조 표시 색상에 검은색을, 알파에 1.0을, 혼합 모드에 BlendMode.DIFFERENCE를 사용합니다. 커서의 깜박임 속도는 500밀리초입니다.

pointAlpha 및 rangeAlpha 속성을 0으로 설정하면 선택 항목 강조 표시가 해제됩니다.

0이 아닌 깜박임 속도는 EditManager를 TextFlow에 연결할 때에만 사용합니다.

매개 변수
rangeColor:uint (default = 0xffffff) — 강조 표시를 그리는 데 사용할 색상입니다.
 
rangeAlpha:Number (default = 1.0) — 강조 표시를 그리는 데 사용할 투명도 값입니다. 유효한 값은 0(완전 투명)에서 1(완전 불투명, 기본값) 사이의 수입니다.
 
rangeBlendMode:String (default = "difference") — 강조 표시를 그리는 데 사용할 혼합 모드입니다. BlendMode 클래스에서 지정된 상수를 사용하여 이 매개 변수를 설정합니다.
 
pointColor:uint (default = 0xffffff) — 커서를 그리는 데 사용할 색상입니다.
 
pointAlpha:Number (default = 1.0) — 커서를 그리는 데 사용할 투명도 값입니다. 유효한 값은 0(완전 투명)에서 1(완전 불투명, 기본값) 사이의 수입니다.
 
pointBlendMode:String (default = "difference") — 커서를 그리는 데 사용할 혼합 모드입니다. BlendMode 클래스에서 지정된 상수를 사용하여 이 매개 변수를 설정합니다.
 
pointBlinkRate:Number (default = 500) — 커서가 깜박이는 속도(밀리초)입니다.
