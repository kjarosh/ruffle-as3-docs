public function DownloadErrorEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, text:String = "", id:int = 0, subErrorID:int = 0)

런타임 버전: AIR 1.5

생성자 함수입니다. 이 함수를 호출하지 마십시오. 업데이트 프레임워크에서 이 함수를 호출하여 이벤트 객체를 만듭니다.

매개 변수
type:String — 이벤트 유형입니다. 이벤트 리스너는 상속된 type 속성을 통해 이 정보에 액세스할 수 있습니다. 오류 이벤트 유형은 ErrorEvent.ERROR 한 가지뿐입니다.
 
bubbles:Boolean (default = false) — Event 객체의 버블링 여부를 결정합니다. 이벤트 리스너는 상속된 bubbles 속성을 통해 이 정보에 액세스할 수 있습니다.
 
cancelable:Boolean (default = false) — Event 객체를 취소할 수 있는지 여부를 결정합니다. 이벤트 리스너는 상속된 cancelable 속성을 통해 이 정보에 액세스할 수 있습니다.
 
text:String (default = "") — 오류 메시지로 표시될 텍스트입니다. 이벤트 리스너는 text 속성을 통해 이 정보에 액세스할 수 있습니다.
 
id:int (default = 0) — 특정 오류와 연결할 참조 번호입니다.
 
subErrorID:int (default = 0) — 오류 이벤트와 errorID에 대해 자세히 설명합니다.

DownloadErrorEvent.DOWNLOAD_ERROR 상수는 downloadError 이벤트에 대한 이벤트 객체의 type 속성 값을 정의합니다.

DownloadErrorEvent 객체의 errorID 속성은 오류 정보를 정의하는 정수입니다(다음 표 참조). 추가 subErrorID 속성은 더 많은 오류 정보를 포함할 수 있습니다.
