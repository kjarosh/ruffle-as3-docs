클래스public class IKManager
상속IKManager EventDispatcher Object

언어 버전: ActionScript 3.0
제품 버전: Flash CS4
런타임 버전: Flash Player 10, AIR 1.5

IKManager 클래스는 문서에 정의되어 있는 모든 역기구학(IK) 트리(뼈대)를 나타내고 런타임에 이러한 뼈대를 관리하는 데 사용되는 컨테이너 클래스입니다.

뼈대는 Flash 제작 도구에서만 만들 수 있습니다. 뼈대와 함께 Flash CS4 문서를 제작할 때 Flash 제작 도구에서는 뼈대와 생성된 일부 ActionScript 코드의 XML 표현을 SWF 파일에 저장합니다. 런타임에 Flash Player는 ActionScript 코드를 실행하고 XML을 파싱하여 뼈대를 다시 만듭니다.

IKManager 클래스의 인스턴스를 만들 필요가 없습니다. 런타임에서 이 클래스의 인스턴스를 자동으로 만듭니다. 또한 IKManager 클래스의 모든 속성 및 메서드는 정적입니다. 따라서 IKManager 클래스의 인스턴스를 참조할 필요가 없습니다.

참고: 문서 클래스 내의 IKArmature 객체를 참조할 때는 먼저 frameConstructed 이벤트를 점검하여 모든 객체가 채워졌는지 확인해야 합니다.

public static function getArmatureAt(index:int):IKArmature

언어 버전: ActionScript 3.0
제품 버전: Flash CS4
런타임 버전: Flash Player 10, AIR 1.5

지정된 인덱스의 뼈대를 반환합니다.

매개 변수

index:int — 0부터 시작하는 검색할 뼈대 인덱스 번호입니다.

반환값
IKArmature — 지정된 인덱스의 IKArmature 인스턴스를 반환합니다.

예제  ( 예제 사용 방법 )
다음 예제에서는 트리의 첫 번째 뼈대를 검색합니다. 문서 클래스 내의 뼈대를 참조하려면 frameConstructed 이벤트를 addedToStage 이벤트 대신 사용합니다. 그래야 모든 IKArmature가 채워집니다. 
         package
         {
            import fl.ik;
            import flash.display.MovieClip;
            import flash.events;
         
            public class IKDocClass extends MovieClip
            {
               public var arm:IKArmature;
         
               function IKDocClass()
               {
                  addEventListener("frameConstructed", onFC);
               }
         
               function onFC(evt:Event):void
               {
                  //Refer to armatures in frameConstructed
                  arm = IKManager.getArmatureAt(0);
                  trace(arm.name);
                  removeEventListener("frameConstructed", onFC);
               }
            }
         }

setStage

()메서드 
public static function setStage(stage:DisplayObjectContainer):void

언어 버전: ActionScript 3.0
제품 버전: Flash CS4
런타임 버전: Flash Player 10, AIR 1.5

이 메서드를 사용하면 로드된 SWF 파일에 있는 심볼의 컨테이너를 전달할 수 있습니다. 뼈대가 들어 있는 SWF 파일을 로드하는 경우 뼈대의 심볼 위치를 부모 SWF 파일에 전달해야 합니다. 자식 SWF 파일의 스테이지에 뼈대 심볼이 있는 경우 stage 또는 스테이지의 변수가 값이 됩니다.

매개 변수

stage:DisplayObjectContainer — 로드된 SWF 파일의 뼈대 심볼 위치입니다.


예제  ( 예제 사용 방법 )
다음 예제에서는 한 SWF 파일에서 뼈대가 들어 있는 다른 SWF 파일을 로드합니다. myAnimation.fla 파일에서 자식 SWF 파일(뼈대 심볼이 들어 있는 파일)이 생성되고, Main.fla 파일에서 부모 SWF 파일(myAnimation.swf 파일을 로드하는 파일)이 생성됩니다. myAnimation.fla 파일에는 스테이지에 배치된 뼈대 심볼 집합이 들어 있습니다. ActionScript 행에서 스테이지에 대해 변수 myAnimationStage를 설정합니다. 이 예제의 경우 스테이지가 뼈대 심볼의 컨테이너입니다. Main.fla 파일의 ActionScript에서 setStage()를 사용하여 자식 SWF 파일의 심볼 컨테이너 위치를 부모 SWF 파일에 전달합니다.

myAnimation.fla 파일의 ActionScript(자식 SWF 파일 생성용):

         import fl.ik.*;
         
         var myAnimationIKM = IKManager;
         var myAnimationArm:IKArmature = IKManager.getArmatureAt(0);
         // set a variable for the stage containing the armature symbols
         var myAnimationStage:DisplayObjectContainer = stage;

Main.fla 파일의 ActionScript(부모 SWF 파일 생성용):

         var myLoader:Loader = new Loader();
         myLoader.contentLoaderInfo.addEventListener(Event.COMPLETE, onCompleteFunc);
         
         function onCompleteFunc(event:Event) {
             // indicate container of the armature symbols
             myLoader.content.myAnimationIKM.setStage(loader.content.myAnimationStage);
             myLoader.content.myAnimationArm.registerElements(loader.content.myAnimationStage);
             myLoader.content.myAnimationIKM.trackAllArmatures();
         }
         
         myLoader.load(new URLRequest("myAnimation.swf"));
         addChild(myLoader);

trackAllArmatures

()메서드 
public static function trackAllArmatures(enable:Boolean = true):void

언어 버전: ActionScript 3.0
제품 버전: Flash CS4
런타임 버전: Flash Player 10, AIR 1.5

모든 뼈대에 대해 실시간 추적을 활성화하거나 비활성화합니다. 실시간 추적이 활성화된 경우 뼈대가 포함된 SWF 파일의 사용자는 뼈대를 클릭하고 드래그하여 이동할 수 있습니다.

매개 변수

enable:Boolean (default = true) — false로 설정된 경우 실시간 추적이 비활성화됩니다. 그렇지 않으면 실시간 추적이 활성화됩니다.

관련 API 요소

trackIKArmature()
trackIKObject()

예제  ( 예제 사용 방법 )
다음 예제에서는 SWF 파일의 사용자가 어떠한 뼈대 부분의 움직임도 제어할 수 없도록 차단합니다. 
             import fl.ik.*;
             // disable live tracking for all armatures in the SWF file
             IKManager.trackAllArmatures(false);

예제  ( 예제 사용 방법 )
다음 예제에서는 SWF 파일의 사용자가 인체 모양에서 몸통 부분을 제외한 모든 부분을 클릭하고 드래그하여 인체의 움직임을 제어할 수 있도록 합니다. 이 모양은 "personTree"라는 IKArmature로 표시되며 인체의 몸통을 나타내는 "torso"라는 IKObject를 포함합니다. 
             import fl.ik.*;
             // turn off tracking for all armatures in the SWF file
             IKManager.trackAllArmatures(false);
             // enable live tracking for the entire armature
             IKManager.trackIKArmature(personTree, true);
             // disable live tracking for the figure's torso
             IKManager.trackIKObject(torso, false);

enable:Boolean (default = true) — false로 설정된 경우 실시간 추적이 비활성화됩니다. 그렇지 않으면 실시간 추적이 활성화됩니다.

예제  ( 예제 사용 방법 )
다음 예제에서는 SWF 파일의 사용자가 인체 모양에서 손 부분만 클릭하고 드래그하여 인체의 움직임을 제어할 수 있도록 합니다. 손은 IK 표시 객체 인스턴스 rightHand 및 leftHand(둘 다 MovieClip 인스턴스임)로 나타납니다.

             IKManager.trackIKObject(leftHand, true);
