클래스public class Rotate
상속Rotate Transition EventDispatcher Object

언어 버전: ActionScript 3.0
제품 버전: Flash CS3
런타임 버전: Flash Player 9, AIR 1.0

Rotate 클래스는 동영상 클립 객체를 회전합니다. 이 효과에는 다음 매개 변수가 필요합니다.
  • ccw: 시계 방향 회전은 false, 반시계 방향 회전은 true인 부울 값입니다.
  • degrees: 객체의 회전 각도를 나타내는 정수 값입니다. 1에서 9999사이의 숫자를 지정하는 것이 좋습니다. 예를 들어 degrees를 1080으로 설정하면 객체를 완전히 세 번 회전시킵니다.

예를 들어 다음 코드에서는 img1_mc 동영상 클립 인스턴스에 Rotate 전환을 사용합니다.

 import fl.transitions.*;
 import fl.transitions.easing.*;
    
 TransitionManager.start(img1_mc, {type:Rotate, direction:Transition.IN, duration:3, easing:Strong.easeInOut, ccw:false, degrees:720});
