상속UIScrollBar ScrollBar UIComponent Sprite DisplayObjectContainer InteractiveObject DisplayObject EventDispatcher Object

언어 버전: ActionScript 3.0
제품 버전: Flash CS3
런타임 버전: Flash Player 9.0.28.0, AIR 1.0

UIScrollBar 클래스는 모든 스크롤 막대 기능을 포함하지만, scrollTarget 속성을 추가하여 이 클래스를 TextField 인스턴스 또는 TLFTextField 인스턴스에 연결할 수 있습니다.

참고: ActionScript를 사용하여 텍스트 레이아웃에 영향을 주는 TextField 또는 TLFTextField 인스턴스의 속성을 업데이트하는 경우 UIScrollBar 구성 요소 인스턴스에서 update() 메서드를 호출하여 스크롤 속성을 새로 고쳐야 합니다. TextField 또는 TLFTextField 인스턴스에 속하는 텍스트 레이아웃 속성에는 width, height, wordWrap 등이 있습니다.

ScrollBarDirection

예제  ( 예제 사용 방법 )
다음 예제에서는 일부 텍스트를 텍스트 필드에 동적으로 로드하고 가로 및 세로 스크롤 막대를 만듭니다. 

import fl.controls.UIScrollBar;
import fl.controls.ScrollBarDirection;

var url:String = "http://www.helpexamples.com/flash/text/lorem.txt";

var uLdr:URLLoader = new URLLoader(new URLRequest(url));
uLdr.addEventListener(Event.COMPLETE, completeHandler);

var tf:TextField = new TextField();
tf.width = 320;
tf.height = 140;
tf.x = 10;
tf.y = 10;
tf.border = true;
tf.multiline = true;
tf.wordWrap = false;
addChild(tf)

var hScrollBar:UIScrollBar = new UIScrollBar();
hScrollBar.direction = ScrollBarDirection.HORIZONTAL;
hScrollBar.move(tf.x, tf.y + tf.height);
hScrollBar.width = tf.width;
hScrollBar.scrollTarget = tf;
addChild(hScrollBar);

var vScrollBar:UIScrollBar = new UIScrollBar();
vScrollBar.direction = ScrollBarDirection.VERTICAL;
vScrollBar.move(tf.x + tf.width, tf.y);
vScrollBar.height = tf.height;
vScrollBar.scrollTarget = tf;
addChild(vScrollBar);

function completeHandler(event:Event):void {
    tf.text = event.target.data;
    hScrollBar.update();
    vScrollBar.update();
}
다음 예제에서는 동적 텍스트 필드를 스크롤하는 데 사용되는 UIScrollBar 구성 요소 인스턴스를 만듭니다. 

import fl.controls.ScrollBarDirection;
import fl.controls.UIScrollBar;

var myTextField:TextField = new TextField();
myTextField.width = 160;
myTextField.height = 120;
myTextField.border = true;
myTextField.x = 10;
myTextField.y = 10;
addChild(myTextField);

var hScrollBar:UIScrollBar = new UIScrollBar();
hScrollBar.direction = ScrollBarDirection.HORIZONTAL;
hScrollBar.scrollTarget = myTextField;
hScrollBar.width = myTextField.width;
hScrollBar.move(myTextField.x, myTextField.y + myTextField.height);
addChild(hScrollBar);

var myURLLoader:URLLoader = new URLLoader();
myURLLoader.addEventListener(Event.COMPLETE, completeHandler);
myURLLoader.load(new URLRequest("http://www.helpexamples.com/flash/lorem.txt"));

function completeHandler(event:Event):void {
    var txt:String = URLLoader(event.currentTarget).data as String;
    myTextField.text = txt;
    vScrollBar.update();
}

현재 구성 요소에 대한 기본 스타일 맵을 가져옵니다. 스타일 맵에는 구성 요소에서 사용하는 스타일에 따라 구성 요소에 적합한 유형이 포함됩니다. 예를 들어 disabledTextFormat 스타일에는 null 값 또는 TextFormat 객체가 포함되어 있습니다. 이러한 스타일을 사용하여 현재 구성 요소에서 setStyle()을 호출할 수 있습니다. 다음 코드는 지정된 구성 요소에서 기본 disabledTextFormat 스타일을 무시합니다. 

componentInstance.setStyle("disabledTextFormat", new TextFormat());

Object — 기본 스타일 객체입니다.

override public function setScrollProperties(pageSize:Number, minScrollPosition:Number, maxScrollPosition:Number, pageScrollSize:Number = 0):void

언어 버전: ActionScript 3.0
제품 버전: Flash CS3
런타임 버전: Flash Player 9.0.28.0, AIR 1.0

ScrollBar 구성 요소의 범위와 뷰포트 크기를 설정합니다. ScrollBar 구성 요소의 상태에 따라 화살표 버튼의 상태와 스크롤 썸의 크기가 업데이트됩니다. 모든 scroll 속성은 minScrollPosition 및 maxScrollPosition의 비율에 상대적입니다. 최소값과 최대값 사이의 각 숫자는 한 번 스크롤하는 위치를 표시합니다.

매개 변수

pageSize:Number — 한 페이지의 크기입니다. 화살표를 클릭할 때 스크롤 막대가 이동하는 거리와 썸의 크기를 결정합니다.
 
minScrollPosition:Number — 스크롤 범위의 아래쪽 끝입니다.
 
maxScrollPosition:Number — 스크롤 범위의 위쪽 끝입니다.
 
pageScrollSize:Number (default = 0) — 트랙을 눌렀을 때 이동하는 거리(픽셀 단위)입니다.

스크롤 막대의 scroll 속성을 즉시 업데이트합니다. 지정한 scrollTarget 텍스트 필드의 텍스트를 ActionScript를 통해 추가함에 따라 스크롤 막대를 새로 고쳐야 하는 경우에 이 객체가 필요합니다.

예제  ( 예제 사용 방법 )

다음 예제에서는 동적 텍스트 필드를 스크롤하는 데 사용되는 UIScrollBar 구성 요소 인스턴스를 만듭니다. 

import fl.controls.UIScrollBar;

var myTextField:TextField = new TextField();
myTextField.wordWrap = true;
myTextField.width = 160;
myTextField.height = 120;
myTextField.border = true;
myTextField.x = 10;
myTextField.y = 10;
addChild(myTextField);

var vScrollBar:UIScrollBar = new UIScrollBar();
vScrollBar.scrollTarget = myTextField;
vScrollBar.height = myTextField.height;
vScrollBar.move(myTextField.x + myTextField.width, myTextField.y);
addChild(vScrollBar);

var myURLLoader:URLLoader = new URLLoader();
myURLLoader.addEventListener(Event.COMPLETE, completeHandler);
myURLLoader.load(new URLRequest("http://www.helpexamples.com/flash/lorem.txt"));

function completeHandler(event:Event):void {
    var txt:String = URLLoader(event.currentTarget).data as String;
    myTextField.text = txt;
    vScrollBar.update();
}
예제 예제 사용 방법
UIScrollBarExample.as

다음 예제에서는 TextField에 UIScrollBar 인스턴스를 추가하는 방법을 보여 줍니다.

예제를 실행하려면 다음 단계를 수행합니다.

  1. 라이브러리에 UIScrollBar 구성 요소를 추가합니다.
  2. 이 코드를 FLA 파일과 같은 디렉토리에 UIScrollBarExample.as로 저장합니다.
  3. FLA 파일의 Document 클래스를 UIScrollBarExample로 설정합니다.
package 
{
    import fl.controls.UIScrollBar;
    import flash.display.Sprite;
    import flash.text.TextField;
    
    public class UIScrollBarExample extends Sprite
    {
        private var tf:TextField;
        private var sb:UIScrollBar = new UIScrollBar();
        private var sampleText:String = "Lorem ipsum dolor sit amet, consectetuer adipiscing elit. Phasellus vestibulum. Praesent in velit. Fusce scelerisque. Nullam fringilla. Aenean ut lacus at mi rutrum elementum. Suspendisse sit amet ligula et ipsum rhoncus sagittis. Aenean quis lacus. Nullam quis lorem. Integer pellentesque dictum lorem. In hendrerit lectus non orci. Sed vestibulum. Maecenas ornare, libero eu laoreet tristique, felis odio tincidunt nibh, et faucibus nibh tellus at metus. In hac habitasse platea dictumst. Nullam aliquam, dolor ut commodo dictum, ante justo dictum nisi, id suscipit elit sapien nec pede."
        
        public function UIScrollBarExample() {
            createTextField();
            assignScrollBar();
        }
        private function createTextField():void {
            tf = new TextField();
            tf.width = 200;
            tf.height = 100;
            tf.wordWrap = true;
            tf.multiline = true;
            tf.border = true;
            tf.text = sampleText;
            addChild(tf);            
        }
        private function assignScrollBar():void {
            sb = new UIScrollBar();
            sb.move(tf.x + tf.width, tf.y);
            sb.setSize(sb.width, tf.height);
            sb.scrollTarget = tf;
            addChild(sb);            
        }
    }
}
