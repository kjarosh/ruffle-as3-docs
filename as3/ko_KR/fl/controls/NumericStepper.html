런타임 버전: Flash Player 9.0.28.0, AIR 1.0

NumericStepper 구성 요소는 사용자가 선택할 수 있는 순서가 지정된 숫자 세트를 표시합니다. 이 구성 요소에는 텍스트 입력을 위한 한 줄짜리 필드와 일련의 값 세트를 차례로 사용하기 위한 한 쌍의 화살표 버튼이 포함되어 있습니다. 위쪽 및 아래쪽 화살표 키를 사용하여 값 세트를 볼 수도 있습니다. NumericStepper 구성 요소는 현재 값이 변경된 경우 change 이벤트를 전달합니다. 이 구성 요소에는 value 속성도 포함되어 있습니다. 이 속성을 사용하여 구성 요소의 현재 값을 가져올 수 있습니다.

사용자가 NumericStepper 구성 요소의 값을 변경할 때 전달됩니다.

참고: ActionScript를 사용하여 값을 변경하는 경우에는 이 이벤트가 전달되지 않습니다.

Event.CHANGE 상수는 change 이벤트 객체의 type 속성 값을 정의합니다.

이 이벤트에는 다음과 같은 속성이 있습니다.

속성값
bubblestrue
cancelablefalse. 취소할 기본 비헤이비어가 없습니다.
currentTarget이벤트 리스너를 통해 Event 객체를 처리하고 있는 객체입니다.
target값이 수정된 객체입니다. target은 표시 목록에서 이벤트 리스너를 등록한 객체가 아닐 수도 있습니다. 표시 목록에서 현재 이벤트를 처리 중인 객체에 액세스하려면 currentTarget 속성을 사용합니다.

NumericStepperExample.as

이 예제에서는 NumericStepper 구성 요소를 동적으로 만들고 구성하는 방법을 보여 줍니다.
  1. 라이브러리에 NumericStepper 구성 요소를 추가합니다.
  2. 이 코드를 FLA와 같은 디렉토리에 NumericStepperExample.as로 저장합니다.
  3. FLA의 DocumentClass를 NumericStepperExample로 설정합니다.
package 
{
    import flash.display.Sprite
    import flash.text.TextField;
    import flash.events.Event;
    import fl.controls.NumericStepper;
    
    public class NumericStepperExample extends Sprite
    {
        private var ns1:NumericStepper;
        private var ns2:NumericStepper;
        private var tf1:TextField;
        private var tf2:TextField;
        private var rowHeight1:Number = 10;
        private var rowHeight2:Number = 50;
        
        public function NumericStepperExample() {        
            setupSteppers();
            setupTextFields();
        }
        private function setupTextFields():void {
            tf1 = new TextField();
            tf1.text = "This stepper increments by 1 between 0 and 5. Disables if it reaches 0.";
            tf1.autoSize = "left";
            tf1.x = 100;
            tf1.y = rowHeight1;
            addChild(tf1);
            
            tf2 = new TextField();
            tf2.text = "This stepper increments by 0.01 between 0 and 20. Disables if it reaches 0.";
            tf2.autoSize = "left";
            tf2.x = 100;
            tf2.y = rowHeight2;
            addChild(tf2);
        }
        private function setupSteppers():void {    
            ns1 = new NumericStepper();
            ns1.stepSize = 1;
            ns1.maximum = 5;
            ns1.minimum = 0;
            ns1.addEventListener(Event.CHANGE,changeOccurred);
            ns1.move(10,rowHeight1);
            addChild(ns1);

            ns2 = new NumericStepper();
            ns2.stepSize = 0.01;
            ns2.maximum = 20;
            ns2.minimum = 0;
            ns2.value = 10;
            ns2.addEventListener(Event.CHANGE,changeOccurred);
            ns2.move(10,rowHeight2);
            addChild(ns2);
        }
        private function changeOccurred(e:Event):void {
            var nsTarget:NumericStepper = e.target as NumericStepper;
            if(nsTarget.value == 0) nsTarget.enabled = false;
        }
    }
}
