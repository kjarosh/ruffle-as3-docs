인터페이스public interface IFocusManager
구현자 FocusManager

언어 버전: ActionScript 3.0
제품 버전: Flash CS3
런타임 버전: Flash Player 9.0.28.0, AIR 1.0

사용자 정의 포커스 관리자를 만들기 위해 IFocusManager 인터페이스를 구현합니다. 포커스 관리자는 응용 프로그램을 활성화하여 사용자가 마우스를 이동하거나 Tab 키를 누를 때 구성 요소 간에 포커스를 전송합니다.

속성 세부 정보

defaultButton

속성
defaultButton:Button

언어 버전: ActionScript 3.0
제품 버전: Flash CS3
런타임 버전: Flash Player 9.0.28.0, AIR 1.0

기본 버튼에 대한 참조를 가져오거나 설정합니다. 기본 버튼은 Enter 키를 누를 때 포커스가 있는 구성 요소의 프록시 버튼 역할을 합니다. Enter 키를 누르면 click 이벤트가 포커스가 있는 구성 요소를 대신하여 기본 버튼에서 전달되도록 트리거됩니다. Button 구성 요소에는 기본 버튼이 필요하지 않습니다. 포커스가 Button 구성 요소로 이동하면 기본 버튼을 트리거할 수 없습니다. 포커스가 Button 구성 요소에서 버튼이 아닌 구성 요소로 이동하면 기본 버튼이 다시 트리거될 수 있습니다.

defaultButtonEnabled

속성 
defaultButtonEnabled:Boolean

언어 버전: ActionScript 3.0
제품 버전: Flash CS3
런타임 버전: Flash Player 9.0.28.0, AIR 1.0

기본 버튼의 활성 여부를 나타내는 값을 가져오거나 설정합니다. 이 값을 true로 설정하면 포커스 관리자가 Enter 키를 모니터링하여 Button 구성 요소가 아닌 구성 요소에 포커스가 있을 때 Enter 키를 누를 경우 기본 버튼에 대해 click 이벤트를 전달합니다. 이 값을 false로 설정하면 포커스 관리자가 Enter 키를 모니터링하지 않습니다. Enter 키를 사용하는 구성 요소는 이 속성을 false로 설정하여 사용자가 Enter 키를 누를 때 click 이벤트가 기본 버튼에 대해 전달되지 않도록 합니다.

showFocusIndicator

속성 
showFocusIndicator:Boolean

언어 버전: ActionScript 3.0
제품 버전: Flash CS3
런타임 버전: Flash Player 9.0.28.0, AIR 1.0

특정 구성 요소에 포커스가 있는지를 나타내도록 사용자 인터페이스를 변경할지 여부를 결정하는 값을 가져오거나 설정합니다.

이 속성을 true로 설정하면 포커스가 있는 구성 요소에 시각적 포커스 표시기가 표시됩니다. 이 속성을 false로 설정하면 시각적 포커스 표시기가 사용되지 않습니다.

사용자가 Tab 키를 누를 때까지 기본적으로 이 속성은 false로 설정되어 있으며, 사용자가 Tab 키를 누르면 true로 설정됩니다.

런타임 버전: Flash Player 9.0.28.0, AIR 1.0

포커스 관리자를 활성화합니다.

여러 DisplayObjectContainer 객체가 화면에 동시에 표시되는 경우 포커스가 특정 컨테이너에서 다음 컨테이너로 이동할 때 시스템 관리자가 해당 객체의 FocusManager 객체를 활성화 및 비활성화합니다. 포커스 관리자가 비활성화된 DisplayObjectContainer 객체에서 구성 요소로 포커스가 이동하면 시스템 관리자는 activate() 메서드를 호출하여 해당 포커스 관리자를 활성화합니다. 한 번에 하나의 포커스 관리자만 활성 상태일 수 있습니다. 따라서 포커스 관리자를 활성화하려는 경우 시스템 관리자는 먼저 deactivate() 메서드를 사용하여 구성 요소가 포커스를 잃은 활성 포커스 관리자를 비활성화해야 합니다.

findFocusManagerComponent

public function findFocusManagerComponent(component:InteractiveObject):InteractiveObject

언어 버전: ActionScript 3.0
제품 버전: Flash CS3
런타임 버전: Flash Player 9.0.28.0, AIR 1.0

지정된 객체를 포함하는 IFocusManagerComponent 객체(있는 경우)를 검색합니다.

Flash Player는 구성 요소 자체뿐 아니라 하위 구성 요소에도 포커스를 설정할 수 있습니다. 이 메서드는 포커스가 있거나 포커스가 있는 하위 구성 요소를 포함하는 구성 요소를 찾는 데 사용됩니다.

매개 변수

component:InteractiveObject — Flash Player 레벨 포커스를 가질 수 있는 객체입니다.

반환값
InteractiveObject — 지정된 객체를 포함하는 IFocusManagerComponent이며 그렇지 않은 경우 이 메서드는 null을 반환합니다.

getNextFocusManagerComponent

()메서드 
public function getNextFocusManagerComponent(backward:Boolean = false):InteractiveObject

언어 버전: ActionScript 3.0
제품 버전: Flash CS3
런타임 버전: Flash Player 9.0.28.0, AIR 1.0

사용자가 Tab 키를 사용하여 포커스를 이동하면 다음에 포커스를 받는 구성 요소를 검색합니다.

이 메서드는 포커스가 앞으로 또는 뒤로 한 요소씩 이동하는 경우 탭 루프에서 포커스를 받을 다음 구성 요소를 검색하는 데 사용됩니다. 응용 프로그램에 다른 유효한 구성 요소가 없는 경우 이 메서드는 포커스가 있는 현재 구성 요소를 검색합니다.

매개 변수

backward:Boolean (default = false) — 포커스가 탭 루프를 통해 뒤로 이동하는지 여부를 나타냅니다. 이 값이 true이면 이 메서드는 사용자가 Shift+Tab 키 조합을 눌러 포커스가 뒤로 이동한 경우 포커스를 받을 구성 요소를 반환합니다.

반환값
InteractiveObject — 다음에 포커스를 받을 구성 요소입니다.
