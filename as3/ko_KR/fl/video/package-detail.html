fl.video 패키지에는 FLVPlayback 및 FLVPlaybackCaptioning 구성 요소와 함께 작동하는 클래스가 들어 있습니다.


인터페이스
 인터페이스설명
 INCManager INCManager는 VideoPlayer 클래스의 flash.net.NetConnection을 만드는 클래스에 대한 인터페이스입니다.
 IVPEvent FLVPlayback 구성 요소 내에 있는 특정 VideoPlayer 객체에 적용되는 비디오 이벤트에서 IVPEvent 인터페이스를 구현합니다.
클래스
 클래스설명
 AutoLayoutEvent Flash® Player에서는 자동으로 비디오 플레이어를 배치하고 크기를 조절할 때 AutoLayoutEvent 객체를 전달합니다.
 CaptionChangeEvent CaptionChangeEvent는 캡션 대상 텍스트 필드에서 캡션이 추가되거나 제거될 때마다 전달됩니다.
 CaptionTargetEvent captionTargetCreated 이벤트가 자동으로 만들어진 다음 캡션이 추가되기 전에 captionTargetCreated 이벤트의 유형이 전달됩니다.
 CuePointType CuePointType 클래스는 type CUE_POINT의 MetadataEvent 인스턴스에 대한 info 객체의 type 속성에 상수 값을 제공합니다.
 FLVPlayback FLVPlayback에서 Sprite 클래스를 확장하고 VideoPlayer 객체를 포함합니다.
 FLVPlaybackCaptioningFLVPlaybackCaptioning은 FLVPlayback 구성 요소의 캡션을 활성화하는 구성 요소입니다.
 LayoutEvent 비디오 플레이어의 크기를 조절할 때 및/또는 비디오 플레이어를 배치할 때 전달되는 이벤트입니다.
 MetadataEvent Flash® Player에서는 사용자가 FLV 파일의 메타데이터 정보 패킷(NetStream.onMetaData)을 요청하고 FLV 파일에서 큐 포인트(NetStream.onCuePoint)를 발견했을 때 MetadataEvent 객체를 전달합니다.
 NCManager VideoPlayer 클래스를 쉽게 사용하도록 돕는 도우미 클래스인 NetConnection 객체를 만듭니다.
 NCManagerNative NCManagerNative 클래스는 NCManager 클래스의 하위 클래스이며 기본 대역폭 검색을 지원합니다. 일부 Flash Video Streaming Service 공급자는 이 기능을 지원하지 않을 수 있습니다.
 SkinErrorEvent Flash® Player에서는 스킨을 로드할 때 오류가 있는 경우 SkinErrorEvent 객체를 전달합니다.
 SoundEvent Flash® Player에서는 사용자가 volumeBar 컨트롤의 핸들을 이동하거나 volume 또는 soundTransform 속성을 설정하여 사운드를 변경할 때 SoundEvent 객체를 전달합니다.
 VideoAlign VideoAlign 클래스는 FLVPlayback.align 및 VideoPlayer.align 속성에 사용할 상수 값을 제공합니다.
 VideoError VideoError 예외는 FLVPlayback 및 VideoPlayer 클래스에서 오류를 보고하기 위한 기본 메커니즘입니다.
 VideoEvent Flash® Player에서는 사용자가 비디오를 재생할 때 VideoEvent 객체를 전달합니다.
 VideoPlayer FLVPlayback 구성 요소를 사용했었던 경우 VideoPlayer 클래스를 사용하면 조금 작은 SWF 파일로 비디오 플레이어를 만들 수 있습니다.
 VideoProgressEvent HTTP를 통해 비디오를 점진적으로 다운로드하는 동안 로드된 바이트 수를 요청하면 Flash® Player에서 VideoProgressEvent 객체를 전달합니다.
 VideoScaleMode VideoScaleMode 클래스는 FLVPlayback.scaleMode 및 VideoPlayer.scaleMode 속성에 사용할 상수 값을 제공합니다.
 VideoState VideoState 클래스는 읽기 전용 FLVPlayback.state 및 VideoPlayer.state 속성에 대한 상수 값을 제공합니다.
