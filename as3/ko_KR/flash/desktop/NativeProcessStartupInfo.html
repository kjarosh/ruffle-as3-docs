클래스public class NativeProcessStartupInfo
상속NativeProcessStartupInfo Object

언어 버전: ActionScript 3.0
런타임 버전: AIR 2

이 클래스는 호스트 운영 체제에서 프로세스를 시작하는 데 사용되는 기본 정보를 제공합니다. 이 클래스가 생성되어 NativeProcess 객체의 start() 메서드로 전달됩니다.

기본 프로세스 액세스는 기본 설치 프로그램으로 설치된 AIR 응용 프로그램(확장 데스크톱 프로파일의 응용 프로그램)에서만 사용할 수 있습니다.

arguments

arguments:Vector.<String>

언어 버전: ActionScript 3.0
런타임 버전: AIR 2

시작 시 프로세스로 전달될 명령줄 인수입니다.

arguments 벡터의 각 문자열은 문자열에 포함된 문자에 관계없이 별도의 인수로 실행 파일에 전달됩니다. 즉, 정확한 일대일 대응이 이루어지므로 재해석이 발생하지 않습니다. 문자열에서 이스케이프해야 할 모든 문자(예: 공백 문자)를 AIR가 자동으로 이스케이프합니다.

executable

속성 
executable:File

언어 버전: ActionScript 3.0
런타임 버전: AIR 2

호스트 운영 체제에서 실행 파일을 참조하는 File 객체입니다. 필요한 모든 확장명을 포함한 실행 파일의 전체 경로여야 합니다.

참고: Mac OS에서 응용 프로그램 번들 내의 실행 파일을 실행하려면 응용 프로그램 파일의 경로뿐 아니라 번들 내 실행 파일의 전체 경로가 File 객체의 경로에 포함되어야 합니다.



구현
    public function get executable():File
    public function set executable(value:File):void

오류
ArgumentError — 지정된 값이 null인 경우, 디렉토리를 참조하는 경우 또는 없는 파일을 참조하는 경우입니다.
