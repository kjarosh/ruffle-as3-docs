클래스public class SQLErrorOperation
상속SQLErrorOperation Object

언어 버전: ActionScript 3.0
런타임 버전: AIR 1.0

이 클래스에는 SQLError.operation 속성에 사용 가능한 값을 나타내는 상수가 포함되어 있습니다. 이러한 값은 시도되어 오류를 발생시킨 작업을 나타냅니다.

각 값은 SQLConnection 클래스 또는 SQLStatement 클래스의 작업 중 하나를 나타냅니다.

관련 API 요소

flash.errors.SQLError.operation
flash.data.SQLConnection
flash.data.SQLStatement
