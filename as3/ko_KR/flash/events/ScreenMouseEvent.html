클래스public class ScreenMouseEvent
상속ScreenMouseEvent MouseEvent Event Object

언어 버전: ActionScript 3.0
런타임 버전: AIR 1.0

SystemTrayIcon 객체는 마우스 상호 작용에 대한 응답으로 ScreenMouseEvent 유형의 이벤트를 전달합니다.

ScreenMouseEvent 객체는 MouseEvent 클래스를 확장하여 응용 프로그램 윈도우 또는 스테이지가 아닌 기본 데스크톱 스크린을 기준으로 마우스 좌표를 보고하는 두 가지 추가 속성인 screenX 및 screenY를 제공합니다.

관련 API 요소

flash.desktop.SystemTrayIcon
flash.display.Screen

ScreenMouseEvent

()생성자
public function ScreenMouseEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, screenX:Number = NaN, screenY:Number = NaN, ctrlKey:Boolean = false, altKey:Boolean = false, shiftKey:Boolean = false, buttonDown:Boolean = false, commandKey:Boolean = false, controlKey:Boolean = false)

언어 버전: ActionScript 3.0
런타임 버전: AIR 1.0

화면 좌표의 마우스 위치가 포함된 ScreenMouseEvent 객체를 만듭니다.

매개 변수
type:String — 이벤트 유형입니다. 이벤트 리스너는 상속된 type 속성을 통해 이 정보에 액세스할 수 있습니다.
 
bubbles:Boolean (default = false) — 화면 좌표를 클릭할 때의 X 위치입니다.
 
cancelable:Boolean (default = false) — 화면 좌표를 클릭할 때의 Y 위치입니다.
 
screenX:Number (default = NaN) — 화면 마우스 이벤트가 버블링되지 않으므로 false로 설정합니다.
 
screenY:Number (default = NaN) — 취소할 기본 비헤이비어가 없으므로 false로 설정합니다.
 
ctrlKey:Boolean (default = false) — Windows 또는 Linux에서는 이 이벤트가 발생할 때 Ctrl 키가 눌려져 있었는지 여부를 나타냅니다. Mac에서는 Ctrl 키 또는 Command 키가 눌려져 있었는지 여부를 나타냅니다.
 
altKey:Boolean (default = false) — 이 이벤트가 발생할 때 Alt 키가 눌려져 있음을 나타내려면 true로 설정합니다.
 
shiftKey:Boolean (default = false) — 이 이벤트가 발생할 때 Shift 키가 눌려져 있음을 나타내려면 true로 설정합니다.
 
buttonDown:Boolean (default = false) — 이 이벤트가 발생할 때 마우스 버튼이 눌려져 있음을 나타내려면 true로 설정합니다.
 
commandKey:Boolean (default = false) — Command 키가 눌려져 있었는지 여부를 나타냅니다(Mac용).
 
controlKey:Boolean (default = false) — Ctrl 또는 Control 키가 눌려져 있었는지 여부를 나타냅니다.

commandKeyMac에서 true이면 Command 키가 활성 상태이고, false이면 비활성 상태입니다. Windows에서는 항상 false입니다.
controlKeyCtrl 또는 Control 키가 활성 상태이면 true이고 비활성 상태이면 false입니다.

public static const RIGHT_CLICK:String = "rightClick"

언어 버전: ActionScript 3.0
런타임 버전: AIR 1.5

ScreenMouseEvent.RIGHT_CLICK 상수는 rightClick 이벤트 객체의 type 속성 값을 정의합니다.

이 이벤트에는 다음과 같은 관련 속성이 있습니다.

속성값
buttonDown마우스 기본 버튼이 눌러져 있으면 true이고, 그렇지 않으면 false입니다.
ctrlKeyWindows 또는 Linux에서 Ctrl 키가 활성 상태이면 true입니다. Mac에서 Ctrl 키 또는 Command 키가 활성 상태이면 true입니다. 그렇지 않으면 false입니다.
currentTarget이벤트 리스너를 통해 Event 객체를 처리하고 있는 객체입니다.
shiftKeytrue이면 Shift 키가 활성 상태이고, false이면 비활성 상태입니다.
commandKeyMac에서 true이면 Command 키가 활성 상태이고, false이면 비활성 상태입니다. Windows에서는 항상 false입니다.
controlKeyCtrl 또는 Control 키가 활성 상태이면 true이고 비활성 상태이면 false입니다.
screenX화면 좌표에서 이벤트가 발생한 지점의 수평 좌표입니다.
screenY화면 좌표에서 이벤트가 발생한 지점의 수직 좌표입니다.
target포인팅 장치 아래에 있는 SystemTrayIcon 객체입니다.
