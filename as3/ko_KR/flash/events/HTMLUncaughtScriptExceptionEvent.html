exceptionValue

포착되지 않은 예외를 발생시킨 throw 문의 표현식을 평가한 결과입니다. exceptionValue 속성은 프리미티브 값, JavaScript 객체에 대한 참조 또는 ActionScript 객체에 대한 참조가 될 수 있습니다.

    

stackTrace

속성 
stackTrace:Array

런타임 버전: AIR 1.0

포착되지 않은 예외를 발생시킨 throw 문이 실행되었을 때 스택 추적을 나타내는 객체의 배열입니다. 배열의 각 객체에는 세 가지 속성이 있습니다.

  • sourceURL(문자열): 호출 스택 프레임의 스크립트 URL입니다.
  • line(숫자): 호출 스택 프레임의 sourceURL 리소스에 있는 행 번호입니다.
  • functionName(문자열): 호출 스택 프레임에 대한 함수 이름입니다.

이벤트 리스너에 매개 변수로 전달할 HTMLUncaughtScriptExceptionEvent 객체를 만듭니다.

매개 변수
exceptionValue:* — JavaScript 프로세스에 포착되지 않은 예외가 발생하면 exceptionValue는 포착되지 않은 예외를 발생시킨 throw 문의 표현식을 평가한 결과입니다. exceptionValue 속성은 프리미티브 값, JavaScript 객체에 대한 참조 또는 ActionScript 객체에 대한 참조가 될 수 있습니다.

HTMLUncaughtScriptExceptionEvent.UNCAUGHT_SCRIPT_EXCEPTION 상수는 uncaughtScriptException 이벤트 객체의 type 속성 값을 정의합니다.

이 이벤트에는 다음과 같은 속성이 있습니다.

속성값
bubblesfalse
cancelablefalse. 취소할 기본 비헤이비어가 없습니다.

stackTrace포착되지 않은 예외를 발생시킨 throw 문이 실행되었을 때 스택 추적을 나타내는 객체의 배열입니다.
targetHTMLLoader 객체입니다.
