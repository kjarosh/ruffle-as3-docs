런타임에 다음과 같은 오류가 발생할 수 있습니다. ActionScript 3.0에서는 Strict 모드 또는 경고 모드에서 컴파일할 때 런타임 유형을 확인합니다.


 코드메시지설명
 1000시스템의 메모리가 부족합니다. Flash에서 코드를 컴파일하기 위해 시스템의 가용 메모리보다 많은 메모리가 필요합니다. 시스템에서 실행되는 응용 프로그램 또는 프로세스 중 일부를 닫습니다.  
 1001%m 메서드가 구현되지 않았습니다. 
 1002precision 인수의 범위는 0 - 20이지만 %d을(를) 받았습니다. 속성 또는 메서드의 precision 인수에 대해 0보다 작거나 20보다 큰 값을 지정했습니다. 0에서 20 사이의 값을 precision 인수로 전달하십시오.  
 1003radix 인수의 범위는 2 - 36이지만 %d을(를) 받았습니다. 속성 또는 메서드의 radix 인수에 대해 2보다 작거나 36보다 큰 값을 전달했습니다. 2에서 36 사이의 값을 radix 인수로 전달하십시오.  
 1004호환되지 않는 객체에 '%t'의 메서드를 호출했습니다. 지정된 객체에 사용할 수 없는 메서드를 호출하려고 했습니다. 한 객체에서 다른 객체로 프로토타입 함수를 복사한 다음 이를 호출했으나 대상 객체의 유형과 원래 객체의 유형이 다른 경우에 이 오류가 발생합니다. 대상 객체의 유형과 원래 객체의 유형이 같은지 확인합니다. 세부 사항은 ECMAScript 버전 3의 15장을 참조하십시오.

var m0:uint = 0xFF;
var m1:uint = uint(m0<<24);
