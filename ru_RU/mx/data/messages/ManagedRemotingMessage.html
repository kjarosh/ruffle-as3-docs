getOperationAsString

()метод
public static function getOperationAsString(op:uint):String

Язык версии: ActionScript 3.0
Версия продукта: Adobe Digital Enterprise Platform Data Services for Java EE 4.6
Версии среды выполнения: Flash Player 10.2, AIR 2.6

Предоставляет описание указанной операции. Этот метод используется в операциях toString() данного сообщения.

Параметры

op:uint — Операция.

Возвращает
String — Представление операции в виде строки, либо значение unknown, если операция не определена.

Пример  ( Использование этого примера ) 
         var msg:ManagedRemotingMessage = ManagedRemotingMessage(event.message);
         trace("Current operation -'"+
                ManagedRemotingMessage.getOperationAsString(msg.operation)+ "'.");

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
