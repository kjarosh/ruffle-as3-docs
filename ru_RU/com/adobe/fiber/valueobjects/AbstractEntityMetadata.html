Классpublic class AbstractEntityMetadata
НаследованиеAbstractEntityMetadata EventDispatcher Object
Реализует IModelType, IModelInstance

Язык версии: ActionScript 3.0
Версия продукта: Adobe Digital Enterprise Platform Data Services for Java EE 3
Версии среды выполнения: Flash Player 9, AIR 1.1

Этот класс определяет заданные по умолчанию реализации передаваемых ошибок всех функций, объявленных в интерфейсах IModelType и IModelInstance. Кроме того, все создаваемые расширения этого класса будут представлять следующие созданные API-интерфейсы для каждого из свойств сущности: 
      function get property-nameStyle():IStyle

Привязываемый получатель для экземпляра свойства com.adobe.fiber.styles.IStyle. Для свойств, не имеющих связанного стиля, возвращается значение null.

      function get isproperty-nameAvailable():Boolean

привязываемый получатель состояния доступности свойства с учетом вариантов. Всегда возвращает значение true для свойств, не находящихся в варианте.

      function invalidateDependentproperty-name():void

Функции для каждого из свойств, инициирующие повторное вычисление всех производных свойств, зависящих от этого свойства. После завершения вычисления каждого из производных свойств будут запущены события PropertyChanged. Следует отметить, что эти функции создаются только для свойств данных, имеющих зависимые производные свойства.

Определяет, является ли свойство доступным, как определено в вариантах. При передаче имени свойства, не являющегося допустимым свойством объекта значения, возникает ошибка. Для добавления привязок к флагу доступности определенного свойства следует использовать функцию propertyNameAvailable(). Эта функция существует для каждого свойства и поддерживает привязку.

Параметры

propertyName:String — Свойство, проверяемое на предмет доступности

Возвращает
Boolean — имеет значение true, если в текущий момент эти свойства доступны в соответствии с определением в вариантах

Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
