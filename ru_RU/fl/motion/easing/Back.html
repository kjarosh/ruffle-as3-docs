Сведения о методе

easeIn

()метод
public static function easeIn(t:Number, b:Number, c:Number, d:Number, s:Number = 0):Number

Язык версии: ActionScript 3.0
Версия продукта: Flash CS3
Версии среды выполнения: Flash Player 9.0.28.0, AIR 1.0

Метод easeIn() начинает движение в сторону от цели, затем изменяет его на противоположное и выполняет движение к цели.

Параметры

t:Number — Задает текущее время в интервале между 0 и продолжительностью включительно.
 
b:Number — Задает начальное значение свойства анимации.
 
c:Number — Задает общее изменение в свойстве анимации.
 
d:Number — Задает продолжительность движения.
 
s:Number (default = 0) — Задает величину опережения. Чем выше значение, тем сильнее опережение.

Возвращает
Number — Значение свойства интерполяции в определенное время.

Язык версии: ActionScript 3.0
Версия продукта: Flash CS3
Версии среды выполнения: Flash Player 9.0.28.0, AIR 1.0

Метод easeInOut() сочетает движения методов easeIn() и easeOut(). Метод начинает движение в сторону от цели, затем изменяет направление на противоположное и выполняет перемещение к цели с небольшим опережением, затем снова изменяет направление и выполняет перемещение назад к цели.

Параметры

t:Number — Задает текущее время в интервале между 0 и продолжительностью включительно.
 
b:Number — Задает начальное значение свойства анимации.
 
c:Number — Задает общее изменение в свойстве анимации.
 
d:Number — Задает продолжительность движения.
 
s:Number (default = 0) — Задает величину опережения. Чем выше значение, тем сильнее опережение.

Возвращает
Number — Значение свойства интерполяции в определенное время.

[ X ]Почему по-английски?
Содержимое Справочника ActionScript 3.0 отображается на английском языке

Не все части Справочника ActionScript 3.0 переводятся на все языки. Если какой-то текстовый элемент не переведен, он отображается на английском языке. Например, компонент ga.controls.HelpBox не переведен ни на один из языков. Это значит, что в русской версии справки компонент ga.controls.HelpBox будет отображаться на английском языке.
