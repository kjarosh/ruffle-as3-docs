Sprachversion: ActionScript 3.0

Konstruktor.

Parameter
dest:ConcreteDataService — Ziel, auf das dieser Eigenschaftenbezeichner angewendet werden soll.
 
mode:int — Modus des Eigenschaftenbezeichners. Muss einen dieser Werte aufweisen: 0 (INCLUDE_DEFAULT) 1 (INCLUDE_ALL) 2 (INCLUDE_DEFAULT_PLUS_LIST) 3 (INCLUDE_LIST)
 
extra:Array — Array von Namen zusätzlicher Eigenschaften, die in Modus 2 und 3 aufgenommen werden.
