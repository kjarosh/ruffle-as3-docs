Konstruktor.

Parameter
type:String — Der Ereignistyp. Folgende Werte sind möglich:
  • „rslProgress“ (RSLEvent.RSL_PROGRESS);
  • „rslComplete“ (RSLEvent.RSL_COMPLETE);
  • „rslError“ (RSLEvent.RSL_ERROR);
 
bubbles:Boolean (default = false) — Bestimmt, ob das Ereignisobjekt an der Bubbling-Phase des Ereignisablaufs teilnimmt.
 
cancelable:Boolean (default = false) — Legt fest, ob das Ereignisobjekt abgebrochen werden kann.
 
bytesLoaded:int (default = -1) — Die Anzahl der geladenen Bytes während der Verarbeitung durch den Listener.
 
bytesTotal:int (default = -1) — Die Gesamtzahl der Bytes, die bei einem erfolgreichen Ladevorgang geladen werden.
 
rslIndex:int (default = -1) — Die Indexpositionsnummer des ausgewählten Elements in Relation zur Gesamtzahl. Dieser Wert sollt zwischen 0 und total - 1 liegen.
 
rslTotal:int (default = -1) — Die Zahl der insgesamt geladenen RSLs.
 
url:URLRequest (default = null) — Die Position der RSL.
 
errorText:String (default = null) — Die Fehlermeldung, wenn der Typ RSLEvent.RSL_ERROR entspricht.
 
isResourceModule:Boolean (default = false) — Wert „true“, wenn das Ereignis beim Laden eines ResourceModule auftrat.
 
loaderInfo:LoaderInfo (default = null) — Das mit diesem Ereignis verknüpfte LoaderInfo-Objekt.
Konstantendetails

RSL_ADD_PRELOADED

Konstante
public static const RSL_ADD_PRELOADED:String = "rslAddPreloaded"

Sprachversion: ActionScript 3.0
Produktversion: Flex 4.5
Laufzeitversionen: Flash Player 10, AIR 1.5

Die RSLEvent.RSL_ADD_PRELOADED-Konstante definiert den Wert der type-Eigenschaft des Ereignisobjekts für ein rslAddPreloaded-Ereignis. Dieses Ereignis wird aus einer IFlexModuleFactory ausgelöst, nachdem eine untergeordnete IFlexModuleFactory eine RSL in ihre Anwendungsdomäne vorab geladen hat.

Die Eigenschaften des Ereignisobjekts haben die folgenden Werte:

EigenschaftWert
bubblesfalse
cancelablefalse
currentTargetDas Objekt, das den Ereignis-Listener definiert, der das Ereignis übernimmt. Wenn Sie beispielsweise myButton.addEventListener() zur Registrierung eines Ereignis-Listeners verwenden, weist currentTarget den Wert „myButton“ auf.
loaderInfoDie mit dieser RSL verknüpfte LoaderInfo-Instanz.
targetDas Objekt, von dem das Ereignis ausgelöst wurde. Dies ist nicht immer das Objekt, das auf das Ereignis wartet. Verwenden Sie die currentTarget-Eigenschaft, damit immer auf das Objekt, das auf das Ereignis wartet, zugegriffen wird.

Inhalt des ActionScript 3.0-Referenzhandbuchs wird in englischer Sprache angezeigt

Nicht alle Teile des ActionScript 3.0-Referenzhandbuchs wurden in alle Sprachen übersetzt. Wenn der Text zu einem Sprachelement nicht übersetzt wurde, wird er auf Englisch angezeigt. Zum Beispiel wurden die Informationen zur ga.controls.HelpBox-Klasse nicht in andere Sprachen übersetzt. In der deutschen Version des Referenzhandbuchs erscheint der Abschnitt zur ga.controls.HelpBox-Klasse deshalb auf Englisch.
