Klassen x


Das mx.validators-Paket enthält die Klassen, um das Validieren von Daten zu unterstützen.


Schnittstellen
 SchnittstelleBeschreibung
 IValidator Diese Schnittstelle legt die Methoden und Eigenschaften fest, die ein Validator-Objekt implementieren muss.
 IValidatorListener Von den Komponenten implementierte Schnittstelle zur Unterstützung des Flex-Datenüberprüfungsmechanismus.
Klassen
 KlasseBeschreibung
 CreditCardValidator Die CreditCardValidator-Klasse überprüft die Länge der Kreditkartennummer sowie die Richtigkeit des Präfix und übergibt den Luhn mod10-Algorithmus für den jeweiligen Kartentyp.
 CreditCardValidatorCardType Die CreditCardValidatorCardType-Klasse definiert Konstanten für die Angabe der zu überprüfenden Kreditkarte.
 CurrencyValidator Die CurrencyValidator-Klasse überprüft, ob ein String einen gültigen Währungsausdruck darstellt.
 CurrencyValidatorAlignSymbol Die CurrencyValidatorAlignSymbol-Klasse gibt Wertkonstanten für die Ausrichtung der Währungssymbole an.
 DateValidator Die DateValidator-Klasse überprüft, ob ein String, Datum oder Objekt ein zulässiges Datum und ein vorgeschriebenes Format aufweist.
 EmailValidator Die EmailValidator-Klasse überprüft, ob ein String ein einzelnes @-Zeichen sowie einen Punkt im Domänennamen aufweist und ob das Domänensuffix der höchsten Ebene zwei, drei, vier oder sechs Zeichen enthält.
 NumberValidator Die NumberValidator-Klasse stellt sicher, dass ein String eine gültige Zahl darstellt.
 NumberValidatorDomainType Die NumberValidatorDomainType-Klasse definiert die Werte für die domain-Eigenschaft der NumberValidator-Klasse.
 PhoneNumberValidator Die PhoneNumberValidator-Klasse stellt sicher, dass ein String eine gültige Telefonnummer ist.
 RegExpValidationResult Die RegExpValidator-Klasse löst die valid- und invalid-Ereignisse aus.
 RegExpValidator Mit der RegExpValidator-Klasse kann ein regulärer Ausdruck zur Überprüfung eines Felds verwendet werden.
 SocialSecurityValidator Die SocialSecurityValidator-Klasse überprüft, ob ein String eine gültige deutsche Sozialversicherungsnummer ist.
 StringValidator Die StringValidator-Klasse überprüft, ob die Länge eines Strings innerhalb eines angegebenen Bereichs liegt.
 ValidationResult Die ValidationResult-Klasse enthält die Ergebnisse einer Überprüfung.
 Validator Die Validator-Klasse ist eine Basisklasse für alle Flex-Validators.
 ZipCodeValidator Die ZipCodeValidator-Klasse überprüft, ob die Länge und das Format des Strings für eine fünfstellige PLZ, eine fünfstellige + vierstellige PLZ der USA oder eine kanadische PLZ richtig sind.
 ZipCodeValidatorDomainType Die ZipCodeValidatorDomainType-Klasse definiert die Werte für die domain-Eigenschaft der ZipCodeValidator-Klasse, die zur Angabe des zu überprüfunden PLZ-Codes verwendet wird.

Inhalt des ActionScript 3.0-Referenzhandbuchs wird in englischer Sprache angezeigt

Nicht alle Teile des ActionScript 3.0-Referenzhandbuchs wurden in alle Sprachen übersetzt. Wenn der Text zu einem Sprachelement nicht übersetzt wurde, wird er auf Englisch angezeigt. Zum Beispiel wurden die Informationen zur ga.controls.HelpBox-Klasse nicht in andere Sprachen übersetzt. In der deutschen Version des Referenzhandbuchs erscheint der Abschnitt zur ga.controls.HelpBox-Klasse deshalb auf Englisch.
