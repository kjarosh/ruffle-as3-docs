MediaFactoryItem ist die Kapselung aller Informationen, die zum dynamischen Erstellen und Initialisieren eines MediaElement aus einer MediaFactory benötigt werden.

MediaFactoryItem-Objekte werden von Plugins dargestellt (in der PluginInfo-Klasse) und vom Framework zum Erstellen der vom Plugin angegebenen MediaElement(s) verwendet.

public function MediaFactoryItem(id:String, canHandleResourceFunction:Function, mediaElementCreationFunction:Function, type:String = null)

Sprachversion: ActionScript 3.0
Produktversion: OSMF 1.0
Laufzeitversionen: Flash Player 10, AIR 1.5

Konstruktor.

Parameter
id:String — Ein Bezeichner, der dieses MediaFactoryItem darstellt. Die Bezeichner müssen den Hersteller des Plugins sowie den spezifischen Namen des generierten Elements wiedergeben. Laut Konvention wird das Schema für den Paket-Namespace verwendet. Zwei Beispiele: com.example.MyAdPlugin com.example.MyAnalyticsPlugin Hinweis: org.osmf ist zu vermeiden, da die MediaFactory Nicht-OSMF-Plugins Vorrang gibt.
 
canHandleResourceFunction:Function — Funktion, mit deren Hilfe bestimmt wird, ob dieses MediaFactoryItem eine bestimmt Ressource verarbeiten kann. Die Funktion muss einen einzelnen Parameter vom Typ of type MediaResourceBase annehmen und einen Boolean-Wert zurückgeben.
 
mediaElementCreationFunction:Function — Funktion, die eine neue Instanz des gewünschten MediaElement erstellt. Die Funktion darf keine Parameter annehmen und muss ein MediaElement zurückgeben.
 
type:String (default = null) — Der Typ dieses MediaFactoryItem. Wenn null, lautet der Standardwert MediaFactoryItemType.STANDARD.

Auslöser
ArgumentError — Wenn ein beliebiges Argument (außer „type“) null ist.
