Klassepublic class NetMonitor
VererbungNetMonitor EventDispatcher Object

Sprachversion: ActionScript 3.0
Laufzeitversionen: Flash Player 10.3, AIR 2.7

Die NetMonitor-Klasse verwaltet eine Liste von NetStream-Objekten.

Verwenden Sie die NetMonitor-Klasse, um die in einer Anwendung verwendeten NetStream-Objekte zu verfolgen. Eine Instanz dieser Klasse setzt jedes Mal ein netStreamCreate-Ereignis ab, wenn ein neues NetStream-Objekt erstellt wird.

Mit Unterstützung der NetMonitor-Klasse können Sie die Videowiedergabe und verwandte Ereignisse verfolgen, unabhängig vom verwendeten Videoplayer. Diese Funktion ist hilfreich, wenn Sie Bibliotheken zur Medienmessung, Analyse und Nutzungsverfolgung implementieren möchten.

Hinweis: NetStream-Überwachung wird nicht unterstützt von Flash Player im Browser unter Android und Blackberry Tablet OS oder von AIR unter iOS.

Methodendetails

listStreams

()Methode
public function listStreams():Vector.<NetStream>

Sprachversion: ActionScript 3.0
Laufzeitversionen: Flash Player 10.3, AIR 2.7

Ruft alle NetStream-Objekte ab, die zu dem Sicherheitskontext dieses NetMonitor-Objekts gehören.

Vermeiden Sie das Zwischenspeichern der Liste der NetStream-Objekte. Wenn Sie einen Verweis auf diese NetStream-Objekte beibehalten, können Speicherlecks in eine Anwendung eingeführt werden, da der Garbagekollektor daran gehindert wird, die Ressourcen eines Objekts freizugeben, wenn sie nicht mehr verwendet werden.

Hinweis: Wenn die NetStream-Überwachung auf der aktuellen Plattform nicht unterstützt wird, ist die von dieser Funktion zurückgegebene Liste immer leer.

Rückgabewerte
Vector.<NetStream> — Vektor von NetStream-Objekten
Ereignisdetails

netStreamCreate

 Ereignis
Ereignisobjekttyp: flash.events.NetMonitorEvent
Eigenschaft NetMonitorEvent.type = flash.events.NetMonitorEvent.NET_STREAM_CREATE

Sprachversion: ActionScript 3.0
Laufzeitversionen: Flash Player 10.3, AIR 2.7

Wird abgesetzt, wenn ein neues NetStream-Objekt innerhalb des Sicherheitskontextes dieser NetMonitor-Instanz erstellt wird.

Hinweis: Wenn die NetStream-Überwachung auf der aktuellen Plattform nicht unterstützt wird, werden keine netStreamCreate abgesetzt.

Die NetMonitorEvent.NET_STREAM_CREATE-Konstante definiert den Wert der type-Eigenschaft eines netStreamCreate-Ereignisobjekts.

Das netStreamCreate-Ereignis hat folgende Eigenschaften:

EigenschaftWert
netStreamDas NetStream-Objekt, das erstellt wurde.
bubblesfalse
cancelablefalse. Es gibt kein Standardverhalten, das abgebrochen werden kann.
currentTargetDas Objekt, welches das Ereignisobjekt aktiv mit einem Ereignis-Listener verarbeitet.
targetDas Objekt, das eine Sitzung beginnt oder beendet.
