Sprachversion: ActionScript 3.0
Produktversion: Flash CS3
Laufzeitversionen: Flash Player 9.0.28.0, AIR 1.0

Die KeyframeBase-Klasse definiert den visuellen Zustand zu einer bestimmten Zeit in einem Bewegungs-Tween. Die folgenden Eigenschaften sind die primären Animationseigenschaften: position, scale, rotation, skew und color. Um KeyframeBase zu verwenden, müssen alle Eigenschaften Werte für jede KeyframeBase aufweisen, und für jedes Bild in der Bewegung muss eine KeyframeBase definiert sein. Bei einem Bewegungs-Tween kann jede Zeitindexposition nur ein Schlüsselbild haben. Ein Schlüsselbild verfügt noch über weitere Eigenschaften, wie z. B. blend mode, filters und cacheAsBitmap. Diese Eigenschaften sind immer verfügbar. So verfügt ein Schlüsselbild zum Beipiel immer über einen Mischmodus.

rotateTimes

Eigenschaft 
public var rotateTimes:uint = 0

Sprachversion: ActionScript 3.0
Produktversion: Flash CS3
Laufzeitversionen: Flash Player 9.0.28.0, AIR 1.0

Fügt im Verlauf eines Bewegungs-Tweens zusätzlich zu einer vorhandenen Drehung dem Zielobjekt eine Drehung hinzu. Diese Drehung hängt von dem Wert der rotateDirection-Eigenschaft ab, die auf RotateDirection.CW oder RotateDirection.CCW gesetzt sein muss. Bei dem Wert rotateTimes muss es sich um eine Ganzzahl handeln, die größer oder gleich Null ist.

Beispiel: Wenn das Objekt normalerweise von 0 auf 40 Grad gedreht wird, fügt ein rotateTimes-Wert von 1 und eine auf RotateDirection.CW gesetzte rotateDirection-Eigenschaft eine vollständige Drehung hinzu. Das Ergebnis ist eine Gesamtdrehung um 400 Grad.

Wenn die rotateDirection-Eigenschaft auf RotateDirection.CCW gesetzt ist, werden 360 Grad von der ursprünglichen Drehung abgezogen. Das Ergebnis ist eine Drehung gegen den Uhrzeigersinn um 320 Grad.

setAdjustColorProperty

()Methode 
public function setAdjustColorProperty(filterIndex:int, propertyName:String, value:*):void

Sprachversion: ActionScript 3.0
Produktversion: Flash CS3
Laufzeitversionen: Flash Player 9.0.28.0, AIR 1.0

Stellt eine der vier AdjustColor-Eigenschaften auf das AdjustColor-Objekt für den angegebenen Filterindex ein. Erstellt die AdjustColor-Instanz, falls für diesen Index noch keine vorhanden ist. Die folgenden vier Eigenschaften können festgelegt werden:

  • adjustColorBrightness
  • adjustColorContrast
  • adjustColorSaturation
  • adjustColorHue
Es müssen alle vier AdjustColor-Eigenschaften festgelegt werden, damit der Filter erstellt werden kann. Nachdem alle vier Eigenschaften für eine AdjustColor-Instanz festgelegt wurden, ruft die Funktion vom AdjustColor-Objekt ein flaches Array mit 20 Werten ab, das alle vier Eigenschaften repräsentiert, und erstellt eine ColorMatrixFilter-Instanz, die das flache Array benötigt.

Parameter

filterIndex:int — Die Indexposition des Filters im Filterarray, um die propertyName-Eigenschaft hinzuzufügen.
 
propertyName:String — Einer der vier zulässigen Eigenschaftswerte: "adjustColorBrightness", "adjustColorContrast", "adjustColorSaturation" oder "adjustColorHue".
 
value:* — Der Wert, der für die angegebene Eigenschaft festgelegt werden soll.

In der deutschen Version des Referenzhandbuchs erscheint der Abschnitt zur ga.controls.HelpBox-Klasse deshalb auf Englisch.
