Klassepublic class MatrixTransformer
VererbungMatrixTransformer Object

Sprachversion: ActionScript 3.0
Produktversion: Flash CS3
Laufzeitversionen: Flash Player 9.0.28.0, AIR 1.0

Die MatrixTransformer-Klasse enthält Methoden zum Modifizieren individueller Eigenschaften einer Transformationsmatrix: horizontale und vertikale Skalierung, horizontale und vertikale Neigung sowie Drehung. Diese Klasse enthält auch Methoden zum Drehen eines gegebenen Transformationspunkts anstelle des Standardpunkts (0, 0).

matchInternalPointWithExternal

()Methode 
public static function matchInternalPointWithExternal(m:Matrix, internalPoint:Point, externalPoint:Point):void

Sprachversion: ActionScript 3.0
Produktversion: Flash CS3
Laufzeitversionen: Flash Player 9.0.28.0, AIR 1.0

Verschiebt eine Matrix nach Bedarf, um einen internen Punkt an einen externen Punkt auszurichten. So kann ein Punkt in einem transformierten Movieclip an einem Punkt im übergeordneten Clip ausgerichtet werden.

Parameter

m:Matrix — Eine Matrix-Instanz.
 
internalPoint:Point — Eine Point-Instanz, die eine Position innerhalb des Transformationsraums der Matrix definiert.
 
externalPoint:Point — Eine Point-Instanz, die eine Referenzposition außerhalb des Transformationsraums der Matrix definiert.

Verwandte API-Elemente

flash.geom.Matrix

setRotation

()Methode 
public static function setRotation(m:Matrix, rotation:Number):void

Sprachversion: ActionScript 3.0
Produktversion: Flash CS3
Laufzeitversionen: Flash Player 9.0.28.0, AIR 1.0

Ändert den Drehwinkel in einer Matrix. Wenn die horizontale und die vertikale Neigung nicht identisch sind, wird die vertikale Neigung auf den Wert der Drehung gesetzt, und die horizontale Neigung wird um die Differenz zwischen dem alten Drehungswert und dem neuen Drehungswert erhöht. Dies entspricht dem Drehungsverhalten in Flash Player.

Parameter

m:Matrix — Eine Matrix-Instanz.
 
rotation:Number — Der Drehwinkel, in Grad angegeben.
