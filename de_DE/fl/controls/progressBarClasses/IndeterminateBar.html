VererbungIndeterminateBar UIComponent Sprite DisplayObjectContainer InteractiveObject DisplayObject EventDispatcher Object

Sprachversion: ActionScript 3.0
Produktversion: Flash CS3
Laufzeitversionen: Flash Player 9.0.28.0, AIR 1.0

Die IndeterminateBar-Klasse übernimmt das Zeichnen der Fortschrittleiste, wenn die Größe der geladenen Quelle unbekannt ist. Diese Klasse kann durch jede andere UIComponent-Klasse ersetzt werden, wenn die Leiste auf andere Weise dargestellt werden soll. Bei der Standardimplementierung wird die von API erstellte Zeichung verwendet; eine gestreifte Füllung gibt den Fortschritt des Ladevorgangs an.

Ruft die Standardstilmatrix der aktuellen Komponente ab. Die Stilmatrix enthält den für die Komponente geeigneten Typ und hängt von dem Stil ab, der von der Komponente verwendet wird. Der Stil disabledTextFormat enthält beispielsweise den Wert null oder ein TextFormat-Objekt. Sie können diese Stile verwenden und setStyle() für die aktuelle Komponente aufrufen. Mit folgendem Code wird der Standardstil disabledTextFormat der angegebenen Komponente überschrieben: 

componentInstance.setStyle("disabledTextFormat", new TextFormat());

function showStyleDefinition(e:Event):void {
    var componentClass:Class = e.target.selectedItem.data as Class;
    var styles:Object = componentClass["getStyleDefinition"].call(this);
    trace(styles.toString());
    var styleData:DataProvider = new DataProvider();
    for(var i:* in styles) {

        styleData.addItem( { StyleName:i, DefaultValue:styles[i] } );

    styleData.sortOn("StyleName");

Beispiele Verwendung dieses Beispiels
IndeterminateBarExample.as

Dieses Beispiel zeigt, wie Sie eine individuelle Version der unbestimmten Leiste erstellen.

Gehen Sie wie folgt vor, um das Beispiel auszuführen:

  1. Fügen Sie die ProgressBar-Komponente zur Bibliothek hinzu.
  2. Speichern Sie diesen Code unter "IndeterminateBarExample.as" im gleichen Verzeichnis wie die FLA-Datei.
  3. Legen Sie in der FLA-Datei für die Dokumentklasse den Wert "IndeterminateBarExample" fest.
package
{
    import fl.controls.ProgressBar;
    import fl.controls.ProgressBarMode;
    import flash.display.Sprite;
    
    public class IndeterminateBarExample extends Sprite
    {
        private var progressBar:ProgressBar;
        
        public function IndeterminateBarExample() {
            progressBar = new ProgressBar();
            progressBar.mode = ProgressBarMode.MANUAL;
            progressBar.indeterminate = true;
            progressBar.setSize(300,100);
            progressBar.x = 10;
            progressBar.y = 10;
            progressBar.setStyle("indeterminateBar",MyProgressBar);
            addChild(progressBar);
        }
    }
}

import fl.controls.progressBarClasses.IndeterminateBar;
import flash.events.Event;
import flash.display.*;

class MyProgressBar extends IndeterminateBar 
{
    private var overlay:Shape;
    private var overlayAlpha:Number = 0;
    
    public function MyProgressBar() {
        overlay = new Shape();
        addChild(overlay);
    }
    override protected function handleEnterFrame(event:Event):void {
        super.handleEnterFrame(event);
        drawOverlay();
    }    
    private function drawOverlay():void {
        var alphaLevel:Number = Math.abs(Math.sin(overlayAlpha));
        var softenedAlpha:Number = alphaLevel * 0.4;
        var g:Graphics = overlay.graphics;
        g.clear();
        g.beginFill(0x99FF33,softenedAlpha);
        g.drawRect(0,0,_width,_height);
        g.endFill();
        overlayAlpha += 0.15;
    }
}
