Die List-Komponente zeigt listenbasierte Informationen an und ist besonders für die Anzeige von Informations-Arrays geeignet.

Die List-Komponente beinhaltet Elemente, Zeilen und einen Datenprovider, die wie folgt definiert werden:

  • Element: Ein ActionScript-Objekt, das in der Regel die beschreibende Eigenschaft label und die Eigenschaft data enthält, die die mit dem Element verknüpften Daten speichert.
  • Zeile: Eine Komponente, die zum Anzeigen eines Elements verwendet wird.
  • Datenprovider: Eine Komponente, die die in der List-Komponente angezeigten Elemente bestimmt.

Standardmäßig wird von der List-Komponente die CellRenderer-Klasse zum Bereistellen der Zeilen verwendet, in denen die Listenelemente angezeigt werden. Diese Zeilen können programmgesteuert erstellt werden, indem Sie eine Unterklasse der CellRenderer-Klasse erstellen. Die CellRenderer-Klasse implementiert die ICellRenderer-Oberfläche, welche die Eigenschaften- und Methodensätze liefert, die von der List-Komponente verwendet werden, um ihre Zeilen zu bearbeiten und um Daten und Statusinformationen an die einzelnen Zeilen zu senden, wo diese angezeigt werden. Hierzu zählen auch Informationen zu Größenanpassung und Auswahl von Daten.

Die List-Komponente liefert Methoden, die als Proxy für ihren Datenprovider dienen, z. B. addItem() und removeItem(). Sie können diese und andere Methoden zum Bearbeiten von Daten in einem beliebigen Array verwenden, das sich im gleichen Frame wie die List-Komponente befindet, und die Änderungen dann in mehreren Ansichten veröffentlichen. Wenn eine List-Komponente keinen externen Datenprovider aufweist, wird von diesen Methoden automatisch eine Datenprovider-Instanz erstellt und mithilfe der Eigenschaft List.dataProvider eingeblendet. Die List-Komponente gibt die einzelnen Zeilen mithilfe eines Sprite-Renderers wieder, das die ICellRenderer-Oberfläche implementiert. Verwenden Sie die Eigenschaft List.cellRenderer, um diesen Renderer anzugeben. Sie können auch eine Array-Instanz erstellen oder von einem Server abrufen und als Datenmodell für mehrere Listen, Kombinationsfelder, Datenraster usw. verwenden.

Beispiele Verwendung dieses Beispiels
ListExample.as

In diesem Beispiel wird gezeigt, wie Sie eine List-Komponente erstellen und andere List-Instanzen mit dem ausgewählten Inhalt füllen.
  1. Fügen Sie die List-, Button- und Label-Klassen zur Bibliothek hinzu.
  2. Speichern Sie diesen Code unter „ListExample.as“ im gleichen Verzeichnis wie die FLA-Datei.
  3. Legen Sie für „DocumentClass“ in der FLA-Datei „ListExample“ fest.
package
{
    import flash.display.Sprite
    
    public class ListExample extends Sprite
    {
        import flash.events.*;
        import fl.data.DataProvider;
        import fl.controls.List
        import fl.controls.Label
        import fl.controls.Button
        
        private var clearButton:Button;
        private var availableItems:List;
        private var selectedItemList:List;
        private var selectedItemsList:List;

        public function ListExample() {
            createComponents();
            setupComponents();
        }

        private function setupComponents():void {
            var dp:Array = new Array();
            var i:uint;
            var count:uint = availableItems.rowCount * 2;
            
            for (i = 0; i < count; i++) {
                dp.push({label:"Item " + i});
            }
            
            availableItems.allowMultipleSelection = true;
            availableItems.dataProvider = new DataProvider(dp);
            availableItems.addEventListener(Event.CHANGE, updateLists);
            clearButton.addEventListener(MouseEvent.CLICK, clearHandler);            
        }
        
        private function clearHandler(event:MouseEvent):void {
            availableItems.clearSelection();
            // clear data providers
            selectedItemList.dataProvider = new DataProvider();
            selectedItemsList.dataProvider = new DataProvider();
        }
        
        private function updateLists(e:Event):void {
            selectedItemList.dataProvider = availableItems.selectedItem ? new DataProvider([availableItems.selectedItem]) : new DataProvider();
            selectedItemsList.dataProvider = new DataProvider(availableItems.selectedItems);
        }

        private function createComponents():void {
            clearButton = new Button();
            availableItems = new List();
            selectedItemList = new List();
            selectedItemsList = new List();
            var availableItemsLabel:Label = new Label();
            var selectedItemListLabel:Label = new Label();
            var selectedItemsListLabel:Label = new Label();
            
            clearButton.move(10,142);
            availableItems.move(10,32);
            selectedItemList.move(120,32);
            selectedItemsList.move(230,32);
            availableItemsLabel.move(10,10);
            selectedItemListLabel.move(120,10);
            selectedItemsListLabel.move(230,10);
            
            clearButton.label = "Clear Selection"
            availableItemsLabel.text = "Available Items";
            selectedItemListLabel.text = "Selected Item";
            selectedItemsListLabel.text = "All Selected Items";
            
            addChild(clearButton);
            addChild(availableItems);
            addChild(selectedItemList);
            addChild(selectedItemsList);
            addChild(availableItemsLabel);
            addChild(selectedItemListLabel);
            addChild(selectedItemsListLabel);
        }
    }
}
