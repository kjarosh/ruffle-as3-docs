패키지flashx.textLayout.container
클래스public class TextContainerManager
상속TextContainerManager EventDispatcher Object
구현 ISWFContext, IInteractionEventHandler, ISandboxSupport

언어 버전: ActionScript 3.0
런타임 버전: Flash Player 10, AIR 1.5

컨테이너의 텍스트를 관리합니다. 컨테이너의 모든 자식을 관리한다고 가정합니다. TextFlow당 하나의 컨테이너가 있고 TextFlow가 기본 포커스가 아니거나 정적 텍스트이거나 가끔씩 선택되는 경우, 성능 향상을 위해 TextContainerManager를 사용하는 것이 좋습니다. 양식 필드 등의 텍스트에 적합합니다.
