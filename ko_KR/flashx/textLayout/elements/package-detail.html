클래스 x


flashx.textLayout.elements 패키지에는 Text Layout Framework에서 텍스트 내용을 표현하는 데 사용되는 핵심 클래스가 들어 있습니다.


인터페이스
 인터페이스설명
 IConfiguration 구성 객체의 읽기 전용 인터페이스입니다.
 IFormatResolver 서식 확인 프로그램의 인터페이스입니다.
클래스
 클래스설명
 BreakElement BreakElement 클래스는 줄 분리를 정의합니다. 이 클래스를 사용하여 새 단락을 만들지 않고 텍스트의 줄 분리를 만들 수 있습니다.
 Configuration Configuration 클래스는 Text Layout Framework와 응용 프로그램 간 상호 작용을 위한 기본 지점입니다.
 ContainerFormattedElement ContainerFormattedElement는 모든 컨테이너 수준 블록 요소(예: DivElement 및 TextFlow 객체)의 루트 클래스입니다.
 DivElement DivElement 클래스는 단락 그룹화를 위한 요소(ParagraphElement 객체)를 정의합니다.
 FlowElement 흐름의 텍스트는 텍스트 내 논리적 구분을 나타내는 트리 요소와 함께 트리 형식으로 저장됩니다.
 FlowGroupElement FlowGroupElement 클래스는 자식 배열을 가질 수 있는 FlowElement 객체의 기본 클래스입니다.
 FlowLeafElement FlowElements의 기본 클래스이며, 흐름 계층 구조에서 최하위에 있습니다.
 GlobalSettings 모든 TextFlow 객체에 적용되는 구성입니다.
 InlineGraphicElement InlineGraphicElement 클래스는 텍스트에 인라인을 표시하는 그래픽 객체를 처리합니다.
 InlineGraphicElementStatus InlineGraphicElementStatus 클래스는 InlineGraphicElement.status의 값을 확인하는 상수 집합을 정의합니다.
 LinkElement LinkElement 클래스는 사용자가 클릭하면 실행되는 URI(Universal Resource Identifier)의 링크를 정의합니다.
 LinkState LinkState 클래스는 LinkElement 클래스의 linkState 속성에 대한 상수 집합을 정의합니다.
 ListElement List 클래스는 번호를 매기거나 매기지 않은 목록으로 항목을 그룹화하는 데 사용합니다.
 ListItemElement ListItemElement는 목록의 항목입니다.
 OverflowPolicy OverflowPolicy 클래스는 IConfiguration 클래스의 overflowPolicy 속성에 대한 상수 집합을 정의합니다.
 ParagraphElement ParagraphElement 클래스는 텍스트 흐름 계층 구조의 단락을 나타냅니다.
 ParagraphFormattedElement ParagraphFormattedElement 클래스는 단락 속성을 포함하는 FlowElement 클래스의 추상 기본 클래스입니다.
 SpanElement SpanElement 클래스는 적용된 서식 속성의 단일 집합을 포함하는 텍스트 블록을 나타냅니다.
 SpecialCharacterElement SpecialCharacterElement 클래스는 특수 문자를 나타내는 요소의 추상 기본 클래스입니다.
 SubParagraphGroupElement SubParagraphGroupElement는 FlowLeafElements 및 SubParagraphGroupElementBase를 확장하는 기타 클래스의 그룹화 요소입니다.
 SubParagraphGroupElementBase SubParagraphGroupElementBase 클래스는 FlowLeafElements를 그룹화합니다.
 TabElement TabElement 클래스는 텍스트 흐름의 <tab/>을 나타냅니다.
 TCYElement TCYElement (Tatechuuyoko - ta-tae-chu-yo-ko) 클래스는 세로줄 내에 텍스트를 가로로 그리도록 하는 SubParagraphGroupElementBase의 하위 클래스입니다.
 TextFlow TextFlow 클래스는 스토리의 모든 텍스트 내용을 관리하는 역할을 합니다.
 TextRange 연속 텍스트 범위를 설명하는 읽기 전용 클래스입니다.
