generateRandomBytes

()함수
public function generateRandomBytes(numberRandomBytes:uint):ByteArray

언어 버전: ActionScript 3.0
런타임 버전: Flash Player 11, AIR 3

임의 바이트 시퀀스를 생성합니다.

generateRandomBytes()를 사용하여 암호화 키, 강력한 식별자, 세션 ID 등을 생성합니다. 운영 체제에서 제공하는 강력한 암호의 함수를 사용하여 임의 시퀀스가 생성됩니다. 개별 클라이언트 컴퓨터나 장치에서 사용할 수 있는 적합한 함수가 없는 경우 오류가 발생합니다.

매개 변수

numberRandomBytes:uint — 1에서 1024 사이의 생성할 임의 바이트 수입니다.

반환값
ByteArray — 생성된 바이트를 포함하는 ByteArray입니다.

오류
Error — 임의 바이트를 생성할 수 없는 경우

관련 API 요소

Math.random()
