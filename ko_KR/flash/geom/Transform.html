클래스public class Transform
상속Transform Object
하위 클래스 Transform

언어 버전: ActionScript 3.0
런타임 버전: AIR 1.0, Flash Player 9, Flash Lite 4

Transform 클래스를 통해 표시 객체에 적용 가능한 색상 조정 속성 및 2차원 또는 3차원 변형 객체에 액세스할 수 있습니다. 변형을 수행하면 표시 객체의 색상 또는 방향 및 위치가 현재 값이나 좌표에서 새 값이나 좌표로 조정(오프셋)됩니다. 또한 Transform 클래스는 표시 객체와 모든 부모 객체에 적용된 색상 및 2차원 행렬 변형에 대한 데이터를 수집합니다. 이러한 결합된 변형에 액세스하려면 concatenatedColorTransform 및 concatenatedMatrix 속성을 사용합니다.

색상 변환을 적용하려면 ColorTransform 객체를 만들고 객체의 메서드와 속성을 사용하여 색상 조정을 설정한 다음 colorTransformation 속성(표시 객체의 transform 속성)에 새 ColorTransformation 객체를 할당합니다.

2차원 변형을 적용하려면 Matrix 객체를 만들고 행렬의 2차원 변형을 설정한 다음 표시 객체의 transform.matrix 속성에 새 Matrix 객체를 할당합니다.

3차원 변형을 적용하려면 우선 3차원 표시 객체를 준비합니다. 3차원 표시 객체의 z 속성 값은 0이 아닙니다. Matrix3D 객체는 만들 필요가 없습니다. 표시 객체에 z 값을 할당하면 모든 3차원 객체에 대해 Matrix3D 객체가 자동으로 만들어집니다. 표시 객체의 transform 속성을 통해 표시 객체의 Matrix3D 객체에 액세스할 수 있습니다. Matrix3D 클래스의 메서드를 사용하여 기존 변형 설정을 추가하거나 수정할 수 있습니다. 사용자 정의 Matrix3D 객체를 만들고 사용자 정의 Matrix3D 객체의 변형 요소를 설정한 다음 transform.matrix 속성을 사용하여 새 Matrix3D 객체를 표시 객체에 할당할 수도 있습니다.

스테이지나 루트 객체의 원근 투영을 수정하려면 루트 표시 객체의 transform.matrix 속성을 사용하여 PerspectiveProjection 객체에 액세스합니다. 또는 표시 객체 부모의 원근 투영 속성을 설정하여 표시 객체에 다른 원근 투영 속성을 적용합니다. 자식 표시 객체는 새 속성을 상속합니다. 구체적으로 설명하면, PerspectiveProjection 객체를 만들고 속성을 설정한 다음 PerspectiveProjection 객체를 perspectiveProjection 속성(부모 표시 객체의 transform 속성)에 할당합니다. 이렇게 하면 지정된 투영 변형이 표시 객체의 모든 3차원 자식에 적용됩니다.

PerspectiveProjection과 Matrix3D 객체는 모두 원근 변형을 수행하므로 이러한 두 객체를 표시 객체에 동시에 할당하지 마십시오. PerspectiveProjection 객체는 초점 길이 및 투영 중심을 변경할 때 사용합니다. 투영 변형을 보다 자세히 제어하려면 원근 투영 Matrix3D 객체를 사용합니다.

public function getRelativeMatrix3D(relativeTo:DisplayObject):Matrix3D

언어 버전: ActionScript 3.0
런타임 버전: Flash Player 10, AIR 1.5

현재 표시 객체의 공간을 기준으로 지정된 표시 객체의 공간을 변형할 수 있는 Matrix3D 객체를 반환합니다. getRelativeMatrix3D() 메서드를 사용하면 다른 3차원 표시 객체를 기준으로 3차원 표시 객체를 이동할 수 있습니다.

매개 변수

relativeTo:DisplayObject — 변형할 기준이 되는 표시 객체입니다. 스테이지를 기준으로 Matrix3D 객체를 가져오려면 매개 변수를 root 또는 stage 객체로 설정합니다. 표시 객체의 월드 기준 행렬을 가져오려면 매개 변수를 원근 변형이 적용된 표시 객체로 설정합니다.

반환값
Matrix3D — relativeTo 표시 객체의 공간을 현재 표시 객체의 공간으로 변형하는 데 사용할 수 있는 Matrix3D 객체입니다.

관련 API 요소

flash.geom.Matrix3D
예제 예제 사용 방법
TransformExample.as

다음 예제에서는 TransformExample 클래스를 사용하여 그래디언트 패턴으로 채워진 사각형 스프라이트의 아래쪽을 기울입니다. 사용자가 사각형을 클릭할 때마다 응용 프로그램에서 다음 단계에 따라 스프라이트를 기울여 변형합니다.
  1. TransformExample() 생성자가 새 Sprite 객체 target을 만듭니다.
  2. TransformExample() 생성자가 Sprite에 그래디언트 사각형을 그리는 draw() 메서드를 호출합니다.
  3. TransformExample() 생성자가 해당 Sprite에 대한 click 이벤트 리스너를 추가합니다. 이 리스너는 clickHandler() 메서드에 의해 처리됩니다.
  4. clickHandler() 메서드가 기울임 효과를 적용하도록 설정된 새 Matrix 객체인 skewMatrix를 만듭니다. 다른 행렬인 tempMatrix를 스프라이트의 현재 변형 행렬에 할당한 다음 skewMatrix와 결합합니다. 이때 concat() 메서드를 사용합니다. 이 행렬을 사각형 스프라이트의 transform.matrix 속성에 할당합니다. 사용자가 사각형을 클릭할 때마다 clickHandler()가 호출되어 사각형의 모양을 기울여 변경합니다.
  5. 또한 clickHandler() 메서드는 새 ColorTransform 객체를 만듭니다. 새 ColorTransform의 redOffset 속성은 redOffset의 현재 값으로 설정된 후 25가 더해집니다. 마찬가지로 blueOffset 속성은 25만큼 감소합니다. 따라서 클릭할 때마다 스프라이트의 색상이 변합니다. 
package {
    import flash.display.Sprite;
    import flash.display.GradientType;
    import flash.geom.Matrix;
    import flash.geom.ColorTransform;
    import flash.events.MouseEvent;

    public class TransformExample extends Sprite {
        public function TransformExample() { 
            var target:Sprite = new Sprite();
            draw(target);
            addChild(target);
            target.useHandCursor = true;
            target.buttonMode = true;
            target.addEventListener(MouseEvent.CLICK, clickHandler)
        }
        public function draw(sprite:Sprite):void {
            var red:uint = 0xFF0000;
            var green:uint = 0x00FF00;
            var blue:uint = 0x0000FF;
            var size:Number = 100;
            sprite.graphics.beginGradientFill(GradientType.LINEAR, [red, blue, green], [1, 0.5, 1], [0, 200, 255]);
            sprite.graphics.drawRect(0, 0, 100, 100);
        }
        public function clickHandler(event:MouseEvent):void {
            var skewMatrix:Matrix = new Matrix();
            skewMatrix.c = 0.25;
            var tempMatrix:Matrix = this.transform.matrix;
            tempMatrix.concat(skewMatrix);
            this.transform.matrix = tempMatrix;
            
            var rOffset:Number = this.transform.colorTransform.redOffset + 25;
            var bOffset:Number = this.transform.colorTransform.blueOffset - 25;
            this.transform.colorTransform = new ColorTransform(1, 1, 1, 1, rOffset, 0, bOffset, 0);            
        }
    }
}
