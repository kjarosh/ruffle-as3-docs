클래스public final class NetStreamMulticastInfo
상속NetStreamMulticastInfo Object

언어 버전: ActionScript 3.0
런타임 버전: Flash Player 10.1, AIR 2

NetStreamMulticastInfo 클래스는 NetStream의 기본 RTMFP 피어 투 피어 및 IP 멀티캐스트 스트림 전송과 관련된 여러 QoS(서비스 품질) 통계를 지정합니다. NetStreamMulticastInfo 객체는 NetStreamMulticastInfo 속성에서 반환됩니다.

숫자를 반환하는 속성은 멀티캐스트 스트림의 시작 부분에서 계산된 총계를 나타냅니다. 이러한 속성 유형에 전송한 미디어 바이트 수 또는 수신한 미디어 조각 메시지 수가 포함됩니다. 속도인 속성은 몇 초 동안의 평균이 계산된 현재 속도의 스냅샷을 나타냅니다. 이러한 속성 유형에는 로컬 노드가 데이터를 수신하는 속도가 포함됩니다.

NetStreamMulticastInfo 객체에 포함된 값 목록을 보려면 NetStreamMulticastInfo.toString() 메서드를 사용합니다.

ActionScript 3.0 참조 설명서 중 일부는 전체 언어로 번역되지 않았습니다. 언어 요소가 번역되지 않은 경우 영어로 나타납니다. 예를 들어 ga.controls.HelpBox 클래스는 어떤 언어로도 번역되지 않았습니다. 그러므로 한국어 버전의 참조 설명서에서 ga.controls.HelpBox 클래스는 영어로 나타납니다.
